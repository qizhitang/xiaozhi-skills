# 5套真实场景完整对话脚本

> 本文档为 `xiaozhi-english-speaking-coach` 的配套参考资源，提供角色扮演训练的5套内置场景完整对话脚本与复盘标注。

---

## 使用说明

```
角色扮演操作流程：
  学生给出或选择场景 → 小龙虾确认角色 → 开始对话 → 自然推进 → 结束后复盘

对话中的规则：
  - 全程英语，不插入中文
  - 遇到学生说不出的词，给选择："是想说 A 还是 B？"
  - 对话自然结束后，给整体反馈（不逐句纠错）
  - 学生卡壳时：给一个提示词，不给整句

场景难度标记：
  ⭐ 简单（小学/初中初级）：句式短，词汇基础
  ⭐⭐ 中等（初中）：含复合句，需要一定的策略性表达
  ⭐⭐⭐ 较难（高中/初中高级）：需要解释、说服、处理意外情况
```

## 场景①：机场值机+过安检 ⭐⭐

### 角色分配

```
学生角色：旅客（Passenger）
小龙虾角色：地勤人员 → 安检员

地点：国际航班出发大厅 → 安检通道
```

### 第一阶段：值机柜台

**关键词汇清单：**

```
boarding pass, check in, luggage/suitcase, carry-on, overweight,
aisle seat, window seat, direct flight, layover, gate, departure
```

**对话示例脚本（小龙虾引导版）：**

```
🦞：Good morning! Where are you flying today?

[学生回答目的地]

🦞：Great. Can I see your passport and ticket, please?

[学生出示证件——可以用 "Here you go" 或 "Here's my passport"]

🦞：Thank you. How many pieces of luggage are you checking in?

[学生回答数量]

🦞：Would you prefer a window seat or an aisle seat?

[学生选择]

🦞：Your boarding pass is ready. You're departing from Gate 12.
    boarding starts at 9:30. Have a good flight!
```

### 第二阶段：安检通道

**关键词汇清单：**

```
security check, belt, remove, liquids, electronic devices,
 tray, metal detector, prohibited items
```

**对话示例脚本：**

```
🦞（安检员）：Please place all electronic devices and liquids in the tray.
    Remove your belt and any metal items.

[学生操作——可以用 "Okay, I'm putting my phone in the tray" 等]

🦞：Do you have any liquids in your bag?

[学生回答——练习 "No, I don't" 或 "Just a small bottle of water" 等]

🦞：Please step through the metal detector.

[学生走过去]

🦞：All clear! You can collect your items. Have a safe trip!
```

**常见卡壳点与提示词：**

```
卡壳1：不知道怎么说出 "托运行李"
  → 提示词："check in luggage" 或选择："check in or carry-on?"

卡壳2：被问座位偏好时不知如何表达
  → 提示词："window — you can see outside, aisle — easier to get out"

卡壳3：安检环节不知如何描述动作
  → 提示词："put in the tray" / "take off" / "step through"
```

**复盘重点标注：**

```
本次场景核心表达：
  ① "I'd like a window/aisle seat" — 座位选择
  ② "Here's my passport" — 递交证件
  ③ "I'm checking in [N] pieces of luggage" — 托运数量
  ④ "All clear" — 安检通过

常见升级机会：
  初级："I want window" → 升级："I'd prefer a window seat if possible"
  初级："Put this here" → 升级："I'm placing my devices in the tray"
```

## 场景②：商店购物比较商品 ⭐

### 角色分配

```
学生角色：顾客（Customer）
小龙虾角色：店员（Shop assistant）

地点：电子产品商店 / 服装店（学生自选）
```

### 对话示例脚本（电子产品商店版）

**关键词汇清单：**

```
price, compare, feature, recommend, warranty, discount,
model, size, color, return policy, receipt, try out
```

```
🦞：Hi there! Can I help you find anything today?

[学生说明需求——如 "I'm looking for a phone / a laptop"]

🦞：What's your budget? And what do you mainly use it for?

[学生回答——练习 "I need it for..." + "My budget is around..."]

🦞：I'd recommend this model. It has [特点] and costs [价格].
    There's also this one—it's a bit more expensive,
    but it has [更好的特点]. Which one sounds better for you?

[学生比较和选择——练习比较句式 "This one is... but that one has..." ]

🦞：Great choice! Would you like to see how it works first?

[学生确认——练习 "Can I try it out?" 或 "I'll take it"]

🦞：It comes with a one-year warranty. Would you like extended coverage?
    That's an extra [价格].

[学生决定——练习 "I'll just take the standard warranty" 或 "Yes, I'd like that"]

🦞：Here's your receipt. You can return it within 30 days
    if you're not satisfied. Enjoy your new [产品]!
```

**常见卡壳点与提示词：**

```
卡壳1：比较两个产品时不知如何表达差异
  → 提示词："more expensive / cheaper / better / lighter / faster"

卡壳2：不知如何询问保修
  → 提示词："warranty" — 选择："one year or extended?"

卡壳3：砍价/问折扣表达困难
  → 提示词："Is there any discount?" / "Can you give me a better price?"
```

**复盘重点标注：**

```
本次场景核心表达：
  ① "I'm looking for..." — 表达需求
  ② "What's the difference between these two?" — 比较
  ③ "I'll take this one" — 决定购买
  ④ "Does it come with a warranty?" — 询问保障

常见升级机会：
  初级："I want this" → 升级："I'd like to go with this model"
  初级："How much?" → 升级："What's the price on this one?"
```

## 场景③：西餐厅点餐结账 ⭐

### 角色分配

```
学生角色：顾客（Customer）
小龙虾角色：服务员（Waiter/Waitress）

地点：西式餐厅
```

### 对话示例脚本

**关键词汇清单：**

```
menu, order, appetizer, main course, dessert, beverage,
recommend, special, spicy, vegetarian, bill, tip, change,
fork/knife, napkin, dressing
```

```
🦞：Good evening! Welcome to [restaurant name].
    Here's the menu. Can I get you started with something to drink?

[学生点饮料——练习 "I'd like a..." 或 "Can I have..." ]

🦞：Great. Are you ready to order, or do you need a few more minutes?

[学生决定——练习 "I'm ready" 或 "Can I have a few more minutes?" ]

🦞：What would you like for your main course?

[学生点主菜——练习描述偏好 "I'd like the... / Do you have anything vegetarian?" ]

🦞：Excellent choice! How would you like your steak cooked?
    Rare, medium, or well-done?

[学生选择熟度——如适用]

🦞：Would you like to see our dessert menu as well?

[学生决定——练习 "Yes, please" 或 "Maybe later" ]

[用餐结束]

🦞：Can I get you anything else? ... Ready for the bill?

[学生请求结账——练习 "Could I have the bill, please?" ]

🦞：Here you go. Payment by card or cash?

[学生选择支付方式]

🦞：Thank you! Have a wonderful evening!
```

**常见卡壳点与提示词：**

```
卡壳1：点菜时不知如何描述偏好
  → 提示词："spicy / mild / vegetarian / grilled / fried"
  → 选择法："Do you prefer A or B?"

卡壳2：牛排熟度相关表达
  → 提示词："rare = red inside, medium = pink, well-done = fully cooked"

卡壳3：结账环节表达
  → 提示词："the bill / the check" — 选择："card or cash?"
```

**复盘重点标注：**

```
本次场景核心表达：
  ① "I'd like to order..." — 点菜
  ② "Can I have a few more minutes?" — 延时
  ③ "Could I get the bill, please?" — 结账
  ④ "Keep the change" — 给小费（可选教）

常见升级机会：
  初级："I want beef" → 升级："I'd like the steak, medium please"
  初级："Pay now" → 升级："Could I settle the bill by card?"
```

## 场景④：2分钟即兴演讲+听众提问 ⭐⭐⭐

### 角色分配

```
学生角色：演讲者（Speaker）
小龙虾角色：听众（Audience member）→ 提问者

地点：英语角 / 模拟演讲比赛
```

### 流程脚本

**关键词汇清单：**

```
introduce, topic, opinion, reason, example, conclusion,
agree/disagree, audience, question, respond, summarize
```

```
Step 1：选话题并准备（30秒）
🦞：Your topic is "[随机话题]".
    You have 30 seconds to think, then speak for 1-2 minutes.
    Structure: opening → main points → conclusion.

[学生准备——可以说 "Let me think for a moment"]

Step 2：演讲（1-2分钟）
🦞：Go ahead! I'm listening.

[学生演讲——小龙虾记录：停顿次数/结构完整度/亮点词汇]

Step 3：听众提问（1-2个问题）
🦞：Thank you for that speech! I have a question:
    "[基于演讲内容的追问]"
    如："You said [观点]—can you give a specific example?"
    或："What if [反面假设]—would your opinion change?"

[学生回答问题——练习即兴应对]

Step 4：整体反馈
🦞：[给出复盘反馈——流利度/内容/可改进处]
```

**即兴演讲话题库（10个备选）：**

```
⭐⭐ 话题：
  1. My favorite hobby and why it matters
  2. The best advice I've ever received
  3. Why learning English is important for me
  4. A person who has influenced my life
  5. One thing I'd change about my school

⭐⭐⭐ 话题：
  6. Should homework be optional?
  7. Is social media making us less social?
  8. What does "success" mean to a teenager?
  9. If I could travel anywhere, where and why
  10. Technology: friend or enemy?
```

**复盘重点标注：**

```
本次场景核心表达：
  ① "I'd like to talk about..." — 开场
  ② "For example..." — 举例论证
  ③ "In my opinion..." — 表达观点
  ④ "To sum up..." — 总结

演讲结构检查：
  ✅ 有明确开场（不是直接跳入内容）
  ✅ 有2-3个支撑论点
  ✅ 有结尾（不是突然停下）
  ❌ 全部是事实罗列 → 需加入个人观点
  ❌ 只有一个论点 → 需补充论证层次
```

## 场景⑤：学校社团面试 ⭐⭐

### 角色分配

```
学生角色：申请者（Applicant）
小龙虾角色：面试官（Interviewer—社团负责人）

地点：学校社团招新面试
```

### 对话示例脚本

**关键词汇清单：**

```
apply, position, experience, skills, contribute, team,
schedule, responsibility, enthusiastic, committed,
strength, weakness, goal
```

```
🦞：Hi! Welcome to the [club name] interview.
    First, can you tell me a little about yourself?

[学生自我介绍——练习 "I'm [name], a [grade] student, and I..." ]

🦞：Why are you interested in joining our club?

[学生说明动机——练习 "I'm interested in... because..." ]

🦞：What experience do you have related to [club activity]?

[学生描述经历——练习 "I have done..." / "Last year I..." ]

🦞：What do you think you can contribute to the team?

[学生说明贡献——练习 "I can help with..." / "My strength is..." ]

🦞：Our club meets every [day] after school. Can you commit to that?

[学生确认时间——练习 "Yes, I can make that time" 或 "I might have a conflict on..." ]

🦞：One more question—what's one thing you'd like to improve about yourself?

[学生回答——练习自省表达 "I'd like to be better at..." ]

🦞：Thank you for coming! We'll let you know the result by [date].
    You did great!
```

**常见卡壳点与提示词：**

```
卡壳1：说明动机时只会说 "I like it"
  → 提示词："interested in / passionate about / curious about"
  → 选择法："是因为兴趣还是想学新技能？"

卡壳2：描述自身贡献时表达模糊
  → 提示词："organize / design / help / lead / communicate"
  → 选择法："你擅长组织还是创意？"

卡壳3：回答弱点时过度贬低自己
  → 引导：把 "weakness" 转为 "area to improve"
  → 示例："I'd like to improve my public speaking"
```

**复盘重点标注：**

```
本次场景核心表达：
  ① "I'm interested in joining because..." — 表达动机
  ② "My experience includes..." — 描述经历
  ③ "I can contribute by..." — 说明贡献
  ④ "I'm committed to attending regularly" — 表达承诺

常见升级机会：
  初级："I like this club" → 升级："I'm passionate about [topic] and want to deepen my involvement"
  初级："I'm good at X" → 升级："One of my strengths is [X], which I think could benefit the team"
```

## 六、场景扩展与自适应

```
当学生觉得5套内置场景不够时：
  → 可以用学生提出的新场景，小龙虾即兴扮演对手角色
  → 新场景的质量标准：有明确的角色分工 + 至少3个回合的对话机会

当学生在某个场景中表现特别好时：
  → 下次可以做该场景的"困难版"（如：机场场景增加航班延误、行李超重等意外）
  → 困难版训练目标：处理意外情况的表达能力

当学生角色扮演时间过短时（不到3个回合）：
  → 小龙虾主动推进对话："One more thing—[追加提问/请求]"
  → 避免对话过早结束导致训练量不足
```
