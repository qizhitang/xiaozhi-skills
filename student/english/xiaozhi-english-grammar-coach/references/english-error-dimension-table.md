# 英语错因维度表

> 本文档为 `xiaozhi-english-grammar-coach` 的核心参考资源，定义英语学科包的五类错因维度体系、子类型分类、跨维度关联规则与追踪标准。同时为英语口语陪练、写作教练、听力训练师等英语专项SKILL提供统一的错因分类框架。

---

## 一、维度总览

```
英语错因维度体系（五维×多子类型）

维度①  语法结构错误 ── 规则层（英语语法规则违反）
维度②  词汇应用错误 ── 语义层（词汇选择/搭配/词性偏差）
维度③  发音表达错误 ── 语音层（发音/流利度/语调问题）
维度④  听力理解错误 ── 接收层（听的信息获取偏差）
维度⑤  读写逻辑错误 ── 思维层（中式思维/答题逻辑偏差）

每道错题必须归入且只归入一个主维度。
如存在跨维度因素，记录为"主维度+次要维度"。

学术出处：维度设计参考 Paul & Elder 批判性思维模型，
  将错误分为"信息获取层""理解层""应用层""分析层""表达层"五个层级，
  映射到英语学习的五个具体维度。
```

## 二、维度①：语法结构错误

**定义：** 英语语法规则违反——时态、主谓一致、从句结构、介词搭配等方面的技术性错误。

**判断标准：** 学生知道想表达什么意思，但在用英语表达时违反了语法规则。

### 子类型分类

| 子类型ID | 名称 | 典型表现 | 高发学段 | 关联参考 |
|---------|------|---------|---------|---------|
| G01 | 时态混用 | 用一般现在时描述过去事件；过去时和完成时混用 | 初中全段 | → grammar-patterns §一 |
| G02 | 主谓一致 | 三单现在式遗漏；集合名词单复数不确定 | 初一-初二 | → grammar-patterns §二 |
| G03 | 定语从句错误 | 关系词误用(who/which)；多余代词 | 初三-高中 | → grammar-patterns §四 |
| G04 | 介词搭配错误 | interested at→in；arrive to→at/in | 初中全段 | → grammar-patterns §三 |
| G05 | 长难句结构混乱 | 从句嵌套错误；句子成分不完整 | 初三-高中 | → grammar-patterns §五 |
| G06 | 冠词使用混乱 | a/an/the遗漏或多余；抽象名词前冠词规则 | 初一-初二 | — |
| G07 | 比较级/最高级错误 | double comparative；比较对象不对 | 初一-初二 | — |
| G08 | 被动语态错误 | 被动句结构搭错；被动与主动混用 | 初二-初三 | — |
| G09 | 非谓语动词错误 | to do vs doing 选择偏差；不定式作主语 | 高中 | — |
| G10 | 虚拟语气错误 | 条件句虚拟与现实混用 | 高中 | — |

**严重程度分级：**

```
轻度：G06、G07 → 偶发性小错误，不影响理解
  → 处理：温和提示正确用法，不启动专项训练

中度：G01、G02、G04 → 规则理解有偏差，但经追问可自行修正
  → 处理：苏格拉底四步追问 → 确认理解 → 造句验证
  → 工具：grammar-patterns.md 对应专项追问

重度：G03、G05、G09 → 结构层面系统性偏差
  → 处理：专项训练营（定语从句五步循环法/长难句解剖法）
  → 需要4-6次追问才能建立正确直觉
```

**根因分析层：**

```
G01 时态混用的深层根因：
  → 中文没有时态标记（"我昨天去"和"我明天去"动词相同）
  → 学生大脑默认用中文思维 → 时态意识缺失
  → 不是"不知道规则"，而是"来不及想规则"
  → 治法：不背规则，而是在每次说话/写作前先问"这件事发生在什么时候？"

G02 主谓一致的深层根因：
  → 中文没有主谓一致要求（"他喜欢"和"他们喜欢"动词相同）
  → 三步检查链植入：找主语→判断人称/单复数→确认动词形式

G04 介词搭配的深层根因：
  → 中文介词系统与英语完全不同
  → 不是"搭配不对"，而是"空间逻辑不同"
  → 治法：场景联想法（in=沉浸/at=指向/on=接触表面）
```

## 三、维度②：词汇应用错误

**定义：** 词汇选择、搭配、词性使用偏差——不是"不认识这个词"，而是"认识但用错了"。

**判断标准：** 学生认识这个词的意思，但在具体使用时选择了错误的近义词、搭配或词性。

### 子类型分类

| 子类型ID | 名称 | 典型表现 | 高发学段 | 关联SKILL |
|---------|------|---------|---------|----------|
| V01 | 近义词辨析错误 | make/have/take/get混用；careful/meticulous辨析 | 初中+ | → vocabulary-dna |
| V02 | 词性使用错误 | 把形容词用在动词位置；名词当动词用 | 初一-初二 | → grammar-coach |
| V03 | 固定搭配错误 | do homework→make homework；listen music→listen to music | 初中全段 | → vocabulary-dna |
| V04 | 词汇量不足导致的绕路表达 | 想表达"frustrated"但只能说出"angry and sad" | 初中+ | → vocabulary-dna |
| V05 | 抽象词理解偏差 | 知道success的意思但不知道如何正确使用 | 高中 | → vocabulary-dna |
| V06 | 词根词缀推断错误 | 把un-开头的词全部理解为"否定" | 初三+ | — |

**严重程度分级：**

```
轻度：V04、V06 → 表达不够精确但不影响理解
  → 处理：词汇升级建议（"这里可以换个更精确的词"）
  → 联动词汇DNA：存入推荐升级词

中度：V01、V02、V03 → 使用偏差导致含义变化或语法错误
  → 处理：辨析练习 + 搭配训练
  → 联动词汇DNA：存入正确搭配，启动五轮提醒
  → 注意：V02词性错误可能同时也是G类语法错误 → 记录交叉维度

重度：V05 → 抽象词长期停留在"认识"层面，无法主动使用
  → 处理：从层次①(被动识别)升级到层次②(主动使用)的专项训练
  → 联动词汇DNA五轮提醒第3-4轮（造句/对话植入）
```

**与语法维度(G)的交叉判断：**

```
V02(词性使用错误) vs G02(主谓一致)：
  → 学生用了形容词代替动词 → 既是词性错误(V02)又是语法错误(G02)
  → 判断标准：如果把词性换成正确的，主谓一致还有问题吗？
    没有 → V02为主（词性搞错导致连带语法问题）
    还有 → G02为主（即使词性对了，主谓一致规则还是没掌握）

V03(固定搭配错误) vs G04(介词搭配错误)：
  → "listen music" → 是V03(搭配错)还是G04(介词遗漏)？
  → 判断：学生是否知道"listen to"这个搭配？
    不知道 → V03为主（词汇搭配不熟）
    知道但忘了用 → G04为主（语法规则应用失误）
```

## 四、维度③：发音表达错误

**定义：** 口语发音、流利度、语调方面的错误——不是"说错了内容"，而是"说错了声音/节奏"。

**判断标准：** 学生说的话在文字层面正确，但发音/节奏/语调层面有问题。

### 子类型分类

| 子类型ID | 名称 | 典型表现 | 高发学段 | 关联参考 |
|---------|------|---------|---------|---------|
| P01 | th音替代 | think→sink；this→zis | 全学段 | → pronunciation-issues §2.1 |
| P02 | r/l混淆 | right→light；red→led | 全学段(尤其南方) | → pronunciation-issues §2.2 |
| P03 | 长短元音不分 | sit→seat；live→leave | 初中+ | → pronunciation-issues §2.3 |
| P04 | v/w混淆 | very→wery；vest→west | 全学段 | → pronunciation-issues §2.4 |
| P05 | 词尾辅音遗漏 | and→an；test→tes | 全学段(极高频) | → pronunciation-issues §2.9 |
| P06 | 梅花音/短e不分 | bad→bed；cat→ket | 初中+ | → pronunciation-issues §2.5 |
| P07 | 语调模式错误 | 陈述句用升调；疑问句用降调 | 高中 | → pronunciation-issues §五 |
| P08 | 连读/弱化不足 | 逐词读出，没有英语节奏感 | 高中 | → pronunciation-issues §五 |
| P09 | 重音位置错误 | important重音放错位置 | 初中+ | → pronunciation-issues §五 |

**严重程度分级：**

```
轻度：P07、P08、P09 → 不影响理解但影响自然度
  → 处理：在口语热身复盘时温和指出
  → 联动口语DNA：记录但不启动专项纠音

中度：P03、P06 → 音位混淆导致含义变化
  → sit/seat；bad/bed → 听者可能误解
  → 处理：纠音闭环训练（确认→练习句→跟读→评分循环）
  → 联动口语DNA：标记为"初步弱项"

重度：P01、P02、P05 → 高频系统性发音偏差，严重影响可理解性
  → 几乎100%的中国学生受P01、P05影响
  → 处理：每次热身前提醒重点发音 → 专项纠音训练
  → 联动口语DNA：标记为"顽固弱项"
```

**发音追踪标准（与口语DNA联动）：**

```
出现1次 → 标记"偶发"，暂不进档案
出现2次 → 标记"初步弱项"，下次热身提醒
出现≥3次 → 标记"顽固弱项"，每次热身前必提醒
攻克标准：连续3次未出现 → 移至"已改善"
连续5次未出现 → 标记"攻克"，进口语里程碑
```

## 五、维度④：听力理解错误

**定义：** 听的信息获取偏差——不是"听到的词不懂"，而是"听的方式有问题"。

**判断标准：** 学生听了一段材料后理解偏差，需要区分是词汇/语法/语速哪一层面的问题。

### 子类型分类

| 子类型ID | 名称 | 典型表现 | 高发学段 | 关联SKILL |
|---------|------|---------|---------|----------|
| L01 | 词义不明 | 听清了声音但不知道意思 | 初中全段 | → vocabulary-dna |
| L02 | 语速/连读跟不上 | 词和词连在一起没反应过来 | 初二+ | → speaking-coach |
| L03 | 句子结构听不懂 | 每个词都认识但整句不理解 | 初三+ | → grammar-coach |
| L04 | 关键信息遗漏 | 听了整段但没抓住核心数据/时间/地点 | 初中全段 | → listening-trainer |
| L05 | 信息顺序错乱 | 听到的信息在脑海中的顺序与原文不同 | 初中+ | — |
| L06 | 注意力涣散 | 因话题不感兴趣导致后半段没听进去 | 全学段 | → learning-dna(兴趣) |

**严重程度分级：**

```
轻度：L06 → 注意力问题而非能力问题
  → 处理：调整话题匹配度（使用学生兴趣话题）
  → 联动学习DNA：读取兴趣档案优化话题选择

中度：L01、L04 → 词汇或信息提取层面问题
  → 处理：L01→存入词汇DNA(听力生词入库)
         L04→训练"先听核心信息"的注意力分配
  → 联动听力训练师：卡壳点精准追问(词义/结构/语速判断)

重度：L02、L03 → 语速或结构层面系统性障碍
  → L02→慢速拆解训练 + 连读规律讲解
  → L03→联动语法教练(长难句解剖法)
  → 需要专项训练突破
```

**与语法维度(G)和词汇维度(V)的交叉：**

```
L01(词义不明) vs V04(词汇量不足)：
  → 听力中遇到不认识的词 → 属于L01（听力层面获取问题）
  → 该词存入词汇DNA后 → 如果再次听到还是认不出 → 转为V01/V04（词汇应用问题）

L03(句子结构听不懂) vs G05(长难句结构混乱)：
  → 听不懂长句 → 先判断：看文字版本能理解吗？
    能理解 → L03为主（语速/节奏导致听不懂，不是结构问题）
    不能理解 → G05为主（结构本身理解有偏差）
```

## 六、维度⑤：读写逻辑错误

**定义：** 中式思维导致的表达逻辑偏差——不是"语法错"或"词汇错"，而是"思维模式错"。

**判断标准：** 学生用了正确的语法和词汇，但表达逻辑是中文式的，不符合英语思维习惯。

### 子类型分类

| 子类型ID | 名称 | 典型表现 | 高发学段 | 关联SKILL |
|---------|------|---------|---------|----------|
| T01 | 中式英语表达 | "I very like it"；"open the light" | 初中全段 | → writing-coach |
| T02 | 段落结构中文式 | 先铺垫再结论（英语习惯先结论再展开） | 高中 | → writing-coach |
| T03 | 论证逻辑偏差 | 只举例不分析；列举事实不建立因果链 | 高中 | → writing-coach |
| T04 | 阅读定位错误 | 在原文中找不到题目对应位置 | 初中全段 | → reading-decoder |
| T05 | 完形填空语境忽视 | 只看单句不考虑上下文逻辑 | 初二+ | — |
| T06 | 写作审题偏差 | 偏离题目要求的主题/格式 | 初中+ | → writing-coach |

**严重程度分级：**

```
轻度：T01 → 常见中式表达，可快速纠正
  → 处理：指出地道表达方式，存入词汇DNA

中度：T04、T05 → 读写方法层面问题
  → 处理：T04→联动阅读理解拆解师(定位训练)
         T05→训练"先读全文再逐空填写"的做题习惯

重度：T02、T03 → 写作逻辑层面系统性偏差
  → 处理：需要写作教练深度介入
  → 英语写作习惯：先结论→再展开→最后总结
  → 中文写作习惯：先铺垫→逐步展开→结论在最后
  → 需要多次训练才能建立英语写作逻辑直觉
```

## 七、跨维度关联规则

```
维度间常见交叉（英语特别多交叉）：

G01(时态混用) + V04(词汇量不足导致绕路)：
  → 学生用简单词汇描述过去事件，时态也跟着简化
  → 判断：给了高级词汇后，时态问题是否改善？
    改善 → V04为主（词汇量不足导致简化表达→连带时态简化）
    没改善 → G01为主（时态规则本身没建立直觉）

V02(词性使用错误) + G02(主谓一致)：
  → 用形容词代替动词 → 词性和主谓一致双重问题
  → 判断：纠正词性后主谓一致还有问题吗？
    没有 → V02为主
    还有 → G02为主

P01(th音替代) + L01(词义不明)：
  → 自己发音不准 → 听到正确发音时无法识别
  → 发音问题和听力问题互相强化
  → 处理：同时纠音+听力训练

T01(中式英语) + G类(语法错误)：
  → "I very like it" → 既是中式思维(T01)也是语法错误(G类)
  → 判断：学生是否知道"very"不能直接修饰动词？
    不知道 → G类为主（语法规则缺失）
    知道但习惯性犯 → T01为主（中式思维惯性）

跨维度记录格式：
  主维度：[维度ID+子类型ID]
  次要维度：[维度ID+子类型ID]（如有）

优先处理顺序：
  → G类(语法) > V类(词汇) > T类(逻辑) > P类(发音) > L类(听力)
  → 语法是骨架 → 词汇是砖块 → 逻辑是蓝图 → 发音是外观 → 听力是验收
  → 先修骨架(G) → 再补砖块(V) → 再调蓝图(T)
```

## 八、英语专项SKILL维度分配

```
各英语SKILL主要负责的维度：

语法突破教练：G类（主） + V02/V03（辅）
  → 语法DNA记录：G01-G10五类追踪

口语陪练：P类（主） + G类/G01/G02（辅，口语中的语法问题）
  → 口语DNA记录：P01-P09发音弱点追踪

听力训练师：L类（主） + V01（辅，听力生词）
  → 听力能力追踪报告：L01-L06卡壳类型分布

写作教练：T类（主） + G类（辅，写作中的语法问题） + V类（辅，词汇升级）
  → 三维批改法覆盖：T02/T03 + G类语法 + V类词汇

词汇DNA系统：V类（主） + G04（辅，介词搭配属于跨领域）
  → 词汇DNA五轮追踪：V01-V06词汇使用层面

数据流转规则：
  → 各SKILL发现非本职维度的错误 → 推送到对应SKILL
  → 口语中发现语法错误 → 推送到语法突破教练
  → 写作中发现发音相关错误（拼写反映发音） → 推送到口语陪练
  → 听力中发现词汇不足 → 推送到词汇DNA系统

不产生重复记录：
  → 语法DNA只记录G类错误（语法层）
  → 口语DNA只记录P类错误（发音层）
  → 词汇DNA只记录V类错误（词汇层）
  → 跨维度问题 → 主维度SKILL记录完整，辅维度SKILL只标记关联
```

## 九、顽固弱项追踪标准

```
同一子类型（相同维度+子类型ID+具体错误模式）出现次数追踪：

出现1次 → 标记"偶发"，低优先级
出现2次 → 标记"初步弱项"，下次练习提醒
出现≥3次 → 标记"顽固弱项"，启动专项突破

顽固弱项突破方法（按维度）：
  G类顽固弱项 → 苏格拉底四步追问 + 专项训练营
```
