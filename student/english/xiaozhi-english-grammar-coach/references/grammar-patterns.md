# 五类语法错误详细分析与追问话术扩展库

---

## 一、时态系统完整追问库

### 一般过去时（最高频错误）

**触发词识别：** yesterday, last week/month/year, ago, in [过去年份]

```
标准追问：
"你说的这件事，发生在什么时候？"
→ "那动词应该用什么形式——原形、过去式、还是过去分词？"
→ "[动词]的过去式是什么？"

快速确认：
"一句话确认：这件事已经发生了，对吗？
 已经发生的事，英语动词要用[?]"
```

**高频不规则动词速查（追问时可提示）：**

| 原形 | 过去式 | 过去分词 |
|-----|-------|---------|
| go | went | gone |
| see | saw | seen |
| eat | ate | eaten |
| write | wrote | written |
| come | came | come |
| run | ran | run |
| take | took | taken |
| give | gave | given |

### 现在完成时（中国学生最难掌握）

**核心区别：** 一般过去时 vs 现在完成时

```
一般过去时：强调动作发生在过去某个具体时间点
现在完成时：强调过去的动作对现在的影响/结果

追问话术：
"你想强调的是：这件事发生在什么时候（具体时间），
 还是：这件事到现在为止已经发生了（不指定时间）？"

→ 有具体时间：一般过去时（I went to Beijing last year.）
→ 强调已经发生/目前状态：现在完成时（I have been to Beijing.）
```

**追问示例：**

```
学生：I already finished my homework.

追问1："'already'和现在完成时经常搭配——
       你这里用的是一般过去时（finished）。
       换成现在完成时，'finish'应该变成什么形式？"

追问2："'I already finished'和'I have already finished'——
       感觉上有什么区别？
       （提示：前者更像在说一个事实，后者更像在说对现在的影响）"

追问3："用现在完成时造一个句子——
       说你已经完成了某件事，用 'already'。"
```

---

### 将来时（will vs be going to）

```
区别追问：
"你要表达的是：
 A. 临时决定，刚才才想到要做这件事
 B. 已经计划好了，早就打算要做

→ A用 will（spontaneous decision）
→ B用 be going to（prior plan）"

追问：
"这个计划是你早就有的，还是刚想到的？"
```

## 二、主谓一致追问扩展

### 三步检查链（植入到直觉层）

```
每次写句子前，先走三步：
Step 1：找主语（谁/什么是句子的主角？）
Step 2：判断人称和单复数（第几人称？单数还是复数？）
Step 3：确认动词形式（现在时第三人称单数加s/es）

追问植入练习：
"在你说这个句子之前，先告诉我：
 ① 主语是什么？
 ② 它是第几人称，单数还是复数？
 ③ 动词现在应该是什么形式？"
```

### 常见复杂情况追问

```
情况①：不定式/动名词作主语（单数）
错误：Swimming are good for health.
追问："'Swimming'是什么词性？它是单数还是复数？
     谓语动词应该是 are 还是 is？"

情况②：集合名词（视语义而定）
错误：The team are winning. （英式英语接受，美式不接受）
追问："这里'team'作为一个整体还是一群个体？
     （美式：团队作为整体=单数=is；
      英式：团队成员作为个体=复数=are）"

情况③：there is/are 倒装
错误：There is many students in the classroom.
追问："'students'是单数还是复数？
     'there is'后面跟单数，'there are'后面跟什么？"
```

## 三、介词搭配追问扩展

### 场景联想法（替代死记硬背）

```
in：沉浸进去，在某个空间/时间范围内
  → interested in（兴趣沉浸其中）
  → in the morning（在早上这个时间段内）
  → in the classroom（在教室这个空间内）

at：指向一个点/位置
  → arrive at（到达某个点）
  → good at（擅长某个具体技能）
  → at school（在学校这个位置）

on：接触表面，在某天
  → depend on（依靠在...上面）
  → on Monday（在星期一这个点上）
  → on the table（在桌面上）

for：为了，持续时间
  → wait for（为...等待）
  → for two hours（持续两小时）
```

### 高频错误介词对速查

| 错误搭配 | 正确搭配 | 场景联想追问 |
|---------|---------|------------|
| interested at | interested in | "兴趣是沉浸进去，还是指向一个点？" |
| arrive to | arrive at/in | "到达是指向终点（at）还是进入某地（in）？" |
| good on | good at | "擅长是精通某个技能点（at）" |
| married with | married to | "婚姻是连接两人（to）" |
| depend in | depend on | "依靠是站在...上面（on）" |
| talk to/with | talk to（单向）/talk with（双向） | "区别在于是否双向对话" |

## 四、定语从句追问话术完整版

### 关系词选择的完整判断树

```
先行词是人？
├── 是 → who（主语）/ whom（宾语，正式）/ that（非正式）
└── 否 → 先行词是物/事？
         ├── 是 → which / that
         └── 否 → 先行词是地点？
                  ├── 是 → where（表示在那里）/ which/that（其他情况）
                  └── 否 → 先行词是时间？
                           ├── 是 → when（表示在那时）
                           └── 否 → 先行词是原因？→ why
```

### 从句中关系词充当成分的判断

```
关键追问："把从句单独拿出来看，
          去掉关系词后，句子完整吗？
          → 完整：关系词是宾语（可省略）
          → 不完整：关系词是主语（不可省略）"

示例：
The book that I borrowed it...
拿出从句："I borrowed it"→ 完整（it是宾语）
→ 关系词that是宾语 → it多余，删掉

The girl who won the prize...
拿出从句："won the prize"→ 不完整（缺主语）
→ 关系词who是主语 → 不可省略
```

## 五、长难句追问话术

### 句子解剖五步法

```
Step 1：找谓语动词（中心动词）
追问："这个句子里，最核心的动作/状态词是什么？"

Step 2：通过谓语找主语
追问："是谁/什么在做这个动作，或处于这个状态？"

Step 3：找宾语/表语
追问："这个动作/状态的对象是什么？"

Step 4：把修饰成分括号化
追问："哪些部分是在修饰主语/动词/宾语的？用括号括起来。"

Step 5：读主干验证
"只读主语+谓语+宾语，意思通顺吗？"
```

### 四种常见复杂结构识别

```
结构①：主语较长（含修饰从句）
The discovery（which was made by scientists）changed everything.
→ 括号括起来，主干是：The discovery changed everything.

结构②：插入语/同位语
My teacher, （a well-known poet）, writes beautifully.
→ 括号括起来，主干是：My teacher writes beautifully.

结构③：多重定语从句嵌套
The city where I was born, which is now very developed, has changed a lot.
→ 从内到外逐层剥离

结构④：分词短语
（Finishing his homework）, he went out to play.
→ 分词短语是"完成作业"，主语是he
```
