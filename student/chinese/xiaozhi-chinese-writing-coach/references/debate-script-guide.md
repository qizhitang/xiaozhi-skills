# AI 辩论赛话术与赛制详解

> 本文档为 `xiaozhi-chinese-writing-coach` 模块E（AI辩论赛）的配套参考资源。
> 覆盖两套赛制（三轮辩论 + 苏格拉底审讯）的完整规则、裁判话术模板和进阶策略。

---

## 一、赛制一：三轮辩论

### 1.1 赛制规则

```
基本规则：
· 学生选择立场（正方/反方），小龙虾担任对立立场
· 辩论三轮，每轮格式：学生陈述 → 小龙虾反驳 → 学生回应（可选）
· 每轮限时：学生陈述2分钟，小龙虾反驳1分钟
· 小龙虾认真辩，找逻辑漏洞，不会故意让步
· 第三轮结束后，小龙虾立即切换为教练模式进行复盘

禁止行为：
· 小龙虾不能替学生写辩词
· 小龙虾的反驳必须针对学生的具体论点，不能泛泛攻击
· 复盘时不能只说"你输了"，要指出具体薄弱环节
```

### 1.2 适合辩题库

| 辩题 | 正方立场 | 反方立场 | 适合学段 |
|------|---------|---------|---------|
| 学生应该带手机上学 | 带手机便于学习和联系 | 手机导致分心和依赖 | 初中 |
| 课外作业应该取消 | 作业巩固学习效果 | 作业占时间、磨兴趣 | 初中 |
| AI应该替代老师 | AI可个性化教学 | 教师提供情感和判断 | 初中 |
| 考试是最好的评估方式 | 考试公平、可量化 | 考试压力大、不全面 | 初中 |
| 读书比看视频更有价值 | 书籍深度更高 | 视频信息密度大 | 小学高年级+初中 |
| 好成绩比好品格更重要 | 成绩是社会筛选标准 | 品格决定长远发展 | 初中 |

### 1.3 轮次话术模板

**第1轮开场（小龙虾作为反方）：**

```
"好，辩论开始。你是正方，我是反方。
 我不会让步——我会认真找你的漏洞。
 你先说你的立场，2分钟以内。"
```

**第1轮反驳（小龙虾）：**

```
"[复述学生的核心论点]
 我承认你说了[X]，但我要反驳——
 [指出具体逻辑漏洞或反例]
 你怎么回应这个？"
```

**第2轮推进（小龙虾加重反驳力度）：**

```
"你刚才回应了[上一个反驳]，但你仍然有一个问题：
 [提出新的逻辑漏洞或对方论据]
 第2轮，我追问更深——[具体问题]"
```

**第3轮收官：**

```
"最后一轮。
 [总结两轮以来学生最大的逻辑漏洞]
 [给出反方的最强一击]
 你的最终回应？"
```

### 1.4 辩论结束后的教练复盘

```
标准复盘格式：

"辩论结束，我切换成教练视角——

 你这场辩论最薄弱的地方是[具体指出]，
 因为你在这里[具体原因]，
 如果下次遇到这个反驳，可以这样回应：[给方向不给话术]

 但你有一个亮点：[指出学生做得好的地方]

 下次辩论前，建议先做一件事：[具体建议]"
```

## 二、赛制二：苏格拉底审讯

### 2.1 赛制规则

```
基本规则：
· 学生先陈述一个观点（一个完整的论点）
· 小龙虾只追问，不陈述自己的对立观点
· 每次追问必须针对学生上一句话的具体内容
· 目标：逼学生把每一步论证说得无懈可击
· 适合场景：考试前检验论点是否真的站得住脚

追问节奏：
· 不连续发出两个追问，等学生回答后再问下一个
· 不评判"对错"，只追问下一层
· 学生说"我不知道"时，给出一个选择："你觉得A还是B更合理？"
```

### 2.2 追问话术模板

```
开场：
"你先完整说出你的论点——我会一直追问，
 直到你说得没有漏洞为止。
 开始吧。"

追问层1（澄清边界）：
"你说[学生的论点]——这是在所有情况下都成立，
 还是只在某些条件下成立？哪些条件？"

追问层2（检验证据）：
"你说的[具体论据]——能举一个真实发生的例子吗？
 这个例子有没有反例？"

追问层3（找漏洞）：
"如果有人对你说[提出最可能的对立观点]，
 你怎么反驳他？他哪句话你可能无法回应？"

追问层4（推结论）：
"综合你说的这些，你的最终立场比一开始有没有变化？
 用一句话重新说你的论点——和刚才的对比一下。"
```

### 2.3 审讯结束后的教练复盘

```
标准复盘格式：

"审讯结束——

 你的论点从[原论点]进化到了[最终论点]。
 进化的地方：[指出]
 还不够稳的地方：[指出]

 下次写议论文时，记住：
 [一条从审讯中提炼的具体建议]"
```

## 三、辩论进阶策略

### 3.1 论点升级技巧

| 原论点（太宽） | 升级论点（有边界） | 升级方式 |
|--------------|-----------------|---------|
| 坚持有意义 | 坚持的意义在于让你发现自己能力的边界，但前提是你坚持的方向是对的 | 加边界条件 |
| 手机影响学习 | 手机对学习的负面影响主要发生在缺乏自律能力的学生身上 | 加适用人群 |
| AI不如老师 | AI在传授知识方面可以超过老师，但在情感支持和价值观引导上不能替代 | 承认部分反面 |

### 3.2 反驳常见模式

| 学生常见弱点 | 小龙虾反驳话术 | 学生可学的回应策略 |
|------------|---------------|-----------------|
| 论点太宽 | "你说的[论点]任何人都能说，它太大了——缩小到什么条件下才成立？" | 加边界条件，变成"有条件的成立" |
| 论据空洞 | "你说了[道理]，但没有具体例子——凭什么相信它？" | 准备2个非课本的自己的例子 |
| 只看一面 | "你只考虑了[A面]，但[B面]你怎么回应？" | 在论证中主动承认反面，再反驳 |
| 论证跳跃 | "你从[A]直接跳到了[C]，中间缺少了[B]——凭什么从A推出C？" | 每一步推理都要写"因为…所以…" |

### 3.3 从辩论到写作的迁移

```
辩论中练到的能力 → 写作中怎么用：

· 找到对方漏洞 → 写议论文时主动承认反面再反驳（让论证更严密）
· 缩小论点边界 → 写作时论点不要太大，加适用条件
· 准备具体例子 → 写作时不用课本老例子，用自己的真实经历
· 每步推理完整 → 写作时每个"因为…所以…"都不能跳步
```
