# 四类物理图景绘制追问手册

> 本文档为 `xiaozhi-physics-problem-coach` 的配套参考资源，提供四类物理图景（受力分析图、运动过程图、等效电路图、光路图）的分步绘制引导与常见错误追问序列。

---

## 一、受力分析图 · 四步法追问

### 绘制四步法

```
Step 1：选对象
  → 确定分析哪个物体的受力情况
  → 只画这个物体受到的力，不画它施加给别人的力

Step 2：画重力
  → 方向竖直向下，从物体重心出发
  → 标注 G=mg（即使题目没给质量也要画）

Step 3：画接触力
  → 找接触面：物体和哪些东西有接触？
  → 每个接触面可能产生什么力？
    · 接触面（面接触）→ 支持力（垂直于面）+ 摩擦力（平行于面）
    · 绳子 → 拉力（沿绳方向，指向绳子收缩方向）
    · 弹簧 → 弹力（沿弹簧方向）
    · 液体 → 浮力（竖直向上）

Step 4：检验平衡
  → 如果物体静止或匀速：合力为零
  → 水平方向力平衡吗？竖直方向力平衡吗？
  → 如果不平衡——是不是漏画了某个力？
```

### 追问序列

**Step 1 选对象追问：**

```
Q1："这道题要分析哪个物体？把它圈出来。"
Q2："只分析这一个物体——它受到的力，不是它施加的力。你能分清吗？"
Q3："如果你选了两个物体，需要分别画受力图，不能画在一起。"
```

**Step 2 画重力追问：**

```
Q1："不管什么情况，先画重力。方向是竖直向下——不是垂直于斜面！"
Q2："你标了G吗？如果题目给了质量，算出重力大小标上去。"
```

**Step 3 画接触力追问：**

```
Q1："这个物体和哪些东西有接触？数一数——地面？绳子？另一个物体？"
Q2："每个接触面可能产生两个力——支持力（垂直于面）和摩擦力（平行于面）。你画了几个？"
Q3："支持力的方向——垂直于接触面，不是垂直于地面！斜面上特别要注意。"
Q4："有没有摩擦力？——看物体有没有相对运动或相对运动趋势。
     静止在斜面上的物体有摩擦力吗？——有，沿斜面向上。"
Q5："绳子只能拉不能推——拉力方向沿着绳子，指向绳子收缩方向。"
Q6："浮力方向——竖直向上。不管物体在液体里什么姿态，浮力永远向上。"
```

**Step 4 检验平衡追问：**

```
Q1："物体是什么状态？静止？匀速直线运动？——如果是，合力为零。"
Q2："水平方向的力平衡了吗？向左的力=向右的力？"
Q3："竖直方向的力平衡了吗？向上的力=向下的力？"
Q4："如果某个方向不平衡——一定有漏画的力。回去检查接触面。"
```

### 常见错误追问

| 错误类型 | 追问话术 |
|---------|---------|
| 漏画重力 | "你的图上没有重力——所有物体都受重力，先画重力" |
| 漏画摩擦力 | "物体在斜面上，你画了摩擦力吗？它有相对运动趋势吗？" |
| 漏画浮力 | "物体在液体里——你画了浮力吗？浮力竖直向上" |
| 支持力方向错 | "支持力垂直于接触面，不是垂直于地面。斜面上特别注意" |
| 画了物体施加的力 | "这个力是你选的物体施加给别人的，不是它受到的——删掉" |
| 把速度当力画 | "速度不是力！力是物体之间的相互作用，速度是运动状态" |
| 多画了"向前的力" | "匀速运动的物体不需要'向前的力'——惯性使它继续运动" |

## 二、运动过程图 · 三步法追问

### 绘制三步法

```
Step 1：标状态
  → 运动过程有几个状态？每个状态的速度是多少？
  → 用圆圈或方框标出每个状态，标注速度值

Step 2：画过程
  → 用箭头连接各状态
  → 标注每个过程是加速/减速/匀速
  → 可以用v-t图辅助表示

Step 3：标转折
  → 转折点发生了什么变化？
  → 力的变化？速度的变化？方向的变化？
  → 前一段的末速度 = 下一段的初速度
```

### 追问序列

**Step 1 标状态追问：**

```
Q1："从开始到结束，运动过程能分成几段？每段运动有什么不同？"
Q2："每一段的初速度和末速度分别是多少？标注出来。"
Q3："有没有'静止'的状态？静止也是状态，速度为零。"
```

**Step 2 画过程追问：**

```
Q1："从第一个状态到最后一个状态，用箭头连起来。"
Q2："每一段是加速、减速还是匀速？——看合力方向和速度方向的关系。"
Q3："你能画一个v-t图吗？匀速是水平线，加速是上升线，减速是下降线。"
```

**Step 3 标转折追问：**

```
Q1："转折点前后，速度一样吗？——必须一样，这是连续性条件。"
Q2："转折点力发生了什么变化？——比如开始受摩擦力了，或者停止施力了。"
Q3："前一段的末速度就是下一段的初速度——你标出来了吗？"
```

### 常见错误追问

| 错误类型 | 追问话术 |
|---------|---------|
| 遗漏中间状态 | "从A到B之间，有没有其他状态？仔细看题目——'3秒后开始减速'说明有两段" |
| 转折速度不连续 | "你上一段末速度是5m/s，下一段初速度标了3m/s——转折点速度必须连续" |
| 过程类型标错 | "匀减速的v-t图是下降直线，不是水平线。画一下v-t图确认" |
| 多阶段只分析一段 | "运动过程有三段，你只画了一段——每段都要分别分析" |
| 混淆路程和位移 | "位移是从起点到终点的直线距离，路程是实际走过的长度。你标注的是哪个？" |

## 三、等效电路图 · 三步法追问

### 绘制三步法

```
Step 1：识别串并联
  → 从电源正极出发，电流有几条路径？
  → 只有一条路径 → 串联
  → 有分支（节点） → 并联
  → 既有串联又有并联 → 混联

Step 2：画等效电路
  → 电压表当断路（内阻很大，几乎无电流通过）
  → 电流表当导线（内阻很小，不影响电路）
  → 每个开关状态画一张等效电路图
  → 用标准符号重新画出简化电路

Step 3：标电流方向
  → 从电源正极出发，标出电流经过的路径
  → 并联时标注各支路电流关系
  → 串联时标注 I=I1=I2
  → 标注各部分电压关系
```

### 追问序列

**Step 1 识别串并联追问：**

```
Q1："从电源正极出发，电流沿着导线走——有分叉吗？"
Q2："如果只有一个开关控制所有用电器，可能是串联。如果各用电器独立工作，可能是并联。"
Q3："短路的用电器相当于导线，没有电流通过。你判断对了吗？"
```

**Step 2 画等效电路追问：**

```
Q1："电压表内阻很大——当断路处理，从电路中拿掉不影响分析。"
Q2："电流表内阻很小——当导线处理，用导线替换不影响分析。"
Q3："开关S1闭合、S2断开时，电路结构是什么？画一张等效电路。"
Q4："开关S1断开、S2闭合时呢？再画一张。每个状态画一张。"
Q5："滑动变阻器的滑片位置不同，等效电路也不同吗？——如果接入的有效电阻变了，就要重新分析。"
```

**Step 3 标电流方向追问：**

```
Q1："从电源正极出发，沿着导线走——电流先过谁？再过谁？"
Q2："到了分叉点，电流分成几路？每路过了哪个用电器？"
Q3："串联电路：I=I1=I2，各处电流相等。你标了吗？"
Q4："并联电路：I=I1+I2，干路电流等于支路电流之和。你标了吗？"
Q5："电压关系——串联U=U1+U2，并联U=U1=U2。标注出来。"
```

### 常见错误追问

| 错误类型 | 追问话术 |
|---------|---------|
| 串并联判断错 | "从正极出发走到负极，如果有分叉就是并联，没有就是串联。你重新走一遍" |
| 电压表没当断路 | "电压表内阻极大，通过的电流忽略不计。把它从电路中拿掉，重新判断串并联" |
| 电流表没当导线 | "电流表内阻极小，用导线替换它。替换后电路结构变了吗？" |
| 没分析每个开关状态 | "开关状态不同，电路结构可能完全不同。每个状态都要画一张等效电路" |
| 短路判断错误 | "被短路的用电器两端电压为零，没有电流通过。你确认哪个被短路了吗？" |
| 电流方向标反 | "电流从电源正极出发，回到负极。在电路外部，从正到负" |

## 四、光路图 · 三步法追问

### 绘制三步法

```
Step 1：确定介质
  → 光线从什么介质进入什么介质？
  → 空气→水？空气→玻璃？玻璃→水？
  → 介质的疏密关系决定了折射方向

Step 2：确定入射面
  → 入射面/反射面/折射面在哪里？
  → 画出法线（垂直于界面）
  → 标注入射角、反射角、折射角

Step 3：画反射/折射路径
  → 反射：入射角=反射角，在法线两侧
  → 折射：从光疏到光密，靠近法线；从光密到光疏，远离法线
  → 透镜：用三条特殊光线作图
```

### 追问序列

**Step 1 确定介质追问：**

```
Q1："光线从哪里来？射到哪里去？——两种介质的界面在哪里？"
Q2："这两种介质哪个是光疏介质（折射率小），哪个是光密介质（折射率大）？"
Q3："光从空气进入水——空气是光疏，水是光密。折射光线会靠近法线还是远离法线？"
```

**Step 2 确定入射面追问：**

```
Q1："界面在哪里？画出法线——垂直于界面。"
Q2："入射角是入射光线和法线的夹角——不是和界面的夹角！"
Q3："你能量出入射角是多少度吗？反射角和入射角相等。"
```

**Step 3 画路径追问：**

```
Q1："反射光线——入射角=反射角，在法线两侧，在同一平面内。画出来。"
Q2："折射光线——从光疏到光密，靠近法线；从光密到光疏，远离法线。你画对方向了吗？"
Q3："如果是凸透镜——用三条特殊光线：
     ① 平行于主光轴的光→折射后过焦点
     ② 过光心的光→方向不变
     ③ 过焦点的光→折射后平行于主光轴
     你用了哪条？"
Q4："如果是凹透镜——
     ① 平行于主光轴的光→折射后反向延长过虚焦点
     ② 过光心的光→方向不变
     你用了哪条？"
Q5："成像是实像还是虚像？
     实像：光线实际交汇，在光屏上能看到，在透镜另一侧
     虚像：光线延长线交汇，光屏看不到，在透镜同侧"
```

### 常见错误追问

| 错误类型 | 追问话术 |
|---------|---------|
| 法线方向画错 | "法线垂直于界面，不是垂直于入射光线。重新画法线" |
| 入射角和界面角混淆 | "入射角是光线和法线的夹角，不是和界面的夹角。量一下" |
| 反射角不等于入射角 | "反射定律：入射角=反射角。你画的两个角一样大吗？" |
| 折射方向搞反 | "从空气进入水，光线靠近法线。你画的是靠近还是远离？" |
| 凸透镜特殊光线画错 | "平行于主光轴的光，折射后一定过焦点。你画过焦点了吗？" |
| 实像虚像判断错 | "实像在透镜另一侧，能用光屏接收。虚像在同侧，光屏接不到" |
| 凹透镜画成凸透镜 | "凹透镜成虚像，一定缩小，一定在同侧。你画对了吗？" |
| 全反射条件不满足 | "全反射需要两个条件：①从光密到光疏 ②入射角大于临界角。你检查了吗？" |

## 六、图景绘制与错误基因档案的映射

```
图景绘制中发现错误时的归档映射：

受力分析图阶段错误 → P01 不画受力分析图 / P05 参考系选取错误
运动过程图阶段错误 → P02 运动过程分析不清
等效电路图阶段错误 → P03 电路图识别错误
光路图阶段错误 → P04 光路图画错
隐含条件未体现在图中 → P06 隐含条件未图景化
多对象/多状态不会分别画图 → P07 多对象关系图缺失

错误发现后的处理：
  轻度（遗漏1个力/1个状态）→ 现场纠正，要求补充后继续
  中度（图景方向性错误）→ 要求擦掉重画，建立正确图景
  重度（完全不会画图）→ 进入分步引导模式，从Step 1开始逐步引导

详见 physics-error-dimension-table.md（P类维度定义）
```
