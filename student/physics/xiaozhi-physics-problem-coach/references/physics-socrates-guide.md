# 物理苏格拉底三层次追问适配指南

> 本文档为 `xiaozhi-physics-problem-coach` 的配套参考资源，提供物理苏格拉底三层次追问（模块B）的完整话术库、学科领域适配策略与学段认知负荷调控方法。
> 基于Paul & Elder批判性思维框架，针对物理学科"先看见再理解"的认知特点进行改编。

---

## 一、三层次追问框架总览

```
物理苏格拉底三层次追问（基于 Paul & Elder 批判性思维模型 · 物理改编版）

第一层  现象层 — "你看到了什么物理现象？"
  目标：确保学生真正"看见"了物理图景，而非只会套公式
  对应批判性思维：Clarity（清晰性）——能否准确感知和描述物理情境
  物理独有：数学追问第一层是"清晰达意"，物理第一层是"看见现象"
  → 物理学习的根本困难是"看不见"，不是"说不清"

第二层  原理层 — "为什么会出现这个现象？"
  目标：从现象上升到规律，确认学生理解了物理原理及其适用条件
  对应批判性思维：Accuracy + Precision（准确性+精确性）
  → 区分"知道公式"和"理解规律"

第三层  迁移层 — "如果改变条件，结果会怎样？"
  目标：从一道题到一类题，确认理解可迁移
  对应批判性思维：Breadth + Depth（广度+深度）
  → 真正的理解是可以预测新情境的

学术出处：
  Paul, R. & Elder, L. (2006). Critical Thinking:
    Tools for Taking Charge of Your Learning and Your Life.
  改编依据：物理认知特殊性——图景建立是理解的前提
```

---

## 二、学段适配总原则

### 认知负荷分级

```
八年级（力学入门，约13-14岁）：
  认知特点：
    → 抽象思维快速发展但尚不稳定，依赖具体图景
    → 物理直觉正在建立，容易被公式迷惑
    → "看懂了"和"真懂了"差距极大
    → 力学是物理的入门，概念正在形成中
  适配策略：
    → 只使用现象层+简化版原理层（2层版）
    → 现象层追问2-3次，原理层追问1-2次
    → 大量使用"闭上眼想象"的引导方式
    → 用"如果...会怎样"替代正式的条件变更
    → 每次只改一个条件，不改多个
    → 多用生活实例类比（走路、骑车、推门等）

九年级（电学进阶，约14-16岁）：
  认知特点：
    → 抽象思维较成熟，能处理多变量问题
    → 已有力学基础，但电学更抽象（看不见电流）
    → 容易停留在"记住公式"而非"理解电路"
    → 对"电路状态变化"分析能力不足
  适配策略：
    → 完整使用三层追问
    → 现象层追问1-2次（已有图景意识），原理层追问2-3次，迁移层追问2-3次
    → 允许学生自己设计条件变更
    → 强调"看不见的电学"→用类比（水流、交通）建立直觉
    → 多状态电路分析是重点训练方向
```

## 三、第一层：现象层追问详解

### 3.1 力学现象层追问（八年级重点）

**目标：** 确保学生能"看见"力的存在和力的作用效果，而不只是记住力的名称。

**追问话术库（8个模板）：**

```
模板①（最基础）：
  "闭上眼睛想象一下——
   题目描述的这个物体，它周围有什么？
   你能'看见'它在什么环境中吗？"

模板②（力的识别）：
  "这个物体受到了哪些力的作用？
   每个力是谁施加的？方向朝哪里？
   你能用手势比划一下吗？"

模板③（力的效果）：
  "如果这些力都消失了，物体会怎样？
   哪个力让物体保持不动？哪个力想让它动？"

模板④（平衡判断）：
  "物体现在是什么状态？静止？匀速？
   这说明了什么？——所有力的效果相互抵消了对吗？"

模板⑤（运动描述）：
  "这个物体是怎么运动的？
   一开始怎样？后来怎样？
   你能用手画出它的运动轨迹吗？"

模板⑥（过程分解）：
  "从开始到结束，运动过程能分成几段？
   每段运动有什么不同？——加速？减速？匀速？"

模板⑦（压强现象）：
  "压力作用在哪里？接触面有多大？
   你能感受到'压强'吗——就是压力的效果？
   为什么图钉尖能扎进去，图钉帽扎不进去？"

模板⑧（浮力现象）：
  "物体在液体里——它为什么没沉到底？
   什么东西在'托'着它？
   你能想象液体对它上下表面的压力差吗？"
```

**力学现象层追问的判断标准：**

```
通过：
  → 学生能说出"物体受到重力和支持力"（力的识别）
  → 学生能说出"合力为零所以静止"（力的效果）
  → 学生能说出"先加速后匀速"（运动过程分解）
  → 学生能用手势或语言描述物理场景

未通过：
  → 学生只说"用F=mg" → 只记住了公式，没看见现象
  → 学生说"重力向下"但说不清谁施加的 → 现象感知不完整
  → 学生无法描述运动过程 → 过程图景未建立
  → 回退S1重新画图
```

### 3.2 电学现象层追问（九年级重点）

**目标：** 确保学生能"看见"电路结构和电流路径，而不只是记住公式。

**追问话术库（8个模板）：**

```
模板①（电路结构）：
  "从电源正极出发，电流有几条路可以走？
   只有一条→串联，有分叉→并联。
   你能用手画出电流的路径吗？"

模板②（开关状态）：
  "开关闭合前后，电流的路径变了没有？
   闭合S1时电流怎么走？断开S1时呢？
   画出两种情况下的等效电路。"

模板③（电流分配）：
  "干路电流到了分叉点——它是怎么分配的？
   哪条路阻力小，哪条路电流就大。
   你能想象成车流分流吗？"

模板④（电压分布）：
  "串联电路里，电源电压被几个用电器'瓜分'了。
   谁电阻大，谁分到的电压就多。
   你能想象成几个人分蛋糕吗？电阻大的胃口大。"

模板⑤（电表读数）：
  "电压表测的是谁两端的电压？——它并联在谁旁边？
   电流表测的是通过谁的电流？——它串联在谁的支路上？
   先搞清楚它们'盯着'谁。"

模板⑥（电功率现象）：
  "灯泡亮度的本质是什么？——实际功率。
   实际功率大就亮，小就暗。
   同一个灯泡，电压变了，亮度会变吗？"

模板⑦（故障现象）：
  "灯不亮了——是断路还是短路？
   如果是断路，和它串联的灯也不亮；
   如果是短路，和它串联的灯会更亮。
   你能用'路通了没'来想吗？"

模板⑧（动态电路）：
  "滑片向右移——电阻怎么变？→总电阻怎么变？
   →总电流怎么变？→各部分电压怎么变？
   你能像多米诺骨牌一样，一个推一个地想吗？"
```

### 3.3 光学现象层追问（八年级）

**追问话术库（6个模板）：**

```
模板①：
  "光线从哪里来？射到什么表面上？
   你能画出光线的'来路'吗？"

模板②：
  "反射面是平的还是弯的？
   如果是平的，反射光线方向不变；
   如果是弯的（凹面镜/凸面镜），反射光线会聚还是发散？"

模板③：
  "光线从空气进入水中——方向变了吗？
   往哪边偏？靠近法线还是远离法线？
   你能想象一根筷子插进水里变弯的样子吗？"

模板④：
  "凸透镜有三条特殊光线——
   平行于主光轴的光，过光心的光，过焦点的光。
   这三条光线你用了哪条？为什么选它？"

模板⑤：
  "成的像是实像还是虚像？
   实像能接到光屏上，虚像接不到。
   你能用手势比划像的位置吗——在透镜哪一侧？"

模板⑥：
  "物体在二倍焦距外、一倍到二倍焦距之间、一倍焦距内——
   成像情况分别是什么？
   你能记住'远缩小近放大，一倍焦距分虚实'吗？"
```

### 3.4 热学现象层追问（八年级）

**追问话术库（5个模板）：**

```
模板①：
  "温度在变化吗？升高还是降低？
   什么导致了温度变化？——吸热还是放热？"

模板②：
  "物态变化了吗？固态→液态？液态→气态？
   这个过程需要吸热还是放热？
   你能联想到生活中的例子吗？——比如冰块融化。"

模板③：
  "两种物质吸收相同的热量，温度升高不一样——
   谁升得少？说明谁的比热容大？
   你能想象'胃口大的人吃一样多的饭，体重增长反而慢'吗？"

模板④：
  "燃料燃烧放出的热量——完全燃烧和不完全燃烧一样吗？
   热值是燃料的属性，和燃烧条件无关。"

模板⑤：
  "热量传递的方向是什么？从高温到低温。
   你能想象两杯水混合后的温度吗？
   一定在两个温度之间。"
```

## 四、第二层：原理层追问详解

### 4.1 八年级知识模块 · 原理层追问

#### 模块A：力与运动

```
追问方向1：牛顿第一定律（惯性）
  "为什么物体不受力时会保持匀速或静止？
   你能举一个惯性的例子吗？——公交车刹车时人往前倾。
   如果力突然消失，物体会怎样？——保持当时的状态不变。"

追问方向2：力的平衡
  "二力平衡的条件是什么？——等大、反向、共线、同体。
   这四个条件缺一个会怎样？
   你能画出一个不满足'共线'的力吗？——物体会转动。"

追问方向3：弹力与弹簧
  "弹力产生的条件是什么？——物体发生弹性形变。
   弹簧测力计的原理是什么？——弹簧的伸长量与拉力成正比。
   这个'成正比'有适用范围吗？——超过弹性限度就不成了。"

追问方向4：摩擦力
  "摩擦力的方向是什么？——与相对运动方向相反。
   '相对运动方向'是什么意思？——是相对于接触面的方向。
   你能分辨'滑动摩擦力'和'静摩擦力'吗？
   人走路时，脚受到的是哪种摩擦力？方向朝哪？——静摩擦力，朝前。"

追问方向5：压强
  "压强公式的物理意义是什么？——单位面积上受到的压力。
   固体压强和液体压强计算方法为什么不同？
   固体：先找压力再算压强；液体：先算压强再算压力。
   你知道为什么吗？——液体的压力不等于液体重力。"

追问方向6：浮力
  "浮力产生的原因是什么？——上下表面的压力差。
   阿基米德原理的适用条件是什么？——液体或气体中。
   物体在液体中一定受浮力吗？——不一定，如果底部与容器紧密接触就没有。"
```

#### 模块B：简单机械与功

```
追问方向1：杠杆
  "杠杆平衡条件的原理是什么？——力矩平衡。
   力臂是什么？——支点到力的作用线的距离。
   力臂一定是杠杆的长度吗？——不是，是最短距离，要画垂线。"

追问方向2：滑轮
  "定滑轮的本质是什么？——等臂杠杆，不省力但改变方向。
   动滑轮的本质呢？——动力臂是阻力臂两倍的杠杆，省一半力。
   为什么实际使用时不能省一半？——摩擦和滑轮自重。"

追问方向3：功和功率
  "做功的两个必要因素是什么？——力和在力方向上的距离。
   '力的方向上的距离'——如果力水平但物体斜着走，算功吗？
   功率的物理意义是什么？——做功的快慢，不是做功的多少。"

追问方向4：机械效率
  "为什么机械效率一定小于1？——有用功总小于总功。
   额外功去哪了？——克服摩擦、提升动滑轮自重。
   机械效率和功率有关系吗？——没有，效率高不一定快。"
```

#### 模块C：声与光

```
追问方向1：声音
  "声音产生的条件是什么？——振动。
   真空能传声吗？——不能，声音需要介质。
   你能区分音调、响度、音色吗？——频率、振幅、波形。"

追问方向2：光的传播
  "光在同种均匀介质中怎样传播？——沿直线。
   '均匀'很重要——不均匀的话光会弯（海市蜃楼）。"

追问方向3：反射与折射
  "反射定律的三条内容是什么？——共面、分居、等角。
   折射定律的关键是什么？——从光疏到光密，靠近法线。
   全反射的条件是什么？——从光密到光疏，入射角大于临界角。"
```

### 4.2 九年级知识模块 · 原理层追问

#### 模块D：欧姆定律

```
追问方向1：电流与电压
  "电流是怎么产生的？——电荷的定向移动。
   电压的作用是什么？——推动电荷定向移动，就像水压推动水流。
   没有电压会有电流吗？——不会，没有推动力。"

追问方向2：电阻
  "电阻的本质是什么？——导体对电流的阻碍作用。
   影响电阻的因素有哪些？——材料、长度、横截面积、温度。
   电阻随电压变化吗？——不随，电阻是导体本身的属性。"

追问方向3：欧姆定律
  "欧姆定律的适用条件是什么？——纯电阻电路。
   什么叫纯电阻电路？——电能全部转化为内能的电路。
   电动机是纯电阻吗？——不是，电能转化为内能+机械能。"

追问方向4：伏安法测电阻
  "伏安法的原理是什么？——R=U/I。
   安培表内接法和外接法有什么区别？——误差来源不同。
   滑动变阻器的作用是什么？——改变电压，多次测量求平均值。"
```

#### 模块E：电功率

```
追问方向1：电功与电功率
  "电功的本质是什么？——电流做的功，电能转化为其他形式能。
   电功率的物理意义？——电流做功的快慢。
   额定功率和实际功率的区别？——额定是标称值，实际看电压。"

追问方向2：焦耳定律
  "焦耳定律的适用条件是什么？——纯电阻电路。
   非纯电阻电路怎么算热量？——Q=W-E其他。
   串联和并联电路中，谁的发热多？
   串联：R大的发热多；并联：R小的发热多。"

追问方向3：测量电功率
  "测量电功率的原理是什么？——P=UI。
   和伏安法测电阻的电路一样吗？——基本相同。
   为什么要测多次？——看额定功率和实际功率的区别。"
```

#### 模块F：电与磁

```
追问方向1：电流的磁效应
  "奥斯特实验说明了什么？——电流周围有磁场。
   通电螺线管的磁场方向怎么判断？——安培定则（右手螺旋定则）。"

追问方向2：电磁感应
  "产生感应电流的条件是什么？——闭合电路+部分导体切割磁感线。
   '切割'是什么意思？——导体运动方向与磁感线不平行。
   发电机和电动机的区别？——发电机：动生电；电动机：电生动。"
```

## 五、第三层：迁移层追问详解

### 5.1 力学迁移层追问

```
条件变更系列1（八年级·力与运动）：
  "如果一个物体在水平面上匀速运动——
   ① 如果把水平面换成斜面，运动状态会怎么变？
   ② 如果在物体上再放一个重物，摩擦力怎么变？速度会变吗？
   ③ 如果把接触面换成冰面（摩擦力几乎为零），物体会怎样？"

条件变更系列2（八年级·压强与浮力）：
  "如果一个物体漂浮在水面上——
   ① 如果换成盐水，浮力变不变？排开液体体积变不变？
   ② 如果把物体按入水中再松手，它会怎样？为什么？
   ③ 如果物体是空心的，和实心相比，浮力一样吗？为什么？"

条件变更系列3（八年级·简单机械）：
  "如果用杠杆撬重物——
   ① 如果动力臂变长，动力怎么变？省力还是费力？
   ② 如果支点移到动力作用点和阻力作用点之间呢？
   ③ 如果动力方向不是竖直的而是斜的，力臂怎么变？"
```

### 5.2 电学迁移层追问

```
条件变更系列1（九年级·欧姆定律）：
  "如果电路中串联一个电阻R1——
   ① 再串联一个R2，总电阻怎么变？总电流怎么变？
   ② 把R2改成并联，总电阻怎么变？各支路电流怎么分？
   ③ 如果R1是滑动变阻器，从最大值调到最小值，
     电路中各物理量怎样变化？——像多米诺骨牌一样推。"

条件变更系列2（九年级·电功率）：
  "如果一个灯泡标有'220V 40W'——
   ① 接在110V电压上，实际功率是多少？（不是20W！）
   ② 和另一个'220V 60W'的灯泡串联，谁更亮？为什么？
   ③ 如果两个灯泡并联呢？谁更亮？为什么？
   → 关键：串联比电阻，电阻大的灯泡更亮
         并联比电阻，电阻小的灯泡更亮"

条件变更系列3（九年级·动态电路）：
  "如果电路中有滑动变阻器——
   ① 滑片向左移，电阻怎么变？→总电阻？→总电流？→各电压？
   ② 如果再并联一个电阻，总电阻变大还是变小？——一定变小
   ③ 如果把电压表从测R1改成测R2，读数怎么变？"
```

### 5.3 光学迁移层追问

```
条件变更系列1（八年级·透镜成像）：
  "如果物体在凸透镜前移动——
   ① 物体从远处靠近透镜，像怎么变？——变大变远
   ② 经过焦点时发生了什么？——不成像（折射光线平行）
   ③ 如果换成凹透镜呢？——永远成虚像，缩小，同侧
   ④ '远缩小近放大，一倍焦距分虚实'——你能解释每句话吗？"
```

### 5.4 热学迁移层追问

```
条件变更系列1（八年级·比热容）：
  "如果用相同的酒精灯加热等质量的水和油——
   ① 谁的温度升高得快？为什么？——油的比热容小
   ② 如果升高相同的温度，谁需要的时间长？——水
   ③ 这个原理在生活中有什么应用？——暖气用水，散热用金属"
```

## 六、各学段操作规范速查

| 规范项 | 八年级力学 | 九年级电学 |
|-------|-----------|-----------|
| 使用层次 | 2层（现象+简化原理） | 3层（现象+原理+迁移） |
| 现象层追问次数 | 2-3次 | 1-2次 |
| 原理层追问次数 | 1-2次 | 2-3次 |
| 迁移层追问次数 | 不使用或1次简化版 | 2-3次 |
| 条件变更方式 | 教练提供具体变更 | 学生自提+教练辅助 |
| 类比使用 | 大量生活类比 | 少量类比，过渡到抽象推理 |
| 逻辑链要求 | 简单因果（因为...所以...） | 推理链（A→B→C→D） |
| 沉淀深度 | "记住这个现象" | "理解这个规律，知道什么时候用" |
| 时间控制 | 5分钟以内 | 8-10分钟 |
| 画图要求 | 每次追问都要求画图辅助 | 需要时画图，鼓励心像 |

## 七、常见问题与处理策略

### 学生说"我懂了"但现象层追问失败

```
诊断方法：立即启动现象层追问
  "你说你懂了——那你告诉我，这道题描述了什么物理现象？"

判断标准：
  → 只能说公式 → 停留在"记住了"层面 → 回退S1画图
  → 能说出物理场景但描述含糊 → 图景建立不完整 → 补充画图
  → 能清晰描述物理场景和过程 → 真正看见了 → 可进入原理层
```

### 学生在追问中反复卡壳

```
八年级处理：
  → 卡壳1次：给提示词（"你看看图上有没有遗漏的力？"）
  → 卡壳2次：把问题拆成更小的问题（"先只看竖直方向的力"）
  → 卡壳3次：换一种说法重新提问（用生活类比替代物理术语）
  → 卡壳超过3次：暂停追问，回到画图本身，重新建立图景

九年级处理：
  → 卡壳1次：给方向性提示（"先判断串并联"）
  → 卡壳2次：给类比案例（"想象电流像车流"）
  → 卡壳3次：把问题拆成两步（"先分析S1闭合时的电路，再分析S2闭合"）
  → 卡壳超过3次：暂停追问，记录卡壳点到物理错误DNA
```

### 学生回答"不知道"的策略

```
层级1（完全不知道）：
  → 降低问题难度：从"为什么"降到"是什么"
  → 例：从"为什么电流变小" → "总电阻变大了还是变小了？"

层级2（有模糊想法但说不清）：
  → 给选择法："电流变小是因为电阻变大了还是电压变小了？"
  → 给半句提示："串联电路中，电阻越大的用电器分到的电压..."

层级3（方向对了但表述不完整）：
  → "你说了一半——后半句是什么？"
  → "你的意思我理解了，但更准确的说法是..."
```

### 学生只背公式不理解原理

```
典型表现：
  → 学生说"F=mg"但不知道m是什么，g是什么
  → 学生说"I=U/R"但不知道适用条件
  → 学生说"串联分压"但不会推导

处理方法：
  → 原理层追问必须追问到适用条件
  → "I=U/R，这个公式用在什么电路上？——纯电阻电路。
     什么叫纯电阻？——电能全部转化为内能。
     电动机是纯电阻吗？——不是。那I=U/R还能用吗？——不能直接用。"

  → 要求推导而非记忆
  → "串联分压你能推导吗？——I相同，U=IR，R大的U大。
     不用推导光记住结论，换道题你可能就忘了。"
```

## 八、三层次追问与四步法的衔接

```
三层次追问的触发时机：
  ① 学生完成四步法Step 4（三重验证通过）后
  ② 学生说"我懂了"后——启动现象层验证是否真懂
  ③ S4验证发现问题但不严重时——用追问帮助理解

衔接话术：
  "你算对了——但算对不等于真懂。
   我问你几个问题，帮我确认你是不是真的理解了。
   放轻松，就当我们在聊天。"

三层次追问与四步法的分工：
  四步法：解决"怎么做"（画图→建模→列式→验证）
  三层次追问：解决"为什么这么做"（看见→理解→迁移）

时间分配：
  → 四步法为主（占70%时间）
  → 三层次追问为辅（占30%时间，仅在验证后或"说懂了"后启用）
  → 不在同一道题上同时进行四步法和追问

回退规则：
  → 现象层失败 → 回退到Step 1（重新画图）
  → 原理层失败 → 回退到Step 2（重新建模）
  → 迁移层失败 → 回退到Step 3（重新列式分析）
  → 回退时不重做整个步骤，只针对失败点补充
```

## 九、特殊场景处理

### 实验题的三层次追问

```
现象层改编：
  → "这个实验要验证什么？实验现象是什么？"
  → 不问"你看到了什么物理现象"，而是"实验装置在做什么？"

原理层改编：
  → "为什么这个实验设计能验证结论？控制变量法用对了吗？"
  → 实验题的原理层更关注实验方法而非物理规律本身

迁移层改编：
  → "如果换一个实验器材，还能验证同样的结论吗？"
  → "如果增加一个实验步骤，能排除什么干扰因素？"

注意：实验题应联动物理实验思维教练SKILL
```

### 计算纯算对但理解不到位的场景

```
识别信号：
  → 学生算对了但说不出为什么这么做
  → 学生说"我套公式算的"
  → 学生换一道同类题就不会了

处理策略：
  → 必须启动三层次追问，即使算对了
  → 重点放在现象层和原理层
  → "你算对了，但我想确认你是不是真理解了。
     如果这道题的条件换一下，你还能做对吗？"

这是最常见的"伪理解"场景——物理学习中最危险的情况
```

### 学生在迁移层表现出色的处理

```
识别信号：
  → 学生能自己设计条件变更并正确预测
  → 学生说"如果再并联一个电阻，总电阻变小，总电流变大"
  → 学生能举一反三

处理策略：
  → 肯定并深化："你刚才的推理非常棒——
     你已经不只是在做一道题，而是在理解一个规律。"
  → 让学生自己出题："你能不能出一道类似的题考考我？"
  → 记录到学习DNA：物理解题能力维度标记为"迁移能力优秀"
```

## 十、追问话术质量自检

```
好的追问话术特征：
  ✅ 引导学生"看见"而非"记住"
  ✅ 用物理图景而非抽象概念开头
  ✅ 允许多种正确回答而非唯一答案
  ✅ 逐步深入而非一步到位
  ✅ 适合学生当前认知水平

坏的追问话术特征：
  ❌ "你知道F=mg吗？"（只是检验记忆）
  ❌ "答案是不是5N？"（是/否判断，不激发思考）
  ❌ "你觉得这用欧姆定律对吗？"（暗示太强）
  ❌ "为什么？为什么？为什么？"（连续追问无方向）
  ❌ 用高中概念解释初中问题（超出认知水平）

自检方法：
  每次追问前问自己：
  ① 这个问题是引导学生"看见"还是"记住"？
  ② 学生能用图景来回答吗？还是只能用公式？
  ③ 如果学生回答不上来，我能拆成更小的问题吗？
```
