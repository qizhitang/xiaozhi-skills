# CLAW模板扩展话术与苏格拉底五问各学段适配指南

---

## 一、5套CLAW模板的变体场景与扩展话术

### 模板①变体：概念关联型

当学生学了多个相关概念，容易混淆时：

```
"我在学[学段]的[概念A]和[概念B]。
 我知道它们都[共同点]，
 但我在[具体场景/题型]里总是搞混。
 帮我找一个能区分它们的'标志性特征'，
 不要只给我定义，要给我一个判断方法。"
```

常见易混淆概念对：
- 一次函数 vs 正比例函数
- 等差数列 vs 等比数列（初中涉及时）
- 全等三角形 vs 相似三角形判定条件
- 整式 vs 分式的运算规则
- 勾股定理 vs 射影定理

---

### 模板②变体：系统性错误型

当学生发现自己在一类题上反复出错，且不只是一道题时：

```
"我在[知识模块]的题上总是出错，
 已经出错了[N次]。
 我把最近三道错题都发给你：[题目1][题目2][题目3]。
 帮我找出这三道题里错误的共同根因，
 不要逐一分析，而是找出它们之间的规律。"
```

小龙虾收到后的处理方式：
1. 不逐一解释，而是横向对比三道题
2. 找到共同的"卡壳点"
3. 一句话概括根因，比如：
   "这三道题你都在'设元'那一步出问题——
   不是不会解方程，是不知道该把什么设为x。"

---

### 模板③变体：几何卡壳型

几何题卡壳有其特殊性：往往是"辅助线画不出"或"条件用不上"。

```
"这道几何题[图片/描述]，
 我已经标出了所有已知条件，
 但我不知道下一步该添加什么辅助线，
 或者图里有什么条件我没有用到。
 帮我盘点一下：我还有哪些条件没有用，
 这些条件可能指向什么。"
```

小龙虾几何追问专用序列：
1. "图里有几条线/几个角？哪些你已经用上了？"
2. "题目给的条件里，有没有你看到了但还没有转化成数学关系的？"
3. "要证明/求的结论，你知道需要什么前提吗？从结论往回推。"

---

### 模板④变体：弱项针对型

已知学生的弱项，专门针对这个弱项出题：

```
"我的错误基因档案显示我在[弱项X]上出错最多。
 请给我出5道专门针对[弱项X]的练习题，
 难度从'刚好能做对但需要注意'开始，
 逐步增加到'需要结合[另一个知识点]'。
 每道题出完后先不给我答案，等我做完再说。"
```

---

### 模板⑤变体：知识图谱型

考前需要看清楚整个知识模块的全貌时：

```
"考试前[N]天，帮我画出[知识模块]的知识图谱：
 核心概念有哪些，它们之间什么关系，
 哪些是'树根'（基础），哪些是'树叶'（应用）。
 然后告诉我：从'叶'追溯到'根'，
 哪一段路最容易在考试里出题？"
```

## 二、数学苏格拉底五问链各学段适配指南

### 小学高年级（4-6年级）适配

**调整原则：** 问题语言更具体，不用抽象词汇

```
第1问（清晰达意）：
原版："用自己的话把解题过程说出来"
小学版："你是怎么一步一步算的？
        每一步你做了什么，告诉我。"

第2问（假设检验）：
原版："如果把条件改变，解法还能用吗？"
小学版："如果这里不是[3]而是[5]，
        你还是这样算吗？算算看。"

第3问（证据链）：
原版："最关键的一步是什么，为什么成立？"
小学版："你这一步为什么要[除以/乘以/加上]这个数？"

第4问（视角切换）：
原版："另一种方法"
小学版："有没有另一种方式算？
        比如先画图试试，或者列表格找规律？"

第5问（个性化沉淀）：
原版："适用哪一类题？给它起个名字"
小学版："以后碰到什么样的题，你会想到用这个方法？"
```

---

### 初中阶段（7-9年级）适配

**调整原则：** 加入方程思维和代数思维的培养

```
第1问（清晰达意）：
"用一句话说：这道题的'核心思路'是什么？
 如果不提具体数字，只说方法。"

第2问（假设检验）：
"如果题目改成：[类型改变，不只是数字改变]，
 你的方法还适用吗？为什么？"
（注意：初中改的是类型，不只是数字）

第3问（证据链）：
"在你的解题过程中，你用了[定理/公式]。
 这个定理的适用条件是什么？
 这道题满足这些条件吗？"

第4问（视角切换）：
"方程法、图像法、代入消元、加减消元——
 这道题你用了[X]，
 有没有另一种方法也能解？哪种更快？"

第5问（个性化沉淀）：
"这道题属于[题型X]的哪个变种？
 你能描述一下'识别这类题'的特征是什么吗？"
```

---

### 高中阶段（10-12年级）适配

**调整原则：** 强化逻辑严密性和多解思路比较

```
第1问（清晰达意）：
"不看你的解题步骤，重新给我讲一遍解题思路。
 从'为什么想到用这个方法'开始。"

第2问（假设检验）：
"这道题的解法依赖于[某个关键条件]。
 如果这个条件不成立，解法会在哪里失效？"

第3问（证据链）：
"你的解题中有没有一步是'直觉感觉对的'但没有严格证明的？
 找出来，把它的证明补全。"

第4问（视角切换）：
"这道题至少有几种解法？
 你这种方法的优缺点是什么？
 什么情况下另一种方法更好？"

第5问（个性化沉淀）：
"这道题用到了[定理A]和[方法B]的组合。
 你能想到3道需要同样组合的题吗？
 把这个组合模式记入你的'方法模式库'。"
```

## 三、高频卡壳场景与破局追问速查

| 卡壳场景 | 追问话术 |
|---------|---------|
| 不知道用哪个公式 | "这道题里出现了几个变量？它们之间有什么关系？有没有哪个关系式你学过？" |
| 计算过程中算错 | "从第[N]步往后逐步说给我听，我帮你找第一个不对的地方" |
| 几何不知从哪下手 | "先把题目给的所有条件都标在图上，哪些是角，哪些是边，哪些是比例关系" |
| 应用题建不起方程 | "先不管公式。用中文说：[未知量]和[已知条件]之间的数量关系是什么" |
| 证明题不知写什么 | "从结论往回推：要证[结论]，你需要先证[什么]？那个'什么'你有吗？" |
| 概念混淆用错方法 | "你现在用的是[方法X]，这个方法要求[条件]。这道题满足这个条件吗？" |
| 做对了但不知道为什么 | "你这道题做对了，但我要问你：如果答案改成[另一个数]，你能判断对错吗？" |
