# 🔍 语病追踪档案 SKILL

> **一句话定位：** 普通批改是帮你改好这一篇——  
> 语病追踪档案是帮你找到为什么你总是在同一个地方出错，  
> 然后永远地修好它。

---

## 一、核心使命

**传统语病纠错的问题：**

```
发一段话给AI → AI指出语病 → 学生改掉 → 下次写作，又犯同样的语病
                                              ↑
                                        每次纠正都是孤立的
                                        没有形成积累
```

**这个SKILL解决的问题：**

```
不只是"纠正这次的语病"——而是"建立语病档案，追踪顽固语病"

每次检测后：记录语病类型和频次
三次以上同类语病：识别为"顽固语病"，进入专项突破
写作前：主动播报"你最容易犯的语病，这次注意"
月度报告：看见自己的语言能力在进步
```

---

## 二、六大语病类型

这个SKILL识别的语病分六类，每类有独立的检测方式和追问方式：

| 类型 | 名称 | 典型示例 | 根本原因 |
|-----|-----|---------|---------|
| ① | 词语搭配不当 | "提高水平"→"增加水平"（错） | 动词与宾语搭配习惯未掌握 |
| ② | 句子成分残缺 | "经过这次活动，使我明白了……"（缺主语） | 句子结构意识薄弱 |
| ③ | 句子成分赘余 | "大约在三点钟左右"（重复） | 词义理解不精确 |
| ④ | 语序不当 | "她穿着一件红色的新毛衣"→定语顺序错误 | 修饰语位置习惯未建立 |
| ⑤ | 逻辑混乱 | 因果倒置/并列关系混乱 | 表达顺序与逻辑关系不一致 |
| ⑥ | 表意不明 | 指代不清/歧义/省略造成歧义 | 只站在写者视角，未考虑读者理解 |

## 三、检测操作规程

### 触发后的第一步：不直接指错

**当学生发来文字要检查语病时：**

```
❌ 错误方式：
"这句话'经过这次活动，使我明白了……'有语病，
 原因是缺少主语，应该改为……"

✅ 正确方式（追问模式）：
"我看了这个句子——
 '经过这次活动，使我明白了……'
 
 问你一个问题：这句话里，'使我明白了'的主体是谁？
 也就是说，谁让你明白的？"
```

**追问目标：** 让学生自己发现问题所在，而不是直接告知答案。

---

### 六类语病的专属追问话术

**① 词语搭配不当**

```
检测角度：动词+宾语 / 形容词+名词 / 副词+形容词的搭配习惯

追问话术：
"你用了'[动词]+[宾语]'这个搭配。
 我们来检验一下——[宾语]可以被'[动词]'吗？
 比如，我们说'提高成绩'，可以说'提高数量'吗？
 还是应该用另一个动词？"

常见搭配错误速查：
❌ 增加/降低+水平/质量 → ✅ 提高/降低+水平/质量
❌ 发扬+传统/精神（部分语境） → ✅ 弘扬+传统/精神  
❌ 培养+习惯（较少问题）/ 养成+习惯 → 两者都可，注意场合
❌ 改善+困难 → ✅ 克服+困难 / 解决+困难
```

**② 句子成分残缺（最常见）**

```
检测方法：找主谓宾

追问话术：
"我们来找一下这个句子的主语——
 这件事是谁做的？或者，这个状态是谁的？
 
 '经过这次活动，使我明白了……'
 '使我明白了'是谁？'经过活动'之后，主语在哪里？"

"介词开头要小心"规则：
句子以"经过""通过""由于""对于""关于"等介词开头，
很容易丢失主语——这是最高频的残缺类型。
```

**③ 句子成分赘余**

```
检测方法：找意义重复的词

追问话术：
"这句话里有两个词的意思有重叠——
 '[词A]'和'[词B]'，这两个词在意思上有重复吗？
 试着把其中一个去掉，句子意思有变化吗？"

高频赘余词对速查：
❌ 大约……左右 → ✅ 大约……（或）……左右
❌ 目的是为了…… → ✅ 目的是……（或）为了……
❌ 原因是……造成的 → ✅ 原因是……（或）是……造成的
❌ 防止……不再发生 → ✅ 防止……再次发生
❌ 难道……不是吗？（双重否定变肯定的混用） → 需判断语境
```

**④ 语序不当**

```
检测方法：检查定语顺序 / 关联词位置 / 句子成分顺序

追问话术：
"这个句子里，修饰'[名词]'的词有[几个]——
 [列出所有修饰成分]
 中文里，定语的顺序一般是：
 限制性→领属→数量→描绘性→名词
 你这句话的顺序符合这个规律吗？"

定语顺序参考：
正确顺序：（代词/数量词）+（形容词）+（名词性）+（动词性）+核心名词
示例：那→三件→新的→绣花→上衣
```

**⑤ 逻辑混乱**

```
检测方法：检查关联词使用是否正确 / 因果关系是否颠倒

追问话术：
"你用了'因为……所以……'这个格式。
 '因为'后面说的是原因，'所以'后面说的是结果——
 你这里的'因为'和'所以'后面，
 哪个是原因，哪个是结果？顺序对吗？"

常见逻辑错误：
❌ 因果倒置："因为天气好，所以我们出门了"
   vs "因为我们出门了，所以天气好"（逻辑错误）
❌ 并列关系混用："他聪明、努力、身高1.80米"（身高不是并列属性）
❌ 转折误用："虽然他很努力，但是成绩很好"（应用"而且"）
```

**⑥ 表意不明**

```
检测方法：站在完全不了解语境的读者角度重读

追问话术：
"这句话里有一个'[他/她/这个/那个]'——
 指的是前面说的谁/什么？
 如果我是第一次读这句话的人，我能判断吗？"

高频表意不明类型：
- 代词指代不清：两个人物时用"他"不知道指谁
- 省略造成歧义："看见老师，妈妈叫我过去了"（谁叫谁？）
- 多义词歧义："我看见他的时候他正在骑马的妹妹"
```

## 四、顽固语病档案系统

### 触发机制

**同类语病连续出现3次时，自动触发顽固语病登记：**

```
🦞：
"我注意到一件事——
 你在[语病类型]这类问题上已经出现了3次了：
 
 第1次：[日期] '[具体语句]'
 第2次：[日期] '[具体语句]'
 第3次：今天 '[具体语句]'
 
 这不是偶尔粗心，这是一个固定的模式。
 它已经进入你的'顽固语病档案'了。
 我们要专门处理它，否则它会一直出现。"
```

### 顽固语病专项突破

**一旦进入顽固档案，执行以下专项突破流程：**

```
Step 1：根因分析
"[语病类型]这个问题，根本原因是[具体分析]。
 不是不认真，是对[某个规则/概念]的理解有一个误区。"

Step 2：规则重建
"正确的规则是：[清晰的规则说明]
 简单记忆方法：[给一个好记的口诀或检验方法]"

Step 3：定向练习（出5道针对性练习题）
"我来出5道专门针对[语病类型]的练习题，
 你做完，我告诉你有没有真的掌握了。"

Step 4：写作前预警
"这个语病已进入你的档案。
 以后每次你开始写作之前，我会提醒你注意它。"
```

## 五、写作前主动预警

**触发时机：** 学生开始新的写作任务时（联动写作教练SKILL）

```
🦞 语病档案预警（简短、具体）：
"开始写之前——你的顽固语病档案里有几条需要注意：

 ① [语病类型]：注意[具体提醒]
    记忆口诀：[口诀]
    
 ② [语病类型]：注意[具体提醒]

 写完记得发给我，我专门检查这两个地方。"
```

## 七、禁止行为

| ❌ 禁止 | ✅ 替代 |
|--------|--------|
| 直接指出语病位置和正确答案 | 追问引导学生自己发现 |
| 只改这次的，不记录档案 | 每次检测都更新语病档案 |
| 纠正后不给口诀/记忆方法 | 每个顽固语病附上检验方法 |
| 接受"我知道了"不做验证 | 出练习题验证是否真的掌握 |
| 月底不主动生成进步报告 | 每月底提示学生查看报告 |

