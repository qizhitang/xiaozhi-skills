# 📖 智能词汇DNA系统 SKILL

> **一句话定位：** "今天背，明天忘"不是你的错——是你背单词的时间节点错了。  
> 正确时间复习，遗忘就会停下来。  
> 这个SKILL做的事：帮你找到正确的时间节点，然后主动来提醒你。

---

## ⚠️ 技术实现边界声明

> **关于“长期记忆、主动提醒与多模态识别”机制：**
> 1. 本模块对学生词汇遗忘曲线的跨次长期记忆，依托于 OpenClaw 的「本地化持久记忆机制」，并非 LLM 自身的超长上下文。
> 2. 定时的主动推送泛化支持任何已接入 OpenClaw 平台的 IM 软件（如微信、QQ等），并非针对单一软件定制，其定时唤醒能力依托于定制化平台（如 QClaw）的「定时任务机制」。
> 3. 拍照提取单词功能，强依赖于多模态大语言模型（Multimodal LLMs）的视觉能力或专属 SKILL 配置的本地 OCR 脚本扩展。

## 一、核心使命

**艾宾浩斯遗忘规律（全自动执行）：**

```
学完新词后：
  20分钟 → 记忆保留58%
  1天后  → 记忆保留33%（第1轮提醒触发）
  3天后  → 记忆保留约28%（第2轮提醒触发）
  7天后  → 记忆保留约25%（第3轮提醒触发）
  14天后 → 记忆保留约22%（第4轮提醒触发）
  30天后 → 记忆保留约21%（第5轮提醒触发）

每一轮复习，都在遗忘发生之前刷新记忆——
经过5轮，这个词就进入了长期记忆。
```

## 三、模块A：词汇DNA入库

### 三种入库方式

**方式①：拍照提取**

```
触发：学生拍照或截图发来单词表、课文词汇
操作：
"我看到这里有[N]个词——我帮你把它们都存入DNA。
 你现在想先标记几个'重点词'吗？（这些会优先提醒）
 
 或者全部均等存入，都可以。"

存入格式（内部）：
批量入库，每个词标注：
- 入库日期
- 来源（哪节课/哪篇课文）
- 优先级（学生标记的重点）
```

**方式②：对话标记**

```
触发：口语/听力练习中遇到好词
触发词："把这个词存入DNA" / "记下这个" / "存这个"

存入时自动附带：
- 当前对话的语境（这个词在什么情境下出现的）
- 入库日期和场景标签

反馈：
"[词汇]已经存入你的词汇DNA！
 语境：[当前对话语境摘要]
 第一轮复习：24小时后我会发来一道闪卡测验。"
```

**方式③：手动添加**

```
触发：学生主动说出想记的词
格式示例：
"把 'meticulous' 加入我的词汇DNA，
 我想用它描述做事认真细致的人"

存入时记录：
- 词汇本身
- 学生提供的使用场景
- 标注：主动学习意愿强，优先级高

反馈：
"'meticulous' 已存入！这个词形容做事极其仔细、注重细节的人。
 例：She is meticulous about her work.
 
 24小时后我会来考你——
 到时候你能用它造一个描述自己认识的人的句子吗？想想看。"
```

## 四、模块B：五轮IM提醒系统

### 五轮递进设计

每一轮的提醒形式不同，难度递进，从"被动识别"升级到"主动使用"：

**第1轮（24小时后）：闪卡测验**

```
IM推送（微信/QQ等）：
"📖 词汇提醒
 
 [词汇] 是什么意思？
 回复你的答案，不用查——直接说你记得的。"

学生回答后：
→ 答对：
  "✅ 对！[词汇] = [释义]。
   再巩固一下：你能用这个词造个短句吗？"

→ 答错或不记得：
  "没关系！[词汇] = [释义]
   例句：[例句]
   3天后我再问你一次，这次会记住了。"
```

**第2轮（3天后）：造句激活**

```
IM推送（微信/QQ等）：
"📖 词汇提醒（第2轮）
 
 用 [词汇] 造一个关于你自己生活的句子——
 不用复杂，用你真实的情况造就好。"

处理：
→ 学生造句后，评价是否正确使用
→ 如有更地道的表达，提供升级版本
→ 记录这个造句到词汇DNA（作为使用记录）
```

**第3轮（7天后）：对话植入**

```
IM推送（微信/QQ等）：
"📖 词汇提醒（第3轮）
 
 今天聊聊[和词义相关的话题]，试着用上 [词汇]——
 不用刻意，就在对话里自然用出来。"

操作：
→ 开启一段简短对话，话题与词汇语义相关
→ 对话结束后，检查学生是否自然使用了这个词
→ 如果用了：记录为"主动使用一次"，提升熟悉度评级
→ 如果没用：温和提示："这里其实可以用 [词汇] 来表达——试试？"
```

**第4轮（14天后）：语境扩展**

```
IM推送（微信/QQ等）：
"📖 词汇提醒（第4轮）
 
 [词汇] 的近义词/反义词/常见搭配有哪些？
 你知道它们有什么微妙的差别吗？"

回复内容：
→ 近义词对比（如 meticulous vs careful vs thorough）
→ 常见搭配（collocations）
→ 使用场景的细微差别
→ 出一道辨析题让学生练
```

**第5轮（30天后）：月度综合复习**

```
IM推送（微信/QQ等）：
"📖 月度词汇复习
 
 这个月你学了这些词：[列举本月入库词汇]
 来做一次综合测验——
 我出5道题，看看你还记得多少。"

测验形式：
→ 混合题型：选义、造句、辨析、填空
→ 测验后给出"月度词汇留存率"
→ 留存率<60%的词：重新归入第1轮，重新开始五轮
→ 留存率>80%的词：标记为"长期记忆"，进入季度复习模式
```

## 五、模块C：新词汇雷达（课前7天预热）

### 触发方式

```
学生发送："下周我要学第[X]课，这是单词表[粘贴/拍照]，
          帮我启动7天预热计划。"

小龙虾确认：
"收到！我看到有[N]个新词。
 7天预热计划启动——
 你不需要做任何事，IM消息会在正确的时间找到你。
 
 预热进度：
 Day 7 → 词形+发音初见
 Day 5 → 图像/场景联想
 Day 3 → 轻松开口使用
 上课前 → 自信提醒"
```

### 四阶段预热内容

**Day 7（第一次见面）：建立"见过"的印象**

```
IM推送（微信/QQ等）：
"📡 新词预热 · 第1天（距上课还有7天）

 下周你会遇到这些词，我们先认识一下：
 
 [词汇1] /[音标]/ 读作：[拼音辅助]
 [词汇2] /[音标]/ 读作：[拼音辅助]
 ...
 
 今天任务：把这些词的发音读三遍——
 不需要记意思，只是让它们变得不陌生。"
```

**Day 5（场景联想）：从符号变成有画面的概念**

```
IM推送（微信/QQ等）：
"📡 新词预热 · 第2天

 给这几个词配上画面——
 
 [词汇1]：[一张有画面感的场景描述或图示引导]
 [词汇2]：[场景描述]
 
 想象这个场景，把词和画面连在一起。
 下次见到这个词，画面会先出现，意思就跟着来了。"
```

**Day 3（轻松使用）：主动激活**

```
IM推送（微信/QQ等）：
"📡 新词预热 · 第3天

 用这3个词，和我聊3-5句话——
 随便聊，不用都用上，用到哪个算哪个：
 [词汇1] / [词汇2] / [词汇3]"
```

**上课前（自信提醒）**

```
IM推送（微信/QQ等）：
"📡 新词预热 · 最后提醒

 今天上课会遇到这些词——
 你已经见过它们[3]次了，不会陌生的。
 
 特别提示：[2-3个最重要的词] 可能是这节课的重点，
 注意听老师怎么用它们造句。"
```

## 六、模块D：个性化遗忘速度追踪

### 遗忘速度分类

根据学生在五轮提醒中的表现，自动标定每个词的个人遗忘速度：

```
快速遗忘型（1天后就不记得）：
→ 第1轮提醒提前到12小时后
→ 加入"强化模式"：前三轮间隔缩短一半

正常遗忘型（标准5轮时间）：
→ 按默认五轮进行

慢速遗忘型（7天后仍记得）：
→ 跳过第1-2轮，直接进入第3轮
→ 节省提醒次数，专注真正需要的词
```

### 词汇危险指数计算

```
危险指数 = （距上次复习天数 / 预计遗忘天数）× 100%

危险指数 > 80%：今日必须复习（高优先级推送）
危险指数 50-80%：本周建议复习（正常推送）
危险指数 < 50%：暂时安全（不推送）
```

## 八、词汇DNA两个层次追踪

```
层次①：被动识别（看到认识）
  标准：看到这个词能理解意思
  
层次②：主动使用（能自己说/写出来）
  标准：在没有提示的情况下能用这个词表达想法

两个层次分开追踪，分开复习：
→ 层次①的词：闪卡测验为主
→ 层次②的词：造句/对话植入为主

目标：把词库中20%的核心词汇升级到层次②
     剩余80%维持层次①即可
```
