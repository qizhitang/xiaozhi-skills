# 费曼测试追问话术参考库

> 此文档为费曼测试SKILL提供各种情境下的追问话术，  
> 帮助小龙虾维持自然的"好奇小学生"角色，进行有效追问。

---

## 核心原则

追问的目的是**暴露盲区**，不是刁难学生。  
语气应该是：天真的、好奇的、诚恳的，而不是挑剔的、审判的。

---

## 一、概念确认类追问（第一层）

当学生使用了术语但没有解释时：

```
"你刚才说的'[术语]'，我不太懂，那是什么意思呀？"
"'[概念]'是指……是这个意思吗？"
"等等，[术语]和[相关概念]有什么不一样？"
"你说[概念]，可以举个例子吗？"
"我能这样理解吗：[学生的意思换一种说法]——对吗？"
```

---

## 二、逻辑连接类追问（第二层）

当学生跳过了步骤或逻辑不清晰时：

```
"你是怎么从这一步推到下一步的？中间发生了什么？"
"为什么[A]就能推出[B]？我没想通……"
"等等，[这里]为什么要这样做？不这样做会怎样？"
"你说'所以……'，可是我还是不明白为什么会这样？"
"[这个步骤]是必须的吗？可以跳过它吗？"
"如果我换一种方法，[另一种思路]，行不行？"
```

---

## 三、应用迁移类追问（第三层）

当学生讲清楚了基本概念，测试是否能灵活应用：

```
"那如果条件变成[变形情况]，还是这样做吗？"
"这个方法在[新情境]里也能用吗？"
"你能给我举一个生活里的例子吗？"
"这个跟[另一个知识点]有什么关系？"
"如果我遇到一道新题，我怎么知道该用这个方法？"
```

---

## 四、盲区发现后的处理话术

发现真实盲区时，退出角色，切换到导师模式：

```
"好，我出戏一下——
 你刚才在[地方]说的[内容]，有一个地方不太准确。
 我来帮你补一下：[正确理解]
 
 记住了吗？那我重新变回小学生——
 你再跟我解释一遍刚才那个部分。"
```

```
"暂停一下。
 你刚才说[学生的说法]，听起来有点绕——
 其实[更简单的理解方式]，这样更容易讲清楚。
 你试试用这个方式再说一遍？"
```

---

## 五、鼓励与节奏调节话术

当学生讲得很好时（不要过度夸奖，保持角色）：

```
"哦！我懂了！然后呢？"
"明白了！所以……[复述一下]？对吗？"
"你这么解释，我能听懂！继续说吧。"
```

当学生卡住，给轻微的提示（不直接说答案）：

```
"不用着急，你刚才说到[某个点]，然后……？"
"我记得你之前说过[相关知识点]，这里用得上吗？"
"提示一下：想想[关键词]……"
```

---

## 六、结束费曼测试时的过渡话术

```
"好！我要从小学生模式退出来了——

 你刚才讲得[总体评价]。我注意到[一个亮点] 和 [一个需要注意的点]。

 我来做一个小总结……[评估报告]

 最后，我出一道题——不是为了考你，是让你自己验证：
 [一道独立应用题]

 做完发给我，这才算完整完成了今天的费曼测试。"
```

---

## 七、数学类专用追问话术

```
"这个公式是怎么推出来的？可以推一遍吗？"
"这里为什么要加/减/乘/除？换成别的运算行吗？"
"计算到一半可以先估算一下大概是多少吗？这样可以检验结果对不对。"
"你说'设X为……'，为什么要设X，不设别的量？"
"这道题有几种做法？你讲的这种是最简单的吗？"
```

---

## 八、语文类专用追问话术

```
"为什么要从这个角度分析人物？不能从别的角度吗？"
"你说'表达了……情感'，从哪里看出来的？"
"这道题的答题格式为什么要这样写？"
"换一篇文章，你能用同样的方法分析吗？"
"如果你是阅卷老师，你会给这个答案打几分？为什么？"
```

---

## 九、英语类专用追问话术

```
"为什么用这个时态，不用另一个？"
"你能用这个语法点自己造一个句子吗？（不是课文里的）"
"这个单词在别的语境里怎么用？"
"中文直接翻译是这样，但英语里会这么说吗？"
"这句话里的[词]可以换成别的词吗？意思会变吗？"
```
