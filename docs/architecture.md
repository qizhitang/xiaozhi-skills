## 📖 方法论依据

本 SKILL 包的设计基于《龙虾高效学习法》第一章的核心方法论：

**记忆飞轮五转：**
1. 记录首次错误 → **错题本 SKILL**
2. 主动提醒复习 → **IM智能提醒 SKILL**
3. 精准出题检验 → **错题本 SKILL + IM提醒 SKILL**
4. 发现错误规律 → **错题本 SKILL + 学习DNA档案 SKILL**
5. 生成弱项报告 → **每周复盘 SKILL**

**费曼学习法：**
- 用"教会小龙虾"验证真实理解 → **费曼测试 SKILL**

**AI学习铁律：**
- 用AI辅助思考，不用AI替代思考 → 所有 SKILL 的设计原则

**语文专项能力链：**
- 写作表达升级 → **语文写作教练 SKILL**
- 阅读拆题拿分 → **阅读理解拆解师 SKILL**
- 古诗文理解与背诵 → **文言文复活计划 SKILL**
- 长期素材沉淀 → **语文素材库2.0 SKILL**
- 语言表达纠偏 → **语病追踪档案 SKILL**

**数学专项能力链：**
- 日常错题拆解 → **数学解题教练 SKILL**
- 错因模式沉淀 → **数学错误基因档案 SKILL**
- 概念与几何直觉重建 → **数学概念解释器 SKILL**
- 应用题读题建模 → **应用题建模教练 SKILL**
- 天花板突破与进阶 → **思维梯度训练师 SKILL**

**英语专项能力链：**
- 口语开口与表达升级 → **英语口语陪练 SKILL**
- 词汇长期积累与提醒 → **智能词汇DNA系统 SKILL**
- 语法结构定位与突破 → **英语语法突破教练 SKILL**
- 听力材料个性化训练 → **个性化英语听力训练师 SKILL**
- 写作表达进化 → **英语写作进化教练 SKILL**

**第六章系统升级：**
- 学会自己造工具 → **SKILL创建教练 SKILL**
- 让笔记主动找你 → **智能康奈尔笔记档案 SKILL**
- 让核心工具自动联动 → **五SKILL联动协调器 SKILL**
- 错题本升级为预警系统 → **错题本 v1.1**
- 费曼测试升级为深度档案 → **费曼测试 v1.1**
- 每周复盘升级为量化存折 → **每周学习复盘 v1.1**

**第七章、第八章系统升级：**
- 学会基于真实数据制定计划 → **30天学习计划制定师 SKILL**
- 看清时间流向并提升专注 → **时间与专注力教练 SKILL**
- 打破学科孤岛建立联结 → **跨学科侦探周 SKILL**
- 区分浅层喜好与真正兴趣 → **兴趣成长探索计划 SKILL**
- 把复盘升级为六模块追问系统 → **每周学习复盘 v1.1**
- 把批判性验证绑定到理解检查 → **费曼测试 v1.1**
