# 🗺️ 安装指南与发布信息

本文档包含完整的 35 步分阶段加装指引，以及发布至 ClawHub 的官方上架元数据配置建议。

## 🚀 推荐安装顺序（35步长序列）

> ⚠️ **重要提示：** SKILL 并不是装得越多越好！  
> 先把核心五件套用熟，再补上第六章的方法论系统层和第七八章的自我管理层，最后按需安装各学科 SKILL。

### 第一阶段：核心起步

```
第一步：安装 🧬 学习DNA档案 → 完成首次建档（最重要！）
第二步：安装 ❌ 错题本 → 把近期一道错题发给小龙虾试试
第三步：安装 ⏰ IM智能提醒 → 设置第一条复习提醒
第四步：安装 🎓 费曼测试 → 选一个刚学的知识点，试着讲给小龙虾听
第五步：安装 📊 每周学习复盘 → 触发第一次复盘（哪怕数据很薄）
```

### 第二阶段：方法论加装

```
第六步：安装 🛠️ SKILL创建教练 → 学会为自己的需求搭建新工具
第七步：安装 📝 智能康奈尔笔记档案 → 把一页课堂笔记拍照提炼并入档
第八步：安装 🔗 五SKILL联动协调器 → 让错题、费曼、笔记开始自动联动
第九步：确认 ❌ 错题本已为 v1.1 → 使用拍题三信息法、弱项预警与学期报告
第十步：确认 🎓 费曼测试已为 v1.1 → 使用三层理解深度档案与挑战者模式
第十一步：确认 📊 每周复盘已为 v1.1 → 使用学习存折量化体系
```

### 第三阶段：自我管理与成长加装

```
第十二步：安装 🗓️ 30天学习计划制定师 → 用真实DNA数据生成第一份30天计划
第十三步：安装 ⏱️ 时间与专注力教练 → 开始记录第一周的时间销行账
第十四步：确认 📊 每周学习复盘已整合为 v1.1 → 启用六模块周报与苏格拉底七问
第十五步：确认 🔗 五SKILL联动协调器已整合为 v1.1 → 切换到五SKILL联动
第十六步：安装 🔭 跨学科侦探周 → 选第一个探索主题，完成第一周侦探
第十七步：安装 ☕ 兴趣成长探索计划 → 启动52杯咖啡第一杯
第十八步：确认 🎓 费曼测试已整合为 v1.1 → 启用第五跳批判性验证
第十九步：确认 🧬 学习DNA档案已整合为 v1.1 → 启用成长图谱与兴趣/跨科接口
第二十步：确认 ⏰ IM智能提醒已整合为 v1.1 → 启用计划与探索类提醒
```

### 第四阶段：语文学科加装

```
第二十一步：安装 📚 语文素材库2.0 → 把现有积累入库（哪怕只有5条）
第二十二步：安装 🔍 语病追踪档案 → 把最近一篇作文发给它检测一次
第二十三步：安装 🖊️ 语文写作教练 → 用5步流程做一次完整的作文练习
第二十四步：安装 📖 阅读理解拆解师 → 做一篇现代文阅读，走完4步法
第二十五步：安装 🏛️ 文言文复活计划 → 用复活计划理解一首正在学的古诗
```

> 💡 不需要一次全部安装。有写作作业时用写作教练，做阅读练习时用拆解师，学古文时用复活计划，平时碎片积累用素材库，写作批改时可同时联动语病档案。

### 第五阶段：数学学科加装

```
第二十六步：安装 🧬 数学错误基因档案 → 把近期错题存入档案（这是其他SKILL的数据来源）
第二十七步：安装 📐 数学解题教练 → 用三步拍照法处理一道当前作业中的错题
第二十八步：安装 💡 数学概念解释器 → 选一个模糊的概念，用类比法重建理解
第二十九步：安装 📝 应用题建模教练 → 拿一道做错的应用题，用三步提取法走一遍
第三十步：安装 🎯 思维梯度训练师 → 对一个“会了”的知识点做天花板测定
```

### 第六阶段：英语学科加装

```
第三十一步：安装 📖 智能词汇DNA系统 → 把下周课文单词表存入DNA，启动7天预热
第三十二步：安装 🎙️ 英语口语陪练 → 说今天的第一句英语（就一句就行）
第三十三步：安装 📝 英语语法突破教练 → 把最近一篇英语作业发去，生成语法基因图谱
第三十四步：安装 🎧 个性化英语听力训练师 → 生成第一段匹配你词汇量的听力材料
第三十五步：安装 ✍️ 英语写作进化教练 → 用三维批改法处理一篇旧作文
```

> 💡 五个英语专项 SKILL 以“学习DNA”为中心，每个都在输入数据、也在消费数据。装得越多用得越多，整个系统越来越懂你。

## 📅 分四阶段推荐成长路径

```text
第一阶段（第1-2周）：打好基础
  必装：#1 学习DNA档案 → #2 错题本 → #3 IM提醒 → #4 费曼测试 → #5 每周复盘
  验证：能完成一次三SKILL联动（错题→费曼→笔记）

第二阶段（第3-6周）：建立系统
  安装：#6 SKILL创建教练 → #7 康奈尔笔记档案 → #8 五SKILL联动协调器
  激活：学科SKILL包（语文/数学/英语，根据需求选择）
  验证：能生成第一份有三维度数据的周报

第三阶段（第7-10周）：管理升级
  安装：#9 30天学习计划制定师 → #10 时间与专注力教练
  升级：每周复盘→v1.1（六模块版）、联动协调器→v1.1（五SKILL版）
  验证：第一份六模块周报 + 第一张时间销行账

第四阶段（第11周起）：成长探索
  安装：#11 跨学科侦探周 → #12 兴趣成长探索计划
  升级：费曼测试→v1.1（批判性验证版）
  验证：完成第一次跨学科侦探，启动52杯咖啡第一周
```

> ⚠️ 原则：每个阶段用熟了再进入下一阶段。工具装了不用，不如只装三件用得透。

### ClawHub上架全景建议

**建议以“包”为单位上架，共6个包：**

```text
包①：核心通用基础包（#1-5）→ 必装，任何用户的起点
包②：学习方法论包（#6-8）→ 适合使用3周以上的进阶用户
包③：自我管理与成长包（#9-12）→ 适合使用6周以上的深度用户
包④：语文专项包（#13-17）→ 语文有明确需求的用户
包⑤：数学专项包（#18-22）→ 数学有明确需求的用户
包⑥：英语专项包（#23-27）→ 英语有明确需求的用户
```

**推荐套餐组合：**

```text
轻度用户：包① + 一个学科包（根据最弱学科选）
中度用户：包①② + 一到两个学科包
重度用户：包①②③ + 全部学科包
```
