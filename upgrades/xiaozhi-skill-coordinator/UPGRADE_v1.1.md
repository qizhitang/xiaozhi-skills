## Part 1：三SKILL联动协调器 v1.0 → v1.1

> 核心变化：从"三SKILL"扩展为"五SKILL"协调体系。
> 原有三个SKILL（错题本+费曼测试+康奈尔笔记）保持不变，
> 新增两个维度的数据来源。

### 升级①：扩展为五SKILL协调体系

**原版协调范围：** 错题本 + 费曼测试 + 康奈尔笔记（三SKILL）

**v1.1扩展：** 新增两个数据维度来源

```
五SKILL协调体系：

原有三维度（保持不变）：
  ① 错题本 SKILL → 错误追踪数据
  ② 费曼测试 SKILL → 理解深度数据
  ③ 康奈尔笔记档案 SKILL → 知识记录数据

新增两维度（v1.1新增）：
  ④ 30天学习计划制定师 SKILL → 计划执行数据
  ⑤ 时间与专注力教练 SKILL → 时间管理数据

五维度共同汇入：每周学习复盘SKILL（v1.2六模块周报）
```

### 升级②：五SKILL全景月度报告（扩展"月报"模块）

### 升级③：联动判断规则扩展（新增两类触发）

**原版判断规则：** 错题分析后，判断是否触发费曼测试/笔记推送

**v1.1新增两类判断规则：**

```
新增规则④：计划偏差触发
  触发条件：计划某任务连续两周未完成
  联动响应：
    → 30天学习计划制定师：收到偏差信号，提示调整
    → 错题本：检查未完成任务是否与顽固弱项相关
    → 小龙虾主动出现：
      "你的[任务X]已经连续两周没有按计划完成了。
       原因可能是[分析]，建议[调整方向]。
       我已经通知计划制定师，下周这个任务的安排会调整。"

新增规则⑤：专注力崩溃触发
  触发条件：某周时间管理维度显示有效率<50%
  联动响应：
    → 时间与专注力教练：自动激活专注力诊断模式
    → 30天学习计划制定师：建议减少本周任务量20%
    → 每周复盘：在六模块报告中专门高亮时间维度
    → 小龙虾主动出现：
      "这周的数据显示有效学习时间明显低于正常水平。
       在继续推进学习任务之前，我们先处理一下这个——
       这周让你难以专注的主要原因是什么？"
```

## Part 2：IM智能提醒 轻度扩展

> 在原有提醒系统基础上，新增三类提醒触发场景。

### 新增触发场景①：计划任务定时提醒

**来源：** 与30天学习计划制定师SKILL联动

```
当30天计划制定师生成任务后，自动在IM提醒系统注册：

提醒格式：
"📅 今日学习任务提醒
 
 今天计划中的三个任务：
 ① [任务1]（[学科]·[预计时长]·对应弱项：[弱项名]）
 ② [任务2]
 ③ [任务3]
 
 建议从[任务X]开始（你的历史数据显示这个时段做[学科]效率最高）。
 
 完成后告诉我今日结果，我更新计划完成率。"

发送时机：根据黄金时段数据，在学生开始学习前15-30分钟发送
```

### 新增触发场景②：52杯咖啡探索提醒

**来源：** 与兴趣成长探索计划SKILL联动

```
每周固定一天发送探索提醒（时间由学生设定）：

提醒格式：
"☕ 本周兴趣探索提醒
 
 这周是你兴趣探索计划的第[N]周。
 
 [如果正在探索某领域]：
 上周你在探索[领域X]，这周准备继续还是横向探索新方向？
 
 [如果刚开始]：
 你还没有开始过探索——选一个你最近有点好奇的话题，
 今天花20分钟了解一下。说'开始探索[话题]'就启动。"
```

### 新增触发场景③：时间销行账每日确认提醒

**来源：** 与时间与专注力教练SKILL联动

```
每天学习结束时间（由学生设定）发送：

提醒格式：
"⏰ 今日学习时间确认

 告诉我今天：
 · 完成了哪些学习任务
 · 实际学了多长时间
 · 有没有中途分心，大概几次
 
 用一句话说就行，
 比如：'数学45分钟完成，英语背单词20分钟，中间看了两次手机。'
 我来统计进时间销行账。"
```
