## 升级①：三维度 → 六模块（核心架构升级）

**v1.1三维度：** 错题维度 + 笔记维度 + 费曼测试维度

**v1.2新增三个维度：**

```
新增维度④：时间管理维度（来自时间与专注力教练SKILL数据）
  数据项：
  · 本周总学习时长（实际 vs 计划）
  · 高效时段利用率（黄金时段有没有用对）
  · 各学科时间分配是否合理
  · 分心次数和最主要的中断原因
  
  核心问题：这周时间用到该用的地方了吗？

新增维度⑤：计划执行维度（来自30天学习计划制定师SKILL数据）
  数据项：
  · 本周计划完成率（%）
  · 未完成的任务列表及原因
  · 计划vs实际的偏差来源分析
  · 对下周计划的调整建议
  
  核心问题：计划执行了多少，没执行的在哪里卡住了？

新增维度⑥：兴趣与成长维度（来自兴趣成长探索计划SKILL数据）
  数据项（可选，有探索记录时自动加入）：
  · 本周兴趣探索次数
  · 本周跨学科侦探周进展
  · 兴趣DNA本周信号（挑战反应记录）
  
  核心问题：这周除了任务完成，发现了什么自己感兴趣的方向？
```

## 升级②：苏格拉底自我复盘七问（新增"深度自省"模块）

**原版：** 复盘数据后直接给下周建议。

**v1.2新增：** 在给建议之前，引导学生用七个追问自我反省。

**七问框架（按顺序追问）：**

```
第一问（开放）：
"不看数据，用一句话说说——这周的学习感觉怎么样？"
→ 目的：获得学生的第一感受，后续和数据对比

第二问（事实）：
"这周你认为真正学会了哪一个知识点？
 用你自己的话说，不要说科目名称或章节名。"
→ 目的：让学生说出具体知识点，而不是模糊的"学了数学"

第三问（困难）：
"这周最让你头疼的一个时刻是什么？
 是哪个知识点，还是哪个时刻你特别不想学？"
→ 目的：定位真实障碍，不是数据里显示的问题，而是学生自己感受到的

第四问（行为）：
"这周有没有一个时刻，你做了一件让自己事后觉得'做得好'的事？
 哪怕很小的一件事。"
→ 目的：建立自我肯定的习惯，让学生发现自己的主动性

第五问（原因）：
"有没有一个计划好的任务没有完成？
 原因是什么——是任务太难、时间不够，还是当时状态不对？"
→ 目的：分析未完成的根因，而不是停在"没完成"的事实层面

第六问（联结）：
"这周有没有什么时刻，你感觉不同学科之间有关联？
 或者觉得某个知识点和生活里的什么事情有关系？"
→ 目的：激活跨科思维，培养知识联结的习惯

第七问（展望）：
"如果下周只能做好一件事，最重要的是什么？"
→ 目的：帮学生自己说出优先级，而不是由小龙虾直接告诉他

七问结束后，小龙虾做整合：
"你说的[第七问答案]和我的数据分析高度一致——
 [或者：你说的和数据显示有一些有趣的差异——]
 下面是我基于两者的综合建议……"
```
