## 升级①：三维度复盘框架（升级"数据来源"模块）

**原版：** 复盘数据来源于本周所有学习相关对话记录。

**v1.1新增：明确三个维度的数据来源和权重**

```
三维度复盘数据来源（按优先级）：

【维度一：错题维度】（来自错题本SKILL）
  数据项：
  · 本周新增错题数量和类型分布
  · 本周攻克的顽固弱项
  · 本周新触发的弱项预警
  · 错误类型趋势（比上周改善/恶化/持平）
  
  核心问题：本周哪里出了问题，出的问题和上周一样吗？

【维度二：笔记维度】（来自康奈尔笔记档案SKILL）
  数据项：
  · 本周新增笔记页数
  · 本周笔记被主动调取次数（关键指标！）
  · 新建的跨学科关联数量
  · 沉默笔记（从未被调取的笔记）比例
  
  核心问题：本周记了多少、用了多少？记而不用是无效积累。

【维度三：费曼测试维度】（来自费曼测试SKILL）
  数据项：
  · 本周费曼测试次数
  · 三层理解深度档案新增条目数
    （新增"真正掌握"/新增"能说不精"/新增"未掌握"）
  · 本周费曼测试发现的主要盲区
  
  核心问题：本周理解了多少？"做了"和"真正理解了"是两件事。
```

## 升级②：学习存折量化体系（新增"存折"模块）

**"学习存折"概念：把每周的学习积累量化，让进步看得见**

### 存折记账规则

```
【存入项（正值）】

+ 费曼测试通过（"真正掌握"）
  每通过1个知识点 = +1分

+ 整理并提炼了笔记
  每整理1页有线索问题和总结的笔记 = +0.5分

+ 攻克了一个新的顽固弱项
  每攻克1个 = +2分（权重最高，这是真正的学习飞跃）

+ 主动触发了三SKILL联动（对一道错题完整走三SKILL流程）
  每次完整联动 = +1分

【消除项（计入净值计算，但不是负分）】

消除错误类（与"存入"合并计算净值）：
  每解决一道顽固错题（错误不再出现）= -1（从错误池里减少）
  每次费曼发现盲区并当场处理 = -0.5（盲区减少）

【净存款计算】
  净存款 = 存入总分 - （消除的弱项数）
  目标：每周净存款 > 3

【健康预警】
  净存款连续2周 ≤ 0：
  说明遗忘速度超过学习速度，需要减少新内容输入，专注消化已学内容
```

### 存折可视化（周报中展示）

```
本周学习存折：

存入 ████████████ +5.5分
  ✅ 费曼通过×3     +3
  📝 笔记整理×3页   +1.5
  🏆 攻克弱项×0     +0
  🔗 联动全流程×1   +1

消除 ────────────────
  ❌ 消除顽固弱项   -1（从弱项池减少）
  🔍 处理费曼盲区   -0.5

净存款：+4 📈

[连续N周净存款 > 3，说明学习系统健康运转]
```

## 升级③：三维度行动建议精确化（升级"下周建议"模块）

**原版：** 下周建议列出3-5条通用建议。

**v1.1新增：基于三维度数据，只给一个最高优先级的具体行动**

### 建议精确化规则

```
下周建议的生成规则（按优先级排序）：

规则①：有"未掌握"条目 → 优先处理（最高优先级）
  → 建议格式：
  "下周最重要的一件事：[具体知识点]的费曼测试还是'未掌握'，
   先不要学新内容，把这个彻底弄懂。
   具体做法：[可立刻执行的两步]"

规则②：有预警触发（连续3次同类错误）→ 第二优先级
  → 建议格式：
  "你在[知识点X]上已经[N]次出现了[错误类型]，
   下周集中3次练习这个知识点，
   每次练完做一个5分钟费曼测试。"

规则③：笔记调取率低（沉默笔记>50%）→ 第三优先级
  → 建议格式：
  "你的笔记档案里有[N]页从来没被调取过。
   下周复习时，试着用这些笔记的线索问题来自测，
   而不是重新翻开笔记看。"

规则④：学习存折净存款连续2周≤0 → 第四优先级
  → 建议格式：
  "连续两周净存款未正增长——
   可能是因为[分析原因：新内容太多/复习太少/费曼测试缺失]。
   下周减少新知识输入，专注消化已学的[具体范围]。"

规则⑤：三维度均正常 → 引导进阶
  → 建议格式：
  "本周三个维度都运转良好。
   下周可以尝试提高一个难度——[具体进阶方向]"
```

### 建议精确化示例

```

"下周建议：
 1. 继续保持错题记录
 2. 多整理笔记
 3. 做好复习"

✅ v1.1版（精确可执行）：
"下周最值得做的一件事：

 你在'函数值域'上已经是第3次出现概念模糊类错误了，
 而且上周做的费曼测试结果是'能说不精'，还差一步。
 
 下周三天内，做一次完整的费曼测试，目标：通过'真正掌握'标准。
 
 具体做法：
 · 第一步：把函数值域的笔记线索问题翻出来，自问自答一遍
 · 第二步：给小龙虾当老师，讲解'值域和定义域的区别'
 · 第三步：做一道题验证"
```
