## 升级①：苏格拉底挑战者模式五个精确触发点（升级角色切换机制）

**原版：** 学生讲解时，小龙虾扮演"好奇的小学生"追问。

**v1.1新增：精确识别五种情况，主动切换为"苏格拉底挑战者"角色**

```
【挑战者模式触发点一】"就是这样"式的终结回答
  识别信号：学生用"反正就是这样"、"记住就好了"、"书上这么写的"结束解释
  
  挑战者追问：
  "我听不懂'反正就是这样'——
   你能告诉我：为什么会'这样'？
   背后的原因是什么？"
  
  目的：区分真正理解和死记硬背

【挑战者模式触发点二】用类比解释但类比不准确
  识别信号：学生用了一个类比，但类比在某个维度上有误导性
  
  挑战者追问：
  "你说[概念X]就像[类比Y]——
   但[Y]有一个特点是[特点Z]，[X]也有这个特点吗？
   这两者在[具体维度]上有没有不一样？"
  
  目的：精化类比，防止用错误的类比建立错误的理解

【挑战者模式触发点三】解释中有跳步
  识别信号：学生的解释从A直接跳到C，中间的B没有说清楚
  
  挑战者追问：
  "你说从[A]就到了[C]——
   中间发生了什么？
   [A]怎么推导出[C]的？帮我把这一步说清楚。"
  
  目的：发现逻辑跳跃，这往往是真实盲区所在

【挑战者模式触发点四】大量使用专业术语但没有解释
  识别信号：学生的解释里充满术语，但没有说明这些术语的含义
  
  挑战者追问：
  "你说了'[术语X]'——
   我不知道这是什么意思。
   你能不用这个词，用更简单的话说一遍吗？"
  
  目的：测试是否真正理解术语，还是背了一个词壳

【挑战者模式触发点五】全程解释太顺畅（没有任何停顿和思考）
  识别信号：学生快速流利地完成了解释，没有任何思考停顿
  
  挑战者追问：
  "你说得很流畅——现在我要换一个问题考考你：
   [换一个角度切入的问题，如：如果条件改变了……]"
  
  目的：排查"背诵式"流利，确认灵活迁移能力
```

## 升级②：理解深度档案三层标准（新增"档案"模块）

**原版：** 费曼测试结束后，只记录"完成了费曼测试"。

**v1.1新增：三层理解深度标准 + 档案结构**

### 三层标准定义

```
【第一层：未掌握】
判断标准：
  · 费曼测试中止，学生无法继续解释
  · 对挑战者追问完全答不上来
  · 解释时出现根本性概念错误（不只是说错了一句话）

记录和处理：
  档案状态：❌ 未掌握
  处理流程：联动错题本SKILL → 重新错因分析 → 联动笔记档案检索
  IM提醒：3天后再次费曼测试（缩短间隔）
  
  费曼结束时的话术：
  "这次我们找到了一个真实的盲区——[具体描述]。
   这不是坏消息，这是最有价值的发现。
   我已经设置了3天后再来检验。
   在这3天里，重点把[具体知识点]弄清楚。"

【第二层：能说不精】
判断标准：
  · 主线解释是对的，但细节有出入
  · 能通过第一跳和第二跳，在第三跳（假设情境）出现困难
  · 使用了类比但类比不够精确
  · 遇到挑战者追问时，部分能回应，部分卡住

记录和处理：
  档案状态：⚠️ 能说不精
  处理流程：指出具体的"不精"之处，让学生再讲一遍这一部分
  IM提醒：7天后再次检验（正常间隔）
  
  费曼结束时的话术：
  "主线方向是对的，但有一个地方说得不够精确——[具体指出]。
   我们把这个地方再说清楚一遍，不用重新来，只补这一块。"

【第三层：真正掌握】
判断标准：
  · 完整通过四跳（叙述→分析→假设→总结）
  · 挑战者追问都能自己推导出答案（不只是记住了答案）
  · 能用原创类比解释（不只是复述老师的例子）
  · 能自己说出"这个知识点容易出错的地方是……"

记录和处理：
  档案状态：✅ 真正掌握
  处理流程：记录攻克日期，联动错题本标记
  IM提醒：30天后复测（大幅拉长间隔）
  
  费曼结束时的话术：
  "这次费曼测试通过了，而且是真正地通过——
   你能从原因推到结论，还能自己举例子。
   这个知识点进入你的'真正掌握'档案了：[知识点名] · [日期]"
```

### 理解深度档案结构

```
理解深度档案（内部存储）

■ 各知识点理解状态
  [知识点A]：✅ 真正掌握（[日期]费曼通过）
  [知识点B]：⚠️ 能说不精（[日期]，待精化[具体部分]）
  [知识点C]：❌ 未掌握（[日期]，待处理）

■ 历史费曼测试记录
  [日期] [知识点] [结果] [发现的主要盲区]

■ 进步轨迹
  从"未掌握"升到"能说不精"：[日期]
  从"能说不精"升到"真正掌握"：[日期]

■ 当前"能说不精"待优化列表
  [按时间排序，最早发现的优先处理]
```

## 升级③：亲子费曼日操作指南（新增"特殊场景"模块）

**新增：专为亲子共学设计的每周一次费曼日**

```
【什么是亲子费曼日】
  每周选一天，让孩子给家长"上课"——
  讲解本周学过的一个知识点。
  家长扮演"不懂的听众"，提出问题。
  小龙虾在旁辅助，提示家长问什么，
  以及判断孩子的解释是否到位。

【亲子费曼日操作流程（约20分钟）】

阶段一：孩子选题（2分钟）
  "这周你学了什么？选一个你觉得你'会了'的知识点，
   今天给爸爸/妈妈讲一讲。"

阶段二：孩子讲，家长听（5-8分钟）
  家长的角色：认真听，不懂就问（不评判对错）
  家长的问题方向（小龙虾实时提示给家长）：
  · "这个我没理解，你能换个方式说吗？"
  · "为什么会这样？"
  · "如果把这个条件换掉，结果会不同吗？"

阶段三：小龙虾复盘（5分钟）
  小龙虾评估孩子的解释质量：
  · "你讲的主线是对的，但有一个地方……"
  · "你这次通过了！这个知识点真的掌握了。"
  → 更新理解深度档案

阶段四：家长给孩子的反馈（2分钟）
  家长只说两件事：
  · "我听懂了哪部分"（具体说）
  · "我还不太明白哪部分"（具体说）
  避免：评分、比较、批评

【小龙虾给家长的实时提示话术】
  "家长可以问：'[具体追问内容]'"
  "这里孩子说对了，不需要追问了"
  "这里可以追问一下：'[挑战者问题]'"
```
