# 🧬 学习DNA档案 SKILL 升级补丁 v1.0 → v1.1

> **升级说明：** 基于第八章8.2节"学习成长图谱"内容，
> 对学习DNA档案SKILL进行四项升级。

---

## 升级①：成长图谱四维度（新增"成长图谱"模块）

**原版：** DNA档案六维度（基础信息/学科强弱/学习风格/错误模式/对话历史/成长轨迹）

**v1.1新增：** 成长图谱四维度（比原有维度更具视觉化和可追踪性）

```
成长图谱是对原有学习DNA档案的"升维可视化版本"。
在档案的基础上，增加四个可以跨时间对比的图谱维度：

图谱维度①：错题地图（空间化展示）
  不只是统计各学科错题数，而是按知识点展开：
  
  格式：[学科] → [章节] → [知识点] → [错误次数+状态]
  
  示例：
  数学 → 一次函数 → 象限判断 → 错误3次（已攻克）
        → 斜率计算 → 错误1次（正常）
        → 解析式推导 → 错误5次（顽固弱项⚠️）
  
  更新触发：每次错题本SKILL记录后自动更新
  
  价值：学生能看到知识点层面的"薄厚对比"，
        不只是"数学弱"这种模糊感受

图谱维度②：口语成长轨迹（英语专项）
  对英语口语用户的专属成长轨迹：
  
  格式：[发音弱点] → [出现次数] → [改善曲线]
        [连续热身天数] → [里程碑事件]
  
  每次口语陪练SKILL结束后自动更新
  
  价值：让"英语口语进步"可见化，不再只是感觉

图谱维度③：弱项突破记录
  专门记录每一个从"顽固弱项"到"已攻克"的完整过程：
  
  格式：
  [弱项名称]
  首次发现：[日期]
  历史出错：[N]次
  突破方法：[错题本/费曼测试/专项训练]
  攻克日期：[日期]
  攻克验证：[连续[N]次复测通过]
  
  这是学习成就感最直接的来源，每次攻克主动告知学生

图谱维度④：知识积累树（跨科关联网络）
  在原有"知识点积累"的基础上，增加"跨科关联"标注：
  
  当同一个概念在不同学科出现，或两个学科知识点有联系时，
  自动在知识积累树中添加跨科节点：
  
  格式：
  [知识点A（数学）] ←→ [知识点B（物理）]
  联结发现日期：[日期]
  联结方式：[描述]
  来源：[跨学科侦探周/学习中自然发现/小龙虾推送]
```

## 升级②：里程碑自动标注机制（强化"成长轨迹"模块）

**原版：** 里程碑由学生主动触发或小龙虾对话中识别。

**v1.1新增：** 四类里程碑的自动标注触发机制

```
自动标注触发规则：

触发①：知识点从"顽固弱项"变为"已攻克"
  触发条件：某知识点连续3次复测通过，且之前出错≥3次
  自动记录：[日期] [知识点名称] 从顽固弱项→攻克
  告知话术：
  "等一下——你知道吗？
   [知识点]在你的档案里已经躺了[N]周了，历史出错[N]次。
   今天连续第3次测对了——这个弱项正式攻克了。
   记进你的成长图谱了：[日期] [知识点] ✅已攻克"

触发②：费曼测试首次"真正掌握"
  触发条件：某知识点费曼测试首次达到"真正掌握"标准
  自动记录：[日期] [知识点] 费曼测试通过

触发③：连续N天学习里程碑
  触发条件：连续7天 / 14天 / 30天 / 100天有学习记录
  告知话术：个性化（见原growth-milestones.md）

触发④：学科成长里程碑
  触发条件：某学科错题数连续三周下降 or 连续两周完成率>80%
  自动记录：[日期] [学科] 持续改善里程碑
```

## 升级③：兴趣DNA接口（新增"兴趣维度"）

**原版：** DNA档案没有兴趣探索相关字段。

**v1.1新增：** 增加兴趣维度字段，与兴趣成长探索计划SKILL联动

```
■ 兴趣DNA（新增字段）
  当前正在探索的领域：[列表]
  各领域挑战反应信号（最近更新）：
    [领域A]：[放弃信号/继续信号/确认兴趣]
  已确认浅层喜好（不需要继续探索）：[列表]
  可能真正兴趣（持续追踪）：[列表]
  兴趣DNA最后更新：[日期]

更新触发：
  兴趣成长探索计划SKILL每次更新后，自动同步到学习DNA
```

## 升级④：跨学科知识树接口（新增"知识联结"字段）

**原版：** DNA档案各学科知识独立记录，没有跨科联结标注。

**v1.1新增：** 增加跨学科联结字段

```
■ 知识积累树（升级字段）
  各学科知识点列表：[原有]
  
  新增：跨学科关联节点
  [知识点A（学科）] ↔ [知识点B（学科）]
  关联类型：[概念相同/逻辑相似/因果关联/应用迁移]
  发现时间：[日期]
  发现来源：[侦探周/学习中/小龙虾推送]

更新触发：
  跨学科侦探周SKILL每次生成项目DNA后，自动同步跨科关联节点
```
