# ⏰ IM智能提醒 SKILL

> **一句话定位：** 不是你去找小龙虾，而是它在你快忘的那一刻找到你——  
> 把遗忘曲线变成复习曲线，让记忆自动滚雪球。

---

## 一、SKILL 的核心使命

**人类大脑的遗忘规律（艾宾浩斯遗忘曲线）：**
```
学完后20分钟  → 记忆保留约58%
学完后1小时   → 记忆保留约44%
学完后1天     → 记忆保留约33%
学完后1周     → 记忆保留约25%
学完后1个月   → 记忆保留约21%
```

这意味着：**不复习，就遗忘**。这是大脑的硬件规律，无法改变。

但可以改变的是：**在遗忘发生之前，有人提醒你复习**。

这个SKILL的使命，就是成为那个"在正确时机出现的提醒者"。

## 二、触发时机

| 触发场景 | 示例语句 | 处理方式 |
|---------|---------|---------|
| 学完新内容主动汇报 | "我今天学了二次函数的顶点公式" | 自动设置复习提醒序列 |
| 明确要求设置提醒 | "帮我明天提醒我复习这个" | 立即确认并设置 |
| 错题分析完成后 | （错题本SKILL分析结束） | 自动联动设置强化提醒 |
| 考试备考模式 | "这周五有数学考试" | 启动倒计时复习计划 |
| 主动查看提醒状态 | "我现在有哪些要复习的？" | 列出当前复习队列 |
| 长时间未使用 | 超过3天无对话 | 发送温柔的唤醒提醒 |

## 三、核心功能：提醒序列设计

### 3.1 标准复习提醒序列（普通知识点）

当学生学完一个新知识点，**自动设置以下提醒序列**：

```
知识点：[XX知识点]
记录日期：Day 0

提醒序列：
├── Day 1  （明天）    → 第一次复习提醒
├── Day 3  （三天后）  → 第二次复习提醒
├── Day 7  （一周后）  → 第三次复习提醒
├── Day 14 （两周后）  → 第四次复习提醒
└── Day 30 （一个月后）→ 巩固验证提醒

如果某次提醒后学生确认"已记住"→ 间隔拉长
如果学生确认"忘了" → 间隔缩短，回到上一阶段
```

### 3.2 错题强化提醒序列（高优先级）

来自错题本SKILL的错题，使用**更密集的提醒**：

```
错题类型：[XX错误模式]
分析日期：Day 0

强化提醒序列：
├── Day 1  → 检验是否理解了根因
├── Day 2  → 出一道变形题测试
├── Day 5  → 再次验证，出更难一点的同类题
├── Day 10 → 综合检验
└── Day 21 → 最终巩固确认

固定错误模式（3次以上）→ 优先级提升为"紧急"，每天提醒直到攻克
```

### 3.3 考前冲刺提醒序列

学生告知考试日期后，启动**倒计时复习计划**：

```
考试日期：[日期]
当前剩余天数：[X天]

X > 14天：正常复习节奏 + 重点薄弱点专攻
7 < X ≤ 14天：加强频率，重点放在固定错误模式
3 < X ≤ 7天：每日复习要点，出模拟练习题
X ≤ 3天：轻度提醒，避免焦虑，重点在信心建立
考试当天上午：发送"加油"提醒 + 关键知识点速览卡
```

## 四、提醒消息格式规范

提醒消息**绝对不能**只是"该复习了！"  
每条提醒都应该是**有内容的、个性化的**：

**格式模板（Day 1提醒）：**
```
[提醒] 嘿，昨天学的[知识点名称]，今天来巩固一下。

❓ 快问快答：[一道简单的检验题]

回复你的答案，还是不太确定的话说"再讲一遍"，
我会根据你的回答来决定后续的复习计划。

（如果真的很忙，回复"稍后"，我明天再提醒你。）
```

**格式模板（Day 7提醒）：**
```
[提醒] 一周前我们搞定了[知识点]，现在还记得吗？

我来出一道稍微难一点的题：
[中等难度检验题]

试试看，不用紧张——忘了很正常，复习了就好。
```

**格式模板（固定错误模式提醒）：**
```
[重要提醒] 你上次在[错误类型]上卡过，今天专门练这个：

题目：[针对性练习题]

做完发给我，我帮你看看这次是不是真的绕过去了。
```

## 五、学生响应处理

当学生回复提醒消息时：

| 学生回复 | 小龙虾处理 |
|---------|---------|
| 答对了 | 给予肯定，确认记忆，延长下次提醒间隔 |
| 答错了 | 重新讲解，缩短下次提醒间隔，联动错题本 |
| "记得了" | 快速验证一题，确认后延长间隔 |
| "忘了" | 温和回应，重新讲解，不批评 |
| "稍后" | 设置4小时后二次提醒 |
| "知道了不用提醒了" | 再次确认，并最终停止（尊重学生决定） |
| 无回应（24小时内） | 24小时后发送一次温柔的补发提醒 |

## 六、提醒数量管理

**避免"提醒轰炸"导致学生疲惫：**

```
每日提醒上限规则：
├── 普通复习提醒：每天最多3条（时间间隔>2小时）
├── 考前冲刺模式：每天最多5条
├── 首选发送时间：学生历史活跃时段（从DNA档案读取）
└── 夜间免打扰：22:00—8:00 不发送（除非学生明确要求）

优先级排序（当日提醒过多时）：
1. 考前冲刺提醒（最高优先级）
2. 固定错误模式专项提醒
3. 正常复习序列提醒（按知识点薄弱程度排序）
```

## 七、复习队列可视化

当学生询问"我现在有什么要复习的？"时，生成**复习队列概览**：

```
📋 你的复习队列（今天）

🔴 紧急（固定错误模式专项）
  • 行程问题-审题习惯 → 已提醒3次，今天必须攻克

🟡 今日到期
  • 英语-完成时态用法 → Day 7复习
  • 数学-因式分解公式 → Day 3复习

🟢 本周内
  • 语文-人物形象分析框架 → Day 14复习（还有4天）
  • 英语-介词搭配 → Day 7复习（还有2天）

💡 建议：先搞定红色，再处理黄色，绿色今天不用管。
   预计需要时间：约25分钟
```
