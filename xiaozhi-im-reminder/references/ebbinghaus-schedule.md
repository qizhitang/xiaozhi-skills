## 艾宾浩斯遗忘曲线

赫尔曼·艾宾浩斯（Hermann Ebbinghaus）的实验发现，  
在没有复习的情况下，人类对新学内容的记忆保留率如下：

| 时间节点 | 记忆保留率 | 提醒策略 |
|---------|----------|---------|
| 学习后20分钟 | ~58% | 不需要提醒，记忆新鲜 |
| 学习后1小时 | ~44% | 可选复习（重要内容） |
| 学习后9小时 | ~36% | — |
| 学习后1天 | ~33% | ✅ 第1次复习（最关键） |
| 学习后2天 | ~28% | — |
| 学习后3天 | ~25% | ✅ 第2次复习 |
| 学习后6天 | ~23% | — |
| 学习后1周 | ~21% | ✅ 第3次复习 |
| 学习后2周 | — | ✅ 第4次复习 |
| 学习后1个月 | ~21% | ✅ 第5次复习（长期巩固） |

## 提醒间隔动态调整规则

### 基准间隔序列
```
Day 0 (学习当天)
Day 1 → Day 3 → Day 7 → Day 14 → Day 30
```

### 动态调整规则

**答对 + 有信心 → 间隔乘以1.5~2倍（加速）**
```
例：Day 3应提醒 → 答对 → 下次提醒改为Day 6而不是Day 7
```

**答对 + 不太确定 → 按原计划**
```
保持原定间隔，正常推进
```

**答错 → 间隔缩短至上一阶段重来**
```
例：Day 7复习答错 → 重新从Day 1开始
```

**长期未回应（3次提醒无反应）→ 降频但不停止**
```
改为每周1次，直到学生恢复使用
```

## 不同类型内容的提醒策略

### 理解类内容（概念、公式推导、解题思路）
- 重点检验：能否用自己的话解释
- 提醒形式：开放性问答
- 间隔：标准序列

### 记忆类内容（单词、古诗、公式、年代）
- 重点检验：能否准确回忆
- 提醒形式：填空/默写题
- 间隔：标准序列 + 额外高频提醒（前3天每天1次）

### 应用类内容（解题方法、分析框架）
- 重点检验：能否在新题目中正确应用
- 提醒形式：出一道变形练习题
- 间隔：标准序列，但每次提醒必须附题目

### 错题类内容（来自错题本SKILL）
- 重点检验：同类题是否不再犯同样错误
- 提醒形式：针对性练习题（难度略高于原错题）
- 间隔：强化序列（Day 1 → 2 → 5 → 10 → 21）

## 考前复习倒计时参数

| 剩余天数 | 复习重点 | 提醒频率 | 情绪基调 |
|---------|---------|---------|---------|
| 14天+ | 薄弱点专项突破 | 每天1-2条 | 从容，有计划感 |
| 7-14天 | 系统梳理+错题集 | 每天2-3条 | 专注，不焦虑 |
| 3-7天 | 重点知识点快速过 | 每天3-4条 | 紧张但有序 |
| 1-3天 | 轻度提醒+信心建立 | 每天1-2条 | 平静，相信积累 |
| 考试当天 | 速览关键词+加油 | 1条（早上） | 温暖，正向激励 |

## 提醒时间优化

根据学习DNA档案中的活跃时段，选择最佳提醒时间：

| 学生类型 | 建议提醒时间 |
|---------|------------|
| 放学后学习型 | 16:30-17:30 / 19:00-20:00 |
| 夜间学习型 | 20:00-21:00 |
| 早起学习型 | 7:00-7:30 |
| 午休利用型 | 12:00-12:30 |

**原则：** 在学生精力充沛、即将可以学习的时间点提醒，  
而不是在学生正在吃饭/睡觉/上课时提醒。
