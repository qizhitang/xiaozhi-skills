# 物理建模模式详解

> 本文档是物理建模教练的完整参考手册，涵盖建模通用框架、五大核心模型详解、模型识别决策树以及跨模型综合题策略。
> 所有内容面向初中物理建模教学，注重可操作性和可迁移性。

---

## 一、建模通用三步框架

物理建模的核心思路是将文字描述的物理情境，转化为可计算的数学结构。这个过程分为三个步骤，每一步都有明确的任务和检测标准。

### Step 1：识别现象

**任务：把题目中的自然语言翻译成物理语言——"这道题描述了什么物理过程？"**

识别现象的关键是找到"变化"——物理题的本质就是描述某种变化，而变化就意味着有物理过程发生。

**识别现象三问：**

| 序号 | 追问内容 | 目的 |
|------|---------|------|
| ① | 题目中有哪些物体？ | 确定研究对象 |
| ② | 这些物体在做什么？（运动/静止/形变/发热/发光……） | 确定物理行为 |
| ③ | 有什么变化发生？（速度变化/温度变化/电路状态变化……） | 确定物理过程 |

**补充追问模板：**

- "用你自己的话说说，这道题描述了什么物理过程？"
- "你看到了什么现象？有什么在变化？"
- "题目中提到了哪些物理量？它们之间有什么关系？"
- "如果让你画一幅图来描述这个过程，你会画什么？"

**识别困难时的退回策略：**

- 看不出物理过程 → 退回读题画图，要求学生画草图标注关键信息
- 说不清什么在变化 → 提示"找变化量"：什么量在变？什么量不变？什么量是恒定的？
- 多个过程混在一起 → 提示"数一数，这道题描述了几个阶段？"

**识别成功的判定标准：** 学生能够用自己的话完整描述物理过程，而不是复述题目原文。

---

### Step 2：选择模型

**任务：从五大核心模型中选择——"这个现象属于哪类物理模型？"**

模型选择的本质是分类——将具体的物理情境归入已知的模型类别。每个模型都有明确的适用条件，选择模型就是判断题目条件是否满足某个模型的适用条件。

**模型识别决策树（简化版）：**

```text
这道题涉及的核心物理过程是什么？
│
├── 涉及力与运动
│   ├── 物体受力平衡（静止/匀速直线运动）→ 力学平衡模型
│   ├── 物体受力不平衡（加速/减速）→ 匀变速直线运动模型
│   └── 涉及碰撞/撞击 → 碰撞动量模型
│
├── 涉及电路
│   ├── 分析电流/电压/电阻/功率关系 → 电路分析模型
│   └── 关注电能转化为其他形式能 → 能量守恒模型
│
└── 涉及能量转化
    └── 一种能量变为另一种能量 → 能量守恒模型
```

**模型选择追问模板：**

- "你观察到的现象，属于力的问题、电的问题、还是能量的问题？"
- "物体是静止的还是在运动？运动状态有没有改变？"
- "有没有能量从一种形式变成另一种形式？"
- "这道题的核心物理过程和哪个模型的条件最吻合？"

**选错模型时的纠正策略：**

- 不要直接说"你选错了"，而是引导检查适用条件
- "力学平衡模型要求合力为零，你画一下受力图，合力是零吗？"
- "匀变速运动模型要求加速度恒定，这里的加速度恒定吗？"

---

### Step 3：数学表达

**任务：用数学语言描述模型——"模型告诉你哪些量之间有关系？把关系写成方程。"**

数学表达是建模的最终输出。从物理模型到数学方程，中间的桥梁是"核心关系式"——每个模型都有一个核心关系，把它写成数学形式就是方程。

**从模型到方程三步：**

| 步骤 | 操作 | 检查点 |
|------|------|--------|
| ① | 写出模型的核心关系式（先用符号，不代数字） | 公式形式是否正确？ |
| ② | 代入已知量（注意单位统一） | 单位是否统一？方向正负号是否正确？ |
| ③ | 求解未知量 | 结果是否合理？量纲是否正确？ |

**方程列不出时的追问模板：**

- "你选了[模型名称]，这个模型的核心关系是什么？"
- "模型告诉你[物理量A]和[物理量B]有什么关系？"
- "把这个关系写成数学表达式——左边是什么？右边是什么？"
- "你列的方程里，有几个未知数？需要几个方程才能解？"

**数学表达常见障碍：**

| 障碍 | 表现 | 处理方式 |
|------|------|---------|
| 不会写核心关系式 | "我知道是平衡但列不出式子" | 引导回忆模型的核心关系，从文字描述开始 |
| 方向正负号混乱 | "a到底取正还是取负？" | 强调先选正方向，再确定各量的正负 |
| 单位不统一 | "厘米和米能不能直接加？" | 列式前先统一单位到国际单位制 |
| 未知数太多 | "列了一个方程但有三个未知数" | 引导找补充条件或补充方程 |

## 二、匀变速直线运动模型

### 适用条件判断

使用此模型前，必须逐条检查以下条件是否满足：

- [ ] 物体做直线运动（轨迹是直线，不是曲线）
- [ ] 加速度恒定（大小和方向都不变）
- [ ] 已知量中至少包含 v₀、a、t、s、v 中的三个
- [ ] 运动过程中没有突变（如碰撞、拐弯等需要分段处理）

**注意：** 如果题目描述的是多段运动（如先加速后减速），每一段分别满足匀变速条件，但需要分段建模，不能混在一起。

### 标准建模流程

```text
第1步：画运动过程示意图
  - 画出物体的运动轨迹（直线）
  - 标注初速度 v₀ 的方向和大小
  - 标注加速度 a 的方向和大小
  - 标注运动时间 t 或位移 s
  - 标注末速度 v 的方向和大小（如果已知）

第2步：选择正方向
  - 通常取初速度方向为正方向
  - 减速运动中，加速度 a 取负值
  - 自由落体中，取向下为正方向，a = g
  - 竖直上抛中，取向上为正方向，a = -g

第3步：列运动学方程
  - 从五个运动学量 v₀、v、a、t、s 中选择合适的基本公式
  - 未知量是 v → 用 v = v₀ + at
  - 未知量是 s → 用 s = v₀t + ½at²
  - 未知量是 t → 用 v = v₀ + at 先求 t
  - 不含 t → 用 v² = v₀² + 2as
  - 平均速度法：s = ½(v₀+v)t（适用于已知初末速度求位移）

第4步：代入求解
  - 统一单位后代入已知量
  - 检查结果的正负号是否合理
  - 用量纲检验结果的合理性
```

### 列式模板

**基本公式组（五个量：v₀、v、a、t、s）：**

```text
速度公式：    v = v₀ + at
位移公式：    s = v₀t + ½at²
速度位移关系：v² = v₀² + 2as
平均速度公式：s = v̄t = ½(v₀ + v)t
```

**列式检查清单：**
1. 正方向是否已选定？
2. a 的正负号是否与正方向一致？
3. s 的正负号是否合理（位移方向与正方向的关系）？
4. 单位是否统一（m、s、m/s、m/s²）？

### 常见变形

**变形1：自由落体**

- 条件：初速度为零，只受重力
- 等效替换：v₀ = 0，a = g = 9.8 m/s²（初中常取 10 m/s²），取向下为正
- 公式简化：v = gt，s = ½gt²，v² = 2gs
- 易错点：不要忘记自由落体也是匀变速运动，只是 v₀ = 0 的特例

**变形2：竖直上抛**

- 条件：初速度向上，只受重力
- 等效替换：取向上为正方向，v₀ > 0，a = -g
- 公式：v = v₀ - gt，s = v₀t - ½gt²
- 关键特征：最高点 v = 0，上升时间 t上 = v₀/g，上升高度 h = v₀²/(2g)
- 易错点：上升到最高点和落回抛出点的时间相等；落回抛出点的速度大小等于 v₀，方向向下

**变形3：刹车问题**

- 条件：车辆匀减速至停止
- 等效替换：v₀ > 0，a < 0（减速），末速度 v = 0
- 关键计算：刹车时间 t = v₀/|a|，刹车距离 s = v₀²/(2|a|)
- 易错点：必须先算刹车时间！如果题目问的时间超过刹车时间，车已经停下，不能再代入减速公式计算（a 不会变负号让车倒退）

**变形4：追及问题**

- 条件：两个物体同向运动，后方物体追赶前方物体
- 建模方法：分别对两个物体列运动学方程，找到位移关系
- 关键条件：追上时两物体位移之差等于初始距离（s₂ - s₁ = Δs）
- 临界分析：速度相等时距离最大或最小
- 易错点：需要两个方程联立；注意时间 t 是同一个

### 常见错误类型

| 错误类型 | 表现 | 纠正方法 |
|---------|------|---------|
| 加速度方向搞反 | 减速运动中 a 取了正值 | 先选正方向，减速时 a 与 v₀ 方向相反，取负值 |
| 刹车不停继续减速 | 刹车后用了超过实际刹车时间 | 先算刹车时间 t = v₀/|a|，超过此时间速度为零 |
| 多段运动不分段 | 先加速后减速用同一个 a | 每段运动单独列方程，注意衔接点速度 |
| 位移和路程混淆 | 往返运动中 s 和路程不相等 | 明确位移是矢量，有方向；路程是标量 |

### 追问序列

1. "物体在做直线运动，它的速度是在增大、减小还是不变？"
2. "加速度的方向和速度的方向是什么关系？同向还是反向？"
3. "你选了正方向没有？加速度在这个正方向下是正的还是负的？"
4. "这道题有几个运动阶段？每个阶段的加速度一样吗？"
5. "你算出来的结果合理吗？比如一个物体从2米高落下，落地速度算出来应该是几米每秒的量级？"
6. "如果时间超过刹车时间，车还会继续减速吗？它实际上是什么状态？"

## 三、力学平衡模型

### 适用条件判断

- [ ] 物体处于静止状态，或匀速直线运动状态
- [ ] 物体受到的合力为零
- [ ] 物体可以视为质点（不考虑转动效应时）
- [ ] 如果涉及转动，使用力矩平衡条件

**判断要点：** "静止"和"匀速直线运动"在力学上等价——都是合力为零。如果题目说"匀速运动"而没有说"直线"，要注意确认是否为直线运动（圆周运动的匀速也需要向心力，合力不为零）。

### 标准建模流程

```text
第1步：确定研究对象
  - 明确分析哪个物体（隔离法）或哪几个物体（整体法）
  - 选择使问题最简单的对象

第2步：画受力分析图
  - 按顺序找力：重力 → 弹力（支持力/拉力/压力）→ 摩擦力
  - 每个力画在作用点上，标明方向和符号
  - 不要多画力（如"下滑力"不是独立的力）
  - 不要漏画力（特别是摩擦力）

第3步：建立坐标系
  - 原则：选择使力的分解最少的方向为坐标轴
  - 斜面问题：沿斜面方向和垂直斜面方向
  - 悬挂问题：水平方向和竖直方向
  - 一般问题：水平方向和竖直方向

第4步：分解力
  - 将不沿坐标轴的力分解到 x、y 方向
  - 斜面上的重力：Gx = mg sinθ，Gy = mg cosθ

第5步：列平衡方程
  - x 方向：ΣFx = 0
  - y 方向：ΣFy = 0
  - 力矩平衡（杠杆问题）：ΣM = 0，即 F₁L₁ = F₂L₂

第6步：求解检验
  - 解出未知力
  - 检查力的方向是否合理（如支持力应该指向物体）
  - 特殊角度检验：θ=0°和θ=90°时结果是否符合直觉
```

### 列式模板

**共点力平衡：**

```text
x 方向平衡：F₁x + F₂x + ... = 0
y 方向平衡：F₁y + F₂y + ... = 0

常见写法：
  水平方向：F拉·cosθ = f摩擦（水平方向合力为零）
  竖直方向：F拉·sinθ + F支持 = mg（竖直方向合力为零）
```

**力矩平衡（杠杆）：**

```text
F₁ × L₁ = F₂ × L₂

其中 L 为力臂——支点到力的作用线的垂直距离（不是支点到作用点的距离！）
```

**浮力平衡：**

```text
F浮 = G排 = ρ液 g V排
平衡条件：F浮 = G物  （漂浮/悬浮时）
沉底条件：F浮 < G物
```

### 常见变形

**变形1：悬挂问题**

- 情境：物体被一根或两根绳子悬挂，处于静止
- 分析方法：以悬挂点为研究对象，画受力图
- 单绳悬挂：绳的拉力 T = mg（竖直方向平衡）
- 双绳悬挂：两个拉力的合力等于重力，用平行四边形法则或正交分解
- 关键：绳子只能拉不能推，拉力方向沿绳子方向

**变形2：斜面问题**

- 情境：物体静止在斜面上（或有沿斜面方向的推力/拉力）
- 坐标系选择：沿斜面方向为 x 轴，垂直斜面方向为 y 轴
- 重力分解：Gx = mg sinθ（沿斜面向下），Gy = mg cosθ（垂直斜面向下）
- 平衡方程：沿斜面 F拉 + f = mg sinθ（若物体有上滑趋势），垂直斜面 F支持 = mg cosθ
- 易错点：不要画"下滑力"——它只是重力沿斜面的分力，不是独立的力

**变形3：杠杆问题**

- 情境：硬棒绕支点转动，处于平衡
- 核心关系：F₁L₁ = F₂L₂
- 力臂的确定：从支点到力的作用线的垂直距离
- 易错点：力臂不是支点到作用点的距离！只有力的方向垂直于支点-作用点连线时两者才相等
- 变化分析：移动支点、改变力的方向都会影响力臂

**变形4：浮力问题**

- 情境：物体在液体中漂浮/悬浮/沉底
- 漂浮：F浮 = G物，V排 < V物
- 悬浮：F浮 = G物，V排 = V物
- 沉底：F浮 < G物，F支持 = G物 - F浮
- 关键：漂浮和悬浮都是平衡态，区别在于排开液体的体积不同
- 易错点：浮力只与 ρ液 和 V排 有关，与物体密度、深度无关

### 常见错误类型

| 错误类型 | 表现 | 纠正方法 |
|---------|------|---------|
| 漏画力 | 忘了画摩擦力或支持力 | 按顺序找力：重力→弹力→摩擦力，逐一检查 |
| 多画力 | 画了"下滑力"或"向心力"作为独立力 | 重力已分解，不要再加分量作为独立力 |
| 力臂概念错误 | 把支点到作用点的距离当力臂 | 力臂是支点到力的作用线的垂直距离 |
| 力的分解方向错误 | 随意选分解方向 | 沿坐标轴分解，坐标轴选择使分解最少 |

### 追问序列

1. "你确定研究对象了吗？是哪个物体？"
2. "按顺序数一数，这个物体受到了几个力？重力、弹力、摩擦力——都有吗？"
3. "你建立的坐标系是什么？为什么这样选？"
4. "x 方向有哪些力？它们的和等于零吗？y 方向呢？"
5. "如果斜面角度变大，支持力会怎么变？摩擦力呢？你的方程能反映这个变化吗？"
6. "你画的力臂对不对？力臂是从支点到哪里的距离？"

## 四、碰撞动量模型

### 适用条件判断

- [ ] 涉及两个物体的碰撞、撞击、爆炸等相互作用
- [ ] 系统不受外力，或外力远小于相互作用力
- [ ] 需要分析碰撞前后的状态变化

**初中简化说明：** 初中物理不要求使用动量守恒定律公式计算，但碰撞的思维方式仍然是核心考查内容。初中阶段侧重训练碰撞前后的状态分析——碰撞前各物体是什么状态，碰撞后各物体变成什么状态，以及碰撞过程中能量如何转化。

### 标准建模流程

```text
第1步：确定系统和过程
  - 明确参与碰撞的物体（系统）
  - 明确碰撞前、碰撞中、碰撞后三个阶段
  - 碰撞前的状态 → 初态
  - 碰撞后的状态 → 末态

第2步：分析碰撞前状态
  - 每个物体的速度（大小和方向）
  - 每个物体的质量
  - 画出碰撞前的示意图

第3步：分析碰撞过程
  - 碰撞过程中发生了什么？（能量转化、形变等）
  - 碰撞时间极短，可以忽略位移

第4步：分析碰撞后状态
  - 每个物体的速度（大小和方向）
  - 是否粘在一起？（完全非弹性碰撞）
  - 是否弹开？（弹性碰撞）
  - 画出碰撞后的示意图

第5步：列关系式
  - 高中阶段：列动量守恒方程 m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
  - 初中阶段：定性分析碰撞前后速度变化、能量转化方向
```

### 列式模板

**高中版——动量守恒方程：**

```text
碰撞前总动量 = 碰撞后总动量
m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'

注意事项：
  - 速度是矢量，需要选正方向
  - 碰撞前后的速度必须相对于同一参考系
  - 动能关系判断：碰撞后总动能 ≤ 碰撞前总动能
```

**初中版——碰撞前后状态分析：**

```text
碰撞前：物体A的状态（速度v₁、方向），物体B的状态（速度v₂、方向）
碰撞后：物体A的状态（速度v₁'、方向），物体B的状态（速度v₂'、方向）

关键分析：
  - 谁变快了？谁变慢了？
  - 碰撞过程中有没有能量损失？
  - 碰撞后两物体是否一起运动？

能量分析：
  - 弹性碰撞：动能守恒（碰后总动能 = 碰前总动能）
  - 非弹性碰撞：动能有损失（碰后总动能 < 碰前总动能）
  - 完全非弹性碰撞：动能损失最大（碰后粘在一起）
```

### 常见变形（初中简化版）

**变形1：两球对心碰撞**

- 情境：一个运动的球撞一个静止的球
- 分析重点：碰撞后运动球变慢（或停下），静止球开始运动
- 初中要求：判断碰撞后各球的运动方向和速度大小关系
- 能量视角：碰撞过程中部分动能转化为内能（发热）和形变势能

**变形2：完全非弹性碰撞（碰后粘合）**

- 情境：两物体碰撞后粘在一起以共同速度运动
- 特点：动能损失最大
- 初中要求：分析碰后共同速度的方向和大小（定性判断）
- 公式简化：m₁v₁ = (m₁ + m₂)v共

**变形3：反弹问题**

- 情境：物体撞墙后反弹
- 分析方法：墙壁质量远大于物体，可认为墙不动
- 初中要求：判断反弹速度大小与碰前速度的关系
- 能量视角：完全弹性反弹速度大小不变；非弹性反弹速度变小

**变形4：爆炸问题**

- 情境：一个物体分裂成两部分（反冲运动）
- 特点：分裂前物体可能静止，分裂后两部分向相反方向运动
- 初中要求：分析分裂后两部分的运动方向，理解内力作用
- 能量视角：化学能（或其他内能）转化为动能

### 常见错误类型

| 错误类型 | 表现 | 纠正方法 |
|---------|------|---------|
| 速度方向忽略 | 碰撞后两物体同向运动时没注意方向 | 选正方向，标明每个速度的正负 |
| 碰撞前后状态混淆 | 把碰撞前的速度代入碰撞后的方程 | 画碰撞前后两张图，分别标注 |
| 忽略能量损失 | 认为碰撞后动能一定守恒 | 除非明确"弹性碰撞"，否则动能有损失 |
| 参考系不统一 | 一个速度相对地面，一个相对物体 | 所有速度必须相对于同一参考系 |

### 追问序列

1. "碰撞前，每个物体是什么状态？在运动还是静止？速度方向呢？"
2. "碰撞后，每个物体变成了什么状态？速度变大了还是变小了？方向有没有变？"
3. "碰撞过程中，有没有能量损失？你怎么判断的？"
4. "你能画两张图吗？一张碰撞前的，一张碰撞后的，标出每个物体的速度。"
5. "如果一个球撞墙反弹，碰前碰后速度大小一样吗？什么时候一样？"
6. "碰撞的时间很短，在这段时间内物体的位置变了吗？为什么可以忽略位移？"

## 五、电路分析模型

### 适用条件判断

- [ ] 题目涉及电路，需要分析电流、电压、电阻、功率等量
- [ ] 电路中有明确的串并联结构（或可通过开关状态确定结构）
- [ ] 需要求解某个电学物理量

**判断要点：** 如果题目涉及"电能转化为其他形式能"而不仅仅是电路中的电压电流关系，应考虑使用能量守恒模型而非电路分析模型。两个模型可以组合使用。

### 标准建模流程

```text
第1步：画等效电路图
  - 根据开关状态画出简化电路
  - 每个开关状态画一张等效图（开关通断可能改变电路结构）
  - 滑动变阻器需要标注当前位置

第2步：识别串并联结构
  - 串联特征：电流只有一条路径，电流处处相等
  - 并联特征：电流有多条路径，各支路两端电压相等
  - 混联电路：先识别局部串并联，再逐步简化

第3步：标注已知量和未知量
  - 在电路图上标注每个元件的 R、U、I、P
  - 已知量标数值，未知量标符号
  - 特别标注额定值（额定电压、额定功率）和实际值

第4步：选择规律列方程
  - 欧姆定律：I = U/R（适用于纯电阻电路）
  - 串联分压：U₁/U₂ = R₁/R₂
  - 并联分流：I₁/I₂ = R₂/R₁
  - 电功率：P = UI = I²R = U²/R

第5步：求解检验
  - 检查功率是否超过额定值
  - 检查电流方向和大小是否合理
  - 检查串并联关系是否正确
```

### 列式模板

**欧姆定律：**

```text
I = U / R
变形：U = IR，R = U/I
适用条件：纯电阻电路（金属导体、电阻器等）
```

**串联电路：**

```text
电流：I = I₁ = I₂ = I₃ = ...
电压：U = U₁ + U₂ + U₃ + ...
电阻：R = R₁ + R₂ + R₃ + ...
分压：U₁/U₂ = R₁/R₂
功率：P₁/P₂ = R₁/R₂（串联时功率与电阻成正比）
```

**并联电路：**

```text
电压：U = U₁ = U₂ = U₃ = ...
电流：I = I₁ + I₂ + I₃ + ...
电阻：1/R = 1/R₁ + 1/R₂ + 1/R₃ + ...
分流：I₁/I₂ = R₂/R₁
功率：P₁/P₂ = R₂/R₁（并联时功率与电阻成反比）
```

**电功率：**

```text
P = UI = I²R = U²/R
W = Pt = UIt（电功）
Q = I²Rt（焦耳定律，电热）

注意区分：
  额定功率：P额 = U额 × I额（铭牌上的值）
  实际功率：P实 = U实 × I实（实际工作中的值）
  只有 U实 = U额 时，P实 才等于 P额
```

### 常见变形

**变形1：开关通断问题**

- 情境：电路中有开关，不同通断状态对应不同电路结构
- 解题方法：每个开关状态画一张等效电路图，分别分析
- 关键点：开关断开 = 该支路无电流；开关闭合可能短路某个元件
- 步骤：列出所有开关状态组合 → 逐一画等效图 → 分别列方程
- 易错点：开关闭合后可能将某个电阻短路，该电阻不再起作用

**变形2：滑动变阻器问题**

- 情境：电路中有滑动变阻器，滑片移动改变阻值
- 分析方法：找到滑片移动时接入电路的阻值如何变化
- 串联时：R滑增大 → 总电阻增大 → 总电流减小 → 各元件电压重新分配
- 并联时：R滑增大 → 该支路电流减小 → 总电流减小 → 干路电压变化（有内阻时）
- 极值分析：滑片在最左端和最右端分别对应什么阻值？
- 易错点：滑动变阻器接入电路的阻值取决于接线柱的选择，不是滑片位置

**变形3：多用电器组合问题**

- 情境：多个用电器串联或并联，分析功率分配
- 串联：P₁/P₂ = R₁/R₂（电阻大的用电器实际功率大）
- 并联：P₁/P₂ = R₂/R₁（电阻小的用电器实际功率大）
- 关键判断：用电器正常工作的条件是实际电压等于额定电压
- 功率比较技巧：先判断串并联，再根据功率与电阻的关系比较
- 易错点：额定功率和实际功率不能混淆——"220V 60W"的灯泡在110V下不是30W

### 常见错误类型

| 错误类型 | 表现 | 纠正方法 |
|---------|------|---------|
| 开关状态变化后未重新分析 | 用断开状态的电路算闭合状态的量 | 每个开关状态画一张等效图 |
| 混淆额定功率和实际功率 | 认为灯泡在任何电压下功率都是额定值 | P实 = (U实/U额)² × P额 |
| 串并联判断错误 | 把并联看成了串联（或反过来） | 从电源正极沿电流走一遍，确认路径 |
| 短路识别遗漏 | 闭合开关后某元件被短路但没发现 | 电流走最短路径——有零电阻旁路则元件被短路 |

### 追问序列

1. "你画等效电路图了吗？开关在这个状态下，电流从正极出发怎么走？"
2. "这些用电器是串联还是并联？你怎么判断的？"
3. "滑动变阻器滑片向右移，接入电路的阻值变大了还是变小了？"
4. "串联电路中功率和电阻是什么关系？并联呢？"
5. "这个灯泡的额定功率是60W，但实际电压只有110V，实际功率还是60W吗？"
6. "开关闭合后，有没有哪个元件被短路了？被短路的元件还有电流通过吗？"

## 六、能量守恒模型

### 适用条件判断

- [ ] 题目涉及不同形式能量之间的转化（动能↔势能、电能↔热能、机械能↔内能等）
- [ ] 需要分析能量的来源和去向
- [ ] 系统总能量守恒（输入能量 = 输出能量 + 损失能量）

**判断要点：** 当题目关注的是"能量从哪里来、到哪里去"而非具体的力或电路细节时，优先使用能量守恒模型。能量守恒模型是最高层次的模型，可以与其他模型组合使用——先用力学模型或电路模型分析具体过程，再用能量守恒模型验证结果。

### 标准建模流程

```text
第1步：列出所有能量形式
  - 初态有哪些能量？（E动能、E势能、E电能……）
  - 末态有哪些能量？
  - 有没有能量损失？（摩擦生热、电阻发热等）

第2步：画出能量转化路径图
  - 用箭头表示能量转化方向
  - 标注每种能量的数值（已知量）或符号（未知量）
  - 例：重力势能 → 动能 → 内能（摩擦损失）

第3步：列能量守恒方程
  - E初 = E末 + E损失
  - 或：E输入 = E输出 + E损失
  - 或：ΔE₁ + ΔE₂ + ... = 0（所有能量变化之和为零）

第4步：代入求解检验
  - 检查等式两边量纲是否一致
  - 检查能量损失项是否遗漏
  - 检查效率是否合理（0 < η ≤ 1）
```

### 列式模板

**机械能守恒（无摩擦）：**

```text
E初 = E末
½mv₀² + mgh₀ = ½mv² + mgh
（动能 + 重力势能 = 常数）
```

**机械能不守恒（有摩擦）：**

```text
E初 = E末 + E损失
½mv₀² + mgh₀ = ½mv² + mgh + f·s
（初态机械能 = 末态机械能 + 摩擦力做的功）
```

**电能转化：**

```text
W电 = W有用 + W损失
W电 = UI t（总电能）
W有用 = P有用 · t（有用能量）
W损失 = I²R t（焦耳热损失）
η = W有用 / W电 × 100%（效率）
```

**热力学（初中简化）：**

```text
Q吸 = cmΔt（吸热公式）
Q放 = mq 或 Q放 = Vq（燃料放热）
Q吸 = Q放（热平衡条件）
```

### 常见变形

**变形1：自由落体（重力势能→动能）**

- 情境：物体从高处自由下落（忽略空气阻力）
- 能量转化：重力势能 → 动能
- 守恒方程：mgh = ½mv²（下落高度 h，末速度 v）
- 速度公式：v = √(2gh)
- 易错点：h 是下落的竖直高度，不是路程

**变形2：摩擦做功（机械能→内能）**

- 情境：物体在粗糙水平面上滑动减速
- 能量转化：动能 → 内能（摩擦生热）
- 守恒方程：½mv² = f·s（初动能全部转化为内能）
- 摩擦力做功：W摩擦 = f·s = μmg·s
- 易错点：摩擦力做功的 s 是物体实际滑行的路程（不是位移），如果路程和位移相同则不影响

**变形3：电热器（电能→内能）**

- 情境：电热水器、电炉等纯电阻用电器
- 能量转化：电能 → 内能
- 守恒方程：W电 = Q热（纯电阻电路中电能全部转化为内能）
- W电 = UIt = I²Rt = U²t/R = Pt
- 效率问题：η = Q有用/W电（如加热水的效率，不是所有热量都被水吸收）
- 易错点：纯电阻电路 W电 = Q热；非纯电阻电路 W电 > Q热

**变形4：电动机（电能→机械能+内能）**

- 情境：电动机等非纯电阻用电器
- 能量转化：电能 → 机械能 + 内能
- 守恒方程：W电 = W机械 + Q热
- W电 = UIt，Q热 = I²Rt，W机械 = W电 - Q热
- 效率：η = W机械/W电 × 100%
- 关键区分：电动机的总功率 P = UI，发热功率 P热 = I²R，机械功率 P机械 = P - P热
- 易错点：电动机不能用欧姆定律 U = IR 计算（非纯电阻电路），UI ≠ I²R

### 常见错误类型

| 错误类型 | 表现 | 纠正方法 |
|---------|------|---------|
| 遗漏能量损失 | 自由落体算了动能变化但忘了摩擦 | 列出所有能量转化路径，逐一检查 |
| 电能等于电热 | 电动机问题中 W电 = Q热 | 只有纯电阻电路 W电 = Q热；非纯电阻 W电 > Q热 |
| 高度取错 | 用斜面长度代替竖直高度 | 重力势能 mgh 中的 h 是竖直高度 |
| 效率公式用错 | η = Q热/W电（电热器）套用到电动机 | 电热器效率是 Q有用/Q总；电动机效率是 W机械/W电 |

### 追问序列

1. "这道题涉及几种形式的能量？初态有哪些？末态有哪些？"
2. "能量从哪种形式转化为了哪种形式？你能画出转化路径图吗？"
3. "有没有能量损失？比如摩擦生热、电阻发热？"
4. "你列的能量守恒方程中，左边的能量和右边的能量一一对应了吗？有没有遗漏？"
5. "这是纯电阻电路还是非纯电阻电路？欧姆定律在这里能用吗？"
6. "你算出来的效率合理吗？效率应该在什么范围？"

## 七、模型识别决策树

当学生面对一道物理题不知从何入手时，以下决策树可以帮助快速定位应该使用的模型。决策树以题目中的关键词和核心物理过程为线索进行匹配。

### 完整决策树

```text
第一步：判断题目所属领域
│
├── 题目涉及力与运动
│   │
│   ├── 关键词出现"匀速""静止""平衡""保持不动"
│   │   └──→ 力学平衡模型
│   │       追加确认：物体合力是否为零？
│   │
│   ├── 关键词出现"加速""减速""匀变速""自由落体""上抛""刹车""追及"
│   │   └──→ 匀变速直线运动模型
│   │       追加确认：加速度是否恒定？
│   │
│   └── 关键词出现"碰撞""撞击""弹开""粘在一起""爆炸""反冲"
│       └──→ 碰撞动量模型
│           追加确认：是否涉及两个物体的相互作用？
│
├── 题目涉及电路
│   │
│   ├── 关键词出现"电路""电流""电压""电阻""欧姆""串并联""开关""变阻器"
│   │   └──→ 电路分析模型
│   │       追加确认：是否需要求电压电流功率？
│   │
│   └── 关键词出现"电热器""电动机""电能转化""效率""发热"
│       └──→ 能量守恒模型（电能转化路径）
│           追加确认：关注点在能量转化还是电路参数？
│
├── 题目涉及能量转化
│   │
│   ├── 关键词出现"能量""转化""守恒""动能""势能""内能""效率"
│   │   └──→ 能量守恒模型
│   │       追加确认：能量从哪里来、到哪里去？
│   │
│   └── 关键词出现"摩擦""做功""热"
│       └──→ 能量守恒模型（机械能→内能路径）
│
└── 无法直接归类
    │
    ├── 题目描述了多个过程
    │   └──→ 跨模型综合题策略（见第八节）
    │
    └── 确实无法归类
        └──→ 退回识别现象步骤，引导学生重新读题画图
```

### 关键词速查表

| 关键词 | 对应模型 | 判断理由 |
|-------|---------|---------|
| 匀速、静止、平衡、保持不动 | 力学平衡模型 | 合力为零的条件 |
| 加速、减速、匀变速、加速度 | 匀变速直线运动模型 | 加速度恒定 |
| 自由落体、竖直上抛、下落 | 匀变速直线运动模型 | v₀=0或a=-g的特例 |
| 刹车、停止、制动 | 匀变速直线运动模型 | 匀减速至v=0 |
| 追及、相遇、同向运动 | 匀变速直线运动模型 | 两物体运动学关系 |
| 碰撞、撞击、弹开、粘合 | 碰撞动量模型 | 两物体相互作用 |
| 爆炸、反冲、分裂 | 碰撞动量模型 | 内力作用下的分裂 |
| 电路、电流、电压、电阻 | 电路分析模型 | 欧姆定律与串并联 |
| 开关、通断、闭合、断开 | 电路分析模型 | 电路结构变化 |
| 滑动变阻器、滑片、阻值变化 | 电路分析模型 | 动态电路分析 |
| 串联、并联、分压、分流 | 电路分析模型 | 电路连接方式 |
| 能量、转化、守恒、效率 | 能量守恒模型 | 能量形式变化 |
| 动能、势能、机械能 | 能量守恒模型 | 机械能守恒或转化 |
| 摩擦、做功、发热、内能 | 能量守恒模型 | 机械能→内能 |
| 电热器、电动机、电热 | 能量守恒模型 | 电能→其他形式能 |

### 模型识别的追问模板

当学生无法判断使用哪个模型时，按以下顺序追问：

1. **领域判断：** "这道题主要是在讲力的问题、电的问题、还是能量的问题？"
2. **状态判断：** "物体是静止的还是运动的？运动状态有没有变化？"
3. **过程判断：** "有没有什么物理过程在发生？比如碰撞、能量转化、电路状态变化？"
4. **条件匹配：** "你说的这个现象，和哪个模型的适用条件最吻合？"
5. **验证确认：** "你选了这个模型，它的条件都满足吗？检查一下。"

## 八、跨模型综合题建模策略

综合题是物理考试中的压轴题，其本质是将多个物理过程组合在一起，每个过程对应一个物理模型。解决综合题的关键不是找一个万能模型，而是把复杂问题拆解为多个简单问题。

### 核心策略：分过程建模

```text
第1步：识别过程数量
  - 仔细读题，数一数题目描述了几个物理过程
  - 每个过程有独立的物理规律
  - 关键词："然后""之后""接着""此时"——这些词通常标记过程切换

第2步：逐过程建模
  - 对每个过程独立建模
  - 每个过程选择合适的模型
  - 每个过程独立列方程

第3步：找衔接点
  - 前一个过程的末状态 = 后一个过程的初状态
  - 最常见的衔接量：速度（运动过程之间）、温度（热学过程之间）
  - 衔接点是连接各过程的桥梁

第4步：联立求解
  - 用衔接条件将各过程的方程连接
  - 整体求解或逐步代入
```

### 常见综合题类型

**类型1：力学综合——斜面+水平面**

```text
题目特征：物体从斜面上滑下，进入水平面后继续运动

过程1：斜面上的运动
  → 模型：匀变速运动模型（或力学平衡模型，若匀速下滑）
  → 方程：沿斜面方向的运动学方程
  → 末状态：斜面底部的速度 v₁

过程2：水平面上的运动
  → 模型：匀减速运动模型（有摩擦）或匀速运动模型（无摩擦/力平衡）
  → 初状态：初速度 = v₁（衔接点！）
  → 方程：水平方向运动学方程

衔接点：斜面底部速度 v₁
  过程1的末速度 = 过程2的初速度
```

**类型2：力电综合——电路+力学**

```text
题目特征：电动机提升重物（电流做功转化为机械能）

过程1：电路分析
  → 模型：电路分析模型
  → 方程：求电动机的电流和电压

过程2：能量转化
  → 模型：能量守恒模型
  → 方程：W电 = W机械 + Q热

衔接点：电功率 P = UI
  电路分析得到的 P = 电动机总功率输入
  能量守恒中的 W电 = P × t
```

**类型3：力学+能量综合**

```text
题目特征：物体受力运动，同时涉及能量转化

建模策略：
  方案A：先用力学模型求力和加速度，再用运动学求速度，最后用能量验证
  方案B：直接用能量守恒模型求速度（更简洁）

选择建议：
  - 如果只需要求速度/高度 → 能量守恒模型更简洁
  - 如果需要求力/加速度 → 必须用力学模型
  - 如果需要验证 → 两种方法互相验证
```

**类型4：多过程运动综合**

```text
题目特征：物体经历多段运动（如：自由落体→碰撞→反弹→上升）

过程1：自由下落
  → 模型：匀变速运动模型
  → 末状态：落地速度 v₁

过程2：碰撞
  → 模型：碰撞动量模型
  → 初状态：碰前速度 v₁
  → 末状态：碰后速度 v₂

过程3：反弹上升
  → 模型：匀变速运动模型（竖直上抛）
  → 初状态：初速度 v₂

衔接点：v₁（过程1末→过程2初），v₂（过程2末→过程3初）
```

### 综合题建模追问序列

1. "这道题描述了几个物理过程？你能用序号标出来吗？"
2. "每个过程用什么模型？分别列一下。"
3. "前一个过程结束时的状态，和后一个过程开始时的状态，有什么联系？"
4. "衔接点上的物理量是什么？是速度？温度？还是别的？"
5. "你能把所有过程的方程列在一起，用衔接条件连接起来吗？"
6. "如果只看第一个过程，你能解出来吗？试试看，先解第一个过程。"

### 综合题常见错误

| 错误 | 表现 | 纠正方法 |
|------|------|---------|
| 不分过程 | 多段运动用同一个方程 | 标出过程切换点，每段独立建模 |
| 衔接条件错误 | 过程1末速度≠过程2初速度 | 明确写出衔接量的等式关系 |
| 模型混用 | 同一过程用了两个矛盾的模型 | 每个过程只选一个主要模型 |
| 漏掉中间过程 | 碰撞前后直接列运动方程 | 列出所有过程，检查有无遗漏 |
