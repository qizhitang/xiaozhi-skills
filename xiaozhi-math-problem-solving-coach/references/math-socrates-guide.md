# 数学苏格拉底五问链学段适配指南

> 本文档为 `xiaozhi-math-problem-solving-coach` 的配套参考资源，提供数学苏格拉底五问链（模块B）在各学段的适配策略、追问话术库与年龄认知负荷调控方法。

---

## 一、五问链框架总览

```
数学苏格拉底五问链（基于 Paul & Elder 批判性思维模型）

第1问  清晰达意 — "你能用自己的话说出来吗？"
  目标：检验是否真正理解，而非仅看懂步骤

第2问  假设检验 — "改变条件后，解法还成立吗？"
  目标：测试方法的适用边界，防止"只会这道题"

第3问  证据链 — "最关键的一步为什么一定成立？"
  目标：确认逻辑严密性，找到推理跳跃处

第4问  视角切换 — "能用另一种方法解吗？"
  目标：培养一题多解，加深概念理解

第5问  个性化沉淀 — "这个思路适用于哪一类题？"
  目标：抽象为可迁移的方法模式

学术出处：Paul, R. & Elder, L. (2006). Critical Thinking:
  Tools for Taking Charge of Your Learning and Your Life.
```

---

## 二、学段适配总原则

### 认知负荷分级

```
小学段（4-6年级，约10-12岁）：
  认知特点：
    → 抽象思维刚开始发展，依赖具体形象
    → 逻辑推理链条短（2-3步）
    → 容易被"大问题"吓住，需要拆成小步
  适配策略：
    → 只使用第1问+简化版第2问（共2问）
    → 每问只追问1次，不连续追问
    → 多用生活比喻替代抽象概念
    → 用"如果...会怎样"替代正式的条件变更

初中段（7-9年级，约13-16岁）：
  认知特点：
    → 抽象思维快速发展，能处理3-5步推理
    → 对"为什么"有强烈好奇心
    → 容易把"看懂了"等同于"真懂了"
  适配策略：
    → 使用第1-3问（3问版），第4问视情况使用
    → 可以连续追问2次
    → 引入逻辑连接词（"因此""所以""那么"）
    → 开始训练"假设检验"思维

高中段（10-12年级，约17-19岁）：
  认知特点：
    → 抽象思维成熟，能处理长推理链
    → 需要严密性和完整性
    → 容易停留在"方法记忆"而非"原理理解"
  适配策略：
    → 完整使用5问
    → 可以连续追问3次
    → 要求完整的逻辑链条表述
    → 强调"边界条件""充要条件""反例验证"
```

## 三、小学段适配详解（2问版）

### 第1问：清晰达意（小学版）

**原版：** "你能用自己的话把这道题的解题过程说出来吗？不用公式，就用普通语言。"

**小学版改编：**

```
改编话术：
  "好，你算出了答案——
   现在闭上眼睛，想象你在给一个不会数学的小朋友解释：
   你是怎么一步一步算出来的？
   用生活中的话来说，不要用公式。"

为什么改编：
  → "小朋友"比"自己的话"更具体可操作
  → "闭上眼睛想象"降低紧张感
  → "生活中的话"比"不用公式"更通俗易懂
```

**小学段追问话术库：**

```
情况①：学生只说数字步骤（"我先乘了3，再减了2"）
  → 追问："你为什么先乘3呢？
           题目里有哪个线索让你决定先乘？"

情况②：学生说不出来（"我就是这样算的"）
  → 追问："没关系，那我们来想一下——
           如果不先算这一步，会发生什么？"

情况③：学生用比喻解释（"就像买3个苹果，每个5块，一共15块"）
  → 肯定："这个比喻很好！你已经真懂了。
           你发现了吗——'每个5块'就是题目里的条件，
           '3个'就是你先乘的3。"

注意：小学段追问只追问1次，不连续追问
```

### 第2问：假设检验（小学简化版）

**原版：** "如果把已知条件[X]改成[Y]，你的解法还能用吗？"

**小学简化版：**

```
改编话术：
  "我们做个小实验——
   如果题目里的[数字/条件]换一个，
   比如[具体的新值]，
   你觉得用同样的方法还能算出来吗？
   不用真的算，就猜猜——会变吗？"

为什么改编：
  → "小实验"比"假设检验"更有趣
  → 给具体的新值（不要让学生自己想替代值）
  → "猜猜就行"降低压力，不需要真的算
```

**小学段第2追问话术库：**

```
情况①：学生说"会变"（正确直觉）
  → "对了！为什么你觉得会变？
     是因为[原条件]在这个方法里起了什么作用？"

情况②：学生说"不会变"（可能误判）
  → "我们来试试看——
     如果[新值]，第一步变成了[结果]，
     跟原来一样吗？不一样说明什么？"

情况③：学生说"不知道"
  → "没关系，我们来试一个最简单的变化——
     把[数字]改成0，你的第一步变成了什么？
     这能帮我们看出这个数字的重要性。"

注意：小学段只做1次条件变更，不做多次
```

## 四、初中段适配详解（3-4问版）

### 第1问：清晰达意（初中版）

**原版话术（可直接使用）：**

```
"你能用自己的话把这道题的解题过程说出来吗？
 不用公式，就用普通语言。"
```

**初中段追问话术库：**

```
情况①：学生说出步骤但缺少因果连接（"我先设了方程，然后解出来了"）
  → "你设方程是根据题目哪个条件？为什么设这个方程而不是别的？"

情况②：学生说出步骤但跳跃了关键一步
  → "你从第二步跳到了第四步——中间发生了什么？
     那一步你是怎么想出来的？"

情况③：学生完整表述了因果链条
  → "很好！你说得很清楚。
     这里有个值得注意的地方：你用了[方法/定理]，
     为什么这道题适合用这个方法？
     什么样的题不适合？"
```

### 第2问：假设检验（初中版）

**原版话术：**

```
"如果把已知条件[X]改成[Y]，你的解法还能用吗？"
```

**初中段适配：**

```
引导方式：
  → 让学生自己提出一个条件变更（而非教练直接给）
  → 话术："现在我们来测试你的方法——
           你能不能想一个办法，改变题目中的一个条件，
           让你的解法不能直接用？"

追问层次：
  第1层：学生提出变更 → "很好，为什么这个变更会让方法失效？"
  第2层：学生说明原因 → "那么，如果方法失效了，
                         有没有修改后的版本还能用？"

条件变更类型指引（初中适用）：
  ① 数值变更：把具体数字换成极端值（0, 1, 负数）
  ② 条件增减：去掉一个条件或增加一个条件
  ③ 类型变更：把等腰三角形变成普通三角形
```

### 第3问：证据链（初中版）

**原版话术：**

```
"你求到这个答案，中间最关键的一步是什么？
 为什么这一步一定成立？"
```

**初中段追问话术库：**

```
情况①：学生找出了关键步骤但不能解释"为什么成立"
  → "你说[步骤]是最关键的——那这一步依赖什么？
     是哪个定理/性质保证了它成立？"

情况②：学生说"因为公式就是这样"
  → "公式的成立条件是什么？
     这道题满足这些条件吗？如果不满足会怎样？"

情况③：学生找不出"最关键"的一步
  → "我们来倒推——如果去掉最后一步，前面的结果还能得到吗？
     哪一步是'如果没了它，后面全断了'的那步？"

初中段重点训练：
  → 区分"充分条件"和"必要条件"
  → 不是所有步骤都是关键步骤——找"最不可替代"的那步
```

### 第4问：视角切换（初中选择性启用）

**启用条件：**

```
仅在以下情况启用第4问：
  ① 学生在第1-3问中表现良好（没有卡壳超过2次）
  ② 题目本身有多种解法（如几何题既可代数法也可纯几何法）
  ③ 学生表现出好奇心和探索意愿

不启用的情况：
  ① 学生在第1-3问中已经很吃力
  ② 题目只有一种合理解法
  ③ 时间紧迫（考前模式）
```

**初中版第4问话术：**

```
"这道题，你用了[方法A]来解。
 如果不用[方法A]，还有别的路可以到达同样的答案吗？
 比如，能不能用[提示一个方向]来试试？"

提示方向（按题型）：
  代数题 → "能不能画图看看？"
  几何题 → "能不能用代数方法来算？"
  方程题 → "能不能不设方程，直接推理？"
  应用题 → "能不能用列表法或枚举法？"
```

## 五、高中段适配详解（完整5问版）

### 第1问：清晰达意（高中版）

```
话术同原版，但追问更深：

追问方向：
  → 要求学生明确每一步的因果逻辑
  → 如果学生使用了"显然"或"易得"：
    "你说'显然'——但考试中'显然'不能作为论证步骤。
     这一步'显然'背后的严格依据是什么？"
```

### 第2问：假设检验（高中版）

```
引导方式：
  → 要求学生自行设计条件变更并预判结果
  → 话术："请你设计一个条件变更，使得：
           (a) 原方法仍然可用但答案不同
           (b) 原方法完全失效
           分别给出例子。"

高中段重点训练：
  → 边界条件分析（充要条件/必要条件）
  → 反例构造能力
  → 方法的适用域与失效域
```

### 第3问：证据链（高中版）

```
追问方向：
  → 要求完整逻辑链条，不允许任何跳跃
  → "你从A推导到B——中间的桥梁是什么？
     给出每一步的严格论证。"

高中段重点训练：
  → 区分"必要但不充分"的步骤
  → 找到"证明中最脆弱的一环"
  → 充要条件的判断
```

### 第4问：视角切换（高中版）

```
引导方式：
  → 不提示方向，让学生自己找替代方法
  → 话术："这道题除了[方法A]，你能找到至少一种完全不同的解法吗？"

高中段要求：
  → 替代方法必须是实质性不同（不是同一方法的变体）
  → 能比较两种方法的优劣（适用范围/计算量/简洁度）
  → 能判断哪种方法更"本质"地理解了问题
```

### 第5问：个性化沉淀（高中版）

```
话术：
  "这道题的解题思路，适用于哪一类题？
   你能给它起一个名字或者描述一个'识别特征'吗？
   反过来——哪类题不适合这个思路？
   你能举一个反例吗？"

高中段要求：
  → 沉淀结果必须包含：
    ① 适用题型识别特征
    ② 不适用题型的反例
    ③ 方法名称或标签
  → 结果记录入数学错误DNA（作为"方法模式"条目）
```

## 六、各学段操作规范速查

| 规范项 | 小学段 | 初中段 | 高中段 |
|-------|--------|--------|--------|
| 使用问数 | 2问（第1+简化第2） | 3-4问 | 5问 |
| 每问追问次数 | ≤1次 | ≤2次 | ≤3次 |
| 条件变更方式 | 教练提供具体变更值 | 学生自提+教练辅助 | 学生自行设计并分析 |
| 比喻使用 | 大量使用生活比喻 | 少量使用，过渡到抽象 | 仅用于概念引入，迅速回归抽象 |
| 逻辑词要求 | because（简单因果） | therefore, so that（推理链） | 充要条件, 反例验证, 边界分析 |
| 沉淀深度 | "记住这个方法" | "这个方法什么时候用" | "适用域+失效域+反例" |
| 时间控制 | 5分钟以内 | 8-10分钟 | 10-15分钟 |

## 七、常见问题与处理策略

### 学生说"我懂了"但实际没懂

```
诊断方法：立即启动第1问
  "你说你懂了——那用自己的话说一遍解题过程？"

判断标准：
  → 只能复述步骤 → 停留在"看懂"层面 → 继续追问
  → 能说出因果逻辑 → 真正理解 → 可进入第2问
  → 能说出"为什么这么做" → 深层理解 → 可进入第3问+
```

### 学生在追问中反复卡壳

```
小学段处理：
  → 卡壳1次：给提示词（不给答案）
  → 卡壳2次：把问题拆成更小的问题
  → 卡壳3次：换一种说法重新提问
  → 卡壳超过3次：暂停追问，回到做题本身

初中段处理：
  → 卡壳1次：给方向性提示
  → 卡壳2次：给类比案例
  → 卡壳3次：把问题拆成两步
  → 卡壳超过3次：暂停追问，记录卡壳点到错误基因档案

高中段处理：
  → 卡壳1次：给思路框架（不给具体步骤）
  → 卡壳2次：提示需要的定理/性质名称
  → 卡壳3次：提供部分推理链，让学生补全
  → 卡壳超过3次：暂停追问，要求学生回去复习相关概念
```

### 学生回答"不知道"的策略

```
层级1（完全不知道）：
  → 降低问题难度：从"为什么"降到"是什么"
  → 例：从"为什么这一步成立" → "这一步用了什么公式？"

层级2（有模糊想法但说不清）：
  → 给选择法："你是在想A还是B？"
  → 给半句提示："这一步之所以成立，是因为..."

层级3（方向对了但表述不完整）：
  → "你说了一半——后半句是什么？"
  → "你的意思我理解了，但更准确的说法应该是..."
```

## 八、五问链与四步拍照法的衔接

```
五问链的触发时机：
  ① 学生完成四步拍照法Step 2（CLAW提问）后说"我懂了"
  ② 学生完成模板①（概念理解类）后
  ③ 学生完成模板③（思路卡壳类）后找到思路

衔接话术：
  "你说你懂了——我来帮你确认一下是不是真懂了。
   不用紧张，就是几个小问题。"

五问链与四步拍照法的分工：
  四步拍照法：解决"怎么做"（找到解题路径）
  五问链：解决"为什么这么做"（深层理解与迁移）

时间分配：
  → 四步拍照法为主（占70%时间）
  → 五问链为辅（占30%时间，仅在"说懂了"后启用）
  → 不在同一道题上同时进行四步法和五问链
```
