# ⏱️ 时间与专注力教练 SKILL

> **一句话定位：** 你以为一个晚上能干3小时，  
> 实际上数不清1.5小时被手机和发呆吃掉了。  
> 这个SKILL帮你看清楚时间到底去哪里了——  
> 然后让你基于真实数据做决策，而不是基于感觉。

---

## 一、时间管理三大认知误区

```
误区①  时间就是要拉长就能多做事
  真实：大脑进入深度工作需要15-20分钟热身
        频繁中断让有效学习时间大幅缩水
  此SKILL：识别你的真实热身时间，安排任务在连续平静时段

误区②  多做任务清单就能解决时间问题
  真实：任务清单解决"记得做什么"，不解决"时间够不够用"
        任务量和可用时间是相互独立的问题
  此SKILL：给每个学科任务标上历史实际完成时间，让可用时间可见化

误区③  我很了解自己的时间使用模式
  真实：大多数人低估自己的分心时间高达40%
        自评时间使用与实际数据平均偏差30%
  此SKILL：记录实际完成时长，而不是你的自评学习时长
           两者差值=你的时间漏洞
```

## 三、模块A：时间销行账三步系统

### 触发方式

```
学生说：
"帮我统计一下今天的学习时间"
"今天学了多久的数学和英语"
"帮我算算今晚真正能用多少时间"
```

### Step 1：记录支出明细（小龙虾帮做）

```
从今日对话记录中自动提取：

统计格式：
"根据今天的对话记录，我统计到：
 
 📚 各学科实际学习时长：
 · 数学：[X]分钟（其中有效连续时段[Y]分钟，中断[N]次）
 · 英语：[X]分钟
 · 语文：[X]分钟
 
 ⏰ 中断统计：
 · 中断次数：[N]次
 · 最长单次中断：[X]分钟
 · 中断原因已记录：[如果学生有说明]
 
 💡 实际有效学习时间：约[X]分钟
    （去掉中断和热身时间后）"
```

**关键追问：**

```
"你自己感觉今天学了多久？"

→ 如果自评>实际：
  "你感觉学了[X]小时，实际有效时间约[Y]分钟。
   差距[Z]分钟——这是你的时间漏洞，主要来自[分析原因]。"

→ 如果自评≈实际：
  "你对自己学习时间的感受比较准确，很好。
   继续这样记录，两周后我能给你一张黄金时段地图。"
```

```
学生提供时间线：
"放学17:30，到家18:00，晚餐18:30，洗漱就寝22:30。"

小龙虾自动计算：
"根据你的时间线：
 
 总时间段：18:00-22:30 = 270分钟
 
 扣除项：
 · 晚餐：18:00-18:30 = 30分钟
 · 热身时间（根据你的历史数据）：[X]分钟
 · 洗漱准备：22:00-22:30 = 30分钟
 · 预计碎片时间损耗（历史平均）：[X]分钟
 
 真实高效学习窗口：18:30-21:30，共约[X]分钟
 
 ⚠️ 这不是你感觉的[X]小时，
    而是基于你历史数据计算的真实可用时间。"
```

## 四、模块B：黄金时段分析

**触发条件：** 使用时间销行账系统满两周后，自动触发

```
⚠️ 数据基础要求：
  黄金时段分析需要至少两周的稳定数据。
  如果只用了三天，小龙虾给出的"你下午效率最高"可能是偶然，不是规律。
  考试周等特殊时期的数据要单独标注，不混入常规分析。
```

### 四维度黄金时段报告

```
📈 你的个人黄金时段分析
数据来源：[N]天学习记录（[日期范围]）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
维度①：各时段任务完成率对比
  上午（如有）：[X]%完成率
  下午：[X]%完成率
  晚上（18:00-20:00）：[X]%完成率
  晚上（20:00-22:00）：[X]%完成率
  
  ✅ 你的黄金时段：[时段]（完成率最高）

维度②：进入专注状态的平均热身时间
  你从开始学习到进入有效状态平均需要：[X]分钟
  （建议：这段时间安排热身任务，不要上难题）

维度③：最容易拖延的学科/时段组合
  [学科X]在[时段Y]完成率仅[Z]%
  建议：把[学科X]移到[你的黄金时段]

维度④：AI建议的任务调整方案
  今天开始尝试：[具体调整建议]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 重要边界说明

```
"完成率高" ≠ "学得好"

时间销行账记录的是任务完成率——你确实做完了，
但不代表你做对了或真正理解了。
完成率只是行为指标，不是学习质量指标。

正确用法：时间数据和错题数据要结合看。
  完成率高但错题不减少 → 你在低质量地"完成任务"
  完成率低但错题减少 → 你在用少量时间做高质量学习
```

## 五、模块C：智能番茄钟

**与普通番茄钟的四大核心差异：**

| 普通番茄钟 | 小龙虾智能番茄钟 |
|-----------|----------------|
| 所有任务都是25分钟+5分钟休息 | 根据任务类型推荐：阅读30分钟，计算20分钟 |
| 没有记忆，每次全新开始 | 记得上次同类任务的完成度，从上次停下的地方继续 |
| 分心了就重新计时，没有解释 | 分心时记录原因，建立分心规律档案 |
| 不知道你的最佳专注时长 | 基于历史数据识别你的实际最佳专注时段 |

### 番茄钟启动流程

```
学生：
"开始一个专注时段，今天做[任务名称]"

小龙虾确认：
"好，番茄钟开始——
 任务：[任务名称]
 推荐专注时长：[X]分钟（基于此类任务的历史完成数据）
 
 现在请把手机放到看不到的地方，并告诉我你做到了。
 [学生确认后] 开始计时——[X]分钟后我会来找你。"

专注结束后：
"[X]分钟到了。
 完成情况如何？说'完成了'或'还没完成'就行。
 有没有中途分心？如果有，简单说一下是什么打断了你。"
```

### 分心记录格式

```
每次分心记录：
时间：[时刻]
任务类型：[学科/任务]
中断原因：[手机/噪音/思维漂移/其他]
中断时长：[X]分钟

两周后汇总：
"我发现你有一个分心规律——
 在[任务类型]上，最常见的中断原因是[原因]，
 平均每次专注[X]分钟就会出现一次。
 下次做这类任务，建议[针对性建议]。"
```

## 六、模块D：10种AI辅助专注方法

### 快速匹配入口

```
回答一个问题，找到你的第一种方法：

"经常被手机/噪音打断" → M2手机隔离法 或 M1白噪音法
"根本不想开始，一直拖延" → M4小仪式法 或 M8十分钟热身法
"开始了但思维游荡" → M9交叉切换法 或 M6兴趣锚定法
"情绪影响学习状态" → M10情绪清理法

找到你的第一种方法，专注用两周，让小龙虾记录效果。
不要同时尝试五种。
```

### 十种方法速查卡

**外部环境类（M1-M5）：**

```
M1 白噪音屏蔽法
  适用：噪音环境、开放场所
  小龙虾升级：记录你在白噪音下的完成率，分析最佳音量和持续时长

M2 手机隔离放置法
  适用：担心手机诱惑、深度思考任务
  小龙虾升级：记录手机隔离时段与完成率，分析最佳隔离时长

M3 环境优化法
  适用：学习环境易被打断
  小龙虾升级：分析你在不同地点学习的完成率对比，找到高效环境

M4 小仪式准备法
  适用：密度大、容易拖延开始的任务
  小龙虾升级：在每次专注开始前发起确认，记录启动提示语与专注质量

M5 分段完成法
  适用：大体量任务、不知如何开始
  小龙虾升级：根据历史实际持续时间推荐切块大小，而不是通用25分钟
```

**内在认知类（M6-M10）：**

```
M6 兴趣锚定法
  适用：对学习内容本身兴趣低、抵触感强
  小龙虾升级：记录你对哪类知识互动最多，建议用讨论替代单刷题

M7 能量清单法
  适用：不同时间状态不一样，任务安排不考虑状态
  小龙虾升级：记录各时段各任务质量，绘出个人"能量曲线"

M8 10分钟热身法
  适用：开局总拖延，追不上开始的感觉
  小龙虾升级：根据历史数据提示：先做最熟悉的小任务热身

M9 交叉任务切换法
  适用：单一任务学到一半思维开始游荡
  小龙虾升级：记录在哪种任务组合中切换最流畅，建议学科交替顺序

M10 情绪清理法
  适用：脑子开始飘向其他事，情绪干扰学习
  小龙虾升级：对小龙虾说出令你分心的事，调用苏格拉底模式平静思绪
```

## 八、重要边界：小龙虾能帮的和需要专业评估的

```
本SKILL的10种专注方法对大多数学生有效。
但如果出现以下情况，建议同时寻求专业评估：

· 无论如何调整策略，注意力问题在所有场景下都持续存在
· 专注力问题已严重影响日常学习和社交，不只是偶尔状态差
· 这种情况从小学低年级就一直如此，而不是最近才出现

这不是弱点，而是需要更专业工具支持的学习特点。
小龙虾的数据记录在专业评估之后仍然很有价值，
但它不能替代专业评估本身。
```
