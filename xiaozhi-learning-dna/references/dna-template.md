# 🧬 学习DNA档案模板（首次建档引导用）

> 此文档用于引导学生完成第一次DNA建档对话。  
> 不要直接发给学生——用自然对话的方式逐步引出这些信息。

---

## 基础信息模板

```
【我的学习DNA - 第一版】
建档日期：____年____月____日

■ 基础信息
  年级：
  目标院校/方向（如有）：
  本学期最重要的考试：

■ 学科强弱（1=非常薄弱 → 5=比较自信）
  语文：__ / 具体薄弱点：
  数学：__ / 具体薄弱点：
  英语：__ / 具体薄弱点：
  其他科目：

■ 学习风格
  偏好解释方式：□ 类比举例  □ 分步骤拆解  □ 先看结论再看过程  □ 图示辅助
  对话节奏偏好：□ 快问快答  □ 深度慢聊  □ 先让我自己想想再提示
  容易分心吗：□ 还好  □ 有点容易分心  □ 很容易分心

■ 近期状态
  这周/这段时间最头疼的事：
  有没有近期要备考的内容：

■ 第一个目标
  我希望用小龙虾帮我完成的第一件事：
```

## 首次建档对话示范脚本

**开场（不要用"您好"开头，太正式）：**
```
"嘿，终于见到你了！在我们开始之前，我想多了解你一点——
 放心，不是考试，就是聊聊天。你几年级？"
```

**引出薄弱点（不要问"哪里不好"，太宽泛）：**
```
"如果明天有一门课的考试，你最不希望是哪门？
 然后在那门课里，哪个部分是你一看就头大的那个？"
```

**了解学习风格（用选择题降低门槛）：**
```
"我帮你解释一个概念的时候，你更喜欢哪种方式——
 A：先举个生活里的例子，然后再讲定义？
 B：直接一步一步拆开讲？
 C：让你先自己试一试，遇到卡的地方再帮你？"
```

**确认目标（要具体，不要"我想提高成绩"）：**
```
"说一个你最近最想实现的学习目标——
 不用太大，就是'下次月考数学过80分'这种具体的就好。"
```

## DNA更新触发词表

当学生说以下内容时，**自动触发对应的DNA更新**：

| 学生说 | DNA更新操作 |
|-------|------------|
| "我懂了" / "明白了" | 标记该知识点为"近期掌握"，加入复习队列 |
| "还是不太明白" / "有点迷" | 加深薄弱标记，安排追问 |
| "我升XX年级了" | 更新年级信息，重新评估目标 |
| "我考试考了XX分" | 更新学科状态，调整强弱评级 |
| "我发现我最近XX进步了" | 记录成就，标记为里程碑 |
| "我这周要考XX" | 设置临时高优先级，触发IM提醒SKILL |
| "这道题我做了三次都错" | 识别为固定错误模式，推送错题本SKILL |

## 30天成长里程碑参考

| 里程碑 | 参考判断标准 |
|-------|------------|
| 🌱 DNA种子萌芽（第1-7天） | 完成首次建档，有3+条学习记录 |
| 🌿 DNA初步成型（第8-20天） | 错误模式开始出现规律，能给出个性化建议 |
| 🌳 DNA全面激活（第21-30天） | 能在未被提醒的情况下主动关联历史记录 |
| 🦞 专属导师模式（30天+） | 学生感受到"它真的懂我了"的时刻 |

---

> ⚠️ **注意事项：**  
> DNA的价值 = 信息的真实程度 × 积累的深度  
> 帮助学生理解：诚实地反馈"懂了/没懂"，是让DNA准确的前提。  
> 骗了AI，受损失的是自己的学习DNA。
