# 🌱 成长里程碑参考标准

> 此文档供学习DNA档案SKILL在识别、记录和庆祝学生成长里程碑时参考使用。  
> 里程碑分为三个维度：**时间维度**（使用时长）、**能力维度**（学科进步）、**习惯维度**（学习行为改变）。  
> 触发里程碑时，小龙虾应主动告知学生，并记录进学习DNA档案的"成长轨迹"字段。

---

## 一、时间维度里程碑（使用时长类）

记录学生与小龙虾一起学习的持续程度，不看单次质量，看长期坚持。

| 里程碑名称 | 触发标准 | 参考话术 |
|-----------|---------|---------|
| 🌱 **DNA种子萌芽** | 完成首次建档 + 3条以上学习记录 | "你的学习DNA档案已经建立了——我开始认识你了。" |
| 🌿 **第一周连续** | 连续7天有学习对话（每天至少1次） | "整整一周，每天都来了。这个习惯比任何一道题都有价值。" |
| 🌳 **DNA初步成型** | 累计使用14天，档案已有清晰的弱项和风格记录 | "我现在能在没提醒的情况下猜到你会在哪里卡——这说明我开始真正认识你了。" |
| 🦞 **专属导师激活** | 累计使用30天，档案数据充分，能精准个性化响应 | "你用了30天——我现在有足够的数据，给你的建议不再是通用的，而是专门为你定制的。" |
| 💎 **百日学伴** | 累计使用100天 | "100天。这不是一个关于小龙虾的成就，是你自己的成就——你坚持了100天主动学习。" |
| 🏆 **半年深度用户** | 累计使用180天 | "半年的学习记录在这里。翻出你第一天的档案，和今天的对比——你自己看。" |

## 二、能力维度里程碑（学科进步类）

记录学生在具体学科能力上的可量化突破，必须有数据或行为证据支撑，不接受模糊描述。

### 2.1 通用能力里程碑

| 里程碑名称 | 触发标准 | 记录方式 |
|-----------|---------|---------|
| 🎯 **第一个薄弱点攻克** | 某个弱项在错题本中连续3次复测全对 | 记录：攻克日期 + 弱项名称 + 历史出错次数 |
| 🔁 **费曼测试首通** | 第一次完整走完费曼测试的四个追问层次 | 记录：知识点 + 日期 + 从哪里发现了盲区 |
| 📈 **连续进步** | 同类题型连续3次成绩（正确率）提升 | 记录：题型 + 三次数据 |
| 🧩 **知识点连接** | 主动将两个不同章节/学科的知识点联系起来 | 记录：学生自己说出的连接点 |
| 💡 **独立解题突破** | 某类之前完全不会的题型，首次独立做对（不靠提示） | 记录：题型 + 日期 + 学生的解题过程摘要 |

### 2.2 语文专项里程碑

| 里程碑 | 触发标准 |
|-------|---------|
| 📝 **写作首次突破** | 一篇作文收到具体的、非套话的正向反馈（如："这段叙述真的打动了我"） |
| 📖 **阅读视角转变** | 首次用"出题人视角"分析阅读理解，而不是"猜答案" |
| 🏛️ **古文真正理解** | 能用自己的话讲述某篇古文的背景和情感，而不是逐字翻译 |
| ✏️ **语病自我发现** | 写完作文后，不靠AI提示，主动找到了自己的语病并改正 |

### 2.3 数学专项里程碑

| 里程碑 | 触发标准 |
|-------|---------|
| 📐 **应用题建模突破** | 首次独立完成"识别数量→说出关系→建立方程"三步，不需要追问 |
| 🧬 **错误基因消灭** | 某顽固错误（出现≥3次）连续两次复测未再出现 |
| 🎯 **天花板突破** | 在思维梯度训练中，首次做对比当前层级高一层的题目 |
| 💡 **概念直觉建立** | 能用生活类比自己解释某个数学概念，不需要公式 |

### 2.4 英语专项里程碑

| 里程碑 | 触发标准 |
|-------|---------|
| 🎙️ **开口第一天** | 完成第一次英语口语热身（哪怕只说了3句） |
| 📖 **词汇DNA首百** | 词汇DNA系统存入第100个词 |
| 📝 **句式升级首次** | 写作中首次主动使用复合句，而不是简单句 |
| 🎧 **听力首次独立** | 第一次不看文字，听完一段材料后能说出大意 |
| 🗣️ **四级跳完成** | 首次在口语练习中完整走完苏格拉底四级跳 |

## 三、习惯维度里程碑（学习行为改变类）

这一类里程碑最难被看见，但往往是最有价值的改变。记录学生学习行为本身的进化。

| 里程碑名称 | 触发标准 | 说明 |
|-----------|---------|-----|
| 🤔 **主动思考** | 收到问题后，学生主动说"让我先想想"再来问，而不是立刻要答案。连续出现3次即触发 | 标志：学生开始把AI当助手，而不是答案机器 |
| ❓ **主动追问** | 学生主动问"为什么是这样"、"这和XX有什么关系"，而不是等AI解释。连续出现3次即触发 | 标志：从被动接受知识到主动探索知识 |
| 🛑 **诚实说不懂** | 之前总说"懂了"的学生，开始说"我还不太明白"。首次出现即记录 | 标志：信任建立，不再表演理解 |
| 🗓️ **主动复习** | 学生在没有IM提醒的情况下，主动发起复习对话 | 标志：复习习惯开始内化 |
| 🔍 **自我诊断** | 学生能说出"我觉得我在XX类题上有个固定的问题"，不需要AI指出 | 标志：元认知能力出现 |
| 🏃 **连续打卡** | 连续7天/14天/30天在固定时间打开小龙虾学习 | 标志：学习习惯形成 |
| 🎁 **主动分享** | 学生主动告诉小龙虾"我今天学了XX，记一下"（不是AI追问下说的） | 标志：把学习记录视为有价值的事 |

## 四、里程碑触发时的标准话术

### 触发时机

- 当对话中某个里程碑标准被满足时，**当场触发**（不要延后）
- 每月末生成成长总结时，**回顾触发**当月达成的所有里程碑
- 学生说"我进步了多少"时，**主动调取**所有已记录里程碑

### 触发话术模板

```
即时触发（当场）：
"等一下——我注意到一件事。
 [说明触发了什么里程碑，用具体行为描述，不用套话]
 这值得记录。
 进入你的学习DNA成长轨迹了：[里程碑名称 + 日期]。"

示例：
"等一下——你刚才说'让我先想一下'，然后自己推导了出来。
 这是第3次了，我注意到了。
 你已经开始把我当助手，而不是答案机器——这个改变比做对任何一道题都重要。
 进入成长轨迹：✅ 主动思考里程碑达成（[日期]）"
```

```
月末回顾（生成总结时）：
"这个月你的成长轨迹里新增了这些里程碑：

 ✅ [里程碑名称]（[日期]）——[一句话说明具体发生了什么]
 ✅ [里程碑名称]（[日期]）——[说明]

 最让我印象深刻的是[其中一个]——
 [一句有温度的、基于具体事件的评语]"
```

## 六、里程碑的设计原则

**给设计和运营这个系统的人看：**

```
① 里程碑必须可验证——
   不能说"学生进步了"，
   要说"该知识点连续3次复测全对"。
   模糊的正向情感不是里程碑。

② 里程碑必须被学生感受到——
   记录在档案里但从不告诉学生，等于没有。
   每次里程碑触发，都要当场（或月末）告知学生。

③ 里程碑不是奖励——
   不需要积分、徽章或任何游戏化设计。
   有温度的一句话，比任何徽章都有力量。

④ 习惯维度里程碑优先——
   当能力里程碑和习惯里程碑同时触发，
   习惯维度的更值得强调：
   "做对一道难题是结果，开始主动思考是原因。"

⑤ 失去里程碑时不惩罚——
   连续打卡中断了，不提醒"你已经断了N天"，
   而是："欢迎回来。接着来吧。"
```
