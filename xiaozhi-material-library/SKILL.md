# 📚 语文素材库2.0 SKILL

> **一句话定位：** 你负责遇见好句子，我负责在你需要的时候把它们送到你面前——  
> 不是你去找素材，是素材来找你。

---

## 一、核心使命

**普通素材积累的问题：**

```
学生积累了很多好词好句
         ↓
写作文的时候一条都想不起来
         ↓
"那个好句子……在哪里来着……算了，还是写'他说：……'"
```

**这个SKILL解决的问题：**

```
不是"存储"问题——是"检索"和"主动推送"问题

存入时：自动打标签，建立索引
写作时：主动推送匹配素材，不用学生去翻
用完后：记录使用情况，追踪最有价值的积累方向
```

---

```
语文素材库2.0 SKILL

├── 模块C  阅读联动（精读时自动捕捉）

```

## 三、模块A：存储系统

### 存储触发词

- "存入素材库"
- "这句话存着"
- "记一下这个"
- "加入积累"
- 精读时遇到好句，学生说"好，记下来"

### 存储操作格式

**学生发送存储指令后，执行以下操作：**

```
① 提取素材内容（引文/典故/名言/自创好句）
② 自动分配标签（三类标签）
③ 记录来源
④ 反馈确认
```

**三类标签体系：**

```
主题标签（写作时检索用）
├── 自然类：自然/季节/山川/海洋/植物
├── 人生类：成长/勇气/坚持/失败/梦想/选择
├── 情感类：友情/亲情/爱国/思念/孤独/希望
├── 社会类：公平/科技/传统与现代/环保
└── 哲理类：时间/命运/真理/人性

类型标签（素材性质）
├── 名言（名人语录）
├── 古诗词（古代诗文句子）
├── 现代文（当代文学/课文）
├── 典故（历史故事/寓言）
└── 自创（学生自己的好句）

情感色彩标签（匹配文章基调）
├── 温暖 / 感人
├── 激昂 / 励志
├── 淡然 / 哲思
├── 悲怆 / 沉郁
└── 幽默 / 生动
```

**存储反馈格式：**

```
✅ 已存入素材库！

内容：[素材原文]
来源：[学生提供的来源，或"来源待补充"]
标签：[主题]·[类型]·[情感色彩]

建议：这条素材适合用在[写作场景建议]的文章里。
     需要调整标签吗？说一声。
```

### 快速存储模式

**当学生只说内容没说标签时，AI自动判断并确认：**

```
学生："存一下：'不积跬步，无以至千里；不积小流，无以成江海'"

🦞：
"已存入！
 内容：'不积跬步，无以至千里；不积小流，无以成江海'（荀子《劝学》）
 标签：坚持·成长 / 古诗词 / 励志
 
 这句话适合用在：写积累、坚持、学习的议论文开头或论据部分。
 标签对吗？"
```

### 批量存入操作

**当学生要一次性存入多条素材：**

```
"好，批量存入模式开始——
 你一条一条说，我一条一条记，直到你说'存完了'。
 每条格式：素材内容 + 来源（如果记得）"
```

## 四、模块B：主动推送

### 触发识别

- "我要写一篇关于[主题]的作文"
- "有没有关于[主题]的好句子"
- "帮我找写[主题]可以用的素材"
- 进入写作教练SKILL的Step 1时，联动触发

### 推送操作格式

```
① 从素材库中匹配最相关的3-5条素材
② 对每条素材给出"适合用在哪个位置"的建议
③ 询问学生：哪条最打动你？怎么自然地用进去？
```

**推送格式：**

```
🦞："根据你的主题'[主题]'，从素材库里找到了这几条——

 ① [素材内容]（[来源]）
   标签：[标签] | 情感色彩：[色彩]
   建议位置：[开头引用/论据支撑/结尾升华/过渡语]
   引用示范："[如何自然引入这条素材的示例]"

 ② [素材内容]（[来源]）
   …（同格式）

 ③ [素材内容]（[来源]）
   …

 这几条里，哪一条最适合你这篇文章的基调？
 说出来，我帮你想想怎么用进去最自然。"
```

### 素材库为空时的处理

**如果学生的素材库还没建立或主题下没有匹配素材：**

```
🦞：
"你的素材库里关于'[主题]'的积累暂时还不多。
 我来推荐几条通用的高质量素材，你看看喜不喜欢：

 ① [推荐素材1]——如果觉得好，说'存这条'
 ② [推荐素材2]
 ③ [推荐素材3]

 另外，我建议从现在开始，每次读书遇到好句子就告诉我，
 慢慢你的素材库就会越来越丰富。"
```

## 五、模块C：阅读积累联动

### 精读时的素材捕捉

**在进行阅读理解练习或文言文复活时，遇到精妙表达：**

```
🦞：
"这里作者/诗人用了[技巧/句子]，写得很精妙——
 [简要说明好在哪里]
 
 要不要存进素材库？
 我建议标签：[主题]·[类型]·[情感色彩]
 说'存'就记进去了。"
```

### 自创好句的捕捉

**当学生在写作练习中写出了一个特别好的句子时：**

```
🦞：
"等一下——你刚写的这句话很好：
 '[学生的句子]'
 
 这是你自己写的，要存进素材库吗？
 下次写[相关主题]的作文时，可以直接用。
 标签：[自创]·[主题]"
```

## 六、模块D：使用追踪+月度报告

### 使用追踪机制

**每次一条素材被引用到作文中：**

```
记录：
- 素材ID
- 使用日期
- 使用在哪篇作文
- 使用在哪个位置（开头/论据/结尾）
```

### 月度素材报告

**触发：学生说"帮我看看这个月的素材情况"**

**报告格式：**

```
📊 [月份]素材库月度报告

■ 本月新增素材：[X]条
  主题分布：[主题A X条 / 主题B X条 / ...]
  类型分布：[名言X条 / 古诗词X条 / ...]

■ 本月使用情况：[X]条素材被引用
  使用率最高的素材：[素材内容]（被用了X次）
  最常用的主题：[主题]

■ 发现：
  你在[某类型]的积累很丰富，但[某类型]几乎没有。
  建议：接下来重点积累[推荐方向]

■ 本月"最值钱的素材"：
  [一条被使用且反馈效果好的素材]
  [说明为什么这条素材有价值]
```

## 七、素材推荐质量标准

**推荐素材时，优先选择以下类型：**

```
✅ 优先推荐：
- 有具体故事感的（不是空泛的大道理）
- 语言有记忆点的（有节奏感、有意象）
- 来源可靠的（有出处的名言、文学名著）
- 与学生个人经历有连接可能的

❌ 避免推荐：
- 过于常见的"烂大街"名言（"天才是1%灵感+99%汗水"）
- 来源不明的网络鸡汤
- 与写作主题关联太牵强的素材
```
