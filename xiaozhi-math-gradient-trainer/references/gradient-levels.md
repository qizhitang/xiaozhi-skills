# 初中数学各知识模块梯度分层参考

> 每个知识点按5层出题示例，帮助小龙虾准确判断学生的天花板位置。

---

## 一次函数

| 层级 | 题型描述 | 示例题 |
|-----|---------|--------|
| 第1层（课本） | 直接求函数值或截距 | 已知y=2x-3，求x=2时y的值 |
| 第2层（应用） | 根据条件确定函数解析式 | 已知一次函数过(1,3)和(-1,-1)，求解析式 |
| 第3层（联动） | 函数图像与几何结合 | 一次函数y=kx+b的图像与x轴围成三角形面积为8，求k与b |
| 第4层（变形） | 参数范围或条件判断 | 若一次函数y=(m-1)x+(m²-1)不过第二象限，求m的范围 |
| 第5层（竞赛） | 函数与数论或不等式综合 | 两个一次函数在整数点上的交点问题 |

---

## 二元一次方程组

| 层级 | 题型描述 | 示例题 |
|-----|---------|--------|
| 第1层（课本） | 直接解方程组 | 解方程组：2x+y=5，x-y=1 |
| 第2层（应用） | 根据实际问题列方程组 | 鸡兔同笼：共30只，80条腿，各几只 |
| 第3层（联动） | 带参数或结果有条件限制 | 方程组有整数解，求参数范围 |
| 第4层（变形） | 无解/无穷多解的条件讨论 | 当m取什么值时，方程组有唯一解/无解/无穷多解 |
| 第5层（竞赛） | 方程组与函数/几何综合 | 三直线两两相交的交点坐标与图形面积 |

---

## 三角形全等与相似

| 层级 | 题型描述 | 示例题 |
|-----|---------|--------|
| 第1层（课本） | 直接应用判定定理 | 已知三边或两角一边，判断是否全等 |
| 第2层（应用） | 在综合图形中找全等对 | 平行线切割图形中证明两三角形全等 |
| 第3层（联动） | 全等推出等线段/等角，再证其他 | 用全等证明中线平分，再证面积关系 |
| 第4层（变形） | 添加辅助线构造全等 | 题目中直接条件不足，需要构造辅助全等 |
| 第5层（竞赛） | 多次应用全等或全等与相似混合 | 需要三次以上推导，中间结果互相依赖 |

## 圆

| 层级 | 题型描述 | 示例题 |
|-----|---------|--------|
| 第1层（课本） | 圆心角=2×圆周角，直接应用 | 已知圆心角120°，求同弧圆周角 |
| 第2层（应用） | 切线长定理和切割线定理 | 从圆外一点作切线，求切线长 |
| 第3层（联动） | 圆与三角形全等/相似综合 | 圆内接四边形与相似三角形综合题 |
| 第4层（变形） | 动点问题或参数条件下的圆 | 动点在圆上移动，求面积最大值 |
| 第5层（竞赛） | 根轴、共轭圆等高级概念 | 三圆公共弦交于一点的证明 |

## 二次函数

| 层级 | 题型描述 | 示例题 |
|-----|---------|--------|
| 第1层（课本） | 顶点坐标公式直接求 | y=x²-4x+3，求顶点坐标 |
| 第2层（应用） | 已知条件确定解析式 | 抛物线过三个已知点，求解析式 |
| 第3层（联动） | 二次函数与不等式结合 | 求使二次函数值大于0的x的范围 |
| 第4层（变形） | 含参数的二次函数讨论 | 当参数a变化时，顶点轨迹是什么 |
| 第5层（竞赛） | 二次函数与几何综合 | 抛物线上三点构成的三角形面积最值 |

## 概率与统计

| 层级 | 题型描述 | 示例题 |
|-----|---------|--------|
| 第1层（课本） | 古典概型直接计算 | 从1-10中随机取一个数，是偶数的概率 |
| 第2层（应用） | 列举所有等可能结果 | 两个骰子同时掷，点数之和为7的概率 |
| 第3层（联动） | 频率与概率的关系 | 根据抽样频率估计总体分布 |
| 第4层（变形） | 含条件概率的问题 | 已知A发生的条件下B发生的概率（初中版） |
| 第5层（竞赛） | 组合计数与概率 | 需要用排列组合公式计算的概率 |

---

## 梯度层跨越的思维特征

| 跨越 | 需要掌握的思维方式 |
|-----|----------------|
| 第1→2层 | 从直接代入到"如何设元"和"识别条件" |
| 第2→3层 | 从单知识点到"联系两个知识点" |
| 第3→4层 | 从顺向推导到"逆向推导"和"讨论分类" |
| 第4→5层 | 从已知方法到"创造性地发现新路径" |
