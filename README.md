## 📖 方法论依据

本 SKILL 包的设计基于《龙虾高效学习法》第一章的核心方法论：

**记忆飞轮五转：**
1. 记录首次错误 → **错题本 SKILL**
2. 主动提醒复习 → **IM智能提醒 SKILL**
3. 精准出题检验 → **错题本 SKILL + IM提醒 SKILL**
4. 发现错误规律 → **错题本 SKILL + 学习DNA档案 SKILL**
5. 生成弱项报告 → **每周复盘 SKILL**

**费曼学习法：**
- 用"教会小龙虾"验证真实理解 → **费曼测试 SKILL**

**AI学习铁律：**
- 用AI辅助思考，不用AI替代思考 → 所有 SKILL 的设计原则

**语文专项能力链：**
- 写作表达升级 → **语文写作教练 SKILL**
- 阅读拆题拿分 → **阅读理解拆解师 SKILL**
- 古诗文理解与背诵 → **文言文复活计划 SKILL**
- 长期素材沉淀 → **语文素材库2.0 SKILL**
- 语言表达纠偏 → **语病追踪档案 SKILL**
