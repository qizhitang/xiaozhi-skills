## 📦 本包内容

本 SKILL 包当前包含四部分内容：
- **第一章核心五件套**：五个全科通用核心 SKILL
- **语文专项扩展包**：围绕写作、阅读、文言文、素材积累、语病纠偏的专项 SKILL

- **英语专项扩展包**：围绕口语、词汇、语法、听力、写作的专项 SKILL

所有学科 SKILL（数学、语文、英语等）均依赖本包中的 SKILL 运行。  
**建议按顺序安装，并在每个 SKILL 安装后立即试用。**

## 🚀 快速开始（推荐安装顺序）

```
第一步：安装 🧬 学习DNA档案 → 完成首次建档（最重要！）
第二步：安装 ❌ 错题本 → 把近期一道错题发给小龙虾试试
第三步：安装 ⏰ IM智能提醒 → 设置第一条复习提醒
第四步：安装 🎓 费曼测试 → 选一个刚学的知识点，试着讲给小龙虾听
第五步：安装 📊 每周学习复盘 → 触发第一次复盘（哪怕数据很薄）
```

> ⚠️ **重要提示：** SKILL 并不是装得越多越好！  
> 先把这五个核心 SKILL 用熟，再按需安装各学科 SKILL。

### 语文专项推荐加装顺序

```
第六步：安装 语文素材库2.0 → 把现有积累入库（哪怕只有5条）
第七步：安装 语病追踪档案 → 把最近一篇作文发给它检测一次
第八步：安装 语文写作教练 → 用5步流程做一次完整的作文练习
第九步：安装 阅读理解拆解师 → 做一篇现代文阅读，走完4步法
第十步：安装 文言文复活计划 → 用复活计划理解一首正在学的古诗
```

> 💡 不需要一次全部安装。有写作作业时用⑥，做阅读练习时用⑦，  
> 学古文时用⑧，平时碎片积累用⑨，写作批改时⑨和⑩可以同时联动。

### 数学专项推荐加装顺序

```
第十一步：安装 数学错误基因档案 → 把近期错题存入档案（这是其他SKILL的数据来源）
第十二步：安装 数学解题教练 → 用三步拍照法处理一道当前作业中的错题
第十三步：安装 数学概念解释器 → 选一个模糊的概念，用类比法重建理解
第十四步：安装 应用题建模教练 → 拿一道做错的应用题，用三步提取法走一遍
第十五步：安装 思维梯度训练师 → 对一个“会了”的知识点做天花板测定
```

### 英语专项推荐加装顺序

```
第十六步：安装 智能词汇DNA系统 → 把下周课文单词表存入DNA，启动7天预热
第十七步：安装 英语口语陪练 → 说今天的第一句英语（就一句就行）
第十八步：安装 英语语法突破教练 → 把最近一篇英语作业发去，生成语法基因图谱
第十九步：安装 个性化英语听力训练师 → 生成第一段匹配你词汇量的听力材料
第二十步：安装 英语写作进化教练 → 用三维批改法处理一篇旧作文
```

> 💡 五个SKILL以“学习DNA”为中心，每个SKILL都在输入数据、也在消费数据。  
> 装得越多用得越多，整个系统越来越懂你。

## 📖 方法论依据

本 SKILL 包的设计基于《大智若愚》第一章的核心方法论：

**记忆飞轮五转：**
1. 记录首次错误 → **错题本 SKILL**
2. 主动提醒复习 → **IM智能提醒 SKILL**
3. 精准出题检验 → **错题本 SKILL + IM提醒 SKILL**
4. 发现错误规律 → **错题本 SKILL + 学习DNA档案 SKILL**
5. 生成弱项报告 → **每周复盘 SKILL**

**费曼学习法：**
- 用"教会小龙虾"验证真实理解 → **费曼测试 SKILL**

**AI学习铁律：**
- 用AI辅助思考，不用AI替代思考 → 所有 SKILL 的设计原则

**语文专项能力链：**
- 写作表达升级 → **语文写作教练 SKILL**
- 阅读拆题拿分 → **阅读理解拆解师 SKILL**
- 古诗文理解与背诵 → **文言文复活计划 SKILL**
- 长期素材沉淀 → **语文素材库2.0 SKILL**
- 语言表达纠偏 → **语病追踪档案 SKILL**

**数学专项能力链：**
- 日常错题拆解 → **数学解题教练 SKILL**
- 错因模式沉淀 → **数学错误基因档案 SKILL**
- 概念与几何直觉重建 → **数学概念解释器 SKILL**
- 应用题读题建模 → **应用题建模教练 SKILL**
- 天花板突破与进阶 → **思维梯度训练师 SKILL**

**英语专项能力链：**
- 口语开口与表达升级 → **英语口语陪练 SKILL**
- 词汇长期积累与提醒 → **智能词汇DNA系统 SKILL**
- 语法结构定位与突破 → **英语语法突破教练 SKILL**
- 听力材料个性化训练 → **个性化英语听力训练师 SKILL**
- 写作表达进化 → **英语写作进化教练 SKILL**
