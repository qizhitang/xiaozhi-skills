# 🦞 龙虾高效学习法 · SKILL 包总览

> **龙虾高效学习法 SKILL 系列**  
> 作者：小智伴学 | 适用平台：OpenClaw / ClawHub / SkillHub | 当前版本：v1.1

传统的 AI 往往被当成“给答案的计算器”。  
而这套包含 27 个 SKILL 的生态系统，旨在将 AI 转化为“**追问思路的专属教练**”。

---

## 📦 包含模块

本技能库包含了基于《龙虾高效学习法》全书构建的 6 大学习与管理模块：

- ⚙️ **核心五件套**：建立基础学习档案，实现长效错题追踪与主动式提醒的通用底层系统。
- 🧠 **学习方法论包**：让独立工具协同运作，形成闭环的学习系统。
- 🚦 **自我管理与探索包**：从学习延展至时间管理、习惯养成与深度的跨学科兴趣探索。
- 🈶 **语文学科专项**：覆盖写作、阅读、古文、素材积累与语病纠偏的专项训练。
- ➗ **数学学科专项**：覆盖错误溯源、概念深潜、应用题建模与思维天花板突破。
- 🔤 **英语学科专项**：贯穿口语开口、长期词汇追踪、语法基因分析与听写综合训练。

---

## 🚀 快速开始（一分钟上手）

> ⚠️ **强烈建议：** 不要一次性安装所有 SKILL！请按照以下最小核心顺序进行体验。

1. **安装 [🧬 学习DNA档案](xiaozhi-learning-dna/)** → 完成首次建档（这是激活系统的大脑）
2. **安装 [❌ 错题本](xiaozhi-wrong-answer-notebook/)** → 拍一道你近期做错的题，体验“找根因”而非“找答案”
3. **安装 [⏰ IM智能提醒](xiaozhi-im-reminder/)** → 安排针对这道题的第一次自动复测提醒
4. **安装 [🎓 费曼测试](xiaozhi-feynman-test/)** → 试着给小龙虾讲一遍这个知识点
5. **安装 [📊 每周学习复盘](xiaozhi-weekly-review/)** → 等到周末，召唤一份属于你的数据化成长复盘

*(完整的长序列安装指南与加装路径，请参考 [🗺️ 安装指南与发布信息](docs/installation-guide.md))*

## 🧬 SKILL 核心协作流

在这个生态中，SKILL 不是孤立的，数据会在底层产生流转与共振。

```text
                    ┌──────────────────┐
                    │  🧬 学习DNA档案   │
                    │  （核心底层大脑）   │
                    └────────┬─────────┘
                             │ 为所有SKILL提供学生档案
          ┌──────────────────┼──────────────────┐
          ↓                  ↓                  ↓
  ┌───────────────┐  ┌───────────────┐  ┌───────────────┐
  │  ❌ 错题本    │  │ ⏰ IM智能提醒 │  │ 🎓 费曼测试   │
  └───────┬───────┘  └───────┬───────┘  └───────┬───────┘
          │                  │                  │
          └──────────────────┼──────────────────┘
                             ↓ 汇总所有数据
                    ┌──────────────────┐
                    │ 📊 每周学习复盘  │
                    │  （飞轮周启动器）   │
                    └──────────────────┘
```
