# 低阶→高阶词汇升级对照表

> 按词性和使用场景分类，供英语写作进化教练进行用词维度批改时参考。
> 追问时先问学生"你觉得这两个词有什么区别"，再给出解释。

---

## 一、形容词升级

### very + 形容词 → 精准形容词

| 低阶表达 | 升级替换 | 使用场景 |
|---------|---------|---------|
| very good | excellent / outstanding / remarkable | 正式评价 |
| very good | wonderful / fantastic / amazing | 口语/感叹 |
| very bad | terrible / dreadful / awful | 强烈负面 |
| very big | enormous / vast / massive / huge | 尺寸大 |
| very big | significant / major / substantial | 抽象重要 |
| very small | tiny / minute / microscopic | 尺寸小 |
| very small | minor / negligible / trivial | 抽象不重要 |
| very happy | delighted / thrilled / overjoyed | 非常高兴 |
| very sad | devastated / heartbroken / miserable | 非常难过 |
| very tired | exhausted / drained / worn out | 非常疲惫 |
| very beautiful | stunning / breathtaking / gorgeous | 非常漂亮 |
| very important | crucial / essential / vital / critical | 非常重要 |
| very difficult | challenging / demanding / complex | 非常困难 |
| very easy | effortless / straightforward / simple | 非常容易 |
| very interesting | fascinating / captivating / compelling | 非常有趣 |

## 二、动词升级

### say / think / make → 精准动词

| 低阶动词 | 升级替换 | 语义差别 |
|---------|---------|---------|
| said | explained | 解释说明 |
| said | argued | 有争论地说 |
| said | claimed | 声称（可能有争议） |
| said | insisted | 坚持说 |
| said | admitted | 承认 |
| said | announced | 正式宣布 |
| think | believe | 坚信 |
| think | consider | 仔细考虑 |
| think | argue | 有理由认为 |
| think | suggest | 提出建议 |
| make | create | 创造 |
| make | produce | 生产、制造 |
| make | generate | 产生（无形的） |
| make | cause | 导致（负面结果） |
| get | obtain | 获得（正式） |
| get | acquire | 获取、习得 |
| get | achieve | 实现（通过努力） |
| go | travel | 旅行 |
| go | proceed | 前进、继续 |
| go | navigate | 导航、穿越 |
| show | demonstrate | 展示、证明 |
| show | reveal | 揭示（出人意料） |
| show | indicate | 表明 |

## 三、副词和连词升级

### but / and / also → 精准连接词

| 低阶连词 | 升级替换 | 逻辑关系 |
|---------|---------|---------|
| but | however | 书面语转折 |
| but | nevertheless | 尽管如此 |
| but | on the contrary | 与之相反 |
| and | furthermore | 递进（更重要的是） |
| and | moreover | 递进（此外） |
| and | in addition | 补充 |
| also | additionally | 此外（书面） |
| also | as well as | 也，以及 |
| so | therefore | 因此（书面逻辑） |
| so | consequently | 因此（结果） |
| so | as a result | 结果是 |
| because | due to | 由于（后跟名词） |
| because | as a result of | 因为（后跟名词短语） |
| because | given that | 鉴于（正式） |

## 四、常见低阶习惯词替换指南

### "I think" 的替换

```
I think this is important.
→ I believe this is crucial. （更有把握的观点）
→ I would argue that... （提出论点）
→ In my opinion, ... （表达个人立场）
→ It seems to me that... （较谨慎的观点）
→ There is no doubt that... （强调确定性）
```

### 作文开头句替换

```
低阶开头：
"In today's society, ..." （老套）
"As we all know, ..." （泛泛）
"Nowadays, ..." （常见但可接受）

升级开头：
"[引用名言或有力陈述]..." （开门见山）
"Imagine..." （场景引入）
"Have you ever wondered...?" （问题引入）
"The debate over [话题] has never been more relevant than today." （议题引入）
```

### 结尾句替换

```
低阶结尾：
"In conclusion, I think..." （过于简单）
"So we can see that..." （不够有力）

升级结尾：
"In light of the above, it becomes clear that..." （归纳总结）
"[重申主题]—this is not merely [小事], but [大意义]." （升华意义）
"As [相关领域] continues to evolve, the importance of [主题] will only grow." （展望未来）
```

## 五、追问话术（发现低阶词时使用）

```
发现 "very good" 时：
"你用了 'very good'——
 这里你想表达的是非常出色、令人印象深刻的意思吗？
 如果是，英语里有个更精准的词：'outstanding'。
 感觉上有什么不同？"

发现 "think" 时：
"你说 'I think'——
 你是在表达一个猜测，还是一个有根据的判断？
 如果有根据，可以用 'believe' 或 'argue'——
 哪个更符合你的意思？"

发现重复词汇时：
"我注意到 '[词汇]' 在这篇文章里出现了[N]次——
 英语里避免重复是个好习惯。
 你能想到1-2个表达相近意思的不同词吗？"
```
