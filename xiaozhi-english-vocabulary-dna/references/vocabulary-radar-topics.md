# 词汇雷达话题模板

> 本文档为 `xiaozhi-english-vocabulary-dna` 的配套参考资源，提供新词汇雷达（模块C：课前7天预热）的话题分类模板、词汇分组策略与四阶段预热内容生成指南。

---

## 一、话题分类体系

### 分类原则

```
词汇雷达话题分类的三个维度：
  ① 教材匹配：按教材单元主题分组（最精准的课前预热来源）
  ② 考试高频：按中考/高考高频话题分组（备考优先）
  ③ 兴趣拓展：按学生兴趣档案分组（主动学习时使用）

优先级：教材匹配 > 考试高频 > 兴趣拓展
  → 学生说"下周学新课"时 → 使用教材匹配组
  → 学生说"帮我备考"时 → 使用考试高频组
  → 学生主动存词/兴趣探索时 → 使用兴趣拓展组
```

## 三、考试高频话题组

### 中考高频话题（5大类）

#### 1. 校园与学习

```
核心词汇层：
  基础层（必备）：class, subject, teacher, homework, exam, grade, school
  进阶层（常考）：schedule, curriculum, extracurricular, scholarship, assignment
  高阶层（理解）：academic, comprehensive, compulsory, optional, assessment

话题示例句（预热Day 7用）：
  "School life involves attending classes, completing homework,
   and preparing for exams."
  "Extracurricular activities like clubs and sports are also important."

搭配扩展（预热Day 14用）：
  academic performance / compulsory subject / curriculum design
  extracurricular activity / scholarship program / final exam
```

#### 2. 科技与环境

```
核心词汇层：
  基础层：technology, computer, internet, environment, pollution, protect
  进阶层：digital, sustainable, renewable, emission, ecosystem
  高阶层：innovation, breakthrough, carbon footprint, biodiversity

话题示例句：
  "Technology has changed how we learn and communicate."
  "Protecting the environment means reducing pollution and waste."

搭配扩展：
  digital technology / sustainable development / renewable energy
  environmental protection / carbon footprint / ecosystem balance
```

#### 3. 健康与生活方式

```
核心词汇层：
  基础层：health, exercise, diet, sleep, habit, lifestyle
  进阶层：nutrition, fitness, well-being, balanced, regular
  高阶层：immune system, cardiovascular, mental health, chronic

话题示例句：
  "A healthy lifestyle includes regular exercise and good sleep."
  "Balanced nutrition helps us stay energetic and focused."

搭配扩展：
  healthy lifestyle / balanced diet / regular exercise
  physical fitness / mental well-being / immune system
```

#### 4. 社会与文化

```
核心词汇层：
  基础层：culture, tradition, festival, custom, community, society
  进阶层：heritage, celebration, diversity, respect, belief
  高阶层：multicultural, cultural exchange, social cohesion

话题示例句：
  "Different cultures have different traditions and festivals."
  "Cultural diversity makes our community richer and more interesting."

搭配扩展：
  cultural heritage / traditional festival / social diversity
  multicultural society / cultural exchange / community spirit
```

#### 5. 个人成长与情感

```
核心词汇层：
  基础层：dream, goal, effort, challenge, confident, nervous, proud
  进阶层：ambition, motivation, perseverance, resilience, achievement
  高阶层：self-discipline, determination, fulfillment, self-awareness

话题示例句：
  "Setting goals helps us stay motivated and focused."
  "Overcoming challenges builds confidence and resilience."

搭配扩展：
  personal growth / life goal / self-confidence
  inner motivation / resilience / self-discipline
```

## 四、兴趣拓展话题组

### 动态话题库规则

```
兴趣拓展组不是固定的——它的内容来自：
  ① 学习DNA档案中的"兴趣DNA"字段
  ② 口语进化档案中的"话题偏好"字段
  ③ 学生对话中自发提到的新兴趣

当学生存入一个新词，但这个词不属于当前任何话题组时：
  → 自动创建新话题组
  → 以该词为种子，向上扩展同领域词汇
  → 标记来源为"兴趣拓展"
```

### 常见兴趣话题模板

#### 体育话题

```
种子词示例：basketball
向上扩展：
  基础：team, score, win, lose, player, coach, court, practice
  进阶：league, championship, strategy, defense, offense, dribble, shoot
  高阶：tournament, semifinal, playoff, momentum, underdog, crossover

预热内容模板（Day 7）：
  "下周我们聊 [basketball]！先认识这些词：
   team, score, player, coach, court, practice
   读三遍——今天只听发音，不记意思。"
```

#### 音乐话题

```
种子词示例：guitar
向上扩展：
  基础：song, rhythm, melody, perform, concert, band, singer, stage
  进阶：compose, lyrics, chorus, verse, harmony, acoustic, album
  高阶：composition, repertoire, improvisation, virtuosity, resonance
```

#### 游戏话题

```
种子词示例：Minecraft
向上扩展：
  基础：build, craft, explore, survive, create, block, world, resource
  进阶：strategy, multiplayer, adventure, quest, skill, upgrade, challenge
  高阶：simulation, procedural, sandbox, open-world, immersive
```

## 五、四阶段预热内容生成模板

### Day 7（第一次见面）生成模板

```
模板：
  "📡 新词预热 · 第1天（距上课还有7天）

   下周你会遇到这些词，我们先认识一下：

   [词汇1] /[音标]/ 读作：[中文近似音辅助]
   [词汇2] /[音标]/ 读作：[中文近似音辅助]
   ...

   今天任务：把这些词的发音读三遍——
   不需要记意思，只是让它们变得不陌生。"

音标辅助规则：
  → 使用 IPA 音标
  → 如果学生是初学者，附中文近似音辅助（仅Day 7使用）
  → 如 /θɪŋk/ → 读作类似 "丝-ing-k"（舌尖伸出版）
  → 中文辅助仅供参考，不替代正确音标
```

### Day 5（场景联想）生成模板

```
模板：
  "📡 新词预热 · 第2天

   给这几个词配上画面——

   [词汇1]：[场景描述]
     例：achievement → 想象你站在山顶，手里举着奖杯
   [词汇2]：[场景描述]
     例：perseverance → 想象你跑了很久，终于到达终点线
   ...

   想象这个场景，把词和画面连在一起。
   下次见到这个词，画面会先出现，意思就跟着来了。"

场景联想生成规则：
  → 每个词配一个具体可感知的画面（不是抽象定义）
  → 画面要与学生日常生活可关联
  → 不使用"记住定义是XXX"的方式，而是"想象XXX场景"
```

### Day 3（轻松使用）生成模板

```
模板：
  "📡 新词预热 · 第3天

   用这3个词，和我聊3-5句话——
   随便聊，不用都用上，用到哪个算哪个：
   [词汇1] / [词汇2] / [词汇3]"

对话引导（小龙虾侧）：
  → 开启简短英语对话，话题与词汇语义相关
  → 不强制学生必须用到所有词
  → 学生自然用到了 → 肯定："你用了[X]！这个词你已经会用了"
  → 学生没用到 → 温和提示："这里其实可以用[X]——试试？"
```

### 上课前（自信提醒）生成模板

```
模板：
  "📡 新词预热 · 最后提醒

   今天上课会遇到这些词——
   你已经见过它们[3]次了，不会陌生的。

   特别提示：[2-3个最重要的词] 可能是这节课的重点，
   注意听老师怎么用它们造句。

   加油！你已经准备好了。"
```

## 六、词汇分组策略

### 分组规则

```
每次预热最多包含 8-12 个新词：
  → 分为3组，难度递进
  → 第一组（4-5词）：最基础、最常用的词 → Day 7 必出
  → 第二组（3-4词）：进阶词，需要语境理解 → Day 5 重点
  → 第三组（2-3词）：高阶词或特殊搭配 → Day 3 深度使用

如果单元新词超过12个：
  → 分成两轮预热（Day 7-0 和 Day 7-3）
  → 第一轮：核心高频词（8个）
  → 第二轮：补充和搭配词（余下）
```

### 与词汇DNA五轮提醒的关系

```
词汇雷达预热 ≠ 词汇DNA五轮提醒

词汇雷达：课前7天的预热（"还没学就开始见"）
  → 目标：建立"见过"的印象，降低首次学习负担
  → 结束后：词汇进入正式学习 → 自动转入五轮提醒系统

词汇DNA五轮提醒：学完后的巩固（"学了之后不能忘"）
  → 目标：从短期记忆升级到长期记忆
  → 5轮完成后：进入季度复习模式

衔接流程：
  预热Day 7 → 预热Day 5 → 预热Day 3 → 上课学习 →
  → 五轮Day 1(24h) → 五轮Day 2(3d) → 五轮Day 3(7d) →
  → 五轮Day 4(14d) → 五轮Day 5(30d)
```
