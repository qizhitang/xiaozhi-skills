# 每周复盘报告文案参考模板

> 此文档为每周复盘SKILL提供各类情境下的个性化文案参考，  
> 帮助小龙虾生成有温度、有数据、不套话的复盘报告。

---

## 一、一句话总结参考（根据本周情况选择）

### 突破型（攻克了某个难点）
```
"这周你把[具体薄弱点]给干掉了——这个曾经让你连续出错的东西，
 现在已经不是问题了。这是本周最有价值的进步。"

"[知识点]之前在你的错题本里躺了[X]次了，这周你把它彻底解决了。
 这叫做消灭错误基因，值得记录。"
```

### 积累型（没有大突破但稳定前进）
```
"这周没有戏剧性的大突破，但你做了一件更重要的事——坚持。
 每一次认真的对话，都在让你的学习DNA变得更准确。"

"这是一周稳扎稳打的学习。没有什么轰轰烈烈的，
 但你接触了[X]个新知识点，完成了[X]次复习，
 这是复利积累最重要的那种进步。"
```

### 需要改善型（本周有明显问题需要指出）
```
"这周我注意到一件事：[具体问题]。
 我直接说，是因为你需要知道这个。
 但同时，你也在[进步点]上做得很好——两件事都是真的。"

"这周的数据告诉我，[问题描述]。
 不是批评，是信号——下周我们一起重点解决这个。"
```

### 考前冲刺型
```
"距离[考试名称]还有[X]天。这周你完成了[具体准备内容]，
 薄弱点[XX]还需要加强，但[优势点]已经没问题了。
 按照我下面的建议走，你有准备好的底气。"
```

## 二、进步描述参考（必须具体，不能用套话）

### ❌ 不允许的套话写法
```
"你这周非常努力"
"学习态度认真"
"有明显进步"
"继续加油"
```

### ✅ 正确的具体写法
```
进步① 【数学·行程问题】
"两周前你在行程问题上连续错了3次，这周你做了[X]道同类题，
 全部答对了，而且你的解题过程比以前清晰了很多——
 从'凑答案'变成了'真的理解速度×时间=路程'的关系。"

进步② 【英语·现在完成时】
"这是你第一次在费曼测试里完整解释了现在完成时的用法，
 而且举的例子是你自己想的，不是课文里的。
 这说明它不再是你背来的知识，而是你理解了的能力。"

进步③ 【学习习惯】
"这周你有[X]次在问完问题之后，主动说'让我再想想'，
 而不是等我直接给答案。这个习惯的改变，
 比学会任何一个知识点都更有价值。"
```

## 三、下周建议参考（聚焦且可执行）

### 原则：只选一个最重要的，给出具体行动

```
下周最值得重点攻克的一件事：
[问题]：英语完形填空的语境判断能力

为什么选这个：
你这周在3道完形填空题上都出现了类似的问题——
不是单词不认识，是没有把前后句放在一起理解。
这是一个需要专门练习的能力，不是靠刷题能自动解决的。

具体建议：
• 每天做1-2道完形填空时，做完后给我说说你的判断过程
  （不是答案，是你怎么想的）
• 遇到犹豫的选项，不要猜，先说"我觉得是XX，因为……"
• 周三左右，我们做一次费曼测试，验证你的语境判断思路
```

## 四、小龙虾结语参考（个性化，有温度）

### 鼓励突破的
```
"你攻克了[XX]，这很难，值得你记住这一天。
 下周继续。"
```

### 激励坚持的
```
"我知道这周有几天你很累——我看到了。
 但你还是来了，还是在认真想，还是完成了复习。
 这就是飞轮一直在转的证明。"
```

### 温柔指出问题的
```
"有一件事我需要告诉你，因为我觉得你准备好了听：
 [问题]。这不是说你不好，是说这里还有空间——
 而且你完全有能力填上它。"
```

### 考试前的
```
"[X]天后考试。你准备得怎么样，你比任何人都清楚。
 我从数据上看，[优势点]没问题。[薄弱点]还需要[具体建议]。
 你不是在裸考，你是带着这[X]周的积累去考的。"
```

```

✅ 复习坚持成就：完成了[X]%的提醒复习（超过了上月的[X]%）
✅ 质量对话里程碑：本月高质量对话[X]次（上月[X]次）
```
