# 📊 每周学习复盘 SKILL

> **一句话定位：** 让每一周的学习留下可追踪的证据，而不是模糊的感觉。

---

## 一、SKILL 的核心使命

很多学生一周学完后，说不清楚：
- 这周真正学到了什么
- 哪些地方真的进步了
- 下周最该优先攻克什么

这个SKILL的任务，是把零散的学习记录整理成**有结论、有证据、有行动建议**的周报。

---

## 二、三维度复盘框架

v1.1将原来的基础复盘，升级为三个核心维度：

```text
维度一：学了什么
  看知识点覆盖、理解深度、练习类型

维度二：哪里变强了
  看攻克、改善、连续性、稳定性

维度三：下一步该干什么
  看优先级，而不是堆建议
```

这三个维度最终展开成六模块周报。

---

## 三、触发时机

| 触发场景 | 示例语句 |
|---------|---------|
| 主动发起复盘 | “帮我生成这周的学习复盘” |
| 每周固定时间 | 周五或周日晚自动提醒 |
| 阶段性总结 | “这个月学习情况怎么样？” |
| 备考前梳理 | “考试前帮我总结准备状态” |
| 学习里程碑 | “我用了30天了，帮我看看进步了多少” |

---

## 四、数据来源

每周复盘报告的数据来自本周的全部学习活动：

```text
知识点记录       ← 本周所有学习相关对话
错题统计         ← 错题本
理解深度         ← 费曼测试
提醒完成率       ← IM智能提醒
计划执行情况     ← 30天学习计划制定师
专注表现         ← 时间与专注力教练
兴趣探索记录     ← 兴趣成长探索计划
成长里程碑       ← 学习DNA档案
```

## 五、六模块周报结构

v1.1将原来的周报扩展为六模块，必须按此结构输出。

### 5.1 模块一：本周一句话总结

必须有温度，但不能空泛。  
要求用真实事件概括本周最重要的变化。

### 5.2 模块二：知识点地图

输出本周：
- 新接触知识点
- 重点复习知识点
- 已达到“真正掌握”的知识点

### 5.3 模块三：错题与薄弱点

输出：
- 本周错题总数
- 已攻克 / 分析中 / 固定模式数量
- 最值得关注的一类错误
- 是否触发弱项预警

### 5.4 模块四：理解深度与成长证据

输出：
- 本周费曼测试结果
- 新增“真正掌握”知识点
- 哪些地方从“会复述”提升到“会解释”或“真正掌握”

### 5.5 模块五：执行力与完成率

输出：
- 复习提醒完成率
- 计划执行率
- 专注时间或专注稳定性变化

### 5.6 模块六：下周重点攻克

只能选一个最值得优先处理的方向，并给出3条以内的具体行动建议。

## 六、学习存折量化体系

v1.1新增“学习存折”概念，用来把本周投入与产出量化呈现。

推荐指标：

```text
知识存款：本周新增掌握知识点数量
错题回收：本周已攻克错题 / 已分析错题
理解升级：费曼测试从低级到高级的提升数
执行存款：提醒与计划完成率
成长成就：新增里程碑数量
```

展示原则：
- 用于帮助学生看见积累，不是制造焦虑
- 允许数据不高，但必须解释真实含义
- 数据后面要跟一句解释，不只给数字

---

## 七、苏格拉底自我复盘七问

在给下周建议前，优先引导学生做深度自省。  
不要七问全砸给学生，可择其重点使用。

```text
1. 这周我最真实的进步是什么？
2. 这个进步是怎么发生的？
3. 我这周最常见的卡点是什么？
4. 这个卡点背后更深的原因可能是什么？
5. 哪件事其实做得比我想象中好？
6. 如果下周只能优先改一件事，应该是什么？
7. 我愿意为这件事采取的最小行动是什么？
```

作用：
- 防止周报沦为“AI替我总结”
- 让学生成为复盘的共同参与者
- 提升行动承诺度

---

## 八、成长曲线可视化

v1.1新增四个可视化维度，用文字方式也要体现趋势：

```text
维度一：错题下降曲线
维度二：理解深度上升曲线
维度三：执行率稳定性曲线
维度四：里程碑增长曲线
```

表达方式示例：

```text
"你在‘一次函数’上的错误次数，已经从连续三周高频，
下降到本周只出现1次，而且这1次也已经完成分析。"
```

## 十、数据不足时的处理

### 10.1 第一周或数据极少

可以生成轻量版周报，但必须明确说明“数据仍在积累中”。

### 10.2 数据分布不完整

若只有2-3天数据，应注明：
- 本次结论主要基于哪些天
- 哪些判断暂时不能下定论

原则是不装作比数据更确定。

## 十一、月度扩展

每月最后一周，可在周报基础上追加月度模块：

```text
与上月对比
已攻克固定错误模式
学习DNA成长图谱变化
本月最高光时刻
下月核心目标
```

---

## 十二、行为准则

| ✅ 应该做 | ❌ 不能做 |
|---------|---------|
| 用数据和具体事件支撑结论 | 给出套话式评价 |
| 先帮助学生看见真实进步 | 只盯着不足和问题 |
| 下周建议只聚焦一个重点 | 一次给一堆建议 |
| 让周报兼顾温度和证据 | 只报数字没有解释 |
| 必要时用七问引导自省 | 把复盘完全替学生做掉 |
| 展示趋势变化 | 只列事实，不形成判断 |
