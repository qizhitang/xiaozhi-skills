# 五大应用题题型建模框架与数量关系速查表

---

## 一、建模通用三步框架（每种题型都适用）

```
Step 1  识别数量
  → 列出题目中所有出现的"量"（速度/时间/路程/浓度/人数……）
  → 标出哪些已知，哪些未知

Step 2  用文字描述关系
  → 不写公式，用普通话描述两个量之间的关系
  → 示例："甲的路程加上乙的路程等于总距离"

Step 3  翻译成数学表达式
  → 设未知数 → 用Step 2的文字关系写等式
  → 列出方程
```

---

## 二、行程问题建模框架

### 核心公式
```
路程 = 速度 × 时间
```

### 三种经典情形

| 情形 | 关系式 | 示意 |
|-----|-------|-----|
| 相向而行（相遇） | 甲路程 + 乙路程 = 总距离 | ←A  B→ |
| 同向而行（追及） | 快者路程 - 慢者路程 = 初始差距 | →A  →B（A追B） |
| 来回同一地点 | 去程路程 = 回程路程 | 特殊：速度不同时间不同 |

### 设元建议
- 设"时间"为x：当速度已知，求相遇/追及时间
- 设"距离"为x：当速度和时间已知，求某段距离
- 设"速度"为x：当距离和时间已知，求速度

### 常见陷阱
```
陷阱①：单位不统一（小时vs分钟，km/h vs m/s）
检验："题目里时间用的是什么单位？速度用的是什么单位？统一了吗？"

陷阱②：顺流/逆流（水流问题变体）
顺流速度 = 船速 + 水速
逆流速度 = 船速 - 水速
```

## 三、工程问题建模框架

### 核心公式
```
工作量 = 工作效率 × 工作时间
（设总工作量为1）
```

### 关键设定
```
设总工作量 = 1（标准做法）
→ 甲独立完成需N天，甲每天效率 = 1/N
→ 甲乙合作每天效率 = 1/N₁ + 1/N₂
```

### 建模步骤
```
① 设总工作量为1
② 写出每个人（或每台机器）的日效率 = 1/完成天数
③ 写出合作效率 = 各效率之和
④ 等式：合作效率 × 合作天数 = 1（总工作量）
```

### 常见变体
| 变体类型 | 关系式调整 |
|---------|----------|
| 甲做了一部分后乙加入 | 甲×t₁ + (甲+乙)×t₂ = 1 |
| 交替工作 | 每轮完成量 × 轮数 = 1 |
| 甲比乙快N天 | 设乙需要t天，甲需要t-N天 |

## 四、浓度（混合）问题建模框架

### 核心关系
```
溶质量 = 溶液量 × 浓度
混合/稀释时：溶质总量不变（只加水，不加溶质）
```

### 建模步骤
```
① 识别：操作前后各有多少溶液？各是什么浓度？
② 找不变量：溶质量在操作前后是否改变？
③ 列方程：操作前溶质量 = 操作后溶质量（如果溶质不变）
         或：混合后溶质量 = 溶液1溶质量 + 溶液2溶质量
```

### 追问检验
```
"加水了吗？加水不增加溶质，浓度会降低。
 加浓溶液了吗？溶质增加，浓度会升高。
 到底哪个量守恒？"
```

## 五、利润问题建模框架

### 核心公式链
```
利润 = 售价 - 成本
利润率 = 利润 ÷ 成本 × 100%
售价 = 成本 × (1 + 利润率)
折扣后售价 = 标价 × 折扣系数（如八折 = ×0.8）
```

### 常见混淆点
| 概念 | 说明 | 追问 |
|-----|-----|-----|
| 成本 vs 标价 | 标价≠成本，标价是定的价，成本是买入价 | "题目说的是在什么价格基础上打折？" |
| 利润 vs 利润率 | 利润是金额，利润率是百分比 | "题目求的是赚了多少钱还是赚了百分之几？" |
| 亏损 | 利润为负数 | "售价小于成本时，利润是正还是负？" |

## 六、增长率问题建模框架

### 两种增长类型

| 类型 | 公式 | 特征 |
|-----|-----|-----|
| 简单增长（每次相对原量） | 最终量 = 原量 + 原量×增长率×次数 | 增长的基数不变 |
| 复合增长（每次相对上一次） | 最终量 = 原量 × (1+增长率)ⁿ | 增长的基数每次变化 |

### 建模步骤
```
① 判断是简单增长还是复合增长
② 设初始量或增长率为x
③ 写出n次增长后的量的表达式
④ 根据已知的最终量列方程
```

### 追问
```
"增长率是每次对'原来的量'增长，
 还是对'上一次的量'增长？
 这两种情况，第二次增长的基数一样吗？"
```

## 七、数量关系识别速查卡

收到应用题后，30秒内完成以下识别：

```
□ 有没有"运动/速度/时间"相关词？→ 行程类
□ 有没有"完成/效率/共同"相关词？→ 工程类
□ 有没有"浓度/溶液/加水/混合"相关词？→ 浓度类
□ 有没有"进价/售价/利润/折扣"相关词？→ 利润类
□ 有没有"增长/减少/每年"相关词？→ 增长率类
□ 以上都不是？→ 设元找等量关系，用通用三步框架
```
