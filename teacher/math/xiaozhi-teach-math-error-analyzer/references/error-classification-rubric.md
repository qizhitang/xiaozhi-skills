# 错因分类细则

> 配合 `xiaozhi-teach-math-error-analyzer` 使用。7 类错因识别+溯源参考。

## 1. 错因分类总览

```text
┌──────────┬────────────────────────┬────────────┬──────────────┐
│ 错因      │ 描述                    │ 关键特征    │ 教学策略      │
├──────────┼────────────────────────┼────────────┼──────────────┤
│ 概念模糊  │ 没理解概念              │ 题目陌生    │ 重讲概念      │
│ 规则错误  │ 公式/规则记错           │ 错得"有理"  │ 强调规则      │
│ 审题错误  │ 漏看条件/看错问题       │ 答非所问    │ 训练审题      │
│ 策略错误  │ 方法选择不当            │ 解题思路错  │ 教策略选择    │
│ 计算错误  │ 算错                    │ 过程对结果错│ 加强练习      │

└──────────┴────────────────────────┴────────────┴──────────────┘
```

## 2. 错因识别信号

### 2.1 概念模糊

```text
信号：
  · 学员错得"很没把握"
  · 同一概念反复错
  · 学员自己说"我没懂"
  · 学员不能讲出概念

学员原话：
  · "我没学懂"
  · "我不确定"
  · "什么是 X？"
```

### 2.2 规则错误

```text
信号：
  · 学员说"我记了公式但用错"
  · 错得"有理有据"（学员觉得自己对）
  · 概念清晰但规则错
  · 多次用错同一规则

学员原话：
  · "公式是 X 吗？"
  · "我记得是这样..."
  · "我以为是这样用"
```

### 2.3 审题错误

```text
信号：
  · 答非所问
  · 漏看条件
  · 看错问题（求面积看成周长）
  · 不止一次审题错

学员原话：
  · "我看错了"
  · "我没看到那个条件"
  · "我以为是求 X"
```

### 2.4 策略错误

```text
信号：
  · 解题方向错
  · 学员不知道"用哪个方法"
  · 复杂题不会拆
  · 复杂题完全没思路

学员原话：
  · "我不知从哪下手"
  · "我用了 X 方法但做不下去"
  · "这种题我没见过"
```

### 2.5 计算错误

```text
信号：
  · 过程对，结果错
  · 学员能"指出来错了"
  · 简单题算错
  · 限时练习算错多

学员原话：
  · "我算错了"
  · "我以为是 X"
  · "我重新算一遍"
```

### 2.6 粗心大意

```text
信号：
  · 写错数字
  · 抄错题
  · 时间紧/疲劳
  · 简单题错

学员原话：
  · "我不小心..."
  · "我看花了"
  · "我赶时间"
```

### 2.7 知识漏洞

```text
信号：
  · 缺前置知识
  · 学员听不懂讲解
  · 不是本节内容，是前几节内容
  · 同类问题都错

学员原话：
  · "这个我没学过"
  · "我忘了"
  · "我之前没掌握"
```

## 3. 错因溯源四层

```text
第一层：表面错因
  学员做错了

第二层：错因分类
  7 类之一

第三层：根因
  概念模糊 → 是哪个概念？
  规则错误 → 是哪个规则？
  知识漏洞 → 是哪个前置？

第四层：教学原因
  教法问题？
  学员基础问题？
  复习问题？
```

## 5. 错因追踪规则

```text
■ 3 次同错
  · 启动专项干预
  · 设计针对性训练
  · 标记"顽固错因"

■ 5 次同错
  · 升级警示
  · 学员+家长沟通
  · 标记"高危错因"
  · 需要外部支持

■ 错因消失
  · 标记"已解决"
  · 持续观察 1 个月
  · 防止反弹
```

## 6. 教学干预建议

### 6.1 概念模糊

```text
■ 重新讲解
  · 用更具体的情境
  · 用更直观的图示
  · 用学员能理解的语言

■ 检测
  · 学员能用自己话讲
  · 学员能举出例子
  · 学员能做简单题
```

### 6.2 规则错误

```text
■ 强调规则
  · 写下来
  · 反复读
  · 正反例对比

■ 检测
  · 学员能正确写出
  · 学员能正确使用
```

### 6.3 审题错误

```text
■ 训练审题
  · 画关键词
  · 标问题（求什么）
  · 标条件（已知什么）

■ 检测
  · 学员不漏条件
  · 学员不错看问题
```

### 6.4 策略错误

```text
■ 教策略
  · 给方法框架
  · 给方法选择清单
  · 题组训练

■ 检测
  · 学员能选对方法
  · 学员能说出为什么选这个方法
```

### 6.5 计算错误

```text
■ 加强练习
  · 限时计算
  · 验算习惯
  · 减少跳步

■ 检测
  · 计算正确率提升
```

### 6.6 粗心大意

```text
■ 心理+习惯
  · 适度紧张（不是焦虑）
  · 时间管理
  · 减少干扰

■ 检测
  · 简单题正确率
  · 重复性错题减少
```

### 6.7 知识漏洞

```text
■ 补前置
  · 找出缺哪个前置
  · 单独讲解
  · 加前置练习

■ 检测
  · 前置知识掌握
  · 本节内容能学会
```
