# 数学知识图谱模板

> 配合 `xiaozhi-teach-math-error-analyzer` 使用。知识图谱是错因分析的空间载体，4 层结构（章-节-点-细节）让错因定位更精准。

## 1. 知识图谱 4 层结构

```text
┌──────────┬────────────────────┬──────────────────┐
│ 层级      │ 名称                │ 示例（代数）       │
├──────────┼────────────────────┼──────────────────┤
│ 第 1 层   │ 章                  │ 代数式            │
│ 第 2 层   │ 节                  │ 整式              │
│ 第 3 层   │ 知识点              │ 整式加减          │
│ 第 4 层   │ 细节/易错点         │ 找同类项          │
└──────────┴────────────────────┴──────────────────┘
```

## 2. 知识图谱模板（以"方程与不等式"为例）

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📊 知识图谱：方程与不等式
━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第 1 层：章】
  · 方程与不等式

【第 2 层：节】（3 个）
  · 2.1 一元一次方程
  · 2.2 二元一次方程组
  · 2.3 一元一次不等式

【第 3 层：知识点】（15 个）
  2.1 一元一次方程
    · 2.1.1 方程的概念
    · 2.1.2 等式性质
    · 2.1.3 解方程步骤
    · 2.1.4 应用题
  2.2 二元一次方程组
    · 2.2.1 二元一次方程组的概念
    · 2.2.2 代入消元法
    · 2.2.3 加减消元法
    · 2.2.4 应用题
  2.3 一元一次不等式
    · 2.3.1 不等式概念
    · 2.3.2 不等式性质
    · 2.3.3 解不等式
    · 2.3.4 不等式应用
    · 2.3.5 不等式组

【第 4 层：细节/易错点】（每知识点 3-5 个）
  · 2.1.1 方程的概念
    ① "等式"和"方程"的区别
    ② "未知数"和"字母"的区别
    ③ 方程必须含等号
  · 2.1.3 解方程步骤
    ① 移项要变号
    ② 系数化 1 时除以系数
    ③ 检验步骤（可省略但建议）
  · ...
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 3. 错因定位流程

```text
学员错题 → 在哪一章？ → 哪一节？ → 哪个知识点？ → 哪个细节？
  · 第 1 层定位：宏观模块
  · 第 2 层定位：节级
  · 第 3 层定位：具体知识点
  · 第 4 层定位：易错细节
```

## 4. 知识图谱绘制三法

### 4.1 树状图（最常用）

```text
                    方程与不等式
                          │
        ┌─────────────────┼─────────────────┐
        ↓                 ↓                 ↓
   一元一次方程      二元一次方程组      一元一次不等式
        │                 │                 │
   ┌────┼────┐         ┌────┼────┐         ┌────┼────┐
   ↓    ↓    ↓         ↓    ↓    ↓         ↓    ↓    ↓
  概念 性质 步骤       概念 代入 加减       概念 性质 步骤 ...
```

### 4.2 关系图（突出横向联系）

```text
  ┌────────┐
  │ 方程    │ ←─── 共同基础：等式性质
  └────┬───┘
       │ 推广
       ↓
  ┌────────┐
  │ 不等式  │
  └────┬───┘
       │ 推广
       ↓
  ┌────────┐
  │ 函数    │
  └────────┘
```

### 4.3 螺旋图（突出认知顺序）

```text
  小学：简单方程 → 初中：一元一次方程 → 高中：一元二次方程
         (算术)         (代数)             (函数)
```

## 7. 知识图谱的应用

```text
应用 ① 错因定位
  → 学员错题 → 查图谱 → 定位第 N 层 → 决定补救内容

应用 ② 教学进度
  → 班级图谱 → 找弱项章节 → 重点讲解

应用 ③ 学员档案
  → 个人图谱 → 跟踪学员成长

应用 ④ 阶段报告
  → 章节热力图 → 展示班级整体水平
```
