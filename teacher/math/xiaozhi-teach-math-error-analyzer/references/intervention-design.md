# 教学干预设计模板

> 配合 `xiaozhi-teach-math-error-analyzer` 使用。基于错因分析结果，设计 4 步干预方案（识别-诊断-干预-跟踪）。

## 1. 干预设计 4 步

```text
┌──────────┬────────────────┬────────────────────┐
│ 步骤      │ 名称            │ 关注重心            │
├──────────┼────────────────┼────────────────────┤
│ 第 1 步   │ 识别错因        │ 学员哪里错          │
│ 第 2 步   │ 诊断原因        │ 为什么错            │
│ 第 3 步   │ 设计干预        │ 怎么补              │
│ 第 4 步   │ 跟踪评估        │ 干预是否有效        │
└──────────┴────────────────┴────────────────────┘
```

## 2. 第 1 步：识别错因

### 2.1 错因分类（参见 error-pattern-rubric.md）

```text
7 类错因：概念模糊/规则错误/审题错误/策略错误/计算错误/粗心大意/知识漏洞
```

### 2.2 错因识别三问

```text
问 1：学员在哪个知识点出错？
  → 定位第 3 层知识点

问 2：学员在哪个步骤出错？
  → 定位第 4 层细节

问 3：学员错因属于 7 类中的哪一类？
  → 给出错因标签
```

## 3. 第 2 步：诊断原因

### 3.1 原因分析 5 层

```text
层 1：表面层
  → 学员"算错了" / "不会"

层 2：知识层
  → 哪个公式/规则/概念不熟

层 3：思维层
  → 是逻辑/推理/建模哪一步出问题

层 4：习惯层
  → 是跳过步骤/不检查/字迹潦草等

层 5：情绪层
  → 是畏难/紧张/失去兴趣等
```

### 3.2 学员访谈 5 问

```text
问 1：你能给我讲讲你的思路吗？
问 2：你在哪一步卡住的？
问 3：你当时是怎么想的？
问 4：再给你一次机会你会怎么做？
问 5：你能用另一种方法做吗？
```

## 4. 第 3 步：设计干预

### 4.1 干预方案 4 象限

```text
┌────────────────┬────────────────┐
│  低难度         │  高难度         │
│  学员能自己补   │  学员需老师辅导  │
├────────────────┼────────────────┤
│  · 同类题 5 道  │  · 1 对 1 讲题  │
│  · 微课视频     │  · 概念重讲     │
│  · 思维导图复习 │  · 阶梯练习     │
├────────────────┼────────────────┤
│  低频率         │  高频率         │
│  学员偶发错误   │  学员反复错误   │
├────────────────┼────────────────┤
│  · 一次讲清     │  · 三次复习     │
│  · 不必过度练   │  · 形成性评价   │
└────────────────┴────────────────┘
```

### 4.2 干预 4 类（按错因）

```text
┌──────────┬────────────────────┬──────────────────────┐
│ 错因      │ 干预方法            │ 时长                  │
├──────────┼────────────────────┼──────────────────────┤
│ 概念模糊  │ 概念重讲 + 正反例  │ 1 次课（45 分钟）     │
│ 规则错误  │ 步骤拆解 + 操练    │ 2 次课（90 分钟）     │
│ 审题错误  │ 审题训练 + 模板    │ 1 次课（45 分钟）     │
│ 策略错误  │ 一题多解 + 决策树  │ 2 次课（90 分钟）     │
│ 计算错误  │ 计算专项 + 估算    │ 持续训练（每日 10 题）│
│ 粗心大意  │ 规范训练 + 检查习惯│ 持续训练              │
│ 知识漏洞  │ 前置知识补 + 关联  │ 1-3 次课              │
└──────────┴────────────────────┴──────────────────────┘
```

### 4.3 干预方案样板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 干预方案：学员 A-007 · 解方程步骤错
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 错因类别：规则错误
■ 错因等级：反复（出现 3 次）
■ 错因定位：2.1.3 解方程步骤 / 移项不变号

【干预目标】
  1. 短期（1 周）：移项必变号
  2. 中期（2 周）：复杂方程正确求解
  3. 长期（1 月）：形成解方程规范

【干预步骤】
  · 第 1 步：1 对 1 重讲移项规则（15 分钟）
  · 第 2 步：口诀训练："过桥变号"（5 分钟）
  · 第 3 步：阶梯练习 10 道（30 分钟）
    ① 基础：x + 3 = 7
    ② 进阶：3x - 5 = 10
    ③ 综合：2(3x - 1) = 10
  · 第 4 步：每日 2 道（持续 7 天）

【资源需求】
  · 阶梯练习题 10 道（参见 [题库]）
  · 微课视频 1 个（参见 [视频]）

【跟踪评估】
  · 第 3 天：5 道移项题
  · 第 7 天：10 道综合题
  · 达成标准：90% 正确率
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 5. 第 4 步：跟踪评估

### 5.1 跟踪 3 时间点

```text
T+3 天：短期检测
  → 5 道同类题，看是否"刚学完还记得"

T+7 天：中期检测
  → 10 道综合题，看是否"进入长期记忆"

T+30 天：长期检测
  → 3 道压轴题，看是否"能灵活应用"
```

### 5.2 跟踪记录表

```text
┌──────────┬──────────┬──────────┬──────────┐
│ 学员      │ T+3     │ T+7     │ T+30    │
├──────────┼──────────┼──────────┼──────────┤
│ A-007    │ 80%     │ 90%     │ 100%    │
│ A-008    │ 60%     │ 70%     │ -       │
│ ...      │          │          │         │
└──────────┴──────────┴──────────┴──────────┘
```

### 5.3 干预效果评估

```text
有效：T+30 达到 90%+
部分有效：T+30 在 70-89%
无效：T+30 在 70% 以下
  → 需要调整干预方案
```
