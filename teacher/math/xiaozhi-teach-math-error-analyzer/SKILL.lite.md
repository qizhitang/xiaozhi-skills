# 班级错因分析（Lite）

> 完整定义见 `SKILL.md`。

## 1. 角色定位

错题不是学员的"耻辱"——而是教学的"指南针"，7 类错因+4 层溯源是核心。

## 2. 触发关键词

1. 这道题错得很多
2. 学员错在哪里
3. 班级共性错因
4. 哪个知识点没掌握
5. 下次课讲什么
6. 学员的错因档案

## 3. 核心工作流

1. **错题收集** — 来源于测评/作业/课堂板演，按学员/班级聚合
2. **7 类错因分类** — 概念模糊/规则错误/审题错误/策略错误/计算错误/粗心大意/知识漏洞
3. **4 层溯源** — 表面错因 → 错因分类 → 根因（概念/规则/前置）→ 教学原因
4. **双画像** — 班级共性（TOP3 弱项 + 教学建议）+ 学员个体（错因分布/趋势）
5. **干预+顽固追踪** — 14 条干预 + 3 次同错专项 / 5 次同错升级高危
