# 班级错因分析 · 精简版

> 详细版见 `SKILL.md`。

## 1. 一句话定位

错题不是学员的"耻辱"，是教学的"指南针"——错因分析是从"题目表面"到"教学深层"的桥梁。

## 2. 工作流

```text
错题收集 → 错因分类 → 知识图谱关联 →
班级共性画像 + 学员个体画像 → 教学干预建议
```

## 3. 错因 7 类

```text
概念模糊：没理解概念
规则错误：公式/规则记错
审题错误：漏看条件/看错问题
策略错误：方法选择不当
计算错误：算错
粗心大意：注意力不集中
知识漏洞：缺前置知识
```

## 4. 错因四层溯源

```text
第一层：表面错因（做错了）
第二层：错因分类（7 类之一）
第三层：根因（哪个概念/规则/前置）
第四层：教学原因（教法/基础/复习）
```

## 5. 知识点热力图

```text
🟢 > 70% 掌握
🟡 40-70% 掌握
🔴 < 40% 掌握
```

## 6. 干预样板

```text
概念模糊 → 重新讲解+情境引入
规则错误 → 强调规则+正反例对比
审题错误 → 训练审题+关键词勾画
策略错误 → 教策略选择+题组训练
计算错误 → 加强练习+验算习惯
粗心大意 → 心理干预+时间管理
知识漏洞 → 补前置+回顾旧知
```

## 7. 错因追踪

```text
3 次同错 → 顽固错因 → 专项干预
5 次同错 → 高危错因 → 升级警示
```

## 8. 必触发场景

- 老师："这道题错得很多"
- 老师："学员错在哪里"
- 老师："班级共性错因"
- 老师："哪个知识点没掌握"
- 老师："下次课讲什么"
