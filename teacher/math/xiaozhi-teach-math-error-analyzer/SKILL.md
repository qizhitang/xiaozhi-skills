# 班级错因分析 SKILL

> **一句话定位：** 错题不是学员的"耻辱"，是教学的"指南针"——错因分析是从"题目表面"到"教学深层"的桥梁。

---

## ⚠️ 技术实现边界声明

> **关于"AI 阅卷"边界：** 本 SKILL 输出**错因分析框架**与**教学干预建议**；**不**代替老师阅卷，**不**给学员打分排名。
>
> **关于"数据隐私"边界：** 错因分析基于化名/编号；**不**公开学员真实姓名+错题。
>
> **关于"教学干预"边界：** 本 SKILL **不**替代老师做教学决策；只提供**分析结果**和**建议**。

---

## 一、核心使命

数学错因分析常见的三个误区：

```text
误区① 错因=粗心：把所有错都归为"粗心"，
        错过就过，下次还错。

误区② 错题=个别：把错题归为"学员不认真"，
        没看到教学层面的问题。

误区③ 错题=补差：错题分析完就讲一遍，
        没有系统化、没有持续追踪。
```

本 SKILL 要解决的是：
- **让错因"具体化"**：从"粗心"到"7 类错因"
- **让错题"关联化"**：单个错题连到知识图谱
- **让分析"系统化"**：班级共性 + 学员个体
- **让干预"精准化"**：根据错因给具体建议

## 四、错因分类（7 类）

### 4.1 错因总览

```text
┌──────────┬────────────────────────┬────────────┬──────────────┐
│ 错因      │ 描述                    │ 关键特征    │ 教学策略      │
├──────────┼────────────────────────┼────────────┼──────────────┤
│ 概念模糊  │ 没理解概念              │ 题目陌生    │ 重讲概念      │
│ 规则错误  │ 公式/规则记错           │ 错得"有理"  │ 强调规则      │
│ 审题错误  │ 漏看条件/看错问题       │ 答非所问    │ 训练审题      │
│ 策略错误  │ 方法选择不当            │ 解题思路错  │ 教策略选择    │
│ 计算错误  │ 算错                    │ 过程对结果错│ 加强练习      │
│ 粗心大意  │ 注意力不集中            │ 简单题错    │ 心理干预      │
│ 知识漏洞  │ 缺前置知识              │ 困难题错    │ 补前置        │
└──────────┴────────────────────────┴────────────┴──────────────┘
```

### 4.2 错因识别

```text
■ 概念模糊
  · 学员错得"很没把握"
  · 同一概念反复错
  · 学员自己说"我没懂"

■ 规则错误
  · 学员说"我记了公式但用错"
  · 错得"有理有据"（学员觉得自己对）
  · 概念清晰但规则错

■ 审题错误
  · 答非所问
  · 漏看条件
  · 看错问题（求面积看成周长）

■ 策略错误
  · 解题方向错
  · 学员不知道"用哪个方法"
  · 复杂题不会拆

■ 计算错误
  · 过程对，结果错
  · 学员能"指出来错了"
  · 简单题算错

■ 粗心大意
  · 写错数字
  · 抄错题
  · 时间紧/疲劳

■ 知识漏洞
  · 缺前置知识
  · 学员听不懂讲解
  · 不是本节内容，是前几节内容
```

### 4.3 错因溯源

```text
■ 第一层：表面错因
  · 学员做错了

■ 第二层：错因分类
  · 7 类之一

■ 第三层：根因
  · 概念模糊 → 是哪个概念？
  · 规则错误 → 是哪个规则？
  · 知识漏洞 → 是哪个前置？

■ 第四层：教学原因
  · 教法问题？
  · 学员基础问题？
  · 复习问题？
```

## 五、知识图谱关联

### 5.1 知识点体系

```text
小学：
  数与代数：整数/分数/小数/百分数/比/比例
  图形与几何：平面图形/立体图形/周长/面积/体积
  统计与概率：统计表/统计图/概率

初中：
  数与式：有理数/整式/分式/二次根式
  方程与不等式：方程/不等式/方程组
  函数：一次函数/反比例函数/二次函数
  图形：三角形/四边形/圆/相似/全等
  统计：数据/概率/统计

高中：
  函数：基本初等函数/三角函数/数列
  几何：向量/立体几何/解析几何
  代数：不等式/复数/计数原理
  概率统计：概率/统计/随机变量
```


**禁止行为**：
- 禁止 AI 替学员分析错因（学员自评+老师诊断）
- 禁止 AI 给学员排名
- 禁止 AI 替老师下教学决策（只提供建议）
- 禁止公开学员真实姓名+错题
- 禁止把错因简单归为"粗心"
