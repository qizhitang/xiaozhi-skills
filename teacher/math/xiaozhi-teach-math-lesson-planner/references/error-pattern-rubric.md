# 数学错因分类表（教案层）

> 配合 `xiaozhi-teach-math-lesson-planner` 使用。从教案设计角度区分 7 类常见错因，帮助老师在备课阶段就预防典型错误。

## 1. 7 类错因总览

```text
┌──────┬──────────────┬────────────────────────┐
│ #    │ 错因类别      │ 典型表现               │
├──────┼──────────────┼────────────────────────┤
│ 1    │ 概念模糊      │ 用错公式/混淆符号      │
│ 2    │ 规则错误      │ 算法步骤错/法则记错    │
│ 3    │ 审题错误      │ 看错条件/漏掉关键字    │
│ 4    │ 策略错误      │ 方法选择不当           │
│ 5    │ 计算错误      │ 算术错/符号错          │
│ 6    │ 粗心大意      │ 写错/漏写/字迹潦草    │
│ 7    │ 知识漏洞      │ 前置知识不会           │
└──────┴──────────────┴────────────────────────┘
```

## 2. 七类错因详解

### 2.1 概念模糊

```text
【定义】学员对核心概念理解不透彻
【典型表现】
  · 用错公式（如把长方形面积当周长）
  · 混淆符号（如把 ≠ 当 =）
  · 概念张冠李戴（如把"等腰"当"等边"）
  · 概念边界不清（如不知道"0"是否属于正数）

【教案预防】
  · 在概念引入时增加正/反例对比
  · 给出"概念辨析卡"
  · 设计"找错"练习
```

### 2.2 规则错误

```text
【定义】学员对运算法则/规则记忆错
【典型表现】
  · 解方程移项不变号
  · 分数运算不找最小公倍数
  · 因式分解漏项
  · 函数单调性记反

【教案预防】
  · 把规则拆成最小步骤
  · 设计"我说你做"操练
  · 配合口诀/图示
```

### 2.3 审题错误

```text
【定义】学员没看清题目条件
【典型表现】
  · 看错数字（如把 5 看 6）
  · 漏掉关键字（如"至少"、"整数"）
  · 多余条件没排除
  · 问什么答什么没搞清

【教案预防】
  · 设计"信息提取卡"
  · 训练"圈关键字"习惯
  · 答案前问"题目问什么"
```

### 2.4 策略错误

```text
【定义】学员选择的方法不当
【典型表现】
  · 用复杂方法解决简单题
  · 不会用方程/反用算术
  · 不会分类讨论
  · 不画图分析

【教案预防】
  · 一题多解展示
  · 策略选择"决策树"
  · 强调"先想再算"
```

### 2.5 计算错误

```text
【定义】学员算术/代数运算错
【典型表现】
  · 整数加减错
  · 分数四则错
  · 移项变号错
  · 公式代入错

【教案预防】
  · 强计算专项训练
  · 分步展示避免跳步
  · 估算验证（先估后算）
```

### 2.6 粗心大意

```text
【定义】学员非智力因素错误
【典型表现】
  · 写错答案（如把 8 写 3）
  · 漏写步骤
  · 字迹潦草导致看错
  · 不写单位

【教案预防】
  · 规范答题模板
  · 培养"二次检查"习惯
  · 限时训练（避免拖拉）
```

### 2.7 知识漏洞

```text
【定义】学员前置知识不会
【典型表现】
  · 分数运算不会 → 比例不会
  · 三角形面积不会 → 几何综合题不会
  · 一元一次方程不会 → 一元二次不会

【教案预防】
  · 课前测诊断
  · 关联前置知识复习
  · 阶梯式例题设计
```

## 3. 错因识别流程

```text
学员答错 → 是哪一类？
  · 看懂题目吗？ → 审题错误
  · 选对方法吗？ → 策略错误
  · 公式对吗？ → 概念模糊
  · 步骤对吗？ → 规则错误
  · 算得对吗？ → 计算错误
  · 漏看/漏写？ → 粗心大意
  · 基础会不会？ → 知识漏洞
```

## 5. 错因分布参考（初中数学）

```text
┌──────┬──────────┬──────────────────────┐
│ 错因  │ 占比参考  │ 高发知识点            │
├──────┼──────────┼──────────────────────┤
│ 1 概念│ 20%      │ 函数/几何概念/符号    │
│ 2 规则│ 15%      │ 方程/不等式/运算法则  │
│ 3 审题│ 20%      │ 应用题/几何题         │
│ 4 策略│ 15%      │ 综合题/压轴题         │
│ 5 计算│ 15%      │ 整式/分数/代数式      │
│ 6 粗心│ 10%      │ 填空/计算            │
│ 7 漏洞│ 5%       │ 跨章节综合           │
└──────┴──────────┴──────────────────────┘
```

> 注：占比是参考值，应基于自己班级/学员的实际数据调整。
