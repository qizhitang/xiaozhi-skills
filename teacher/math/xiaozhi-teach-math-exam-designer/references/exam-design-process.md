# 数学测评设计流程

> 配合 `xiaozhi-teach-math-exam-designer` 使用。从测评目标出发，6 步走完"目的→蓝图→题目→审查→拼卷→讲评"全流程。

## 1. 测评设计 6 步流程

```text
┌──────────┬────────────────┬────────────────────┐
│ 步骤      │ 名称            │ 关注重心            │
├──────────┼────────────────┼────────────────────┤
│ 第 1 步   │ 明确测评目的    │ 为什么考            │
│ 第 2 步   │ 编制双向细目表  │ 考什么              │
│ 第 3 步   │ 编写/筛选题目   │ 怎么考              │
│ 第 4 步   │ 试卷审查        │ 试卷质量            │
│ 第 5 步   │ 拼卷定稿        │ 形成终稿            │
│ 第 6 步   │ 讲评设计        │ 考后怎么讲          │
└──────────┴────────────────┴────────────────────┘
```

## 2. 第 1 步：明确测评目的

### 2.1 4 类测评目的

```text
┌──────────┬──────────────────┬────────────────────┐
│ 类型      │ 目的              │ 时机                │
├──────────┼──────────────────┼────────────────────┤
│ 诊断性    │ 摸清学员水平      │ 开学初/单元初       │
│ 形成性    │ 反馈教学效果      │ 单元中段/章节末     │
│ 终结性    │ 评定学习成果      │ 期中/期末           │
│ 选拔性    │ 区分学员水平      │ 升学/竞赛           │
└──────────┴──────────────────┴────────────────────┘
```

### 2.2 测评目的设定

```text
【目标 1】知识目标
  · 哪些知识点必须覆盖
  · 覆盖比例

【目标 2】能力目标
  · 主要测哪种能力
  · 能力层级分布

【目标 3】时长目标
  · 总时长
  · 各题时长估算

【目标 4】难度目标
  · 难度梯度比例
  · 平均分预期
```

## 3. 第 2 步：编制双向细目表

（详见 [blueprint-template.md](blueprint-template.md)）

```text
【双向细目表结构】
  · 纵轴：知识点（章-节-点）
  · 横轴：能力层级（了解/理解/掌握/综合应用）
  · 交叉点：题目数量 + 分数

【细目表填写要素】
  · 题号
  · 题型
  · 分值
  · 预估难度
  · 预估通过率
```

## 4. 第 3 步：编写/筛选题目

### 4.1 题目来源 4 类

```text
来源 ① 自编
  · 优点：完全可控
  · 缺点：耗时
  · 占比建议：30-50%

来源 ② 改编
  · 来源：教材例题/习题改编
  · 占比建议：20-30%

来源 ③ 现成题库
  · 来源：学校题库/历年真题
  · 占比建议：20-40%

来源 ④ 学员错题改编
  · 来源：班级学员常见错题
  · 占比建议：10-20%
```

### 4.2 题目版权标注

```text
┌──────────┬────────────────────┬──────────────┐
│ 状态      │ 来源                │ 标注要求      │
├──────────┼────────────────────┼──────────────┤
│ 自创      │ 老师原创            │ 无需标注      │
│ 授权      │ 已获出版社/作者授权  │ 标注"授权使用"│
│ 公有      │ 超过版权期          │ 标注"公有"   │
│ 未授权    │ 截取未授权资料      │ 禁止使用      │
└──────────┴────────────────────┴──────────────┘
```

### 4.3 编写题目 5 原则

```text
原则 1：题干清晰
  · 一道题只有一个明确问法
  · 避免歧义
  · 字数控制

原则 2：考点单一
  · 一道题突出一个核心考点
  · 避免多个考点混在一起
  · 综合性题放最后

原则 3：难度梯度
  · 由易到难
  · 跳过式设计
  · 让学员有"成就感"

原则 4：公平性
  · 不超出课程标准
  · 不依赖超纲知识
  · 题目信息对所有学员公平

原则 5：可区分性
  · 不同能力层级学员有不同表现
  · 高分段学员有发挥空间
  · 低分段学员有基础分
```

## 5. 第 4 步：试卷审查

### 5.1 试卷审查 6 维

```text
┌──────┬────────────────┬──────┐
│ 维度  │ 检查项          │ 通过  │
├──────┼────────────────┼──────┤
│ 知识  │ 是否覆盖大纲    │ □     │
│ 能力  │ 各层级分布合理  │ □     │
│ 难度  │ 难度梯度合理    │ □     │
│ 题量  │ 时长足够        │ □     │
│ 公平  │ 无超纲/歧视     │ □     │
│ 表述  │ 无歧义/错字     │ □     │
└──────┴────────────────┴──────┘
```

### 5.2 难度梯度参考

```text
· 基础题（容易）：60%
· 进阶题（中等）：30%
· 挑战题（较难）：10%
```

### 5.3 时长测算

```text
公式：单题时长 = 60-90 秒 × 难度系数
· 容易题：30-60 秒
· 中等题：90-150 秒
· 较难题：180-300 秒
```

## 6. 第 5 步：拼卷定稿

### 6.1 拼卷原则

```text
原则 1：由易到难
  · 学员先做会的，再做难的
  · 增强信心

原则 2：同类集中
  · 选择-填空-解答-证明分层
  · 避免题型混排

原则 3：分值合理
  · 选择 2-3 分，填空 3-5 分
  · 解答 5-15 分

原则 4：留白充足
  · 给学员足够的答题空间
```

## 7. 第 6 步：讲评设计

### 7.1 讲评 4 步

```text
第 1 步：错因归类（15 分钟）
  · 班级整体错因分布
  · 重点错因 TOP 3

第 2 步：正解展示（10 分钟）
  · 标准解法
  · 关键步骤

第 3 步：变式训练（10 分钟）
  · 同考点变式 2-3 道
  · 跨章节关联 1 道

第 4 步：巩固练习（10 分钟）
  · 同类题 3-5 道
  · 当堂完成
```

### 7.2 讲评设计卡

