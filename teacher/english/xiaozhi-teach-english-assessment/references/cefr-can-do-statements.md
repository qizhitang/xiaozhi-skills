# CEFR "能做什么" 语句

> 配合 `xiaozhi-teach-english-assessment` 使用。CEFR 6 个等级 × 4 维能力 = 24 个"能做什么"。

## 1. CEFR 等级总览

```text
┌──────────┬────────────────────────┐
│ 等级      │ 描述                    │
├──────────┼────────────────────────┤
│ A1        │ 入门级                  │
│ A2        │ 基础级                  │
│ B1        │ 中级                    │
│ B2        │ 中高级                  │
│ C1        │ 高级                    │
│ C2        │ 精通级                  │
└──────────┴────────────────────────┘
```

## 2. A1 "能做什么"（入门级）

### 2.1 听

```text
· 能听懂慢速、清晰、简单句
· 能听懂简单的指示
· 能听懂数字/时间/价格
```

### 2.2 说

```text
· 能用单词/短句介绍自己
· 能问简单问题
· 能用基本问候语
```

### 2.3 读

```text
· 能看懂简单的短文
· 能识别常见单词
· 能读懂通知/菜单
```

### 2.4 写

```text
· 能写简单句
· 能写明信片
· 能填写简单表格
```

## 9. CEFR 使用

### 9.1 学员水平评估

```text
· 学员能做什么 → 对照 CEFR
· 学员弱什么 → 对照 CEFR
· 学员应做什么 → 对照 CEFR
```

### 9.2 教学目标设定

```text
· 目标：达到 B1
  · 听：能听懂日常对话
  · 说：能讨论日常话题
  · 读：能读懂中长文
  · 写：能写各类短文
```

### 9.3 课程设计

```text
· 当前 A2 → 目标 B1
  · 听：增加日常对话材料
  · 说：增加讨论类任务
  · 读：增加中长文
  · 写：增加各类短文
```

## 10. 测评对照

```text
■ 学员能做什么？
  · 与 CEFR 对照
  · 标记 4 维等级

■ 学员弱什么？
  · 找出最弱维度
  · 重点训练

■ 学员应做什么？
  · 设定目标等级
  · 制定教学计划
```
