# 英语综合测评 · 精简版

> 详细版见 `SKILL.md`。

## 1. 一句话定位

英语测评不是给学员"打分数"，而是给教学"画能力地图"——CEFR 框架是核心。

## 2. 工作流

```text
能力目标 → CEFR 对照 → 4 维测评 → 能力画像 → 教学干预
```

## 3. 4 维能力

```text
听（输入-听觉）
说（输出-口语）
读（输入-视觉）
写（输出-书面）
```

## 4. CEFR 等级

```text
A1 入门级（小学低年级）
A2 基础级（小学高年级）
B1 中级（初中）
B2 中高级（高中）
C1 高级（CET-6）
C2 精通级（接近母语）
```

## 5. 4 维测评设计

```text
听力 30%：主旨/细节/推断
口语 20%：流利/准确/得体
阅读 30%：字面/推断/评价
写作 20%：内容/结构/语言/规范
```

## 6. 学员能力画像

```text
4 维等级（听/说/读/写）
词汇量（主动/被动）
语法掌握
强项/弱项
CEFR 综合等级
```

## 7. 教学干预

```text
听弱 → 听力材料+微技能
说弱 → 任务型+情境练习
读弱 → 阅读策略+分级
写弱 → 写作模板+过程引导
```

## 8. 必触发场景

- 老师："学员英语水平如何"
- 老师："英语综合测评"
- 老师："听说读写怎么测"
- 老师："CEFR 对照"
- 老师："学员能做什么"
