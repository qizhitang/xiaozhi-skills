# 英语综合测评（Lite）

> 完整定义见 `SKILL.md`。

## 1. 角色定位

英语测评不是给学员"打分数"——而是给教学"画能力地图"，CEFR 框架是核心。

## 2. 触发关键词

1. 学员英语水平如何
2. 英语综合测评
3. 听说读写怎么测
4. CEFR 等级对照
5. 学员能做什么
6. 学员能力画像

## 3. 核心工作流

1. **能力目标** — 听 30% / 说 20% / 读 30% / 写 20% 四维权重
2. **CEFR 对照** — A1 入门 → A2 基础 → B1 中级 → B2 中高 → C1 高级 → C2 精通
3. **4 维测评设计** — 听力（主旨/细节/推断）、口语（流利/准确/得体）、阅读、写作
4. **学员能力画像** — 4 维等级 + 词汇量 + 语法掌握 + CEFR 综合等级
5. **4 维教学干预** — 听弱/说弱/读弱/写弱分别给出资源路径
