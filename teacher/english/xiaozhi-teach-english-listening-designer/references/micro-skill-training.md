# 听力微技能训练法

> 配合 `xiaozhi-teach-english-listening-designer` 使用。让听力教学从"刷题"变成"真正能听懂"。

## 1. 微技能总览

```text
┌──────────┬────────────────────────┐
│ 微技能    │ 描述                    │
├──────────┼────────────────────────┤
│ 抓主旨    │ 听出"讲什么"           │
│ 抓细节    │ 听出"具体信息"         │
│ 推断      │ 听出"言外之意"         │
│ 数字/时间 │ 听出"数字/时间"        │
│ 抓语气    │ 听出"说话人态度"       │
│ 抓转折    │ 听出"but/however"      │
│ 抓连接    │ 听出"first/then/finally"│
│ 抓重音    │ 听出"重读词"           │
└──────────┴────────────────────────┘
```

## 2. 抓主旨训练

### 2.1 训练法

```text
■ 听前预测
  · 看标题
  · 看图片
  · 问"会讲什么"

■ 听首尾句
  · 第一句 + 最后一句
  · 主旨常常在这里

■ 不查细节
  · 听 1 遍
  · 不强求"每个词"
```

### 2.2 训练样板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎧 抓主旨训练
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 材料
  短对话（30-60 秒）

■ 训练法
  · 1. 学员看图/标题
  · 2. 预测会讲什么
  · 3. 听 1 遍
  · 4. 学员说主旨
  · 5. 老师反馈

■ 学员层次
  · 基础：能说出主题
  · 中等：能说出细节
  · 提升：能说出说话人态度
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 3. 抓细节训练

### 3.1 训练法

```text
■ 关键词
  · 不强求"每个词"
  · 抓住"实词"

■ 数字/时间/地点
  · 专门听
  · 记笔记

■ 记笔记
  · 简短符号
  · 数字/时间/地点
```

### 3.2 训练样板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎧 抓细节训练
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 材料
  对话（含数字/时间/地点）

■ 训练法
  · 1. 学员准备好笔记
  · 2. 听第 1 遍：抓主旨
  · 3. 听第 2 遍：填细节
  · 4. 学员核对答案
  · 5. 学员反思

■ 学员层次
  · 基础：识别 1-2 个细节
  · 中等：识别 3-5 个细节
  · 提升：识别所有细节
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 4. 推断训练

### 4.1 训练法

```text
■ 听转折词
  · but, however, although
  · 后面的内容更重要

■ 听语气
  · 重音
  · 语调
  · 选词

■ 联系常识
  · 学员已有知识
  · 帮助推断
```

### 4.2 推断样板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎧 推断训练
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 材料
  对话（含言外之意）

■ 听前
  · 学员预测说话人态度
  · 学员预测结果

■ 听中
  · 听转折词
  · 听语气
  · 学员修正预测

■ 听后
  · 学员说"说话人的真实态度"
  · 学员说"为什么这样认为"
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 5. 抓语气训练

### 5.1 训练法

```text
■ 重音
  · 重读词 = 重要
  · 学员关注"重读"

■ 语调
  · 升调 = 疑问
  · 降调 = 陈述

■ 选词
  · 积极/消极词
  · 学员分析
```

### 5.2 抓语气样板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎧 抓语气训练
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 材料
  短句（如 "Oh, that's great."）

■ 训练法
  · 1. 听 3 种不同语气
  · 2. 学员判断说话人态度
  · 3. 学员分析"怎么听出来的"
  · 4. 学员模仿
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 6. 数字/时间训练

### 6.1 训练法

```text
■ 专门听
  · 让学员"准备听数字"
  · 集中注意力

■ 数字清单
  · 价格/时间/日期/数量
  · 学员记录

■ 数字核对
  · 听完核对
  · 学员自己发现错
```

### 6.2 数字训练样板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎧 数字训练
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 材料
  对话（含 3-5 个数字）

■ 训练法
  · 1. 学员准备笔记
  · 2. 听 1 遍：抓主旨
  · 3. 听 2 遍：记数字
  · 4. 学员核对

■ 学员层次
  · 基础：识别 1-2 个数字
  · 中等：识别 3-4 个
  · 提升：识别所有
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 7. 抓转折/连接训练

### 7.1 抓转折

```text
■ 转折词
  · but, however, although
  · 学员先听转折词
  · 重点听转折后

■ 训练
  · 让学员先听转折词
  · 再听转折后的内容
```

### 7.2 抓连接

```text
■ 连接词
  · first, then, next, finally
  · 学员听顺序

■ 训练
  · 让学员听"先发生什么"
  · 让学员听"再发生什么"
  · 让学员听"最后什么"
```

## 8. 听写训练

### 8.1 听写三阶段

```text
■ 阶段 1：短句（5-10 词）
  · 每周 1-2 句
  · 听 1 遍：抓主旨
  · 听 2-3 遍：补全

■ 阶段 2：长句（15-25 词）
  · 每周 1-2 句
  · 同上

■ 阶段 3：段落（50-100 词）
  · 每周 1 段
  · 同上
```

### 8.2 听写法

```text
■ 听 1 遍
  · 抓主旨
  · 让学员有"整体感"

■ 听 2-3 遍
  · 补全
  · 学员记下能听出的

■ 对照原文
  · 学员自己发现错误
  · 不直接告诉

■ 学员反思
  · 哪里没听出来？
  · 为什么会错？
```

## 9. 听写频率

```text
· 每周 1-2 次
· 不频繁（学员压力大）
· 由短到长
· 学员自己发现错
```

## 10. 微技能训练顺序

```text
1. 抓主旨（最基础）
2. 抓细节
3. 抓数字/时间
4. 抓转折/连接
5. 抓语气
6. 推断（最综合）
```

---

> 💡 **小智说：** 听力是 8 个微技能的组合，单独训练比"刷整套题"更有效。
