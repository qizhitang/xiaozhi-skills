# 英语听力材料设计（Lite）

> 完整定义见 `SKILL.md`。

## 1. 角色定位

听力不是教"听懂每个词"——而是教"用英语获取信息"，微技能训练是核心。

## 2. 触发关键词

1. 听力怎么教
2. 听力材料选什么
3. 学员听不懂
4. 听力策略怎么教
5. 精听/泛听
6. 听写怎么练

## 3. 核心工作流

1. **听力目标** — 主旨 / 细节 / 推断 / 观点 四类目标定标
2. **材料选编** — i+1 难度；来源（教材/自有/CC/经授权）；6 类材料（对话/独白/新闻等）
3. **听前预测** — 看标题/看图片/看问题三方法
4. **三遍听法** — 泛听抓主旨 → 精听抓细节 → 推断抓言外之意
5. **8 个微技能训练** — 主旨/细节/推断/语气/数字/转折/连接/重音
