# 英语听力材料设计 · 精简版

> 详细版见 `SKILL.md`。

## 1. 一句话定位

听力不是教学生"听懂每个词"，而是教学生"用英语获取信息"——微技能训练是核心。

## 2. 工作流

```text
听力目标 → 材料选编 → 听前预测 → 听中分层 → 听后任务 → 微技能训练
```

## 3. 听力四类目标

```text
主旨：讲什么
细节：具体信息
推断：言外之意
观点：说话人态度
```

## 4. 材料选编

```text
来源：教材/自有/CC 协议/经授权
难度：i+1（略高于学员）
类型：对话/独白/新闻/故事
```

## 5. 三遍听法

```text
第一遍：泛听，抓主旨
第二遍：精听，抓细节
第三遍：推断，抓言外之意
```

## 6. 听前预测三方法

```text
看标题预测内容
看图片预测场景
看问题预测答案
```

## 7. 听力微技能

```text
抓主旨 / 抓细节 / 推断 / 抓语气
数字/时间 / 抓转折 / 抓连接 / 抓重音
```

## 8. 必触发场景

- 老师："听力怎么教"
- 老师："听力材料选什么"
- 老师："学员听不懂"
- 老师："听力策略怎么教"
- 老师："精听/泛听"
- 老师："听写怎么练"
