# 英语听力材料设计 SKILL

> **一句话定位：** 听力不是教学生"听懂每个词"，而是教学生"用英语获取信息"——微技能训练是核心。

---

## ⚠️ 技术实现边界声明

> **关于"材料版权"边界：** 听力材料必须标注 copyrightStatus（教材原文/自有/CC 协议/经授权）；不复制未授权材料。
>
> **关于"AI 听写"边界：** 本 SKILL 输出**听力教学设计**与**听法训练流程**；**不**替老师做听写评分。
>
> **关于"AI 阅卷"边界：** 本 SKILL **不**替老师给学员听力打分；只提供**评估框架**和**教学建议**。

---

## 一、核心使命

英语听力教学常见的三个误区：

```text
误区① 听力=刷题：让学员做大量听力题，
        学员只关注"对错"，没关注"听懂"。

误区② 听力=逐字听：让学员听写每个词，
        学员压力大，没建立"听"的整体感。

误区③ 听力=只听不教：只放录音、对答案，
        没教听力策略、没训练微技能。
```

本 SKILL 要解决的是：
- **让听力"有目标"**：主旨/细节/推断/观点
- **让材料"有梯度"**：精听/泛听/听写 分层
- **让听法"有策略"**：预测/抓主旨/记关键
- **让微技能"可训练"**：每个微技能都有训练法

## 五、听力材料选编

### 5.1 材料来源

```text
┌──────────┬────────────────────────┬──────────────┐
│ 来源      │ 描述                    │ 适用          │
├──────────┼────────────────────────┼──────────────┤
│ 教材原文  │ 教材配套音频            │ 基础训练      │
│ 自有改编  │ 老师改编/原创           │ 针对本班      │
│ CC 协议  │ 公开可引用              │ 拓展          │
│ 经授权   │ 商业材料已授权          │ 真实材料      │
└──────────┴────────────────────────┴──────────────┘
```

### 5.2 材料难度

```text
■ i+1 原则
  · 略高于学员水平
  · 80-90% 听懂
  · 10-20% 略难

■ 难度评估
  · 词汇量
  · 语速
  · 句式复杂度
  · 主题熟悉度
```

### 5.3 材料类型

```text
┌──────────┬────────────────────────┐
│ 类型      │ 例子                    │
├──────────┼────────────────────────┤
│ 对话      │ 日常对话/电话/购物      │
│ 独白      │ 演讲/讲座/介绍          │
│ 新闻      │ VOA/BBC/专题            │
│ 故事      │ 短篇故事/小说节选       │
│ 影视      │ 电影/电视剧节选         │
│ 教学      │ 教材配套                │
└──────────┴────────────────────────┘
```

### 5.4 选材原则

```text
■ 主题贴近学员
  · 学员生活相关
  · 学员兴趣相关

■ 难度略高于水平
  · i+1 难度
  · 有"跳一跳"感

■ 类型多样
  · 对话 + 独白 + 新闻 + 故事
  · 不同口音/语速

■ 来源合法
  · 标注 copyrightStatus
  · 不复制未授权材料
```

### 5.5 材料版权

```text
✅ 教材原文：标注来源
✅ 自有改编：老师原创
✅ CC 协议：公开可引用
✅ 经授权：商业材料已授权

❌ 禁止：未授权复制音频
❌ 禁止：未授权转载 podcast
```

## 七、听中分层

### 7.1 三遍听法

```text
■ 第一遍（泛听）
  · 抓主旨
  · 不查细节
  · 让学员有"整体感"

■ 第二遍（精听）
  · 抓细节
  · 关键词
  · 数字/时间/地点

■ 第三遍（推断）
  · 推断
  · 言外之意
  · 说话人态度
```

### 7.2 三种听法对应

```text
┌──────────┬────────────────────────┐
│ 听法      │ 任务                    │
├──────────┼────────────────────────┤
│ 泛听      │ 主旨题                  │
│ 精听      │ 细节题                  │
│ 推断      │ 推断题/观点题           │
└──────────┴────────────────────────┘
```

### 7.3 听中任务设计

> 📎 完整样板见 `references/while-listening-task-sample.md`（三遍听法对应的听中任务设计样板）

### 7.4 听中技巧

```text
■ 抓关键词
  · 不强求"每个词都懂"
  · 抓住"实词"

■ 记笔记
  · 数字/时间/地点
  · 关键信息

■ 预测验证
  · 听前预测对吗？
  · 听到什么？
```

## 八、听后任务

### 8.1 三类听后任务

```text
┌──────────┬────────────────────────┐
│ 任务      │ 描述                    │
├──────────┼────────────────────────┤
│ 复述      │ 学员用自己的话讲        │
│ 讨论      │ 围绕主题讨论            │
│ 写        │ 写读后感/概要           │
└──────────┴────────────────────────┘
```

### 8.2 复述任务

```text
■ 原文复述
  · 学员逐字复述
  · 适合精听材料

■ 概要复述
  · 学员用自己的话讲
  · 适合主旨材料

■ 角色复述
  · 学员扮演角色
  · 适合对话材料
```

### 8.3 讨论任务

```text
■ 围绕主题
  · 学员讨论材料主题
  · 联系自己的生活

■ 角色讨论
  · 学员扮演材料中的角色
  · 表达观点
```

### 8.4 写任务

```text
■ 概要
  · 学员写 100-200 字概要

■ 读后感
  · 学员写"我的看法"

■ 同主题写作
  · 学员围绕同主题写
```

## 九、听力微技能训练

### 9.1 微技能总览

```text
┌──────────┬────────────────────────┐
│ 微技能    │ 描述                    │
├──────────┼────────────────────────┤
│ 抓主旨    │ 听出"讲什么"           │
│ 抓细节    │ 听出"具体信息"         │
│ 推断      │ 听出"言外之意"         │
│ 数字/时间 │ 听出"数字/时间"        │
│ 抓语气    │ 听出"说话人态度"       │
│ 抓转折    │ 听出"but/however"      │
│ 抓连接    │ 听出"first/then/finally"│
│ 抓重音    │ 听出"重读词"           │
└──────────┴────────────────────────┘
```

### 9.2 微技能训练法

```text
■ 抓主旨
  · 听前预测
  · 听首尾句
  · 不查细节

■ 抓细节
  · 关键词
  · 数字/时间/地点
  · 记笔记

■ 推断
  · 听转折词
  · 听语气
  · 联系常识

■ 抓语气
  · 重音
  · 语调
  · 选词
```

### 9.3 听写训练

```text
■ 听写三阶段
  · 阶段 1：听写短句（5-10 词）
  · 阶段 2：听写长句（15-25 词）
  · 阶段 3：听写段落（50-100 词）

■ 听写法
  · 听 1 遍：抓主旨
  · 听 2-3 遍：补全
  · 对照原文
  · 学员自己发现错误

■ 听写频率
  · 每周 1-2 次
  · 不频繁（学员压力大）
```
