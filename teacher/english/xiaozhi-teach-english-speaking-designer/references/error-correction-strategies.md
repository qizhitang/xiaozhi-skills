# 英语纠错策略参考

> 配合 `xiaozhi-teach-english-speaking-designer` 使用。基于"纠错时机的 4 类选择"+"纠错方式的 5 种方法"，让老师选择适合学员的纠错策略。

## 1. 纠错时机 4 类

```text
┌──────┬────────────────────┬──────────────────────┐
│ 类型  │ 时机                │ 适用场景              │
├──────┼────────────────────┼──────────────────────┤
│ 即时  │ 学员说完立即纠      │ 反复犯的错            │
│ 延时  │ 学员说完一段再纠    │ 流利度优先的活动      │
│ 学生互纠│ 让其他学员纠错     │ 学员有元认知能力      │
│ 自纠  │ 引导学员自己发现    │ 培养学员自主性        │
└──────┴────────────────────┴──────────────────────┘
```

## 2. 即时纠错 vs 延时纠错决策

```text
【选即时纠错】
  · 学员反复犯同样的错
  · 错误影响意思表达
  · 学员对纠错不焦虑
  · 教学目标是准确性

【选延时纠错】
  · 学员处于"沉默期"
  · 错误不影响理解
  · 教学目标是流利度
  · 学员对纠错敏感
```

## 3. 纠错方式 5 种

### 3.1 方式 ①：明确纠正

```text
【操作】
  · 老师直接给出正确形式
  · "应该说 X，不是 Y"

【适用】
  · 学员水平较高
  · 错误较明显
  · 时间紧张

【注意】
  · 不要打击学员
  · 给学员重复正确形式的机会
```

### 3.2 方式 ②：重述式纠正（Recast）

```text
【操作】
  · 学员说："He go to school."
  · 老师回应："Yes, he goes to school every day, right?"

【适用】
  · 任务型活动
  · 学员处于表达流利阶段
  · 想让学员"在情境中学"

【注意】
  · 学员要能意识到重述
  · 不适合低级错误
```

### 3.3 方式 ③：引导式纠错

```text
【操作】
  · 老师问："He ___ to school yesterday. (go/went/gone)"
  · 学员答："went"
  · 老师："Great!"

【适用】
  · 学员有元认知能力
  · 错误是有规律可循的
  · 培养学员自查能力

【注意】
  · 不让学员长期卡在引导中
```

### 3.4 方式 ④：元语言提示

```text
【操作】
  · 老师："注意时态。"
  · 老师："这里需要第三人称单数。"

【适用】
  · 学员已学过相关规则
  · 想让学员自我修正

【注意】
  · 元语言要简洁
  · 学员不理解元语言时直接纠错
```

### 3.5 方式 ⑤：同伴纠错

```text
【操作】
  · 老师："你觉得他说得对吗？为什么？"
  · 学员 B 答："应该用 went，因为是过去时。"

【适用】
  · 小组活动
  · 学员水平相当
  · 培养互相学习

【注意】
  · 不要让纠错变成攻击
  · 老师要在场引导
```

## 4. 纠错决策流程

```text
学员说错 → 评估错误类型
  · 是反复错误吗？ → 即时纠错
  · 是流利度优先吗？ → 延时纠错
  · 学员能自查吗？ → 引导式纠错
  · 时间紧张吗？ → 明确纠正
  · 任务型活动吗？ → 重述式纠错
  · 小组活动吗？ → 同伴纠错
```

## 5. 错误分类处理

```text
┌──────────┬────────────────────┬──────────────────┐
│ 错误类型  │ 处理建议            │ 原因              │
├──────────┼────────────────────┼──────────────────┤
│ 语音      │ 课后单独纠          │ 不打断流利度      │
│ 词汇      │ 即时纠（重述）      │ 影响理解          │
│ 语法      │ 视情况              │ 看对流利度影响    │
│ 语用      │ 课后讲解            │ 不当场纠以免尴尬  │
│ 内容      │ 不纠（让学员表达）  │ 不评价"内容对错"  │
└──────────┴────────────────────┴──────────────────┘
```

## 6. 纠错后的"重做机会"

```text
【原则】纠错后给学员重说一遍的机会
【操作】
  · 老师："现在请再说一次。"
  · 学员重说（用正确形式）
  · 老师肯定

【意义】
  · 让学员"用对"
  · 比单纯纠错更有效
  · 培养学员的"修正能力"
```

## 7. 纠错的负面作用与避免

```text
负面 ① 学员不敢说
  → 降低纠错频率
  → 强调"尝试"比"准确"重要

负面 ② 学员依赖老师纠
  → 引导学员自查
  → 让学员先自评

负面 ③ 纠错变成攻击
  → 永远私下纠
  → 永远对事不对人
```
