# 英语口语活动设计（Lite）

> 完整定义见 `SKILL.md`。

## 1. 角色定位

口语不是教"发音标准"——而是教"用英语做事"，任务型活动（TBLT）是核心。

## 2. 触发关键词

1. 口语课怎么上
2. 学员不敢开口
3. 口语活动怎么设计
4. 对话练习怎么设计
5. 口语任务怎么设计
6. 发音怎么纠正

## 3. 核心工作流

1. **口语三维目标** — 流利（顺畅）/ 准确（用词语法发音）/ 得体（适合场合）
2. **输入准备** — 听力 i+1 + 关键场景词汇 + 真实/模拟情境
3. **任务型活动** — 信息差/角色扮演/辩论 三类型设计
4. **输出练习** — 独白（自我介绍/演讲）/ 对话（配对/角色）/ 讨论（小组/圆桌）
5. **反馈与档案** — 即时/延时反馈 + 学员自评/同伴反馈 + 重述式纠正 + 口语档案
