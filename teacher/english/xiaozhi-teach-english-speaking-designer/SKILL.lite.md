# 英语口语活动设计 · 精简版

> 详细版见 `SKILL.md`。

## 1. 一句话定位

口语不是教学生"发音标准"，而是教学生"用英语做事"——任务型活动是核心。

## 2. 工作流

```text
口语目标 → 输入准备 → 任务型活动 → 输出练习 → 反馈与纠正 → 口语档案
```

## 3. 口语三维目标

```text
流利：说得顺畅
准确：用词/语法/发音正确
得体：适合场合/对象
```

## 4. 输入准备

```text
听力：i+1 难度
词汇：关键场景词汇
情境：真实/模拟
```

## 5. 任务型活动三类型

```text
信息差：两人信息不同，必须交流
角色扮演：扮演角色完成任务
辩论：不同立场讨论
```

## 6. 输出练习三类

```text
独白：自我介绍/演讲
对话：配对/角色/自由
讨论：小组/圆桌/立场
```

## 7. 反馈三方式

```text
即时反馈 / 延时反馈
学员自评 / 同伴反馈
重述式纠正（不直接说"错"）
```

## 8. 必触发场景

- 老师："口语课怎么上"
- 老师："学员不敢开口"
- 老师："口语活动怎么设计"
- 老师："对话练习怎么设计"
- 老师："口语任务怎么设计"
- 老师："发音怎么纠正"
