# 英语口语活动设计 SKILL

> **一句话定位：** 口语不是教学生"发音标准"，而是教学生"用英语做事"——任务型活动是核心。

---

## ⚠️ 技术实现边界声明

> **关于"AI 陪练"边界：** 本 SKILL 输出**口语活动设计**与**教学流程**；**不**替学员做"AI 陪练对话"（由口语陪练 SKILL 负责）。
>
> **关于"材料版权"边界：** 口语任务素材必须标注 copyrightStatus（自有/改编/教材原文/CC 协议）；不复制未授权材料。
>
> **关于"AI 评估"边界：** 本 SKILL **不**替老师给学员口语打分；只提供**评估框架**和**反馈模板**。

## 一、核心使命

英语口语教学常见的三个误区：

```text
误区① 口语=背诵：让学员背"对话范文"，
        学员说得"流利"但没有真实交际。

误区② 口语=纠错：老师不停纠正学员的错，
        学员不敢开口，焦虑上升。

误区③ 口语=读课文：让学员读课文/对话，
        学员没有"自己说话"的机会。
```

本 SKILL 要解决的是：
- **让口语"有任务"**：让学员在"做事中学英语"
- **让口语"有情境"**：在真实/模拟情境中练习
- **让口语"有梯度"**：从"控制"到"自由"
- **让口语"有反馈"**：纠错不是打断，而是引导

## 四、口语目标（三维）

### 4.1 三维定义

```text
┌──────────┬────────────────────────┬──────────────┐
│ 维度      │ 描述                    │ 学员表现      │
├──────────┼────────────────────────┼──────────────┤
│ 流利      │ 说得顺畅/不卡顿         │ 自然停顿少    │
│ 准确      │ 用词/语法/发音正确      │ 错误少        │
│ 得体      │ 适合场合/对象           │ 礼貌/合适     │
└──────────┴────────────────────────┴──────────────┘
```

### 4.2 学员层次目标

```text
■ 基础层
  · 流利优先：敢开口 > 准确
  · 重点：敢说

■ 中等层
  · 流利 + 准确并重
  · 重点：说得"对"

■ 提升层
  · 三维并重
  · 重点：说得"得体"
```

### 4.3 不同时期目标

```text
· 启蒙期：敢说（流利）
· 成长期：说对（准确）
· 提升期：说好（得体）
```

## 五、输入准备

### 5.1 三类输入

```text
┌──────────┬────────────────────────┬──────────────┐
│ 输入      │ 描述                    │ 任务          │
├──────────┼────────────────────────┼──────────────┤
│ 听力      │ 听地道英语              │ 模仿语调/结构 │
│ 词汇      │ 关键场景词汇            │ 词→句→情境   │
│ 情境      │ 真实/模拟情境            │ 角色代入      │
└──────────┴────────────────────────┴──────────────┘
```

### 5.2 听力输入

```text
■ 选材
  · 略高于学员水平（i+1）
  · 长度：30 秒-3 分钟
  · 主题：贴近生活

■ 听法
  · 第一遍：泛听，抓主旨
  · 第二遍：精听，抓细节
  · 第三遍：模仿，复述

■ 资源
  · 教材配套音频
  · CC 协议 podcast
  · 自有改编材料
```

### 5.3 词汇输入

```text
■ 关键场景词汇
  · 餐厅：order, menu, bill, ...
  · 机场：flight, gate, boarding, ...
  · 购物：price, size, discount, ...

■ 学习路径
  · 词 → 词组 → 句子
  · 例：order → order food → I'd like to order...

■ 不只背单词
  · 在情境中学
  · 在使用中记
```

### 5.4 情境输入

```text
■ 真实情境
  · 学员生活中的真实场景
  · 学员感兴趣的领域

■ 模拟情境
  · 模拟面试
  · 模拟旅行
  · 模拟购物
  · 模拟演讲

■ 学员代入
  · 让学员"成为"角色
  · 让学员"在意"结果
```

## 六、任务型活动

### 6.1 任务型教学法（TBLT）

```text
任务（Task）=
  · 用目标语言做事
  · 有明确的结果
  · 关注意义 > 关注形式
```

### 6.2 任务三类型

```text
┌──────────┬────────────────────────┬──────────────┐
│ 类型      │ 描述                    │ 例子          │
├──────────┼────────────────────────┼──────────────┤
│ 信息差    │ 两人信息不同，需要交流   │ 找差异/拼图   │
│ 角色扮演  │ 扮演角色完成任务         │ 店员顾客      │
│ 辩论      │ 不同立场讨论            │ 校园话题      │
└──────────┴────────────────────────┴──────────────┘
```

### 6.3 信息差任务

```text
■ 定义
  · 两人/多人有不同信息
  · 必须交流才能完成任务

■ 例子
  · 任务 1：两人各持一张图，描述给对方，让对方画出来
  · 任务 2：两人各持一份活动表，找出不同点
  · 任务 3：一人有地图，一人问路

■ 优势
  · 学员必须说话
  · 学员有"目的"说话
  · 自然产生对话
```

### 6.4 角色扮演任务

```text
■ 定义
  · 学员扮演不同角色
  · 完成特定任务

■ 例子
  · 任务 1：餐厅点餐（顾客/服务员）
  · 任务 2：酒店入住（客人/前台）
  · 任务 3：电话咨询（客户/客服）
  · 任务 4：面试（面试官/求职者）

■ 优势
  · 学员"成为"角色
  · 自然使用礼貌用语
  · 适应不同场合
```

### 6.5 辩论任务

```text
■ 定义
  · 学员持不同观点
  · 用英语论证

■ 例子
  · 任务 1：手机对学习的影响（正方/反方）
  · 任务 2：远程工作好不好
  · 任务 3：学生该不该有零花钱

■ 难度
  · 较高（需要论证）
  · 适合中高级学员
```

### 6.6 任务设计原则

```text
■ 有明确结果
  · 学员知道"要做什么"
  · 有"产出"（图/表/答案/决定）

■ 信息差/观点差
  · 必须交流才能完成
  · 不会"无话可说"

■ 难度适中
  · 略高于学员水平
  · 不太简单，不太难

■ 真实/模拟情境
  · 学员有代入感
  · 不是"为说而说"
```

## 七、输出练习

### 7.1 三类输出

```text
┌──────────┬────────────────────────┬──────────────┐
│ 类型      │ 描述                    │ 适合          │
├──────────┼────────────────────────┼──────────────┤
│ 独白      │ 一人讲话                │ 自我介绍/演讲 │
│ 对话      │ 两人交流                │ 日常对话      │
│ 讨论      │ 多人讨论                │ 课堂讨论      │
└──────────┴────────────────────────┴──────────────┘
```

### 7.2 独白练习

```text
■ 自我介绍
  · 1-2 分钟
  · 个人基本信息/兴趣/梦想

■ 演讲
  · 2-3 分钟
  · 观点+论证

■ 故事讲述
  · 复述故事
  · 看图说话
```

### 7.3 对话练习

```text
■ 配对对话
  · 学员两两配对
  · 围绕主题对话

■ 角色对话
  · 一人扮演一个角色
  · 围绕情境对话

■ 自由对话
  · 不指定话题
  · 自由交流
```

### 7.4 讨论练习

```text
■ 小组讨论
  · 3-5 人一组
  · 围绕主题讨论

■ 圆桌讨论
  · 全班一起讨论
  · 老师引导

■ 立场讨论
  · 学员持不同立场
  · 用英语论证
```

## 八、反馈与纠正

### 8.1 反馈三方式

```text
┌──────────┬────────────────────────┬──────────────┐
│ 方式      │ 描述                    │ 适用          │
├──────────┼────────────────────────┼──────────────┤
│ 即时反馈  │ 立刻纠正                │ 关键错误      │
│ 延时反馈  │ 课后/活动后             │ 重要错误      │
│ 学员自评  │ 学员自己发现            │ 培养意识      │
│ 同伴反馈  │ 学员互相反馈            │ 培养敏感度    │
└──────────┴────────────────────────┴──────────────┘
```

### 8.2 重述式纠正

```text
■ 老师不直接说"错"
  · 学员："I go to school yesterday."
  · 老师不直接说"I went"
  · 而是："Oh, you went to school yesterday? When?"

■ 让学员在"对话"中自己发现
  · 不打断学员说话
  · 让学员"自然修正"
```

### 8.3 纠错三原则

```text
■ 不打断
  · 学员说话流畅性 > 准确性
  · 错的不影响意思，先不管

■ 关注"可懂"
  · 让学员"说清楚"
  · 不要求"完美"

■ 关键错误
  · 影响交际的错误要纠正
  · 学员已掌握的规则错要纠正
```
