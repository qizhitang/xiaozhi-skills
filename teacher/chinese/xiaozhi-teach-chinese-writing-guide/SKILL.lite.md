# 写作教学指导 · 精简版

> 详细版见 `SKILL.md`，本文件用于快速启动。

## 1. 一句话定位

写作教学不是教学生"写好作文"，而是教学生"用自己的话说自己的话"——风格 DNA 是核心。

## 2. 工作流

```text
任务设计 → 过程引导 → 三维批改 → 风格 DNA → 讲评
```

## 3. 三维批改速查

```text
结构（5 项）：标题 / 开篇 / 段落 / 详略 / 结尾
语言（5 项）：用词 / 句式 / 具体 / 节奏 / 规范
思维（5 项）：主题 / 深度 / 视角 / 真实 / 共鸣
总分 12 分制。
```

## 4. 过程引导速查

```text
构思 15-20 分钟：5W + 思维导图 + 素材清单
起草 30-40 分钟：边想边写 + 段落感
修改 10-15 分钟：三自问 + 三删 + 一改
```

## 5. 风格 DNA 维度

```text
词汇 / 句式 / 修辞 / 情感 / 思维 / 结构
```

## 6. 讲评三段式

```text
共性观察 5-10 分钟
典型样本 15-20 分钟（优/中/待改 各 1）
可操作建议 5-10 分钟
```

## 7. 必触发场景

- 老师："这周写什么作文"
- 老师："作文怎么批改"
- 老师："写作讲评怎么上"
- 老师："学员作文怎么打分"
- 老师："怎么教写作风格"
