# 写作任务设计模板

> 配合 `xiaozhi-teach-chinese-writing-guide` 使用。避免套路题、虚假情境、单一主题。

## 1. 任务设计三原则

```text
原则 ① 真实情境：学员有真实表达欲望
原则 ② 思维门槛：任务有思考空间
原则 ③ 风格自由：给学员留风格空间
```

## 2. 三类任务对比

```text
┌──────────┬────────────────────────┬────────────┬──────────────┐
│ 任务类型  │ 描述                    │ 难度      │ 适用场景      │
├──────────┼────────────────────────┼────────────┼──────────────┤
│ 命题作文  │ 完全给定题目             │ 中等      │ 单元写作      │
│ 半命题    │ 给出部分题目，学员补充   │ 中等      │ 给学员空间    │
│ 情境写作  │ 给情境和体裁，自定主题   │ 较高      │ 思维训练      │
└──────────┴────────────────────────┴────────────┴──────────────┘
```

## 3. 任务设计样板（5 类）

### 3.1 记叙文 · 真实生活

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 写作任务样板 ①
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 命题方向
  主题：那一刻，我长大了
  真实情境：某个具体的瞬间（不是抽象的"我懂事了"）

■ 任务卡
  体裁：记叙文
  字数：500-600 字
  对象：写给自己
  目的：记录
  时长：2 课时

■ 思维引导
  · 切入点 1：某个具体事件（摔跤、失败、被误解、帮助他人）
  · 切入点 2：当时的具体感受
  · 切入点 3：现在的回望
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 3.2 议论文 · 观点表达

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 写作任务样板 ②
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 命题方向
  主题：手机对学习的影响（自选立场）
  真实情境：学员自己的手机使用经验

■ 任务卡
  体裁：议论文
  字数：600-800 字
  对象：同学
  目的：说服
  时长：2 课时

■ 思维引导
  · 切入点 1：手机的具体作用
  · 切入点 2：手机的具体问题
  · 切入点 3：自己的"边界"
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 3.3 说明文 · 介绍

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 写作任务样板 ③
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 命题方向
  主题：介绍一种你熟悉的事物
  真实情境：自己有相关经验

■ 任务卡
  体裁：说明文
  字数：500-700 字
  对象：陌生读者
  目的：介绍
  时长：2 课时

■ 思维引导
  · 切入点 1：事物是什么
  · 切入点 2：事物怎么运作
  · 切入点 3：事物的意义
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 3.4 应用文 · 实用

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 写作任务样板 ④
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 命题方向
  主题：给某人一封信/邀请/通知
  真实情境：实际需要

■ 任务卡
  体裁：应用文
  字数：300-500 字
  对象：具体收件人
  目的：实用
  时长：1 课时

■ 思维引导
  · 切入点 1：写信原因
  · 切入点 2：具体内容
  · 切入点 3：期待回复
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 3.5 文学创作 · 想象

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 写作任务样板 ⑤
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 命题方向
  主题：基于某个设定的故事
  真实情境：真实情感 + 想象情境

■ 任务卡
  体裁：故事/小说
  字数：800-1000 字
  对象：同学
  目的：娱乐+表达
  时长：3 课时

■ 思维引导
  · 切入点 1：人物设定
  · 切入点 2：核心冲突
  · 切入点 3：情感落点
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 4. 任务设计自检清单

```text
□ 学员有真实表达欲望吗？
□ 任务有思考空间吗？
□ 给学员留了风格空间吗？
□ 主题与学员生活相关吗？
□ 字数/时长是否合适？
□ 评估维度清晰吗？
□ 版权/出处标注了吗？
```

## 5. 题目库版权

```text
✅ 自有：老师原创
✅ 改编：基于教材/经典题目改编
✅ 教材原文：来源教材的题目
✅ 公开可引用：CC 协议资源

❌ 禁止：未授权复制教辅题库
❌ 禁止：未授权转载网络作文题
```
