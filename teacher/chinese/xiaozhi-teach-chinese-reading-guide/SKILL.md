# 阅读教学指导 SKILL

> **一句话定位：** 阅读教学不是教学生"读懂一段话"，而是教学生"和文本对话的能力"——从字面到结构到主题，三阶递进。

---

## ⚠️ 技术实现边界声明

> **关于"AI 阅读"边界：** 本 SKILL 输出**阅读教学设计**与**文本解读框架**；**不**代替老师备课，但能极大缩短老师备课时间。
>
> **关于"教材版权"边界：** 课文选段必须标注 copyrightStatus（教材原文/公有/经授权/CC 协议）；不复制整本教材内容。
>
> **关于"导读材料"边界：** 本 SKILL **不**直接提供完整电子版教材；只输出导读提纲、讨论问题、阅读任务设计。

## 一、核心使命

语文阅读教学常见的三个误区：

```text
误区① 教师讲=学生读：老师把课文"翻译"成白话，
        学员被动接受，没有和文本直接对话。

误区② 只读不思：学员读完不能"入心"，
        读完就忘，更不能用阅读经验。

误区③ 训练=刷题：阅读理解=答题模板，
        学员记住"套路"但不会真正"读"。
```

本 SKILL 要解决的是：
- **让学员直接和文本对话**：老师不再是"翻译机"
- **让阅读有思维层级**：字面 → 结构 → 主题
- **让阅读策略显性化**：6 大策略可教可练
- **让阅读成长可视化**：学员能看到自己"读力"成长

## 四、文本三解

### 4.1 三解定义

```text
┌──────────┬────────────────────────┬────────────┬──────────────┐
│ 层级      │ 描述                    │ 关键问题     │ 学员任务      │
├──────────┼────────────────────────┼────────────┼──────────────┤
│ 字面解    │ 文本说了什么            │ 这段讲什么  │ 提取信息      │
│ 结构解    │ 文本怎么写的            │ 怎么组织的  │ 分析结构      │
│ 主题解    │ 文本想表达什么          │ 为什么这样  │ 思考意义      │
└──────────┴────────────────────────┴────────────┴──────────────┘
```

### 4.2 字面解 5 问

```text
① 写了谁/什么？
② 发生在什么时间/地点？
③ 经历了什么过程？
④ 关键细节是什么？
⑤ 我能用自己的话讲一遍吗？
```

### 4.3 结构解 5 问

```text
① 这段/这篇文章的顺序是什么？
② 哪里是重点/详写？
③ 哪里是次要/略写？
④ 作者用了什么方法（比喻/排比/举例/对比）？
⑤ 如果我来写，会怎么组织？
```

### 4.4 主题解 5 问

```text
① 这篇文章的核心是什么？
② 作者想表达什么情感/观点？
③ 我同意/不同意吗？为什么？
④ 这篇文章和我的生活有什么联系？
⑤ 如果再写一遍，我会怎么改？
```

### 4.5 文本三解样板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📖 文本三解样板 · [《X》]
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 字面解
  · 写了谁/什么：[X]
  · 关键细节：[X]
  · 故事梗概：[X]

■ 结构解
  · 顺序：[顺序/倒叙/插叙]
  · 详写：[X]
  · 略写：[X]
  · 主要方法：[X]

■ 主题解
  · 核心主题：[X]
  · 作者表达：[X]
  · 联系生活：[X]
  · 我的看法：[X]
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 五、阅读策略（6 大策略）

### 5.1 策略总览

```text
┌──────────┬────────────────────────┬────────────┬──────────────┐
│ 策略      │ 描述                    │ 关键动作     │ 适用文体      │
├──────────┼────────────────────────┼────────────┼──────────────┤
│ 预测      │ 读前/读中推测下文        │ 猜一猜      │ 叙事/说明     │
│ 提问      │ 读中提问/找答案          │ 问自己      │ 任何文体      │
│ 可视化    │ 把文字变成图像           │ 画出来      │ 叙事/描写     │
│ 联结      │ 联系旧知/生活            │ 联想        │ 任何文体      │
│ 推断      │ 透过字面推断深层          │ 想一想      │ 任何文体      │
│ 总结      │ 概括/提炼核心            │ 用一句话   │ 任何文体      │
└──────────┴────────────────────────┴────────────┴──────────────┘
```

### 5.2 策略教学法

```text
■ 显性教学
  · 直接告诉学员"我们现在用'提问'策略"
  · 让学员意识到"我在用什么策略"
  · 避免"无意识使用"

■ 示范
  · 老师先做一遍
  · 边做边"出声思维"
  · 学员观察老师的"思考过程"

■ 搭档练习
  · 学员配对合作
  · 一个用策略，另一个观察
  · 然后切换

■ 独立使用
  · 学员独立运用
  · 老师观察+反馈
  · 形成"自动化"
```

### 5.3 策略选择原则

```text
策略不是越多越好。
选 1-2 个策略和文本特点匹配，深度练习。

· 叙事文：预测+可视化
· 议论文：提问+推断
· 说明文：总结+提问
· 文学：联结+主题
```

### 5.4 策略使用样板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📖 阅读策略使用样板 · [《X》]
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 本文主策略：[策略 1] + [策略 2]

■ 策略 1：提问
  · 我读到的：[段落摘录]
  · 我的问题：[X]
  · 我的回答：[X]
  · 答案的"线索"在哪里？

■ 策略 2：可视化
  · 这段我想画：[X 画面]
  · 关键元素：[X]
  · 色彩/位置：[X]

■ 策略使用反思
  · 这个策略帮我理解了什么？
  · 下次我还用吗？
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 六、三阶阅读

### 6.1 三阶定义

```text
┌──────────┬────────────────────────┬────────────┬──────────────┐
│ 阶       │ 描述                    │ 学员表现    │ 老师策略      │
├──────────┼────────────────────────┼────────────┼──────────────┤
│ 基础阶   │ 知道"写了什么"          │ 能复述     │ 教提取信息    │
│ 分析阶   │ 知道"怎么写的"          │ 能分析     │ 教结构方法    │
│ 思辨阶   │ 知道"为什么写/有何意义" │ 能评价     │ 教思维批判    │
└──────────┴────────────────────────┴────────────┴──────────────┘
```

### 6.2 基础阶（提取信息）

```text
■ 任务
  · 找出关键人物/事件
  · 提取关键信息
  · 复述内容

■ 老师方法
  · 给"寻读"任务
  · 画结构图
  · 完成"5W1H"表
```

### 6.3 分析阶（分析结构）

```text
■ 任务
  · 分析作者"怎么写"
  · 找出"为什么这样写"
  · 评价"写得好不好"

■ 老师方法
  · 比较"两种写法"
  · 改写某段，让学员体会差异
  · 讨论"如果是我，会怎么写"
```

### 6.4 思辨阶（评价思辨）

```text
■ 任务
  · 评价"为什么写"
  · 联系现实/历史
  · 形成自己的观点

■ 老师方法
  · 给开放性问题
  · 正反方讨论
  · 联系学员生活
```

### 6.5 三阶练习设计

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📖 三阶练习设计 · [《X》]
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 基础阶（5 分钟）
  · 任务 1：找 X 个关键信息
  · 任务 2：用自己的话讲一遍

■ 分析阶（10 分钟）
  · 任务 1：分析作者怎么写
  · 任务 2：改写一段，对比效果

■ 思辨阶（10 分钟）
  · 任务 1：作者为什么这样写？
  · 任务 2：联系你的生活
  · 任务 3：你同意作者吗？
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 七、阅读任务设计

### 7.1 三段任务

```text
┌──────────┬────────────────────────┬────────────┐
│ 阶段      │ 任务类型                │ 时长        │
├──────────┼────────────────────────┼────────────┤
│ 预习      │ 字面解+1 个问题         │ 15-20 分钟  │
│ 课中      │ 三阶练习+讨论          │ 40-45 分钟  │
│ 课后      │ 拓展阅读+读后任务       │ 30-60 分钟  │
└──────────┴────────────────────────┴────────────┘
```

### 7.2 预习任务设计

```text
■ 字面解
  · 读 X 遍
  · 用自己的话讲一遍内容

■ 提 1-2 个问题
  · 不理解的地方
  · 好奇的地方
  · 反对/怀疑的地方

■ 找 1 个细节
  · 印象最深的细节
  · 划出来

❌ 禁止：预习=抄生字词
❌ 禁止：预习=查工具书完成任务
```

### 7.3 课中任务设计

```text
■ 字面层（5 分钟）
  · 快速复述内容
  · 解决预习中"不理解"的问题

■ 分析层（15 分钟）
  · 重点段落分析
  · 策略使用演练
  · 小组讨论

■ 思辨层（20 分钟）
  · 开放性问题
  · 联系学员生活
  · 表达自己的观点

■ 总结层（5 分钟）
  · 用 1-2 句话总结
  · 收尾
```

### 7.4 课后任务设计

```text
■ 拓展阅读
  · 同主题 1-2 篇
  · 比较阅读

■ 读后任务
  · 写读后感
  · 画思维导图
  · 与同伴/家长讨论

■ 不布置
  · 抄写课文
  · 无意义刷题
```

## 八、群文阅读

### 8.1 群文阅读定义

```text
围绕 1 个议题，选 3-5 篇文本，
在 1-2 课时内集中对比阅读。
```

### 8.2 群文设计四步

```text
① 选议题
  · 主题/作者/体裁/时代/表达方式

② 选文本
  · 3-5 篇
  · 难度有梯度
  · 彼此有关联

③ 设问题
  · 1 个核心问题
  · 2-3 个子问题
  · 开放可讨论

④ 找结构
  · 横向比较
  · 纵向深入
  · 由字面到思辨
```

### 8.3 群文样板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📖 群文阅读设计 · [议题：童年]
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 议题：童年

■ 文本
  · 文本 ①：鲁迅《从百草园到三味书屋》（节选）
  · 文本 ②：林海音《城南旧事》（节选）
  · 文本 ③：萧红《呼兰河传》（节选）

■ 核心问题
  · 三位作家笔下的"童年"有何不同？

■ 子问题
  · ① 童年是什么色彩？
  · ② 童年与成人世界的关系？
  · ③ 你认同谁的"童年"？

■ 结构
  · 横向：比较三个童年
  · 纵向：深入一个童年
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

```text
  （阅读力维度）    （阅读素材入库）  （阅读进步反馈）
```

### 10.2 接口

```text
读：
  lessonPlan.text         → 课文/文本信息
  studentAnalyzer.weaknessRank → 阅读力弱项

写：
  readingGuide.threeSolve → 文本三解
  readingGuide.threeStage → 三阶练习
  readingGuide.bookList   → 书单/读后任务
  → student-analyzer 接收
  → resource-library 接收
```

## 十二、行为准则

| ✅ 应该做 | ❌ 不能做 |
|---------|---------|
| 文本三解（字面/结构/主题） | 老师=翻译机 |
| 显性教阅读策略 | 学员无意识使用 |
| 三阶阅读（基础/分析/思辨） | 训练=答题模板 |
| 群文阅读对比设计 | 单篇孤讲 |
| 阅读成长档案累积 | 学员不积累 |
| 预习=读+提问 | 预习=抄生字 |
| 课后=拓展阅读 | 课后=抄课文 |
