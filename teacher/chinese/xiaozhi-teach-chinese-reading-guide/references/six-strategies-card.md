# 阅读 6 大策略卡片

> 配合 `xiaozhi-teach-chinese-reading-guide` 使用。6 大阅读策略（预测/提问/联想/比较/概括/评价）的卡片化教学工具，老师可裁剪发给学员。

## 1. 6 大策略总览

```text
┌──────┬──────────┬──────────────────┬──────────────┐
│ #    │ 策略      │ 核心问题          │ 适用阶段      │
├──────┼──────────┼──────────────────┼──────────────┤
│ 1    │ 预测      │ 接下来会发生什么  │ 阅读前/阅读中 │
│ 2    │ 提问      │ 我有什么不明白的  │ 阅读中        │
│ 3    │ 联想      │ 这让我想到什么    │ 阅读中/阅读后 │
│ 4    │ 比较      │ 与其他文本有何异同│ 阅读中/阅读后 │
│ 5    │ 概括      │ 这段讲了什么      │ 阅读中/阅读后 │
│ 6    │ 评价      │ 我认同/质疑什么   │ 阅读后        │
└──────┴──────────┴──────────────────┴──────────────┘
```

## 2. 卡片 ①：预测

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔮 策略 1：预测
━━━━━━━━━━━━━━━━━━━━━━━━━━━
【作用】激活先验知识，提升阅读兴趣
【时机】阅读前 / 阅读中

【操作步骤】
  ① 看标题/封面，预测主题
  ② 读前两段，预测走向
  ③ 找关键词，预测情节
  ④ 对比实际，修正预测

【示范问题】
  · "看到标题《______》，你猜会讲什么？"
  · "读到'他愣住了'，你觉得接下来会怎样？"

【学员自检】
  □ 我是否看了标题就猜过？
  □ 我是否每段都猜过？
  □ 我的预测准不准？

【老师使用要点】
  · 不要否定学员的"离谱"预测
  · 把"为什么这样预测"作为追问点
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 3. 卡片 ②：提问

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
❓ 策略 2：提问
━━━━━━━━━━━━━━━━━━━━━━━━━━━
【作用】从被动接收转为主动探究
【时机】阅读中

【操作步骤】
  ① 标注不懂的字词
  ② 标注读不懂的句子
  ③ 标注想跟作者对话的地方
  ④ 写出"我的问题"

【五类好问题】
  · 字面问题：这个词什么意思？
  · 推断问题：作者为什么写这个？
  · 评价问题：我同意吗？
  · 联系问题：这和我的生活有什么联系？
  · 创造问题：如果是我会怎么写？

【学员自检】
  □ 我每段都提了问题吗？
  □ 我的问题有不同层次吗？

【老师使用要点】
  · 鼓励"笨问题"（字面问题也重要）
  · 学员的提问能力比答题能力更重要
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 4. 卡片 ③：联想

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
💭 策略 3：联想
━━━━━━━━━━━━━━━━━━━━━━━━━━━
【作用】激活背景经验，深化理解
【时机】阅读中 / 阅读后

【操作步骤】
  ① 遇到关键词 → 想到自己的经历
  ② 遇到场景 → 想到相似场景
  ③ 遇到情感 → 想到自己的情感
  ④ 联系现实 → 想到当下社会

【三类联想】
  · 经验联想：我有过类似经历
  · 文本联想：我想到了另一篇文章
  · 现实联想：当下也有类似现象

【示范问题】
  · "读到这里你想到什么？"
  · "你的生活中有类似画面吗？"

【学员自检】
  □ 我有联想到自己的经历吗？
  □ 我有联想到其他书/电影吗？

【老师使用要点】
  · 不要让联想"跑太远"
  · 联想后要"拉回文本"
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 5. 卡片 ④：比较

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔄 策略 4：比较
━━━━━━━━━━━━━━━━━━━━━━━━━━━
【作用】通过对比突出特征，加深理解
【时机】阅读中 / 阅读后

【操作步骤】
  ① 找比较对象（另一篇/另一段）
  ② 找比较维度（主题/人物/写法/情感）
  ③ 列出异同
  ④ 总结"为何不同"

【五类比较】
  · 主题比较：两文都讲了什么？异同？
  · 人物比较：两个角色有何不同？
  · 写法比较：叙事/描写/抒情上有什么差异？
  · 情感比较：作者态度有何不同？
  · 结构比较：篇章组织有何差异？

【示范问题】
  · "如果把这段换成另一种写法会怎样？"
  · "这两篇文章你更喜欢哪篇？为什么？"

【学员自检】
  □ 我有比较过类似文本吗？
  □ 我能说出至少 3 个异同吗？
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 6. 卡片 ⑤：概括

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 策略 5：概括
━━━━━━━━━━━━━━━━━━━━━━━━━━━
【作用】提取关键信息，把握文本主干
【时机】阅读中 / 阅读后

【操作步骤】
  ① 划分段落，标出段意
  ② 用一句话概括每段
  ③ 串联段意，得出全文主旨
  ④ 检查是否完整

【三层概括】
  · 段落层：每段 1 句话
  · 文章层：全文 1-2 句话
  · 主题层：作者要表达的核心

【示范问题】
  · "这段能用一句话概括吗？"
  · "如果只能记住全文一句话，是什么？"

【常见障碍】
  · 概括得太详细 → 没概括
  · 概括得太笼统 → 没信息量
  · 概括=抄原文 → 没加工

【学员自检】
  □ 我能一句话说出每段意思吗？
  □ 我能一句话说出全文意思吗？
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 7. 卡片 ⑥：评价

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚖️ 策略 6：评价
━━━━━━━━━━━━━━━━━━━━━━━━━━━
【作用】形成独立判断，培养批判性思维
【时机】阅读后

【操作步骤】
  ① 明确自己的判断（喜欢/不喜欢/同意/不同意）
  ② 找出支持判断的文本依据
  ③ 区分"事实"与"观点"
  ④ 提出"我的不同看法"

【四类评价】
  · 内容评价：主题/立意我认同吗？
  · 形式评价：写法/结构我欣赏吗？
  · 情感评价：是否打动了我？
  · 价值评价：体现的价值观我认同吗？

【示范问题】
  · "你最喜欢哪一段？为什么？"
  · "如果你是作者会怎么改？"
  · "你同意作者的观点吗？"

【老师使用要点】
  · 评价 ≠ 赞美，鼓励学员说"不喜欢"
  · 评价必须基于文本依据
  · 学员的"我不同意"是珍贵的
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 8. 6 大策略使用顺序

```text
阅读前：预测 → 提问
阅读中：预测 → 提问 → 联想 → 概括
阅读后：联想 → 比较 → 评价 → 概括
```

## 10. 老师使用方式

```text
方式 ① 课前发卡片
  → 每次课前发 1 张策略卡
  → 学员读完按策略操作

方式 ② 课中穿插
  → 讲到对应段落时调用策略
  → 现场示范策略使用

方式 ③ 课后作业
  → 选一篇课外文本，用 3 种策略记录

方式 ④ 长期跟踪
  → 用"使用检查表"跟踪学员策略能力
```
