# 文本三解模板（基础 / 深层 / 外延）

> 配合 `xiaozhi-teach-chinese-reading-guide` 使用。从"字面-作者-读者"三层次拆解文本，是阅读教学的基础工具。

## 1. 三解的定义

```text
┌──────────┬──────────────────┬──────────────────────────┐
│ 层次      │ 名称              │ 关注重心                  │
├──────────┼──────────────────┼──────────────────────────┤
│ 基础解    │ 字面层            │ 文章写了什么             │
│ 深层解    │ 作者层            │ 作者为什么这样写         │
│ 外延伸    │ 读者层            │ 读者读后想到什么         │
└──────────┴──────────────────┴──────────────────────────┘
```

## 2. 三解操作流程

```text
┌────────────────────────────────────────┐
│ 第一步：通读 1-2 遍，把握文本大意       │
│ 第二步：基础解 → 拆解文本写了什么      │
│ 第三步：深层解 → 推断作者意图          │
│ 第四步：外延伸 → 联系读者自身          │
│ 第五步：整合 → 三层关系图              │
└────────────────────────────────────────┘
```

## 3. 基础解模板

### 3.1 文本基本信息

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📖 基础解 · 文本基本信息
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 标题：
■ 作者：
■ 出处 / 选自：
■ 体裁：□ 记叙文  □ 散文  □ 议论文  □ 说明文  □ 小说  □ 诗歌
■ 字数：
■ 阅读时长预估：
■ 难度：□ 入门  □ 基础  □ 进阶  □ 挑战
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 3.2 内容结构

```text
【主要内容】
  · 主题一句话：[本文写了……]
  · 关键人物：[化名 1 / 化名 2 / ...]
  · 关键事件：[事件 1] / [事件 2] / [事件 3]
  · 关键地点：[地点 1] / [地点 2]
  · 时间跨度：[起始时间] → [结束时间]

【篇章结构】
  · 结构类型：□ 总分总  □ 时间顺序  □ 空间顺序  □ 倒叙  □ 插叙  □ 并列
  · 段落数：[N] 段
  · 重点段落：第 [N] 段
```

### 3.3 字词句清单

```text
【生字词】5-10 个
  · [生字 1] — 拼音 + 含义 + 文中用法
  · [生字 2] — ...
  · ...

【关键句】3-5 句
  · 句 1：...
  · 句 2：...
  · ...

【修辞】3-5 处
  · 第 [N] 段：[修辞] — 效果
  · ...
```

## 4. 深层解模板

### 4.1 写作背景

```text
【作者背景】
  · 作者：[姓名]
  · 生卒：[年份]
  · 代表作：[作品 1] / [作品 2] / ...
  · 时代背景：[时代] — [时代特点]

【写作背景】
  · 写作时间：[年份]
  · 写作动机：□ 触景生情  □ 有感而发  □ 应邀而作  □ 教学需要
  · 写作对象：□ 公众  □ 特定读者  □ 自己
```

### 4.2 主题与立意

```text
【表层主题】
  · 文本表层在讲什么：[一句话]

【深层主题】
  · 文本真正想表达：[深层立意]
  · 文本的命题方向：[命题]

【价值观取向】
  · 作者认同什么：
  · 作者批判什么：
  · 作者态度倾向：□ 肯定  □ 批判  □ 同情  □ 反思  □ 中立
```

### 4.3 写作手法

```text
【选材】
  · 真实/虚构：□ 真实  □ 虚构  □ 半真半假
  · 典型性：□ 典型  □ 一般
  · 来源：□ 个人经历  □ 听闻  □ 想象  □ 改编

【表达方式】
  · 叙述：[占比]%
  · 描写：[占比]%
  · 抒情：[占比]%
  · 议论：[占比]%

【表现手法】
  · 象征：[象征物] = [象征义]
  · 对比：[对比 1] vs [对比 2]
  · 烘托：[主体] 被 [环境/他人] 烘托
  · 借物抒情：[物] → [情]
```

### 4.4 情感与态度

```text
【作者情感曲线】
  · 开篇情感：[情感 1]
  · 中段情感：[情感 2]
  · 结尾情感：[情感 3]
  · 情感变化：[起点] → [中点] → [终点]

【作者态度】
  · 对 [人物/事件] 的态度：...
  · 对 [社会现象] 的态度：...
```

## 5. 外延伸模板

### 5.1 与读者的连接

```text
【读者共鸣点】
  · 哪一段/句最让读者共鸣：
  · 共鸣原因：[经验/情感/价值观] 上的连接

【读者困惑点】
  · 哪些地方读者可能读不懂：
  · 需要补充的背景知识：

【读者联想点】
  · 读完会想到 [什么类似经历]：
  · 读完会想到 [哪本书/哪篇文章]：
```

### 5.2 与现实的联系

```text
【与个人经历】
  · 学员是否有过类似经历：□ 是  □ 否
  · 学员分享的相关经历：

【与社会现实】
  · 文本反映的社会现象：
  · 当下是否有类似现象：

【与时代精神】
  · 文本体现的价值观：
  · 当下时代是否仍适用：
```

### 5.3 跨文本联系

```text
【同主题作品】
  · 推荐阅读 1：[作品] — 共同点
  · 推荐阅读 2：[作品] — 共同点

【同作者作品】
  · 推荐阅读 1：[作品] — 风格比较

【对比阅读】
  · 推荐对比：[作品 A] vs [本文] — 异同
```

## 6. 三层关系图

```text
┌────────────────────────────────────────┐
│            基础解（字面层）              │
│   · 写了什么                            │
│   · 怎么写的                            │
│   ↓                                    │
│            深层解（作者层）              │
│   · 为什么这样写                        │
│   · 表达什么                            │
│   ↓                                    │
│            外延伸（读者层）              │
│   · 读者如何理解                        │
│   · 与自身/社会的连接                   │
└────────────────────────────────────────┘
```

## 7. 教学应用样板：《背影》(节选)

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📖 文本三解样板 · 《背影》(节选)
━━━━━━━━━━━━━━━━━━━━━━━━━━━

【基础解】
  · 主题：父亲送别儿子，爬月台买橘子
  · 关键人物：父亲、儿子
  · 关键事件：火车站送别、爬月台、买橘子
  · 重点段：第 6 段（爬月台买橘子）

【深层解】
  · 表层主题：父子送别
  · 深层立意：父爱无言 / 家庭代际关系
  · 写作动机：触景生情（回忆父亲）
  · 表达方式：叙述 60% + 描写 30% + 抒情 10%
  · 表现手法：细节描写（爬月台动作）、借物抒情（橘子）
  · 情感曲线：平静 → 不舍 → 流泪

【外延伸】
  · 读者共鸣：父亲为子女奔波的画面
  · 联想：学员自己的父母
  · 跨文本：可对比《我的母亲》(胡适)、《父爱之舟》
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
