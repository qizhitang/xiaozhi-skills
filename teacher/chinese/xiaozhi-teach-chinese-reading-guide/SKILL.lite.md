# 阅读教学指导 · 精简版

> 详细版见 `SKILL.md`。

## 1. 一句话定位

阅读教学不是教学生"读懂一段话"，而是教学生"和文本对话的能力"——从字面到结构到主题，三阶递进。

## 2. 工作流

```text
文本三解 → 策略选择 → 三阶练习 → 阅读档案
```

## 3. 文本三解速查

```text
字面解：写了什么（5W1H）
结构解：怎么写的（顺序/详略/方法）
主题解：为什么写（核心/情感/意义）
```

## 4. 6 大阅读策略

```text
预测（猜下文）/ 提问（问自己）/ 可视化（画出来）
联结（联旧知/生活）/ 推断（透过字面）/ 总结（一句话）
```

## 5. 三阶阅读

```text
基础阶：提取信息（5W1H）
分析阶：分析结构/方法
思辨阶：评价/联系生活
```

## 6. 三段任务

```text
预习：字面解 + 1-2 问题
课中：三阶练习 + 讨论
课后：拓展阅读 + 读后任务
```

## 7. 必触发场景

- 老师："这篇课文怎么讲"
- 老师："阅读理解怎么教"
- 老师："学员读不懂"
- 老师："阅读策略怎么教"
- 老师："群文阅读怎么设计"
- 老师："名著怎么导读"
