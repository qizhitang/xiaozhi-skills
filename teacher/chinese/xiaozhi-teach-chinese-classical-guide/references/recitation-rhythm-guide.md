### 2.3 正音训练步骤

```text
第 1 步：标出多音字
  → 学员在古文中标出所有多音字
  → 标注每个字的正确读音

第 2 步：老师范读
  → 老师读 1 遍，学员听音
  → 重点关注多音字的读音

第 3 步：学员跟读
  → 学员跟读 3 遍
  → 注意字音准确

第 4 步：检验
  → 学员独立读 1 遍
  → 老师纠正错误
```

## 3. 第 2 阶：断句训练

### 3.1 断句 5 法回顾

```text
① 句末虚词：也、矣、乎、哉、焉
② 对称结构："山高 / 水长"
③ 顶真修辞：A...A...
④ 重复结构：X 则 X、X 不 X
⑤ 标点参考：句号/逗号处必断
```

### 3.2 节奏 3 类

```text
· 四言诗：2+2 节奏（如"关关/雎鸠"）
· 五言诗：2+3 节奏（可细分 2+2+1 或 2+1+2，如"床前/明月光"）
· 七言诗：2+2+3 节奏（也可 4+3，如"两个/黄鹂/鸣翠柳"）
· 古文：按"意群"断，无固定节奏
```

### 3.3 断句训练步骤

```text
第 1 步：标虚词
  → 学员先标出"也/矣/乎/哉/焉"
  → 这些字前是停顿

第 2 步：找对称
  → 找对偶/排比/对称结构
  → 对称处可断

第 3 步：划意群
  · 主谓之间
  · 动宾之间
  · 修饰与中心词之间
  · 总分之间

第 4 步：整体调整
  → 检查停顿是否自然
  → 检查是否体现层次

第 5 步：试读
  → 学员按节奏试读
  → 老师反馈
```

### 3.4 断句练习样板

```text
【原文】学而时习之不亦说乎有朋自远方来不亦乐乎人不知而不愠不亦君子乎

【断句后】学而时习之 / 不亦说乎 / 有朋自远方来 / 不亦乐乎 / 人不知而不愠 / 不亦君子乎

【断句依据】
  · "乎"前可断（虚词法）
  · "不亦...乎"重复结构（重复法）
  · "时习之"中"时"修饰"习"（意群法）
```

## 4. 第 3 阶：情感训练

### 4.1 情感分类

```text
┌──────────┬──────────────┬──────────────┐
│ 情感类型  │ 典型篇目      │ 诵读特点      │
├──────────┼──────────────┼──────────────┤
│ 思乡      │ 《静夜思》   │ 缓慢/低沉    │
│ 爱国      │ 《示儿》     │ 激昂/坚定    │
│ 送别      │ 《送杜少府》 │ 平稳/深沉    │
│ 山水      │ 《山居秋暝》 │ 舒缓/悠然    │
│ 哲理      │ 《论语》     │ 平稳/庄严    │
│ 边塞      │ 《凉州词》   │ 雄浑/豪迈    │
│ 闺怨      │ 《闺怨》     │ 幽怨/低回    │
└──────────┴──────────────┴──────────────┘
```

### 4.2 情感训练步骤

```text
第 1 步：理解情感
  · 学员先理解诗文情感
  · 找出情感关键词

第 2 步：标轻重音
  · 重点词重读
  · 修饰词轻读
  · 节奏词连贯

第 3 步：调整语速
  · 欢快：稍快
  · 悲伤：稍慢
  · 激昂：稍快+重音
  · 平静：平稳

第 4 步：试读
  · 学员带情感试读
  · 老师反馈

第 5 步：配乐
  · 选合适背景音乐
  · 学员再次诵读
```

### 4.3 情感诵读样板

```text
【原文】床前明月光，疑是地上霜。举头望明月，低头思故乡。（李白《静夜思》）

【情感分析】
  · 思乡之情
  · 夜晚独处
  · 由景生情

【诵读指导】
  · "床前"平起
  · "明月光"重读"明"
  · "疑是" 略停顿
  · "地上霜" 平读
  · "举头" 轻起
  · "望明月" 略重
  · "低头" 慢起
  · "思故乡" 慢+重+长
```

## 5. 诵读教学法 4 类

```text
方法 ① 老师范读
  · 老师读 3 遍（正音/断句/情感）
  · 学员听

方法 ② 学员跟读
  · 学员跟老师读
  · 多次重复

方法 ③ 配乐诵读
  · 选古风/民乐
  · 让学员"沉浸"

方法 ④ 角色诵读
  · 叙事古文：角色扮演
  · 诗歌：分角色读
```
