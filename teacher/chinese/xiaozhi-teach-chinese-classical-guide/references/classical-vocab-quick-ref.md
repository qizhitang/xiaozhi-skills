## 3. 句式 5 类

```text
┌──────────┬────────────────────────┬──────────────┐
│ 句式      │ 描述                    │ 例句         │
├──────────┼────────────────────────┼──────────────┤
│ 判断句    │ ...者，...也            │ 陈胜者，阳城人也│
│ 被动句    │ ...为...所...           │ 为...所困    │
│           │ 见...于...              │ 见笑于大方   │
│ 倒装句    │ 介词后置                │ 何以...      │
│           │ 主谓倒装                │ 甚矣，汝之不惠│
│           │ 定语后置                │ 求人可使报秦者│
│ 省略句    │ 省略主语/宾语/介词      │ （省略主语） │
│ 固定句    │ 何以...为（要...做什么）│ 何以家为     │
│           │ 如...何（拿...怎么办） │ 奈何         │
│           │ ...孰...（...哪个更...）│ 我孰与城北徐公美│
└──────────┴────────────────────────┴──────────────┘
```

## 5. 常见通假字

├──────────┼────────────────────────┼──────────────┤

## 6. 教学建议

```text
■ 不死记硬背
  · 在语境中理解
  · 联系同主题/同作者

■ 古今对照
  · 古义 vs 今义
  · 让学员体会"语言的变化"

■ 应用检验
  · 完成 X 个例句翻译
  · 学以致用
```
