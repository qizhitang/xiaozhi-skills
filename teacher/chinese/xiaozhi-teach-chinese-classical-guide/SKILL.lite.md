# 文言文教学指导 · 精简版

> 详细版见 `SKILL.md`。

## 1. 一句话定位

文言文不是"古人的语文"，而是"中国文化的活化石"——教学要让它在现代学员心里"活"起来。

## 2. 工作流

```text
诵读正音 → 训诂（字词句式） → 串讲 → 主题 → 文化+现代联结
```

## 3. 诵读三阶

```text
正音：字音/多音字
断句：句读/对称/虚词
情感：轻重/缓急
```

## 4. 训诂三阶

```text
字词：常用义/语境义
句式：倒装/省略/判断
语法：词类活用/虚词
```

## 5. 串讲三步

```text
整体感知（文体/核心）
分段讲解（重点段+重点句）
整合（段间关系/整体结构）
```

## 6. 主题三维度

```text
思想：核心观点
情感：作者/人物情感
价值：对今天的意义
```

## 7. 诗词鉴赏四维

```text
意象：月/酒/柳/雁/...
意境：意象+情感的整体
手法：比兴/夸张/对偶/用典
情感：诗人的情感
```

## 8. 必触发场景

- 老师："这篇文言文怎么讲"
- 老师："古诗怎么上"
- 老师："学员读不懂文言文"
- 老师："实词虚词怎么教"
- 老师："文言文主题怎么把握"
- 老师："诗词鉴赏怎么教"
