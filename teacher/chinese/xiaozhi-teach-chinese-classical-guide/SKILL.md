# 文言文教学指导 SKILL

> **一句话定位：** 文言文不是"古人的语文"，而是"中国文化的活化石"——教学要让它在现代学员心里"活"起来。

---

## ⚠️ 技术实现边界声明

> **关于"教材版权"边界：** 古文原文本身属于公有领域（作者去世 50+ 年）；但**教材版本的注释/翻译/设计**仍受版权保护。
>
> **关于"AI 串讲"边界：** 本 SKILL 输出**串讲框架**与**主题解析**；**不**逐字逐句串讲整篇古文（由老师主导）。
>
> **关于"AI 翻译"边界：** 本 SKILL **不**提供完整现代汉语翻译；只输出关键句段翻译框架（由老师主导）。

---

## 一、核心使命

文言文教学常见的三个误区：

```text
误区① 串讲=翻译：老师把古文"翻译"成现代汉语，
        学员被动接受，没有和古文直接接触。

误区② 训诂=死记：要求学员死记硬背实词虚词，
        没有语境、没有应用。

误区③ 主题=贴标签：把古文主题归结为几个标签
        （思乡/爱国/山水），学员记不住、感受不到。
```

本 SKILL 要解决的是：
- **让古文"可诵读"**：通过节奏、韵律让古文"动"起来
- **让训诂"语境化"**：实词虚词在语境中理解
- **让主题"有共鸣"**：与现代学员生活连接
- **让文化"可累积"**：每篇古文成为学员文化 DNA 的一部分

## 四、诵读正音

### 4.1 诵读三阶

```text
┌──────────┬────────────────────────┬────────────┬──────────────┐
│ 阶       │ 描述                    │ 重点        │ 学员任务      │
├──────────┼────────────────────────┼────────────┼──────────────┤
│ 正音      │ 读准字音                 │ 字音/多音  │ 跟读 3 遍     │
│ 断句      │ 读对节奏                 │ 句读/停顿  │ 划节奏 1 遍   │
│ 情感      │ 读出意味                 │ 轻重/缓急  │ 试读 2 遍     │
└──────────┴────────────────────────┴────────────┴──────────────┘
```

### 4.2 断句 5 法

```text
① 句末虚词
  · 也、矣、乎、哉、焉、耳
  · 多为语气词，可停顿

② 对称结构
  · "山高 / 水长" "天时 / 地利"
  · 对称处可断

③ 顶真修辞
  · "A...A..." 顶真处可断

④ 重复结构
  · "X 则 X" "X 不 X"
  · 重复处可断

⑤ 标点参考
  · 句号/逗号处必断
```

### 4.3 节奏 3 类

```text
· 四言诗：2+2 节奏（如"关关/雎鸠"）
· 五言诗：2+3 节奏（可细分 2+2+1 或 2+1+2，如"床前/明月光"）
· 七言诗：2+2+3 节奏（也可 4+3，如"两个/黄鹂/鸣翠柳"）

古文：按"意群"断，无固定节奏
```

### 4.4 诵读教学法

```text
■ 老师范读
  · 1 遍正音
  · 1 遍断句
  · 1 遍带情感

■ 学员跟读
  · 先无声（看）
  · 后有声（读）
  · 最后有情（演）

■ 配乐诵读
  · 选古风/民乐
  · 让学员"沉浸"

■ 角色诵读
  · 叙事古文：角色扮演
  · 诗歌：分角色
```

## 五、训诂（字词句式）

### 5.1 训诂三阶

```text
┌──────────┬────────────────────────┬────────────┐
│ 阶       │ 描述                    │ 重点        │
├──────────┼────────────────────────┼────────────┤
│ 字词      │ 单字/词的含义           │ 常用义/语境义│
│ 句式      │ 古文句式特点            │ 倒装/省略/判断│
│ 语法      │ 词类活用/虚词          │ 活用/虚词   │
└──────────┴────────────────────────┴────────────┘
```

### 5.2 实词训诂 4 法

```text
① 字形推断
  · 形声字："江"从水，工声
  · 形旁表意

② 组词推断
  · 学=学习/学生
  · 联系现代汉语

③ 语境推断
  · "学而时习之"
  · "学"=学习（语境）

④ 联想推断
  · 联系同主题/同作者
  · 推断通用义
```

### 5.3 虚词训诂 7 类

```text
┌──────────┬────────────────────────┬──────────────┐
│ 虚词      │ 常见义                   │ 例子         │
├──────────┼────────────────────────┼──────────────┤
│ 之        │ 助词/代词/动词           │ 学而时习之   │
│ 者        │ 代词/语气词              │ 仁者爱人     │
│ 也        │ 句末语气词               │ 孔子曰...... │
│ 矣        │ 句末语气词（了）         │ 可行矣       │
│ 乎        │ 句末语气词/介词           │ 不亦说乎     │
│ 哉        │ 句末语气词（感叹）       │ 何其...哉   │
│ 焉        │ 句末语气词/代词          │ 忽焉在...   │
└──────────┴────────────────────────┴──────────────┘
```

### 5.4 词类活用 4 类

```text
① 动词作名词
  · "追亡逐北"（亡=逃亡的人，北=败逃的人）
  · 动词临时表示与该动作相关的人/事物

② 形容词作动词
  · "耻学于师"（耻=以……为耻）
  · 形容词带宾语，表动作或"以……为……"

③ 名词作动词
  · "吾师道也"（师=学习、从师）
  · "驴不胜怒，蹄之"（蹄=用蹄踢）
  · 名词处在谓语位置或带宾语时活用为动词

④ 名词作状语
  · "日积月累"（日/月=每天/每月）
  · 名词直接修饰动词，作状语
```

### 5.5 训诂教学法

```text
■ 不死记硬背
  · 强调"在语境中理解"
  · 反对孤立记忆

■ 联类串讲
  · 同一作者的不同篇目串起来
  · 同一字的不同义项串起来

■ 古今对照
  · 古义 vs 今义
  · 让学员体会"语言的变化"

■ 应用检验
  · 完成 X 个例句翻译
  · 学以致用
```

## 六、串讲

### 6.1 串讲三步

```text
① 整体感知
  · 这篇古文讲什么？
  · 是什么文体？（叙事/写景/说理/抒情）

② 分段讲解
  · 逐段讲解
  · 重点段落+重点句子
  · 不逐字逐句

③ 整合
  · 段间关系
  · 整体结构
```

### 6.2 串讲三问

```text
· 这段讲什么？
· 这段和前一段什么关系？
· 整篇在结构上有什么特点？
```

### 6.3 串讲样板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 文体
  [叙事/写景/说理/抒情]

■ 整体感知
  · 讲什么：[X]
  · 核心：[X]

■ 分段讲解
  · 第 1 段：[X]
    重点句子：[X]

  · 第 2 段：[X]
    重点句子：[X]

  · 第 3 段：[X]
    ...

■ 整合
```

## 七、主题

### 7.1 主题三维度

```text
┌──────────┬────────────────────────┬──────────────┐
│ 维度      │ 描述                    │ 学员任务      │
├──────────┼────────────────────────┼──────────────┤
│ 思想      │ 文章的核心观点           │ 概括思想      │
│ 情感      │ 作者/人物的情感         │ 体会情感      │
│ 价值      │ 对今天的意义             │ 联系现代      │
└──────────┴────────────────────────┴──────────────┘
```

### 7.2 主题教学法

```text
■ 不止于标签
  · 不只说"本文表达了思乡之情"
  · 而是说"为什么思乡？思到什么程度？"

■ 多角度讨论
  · 作者的角度
  · 文中人物的角度
  · 现代学员的角度

■ 联系现代
  · 古代的"思乡"和现代的"乡愁"有什么不同？
  · 古代的"爱国"和现代的"爱国"有什么不同？
```

### 7.3 主题讨论问题设计

```text
■ 思乡类
  · 古代人为什么这么"想家"？
  · 现代人想家是什么感觉？
  · 你有没有"想家"的经历？

■ 爱国类
  · 古代的"国"和现代的"国"有什么不同？
  · 古代"忠臣"的标准是什么？
  · 你认为今天还需要"忠"吗？

■ 山水类
  · 古代人为什么爱"隐居"？
  · 现代人如何与自然相处？
  · 你理想的生活环境是？

■ 哲理类
  · 古代圣人的智慧对我们今天还有用吗？
  · 你最欣赏古人的哪一点？
```
