# 物理解题教学指导 · 精简版

> 详细版见 `SKILL.md`。

## 1. 一句话定位

物理解题不是"套公式"，而是"建模型"——五步解题法是核心。

## 2. 工作流

```text
审题 → 建模 → 过程分析 → 列方程 → 求解 → 反思 → 变式训练
```

## 3. 五步解题法

```text
① 审题：三读（场景/已知/求）
② 建模：模型选择/简化/建构
③ 过程分析：受力/运动/能量
④ 列方程：规律/关系/约束
⑤ 求解：数学/讨论/检验
+ 反思（贯穿始终）
```

## 4. 审题三读

```text
一读：场景（讲什么）
二读：已知（给什么）
三读：求（求什么）
```

## 5. 建模三步

```text
模型选择：什么模型
简化：忽略什么
建构：画图（受力/运动/能量图）
```

## 6. 过程分析三步

```text
受力：画重力/接触力/场力
运动：分解+标参量
能量：找转化
```

## 7. 列方程

```text
规律：牛顿/运动学/能量/动量/欧姆
几何：位移/速度/加速度
约束：绳/杆/接触
```

## 8. 反思

```text
结果：单位/数量级/极限
过程：思路/模型/方程
拓展：一题多解/改条件/联系实际
```

## 9. 变式训练

```text
一题多解（多解法）
多题一解（同方法）
变条件（改条件/改问题/改情境）
```

## 10. 必触发场景

- 老师："这道物理题怎么讲"
- 老师："学员不会做物理题"
- 老师："物理建模怎么教"
- 老师："过程分析怎么教"
- 老师："多过程问题怎么讲"
- 老师："物理解题变式怎么设计"
