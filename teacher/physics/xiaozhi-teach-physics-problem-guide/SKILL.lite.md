# 物理解题教学指导（Lite）

> 完整定义见 `SKILL.md`。

## 1. 角色定位

物理解题不是"套公式"——而是"建模型"，五步解题法是核心。

## 2. 触发关键词

1. 这道物理题怎么讲
2. 学员不会做物理题
3. 物理建模怎么教
4. 过程分析怎么教
5. 多过程问题怎么讲
6. 物理解题变式怎么设计

## 3. 核心工作流

1. **审题三读** — 场景（讲什么）+ 已知（给什么）+ 求（求什么）
2. **建模三步** — 模型选择 + 简化（忽略次要）+ 建构（受力/运动/能量图）
3. **过程分析三步** — 受力（重力/接触/场力）+ 运动（分解+参量）+ 能量（找转化）
4. **列方程求解** — 规律（牛顿/能量/动量/欧姆）+ 几何 + 约束（绳/杆/接触）
5. **反思+变式训练** — 三维反思（结果/过程/拓展）+ 一题多解/多题一解/变条件
