# 物理解题教学指导 SKILL

> **一句话定位：** 物理解题不是"套公式"，而是"建模型"——五步解题法是核心。

---

## ⚠️ 技术实现边界声明

> **关于"AI 讲题"边界：** 本 SKILL 输出**解题思路框架**与**变式训练设计**；**不**替老师讲完整题，**不**直接给"标准答案"。
>
> **关于"题目版权"边界：** 题目必须标注 copyrightStatus（教材原题/改编/自有/CC 协议）；不复制未授权教辅题库。
>
> **关于"AI 评分"边界：** 本 SKILL **不**替老师给学员打分排名；只提供**评估框架**和**改进建议**。

## 一、核心使命

物理解题教学常见的三个误区：

```text
误区① 解题=套公式：学员记公式，
        不会建模型，不会分析过程。

误区② 解题=算对：学员只关心"答案对不对"，
        不关心"思路对不对"、"模型对不对"。

误区③ 解题=刷题：让学员做大量题，
        学员没建立"五步法"，没掌握"建模"。
```

本 SKILL 要解决的是：
- **让解题"有方法"**：五步物理解题法
- **让建模"有思路"**：模型选择+简化+建构
- **让过程"有分析"**：受力+运动+能量
- **让解后"有反思"**：结果+过程+拓展
- **让训练"有变式"**：一题多解+多题一解+改条件

## 四、五步解题法

### 4.1 五步总览

```text
① 审题：明确场景/已知/求
② 建模：选择/简化/建构
③ 过程分析：受力/运动/能量
④ 列方程：规律/关系/约束
⑤ 求解：数学/讨论/检验
+ 反思（贯穿始终）
```

### 4.2 第一步 · 审题

```text
■ 三读
  · 一读：场景（讲什么）
  · 二读：已知（给什么）
  · 三读：求（求什么）

■ 关键动作
  · 划关键词
  · 标已知量
  · 标未知量
  · 标隐藏条件

■ 学员常见问题
  · 不读题
  · 漏看条件
  · 看错问题
```

### 4.3 第二步 · 建模

```text
■ 模型选择
  · 这是什么模型？
  · 质点/刚体/弹簧/流体/...

■ 简化
  · 哪些因素忽略？
  · 哪些是主要的？

■ 建构
  · 画图（受力图/运动图/能量图）
  · 标已知量/未知量

■ 学员常见问题
  · 不会选模型
  · 不会简化
  · 不会画图
```

### 4.4 第三步 · 过程分析

```text
■ 受力分析
  · 画受力图
  · 标力的大小方向

■ 运动分析
  · 描述运动状态
  · 标运动参量

■ 能量分析
  · 找能量转化
  · 标能量初末态

■ 学员常见问题
  · 受力分析错
  · 不会用正交分解
  · 漏力/多力
```

### 4.5 第四步 · 列方程

```text
■ 规律方程
  · 牛顿定律
  · 运动学公式
  · 能量守恒
  · 动量守恒
  · 欧姆定律

■ 几何关系
  · 位移
  · 速度
  · 加速度

■ 约束条件
  · 绳约束
  · 杆约束
  · 接触约束

■ 学员常见问题
  · 不知道用哪个规律
  · 漏列方程
  · 方程列错
```

### 4.6 第五步 · 求解

```text
■ 数学求解
  · 解方程
  · 代入数据
  · 算结果

■ 讨论
  · 结果是否合理
  · 是否多解
  · 特殊情况

■ 检验
  · 单位
  · 数量级
  · 极限情况

■ 学员常见问题
  · 算错
  · 没讨论
  · 没检验
```

### 4.7 反思

```text
■ 结果反思
  · 答案对吗
  · 单位对吗
  · 数量级对吗

■ 过程反思
  · 思路对吗
  · 模型对吗
  · 方程对吗

■ 拓展反思
  · 一题多解
  · 改条件
  · 联系实际
```

## 六、过程分析专项

### 6.1 受力分析三步

```text
① 画重力
② 找接触力（支持力/摩擦力/拉力/弹力）
③ 找场力（电场力/磁场力）

■ 关键
  · 一画一个，不要漏
  · 受力分析完成后要检查
```

### 6.2 运动分析三步

```text
① 判断运动性质
  · 匀速/匀加速/匀减速/曲线

② 分解运动
  · 水平 + 竖直
  · 或：径向 + 切向

③ 标运动参量
  · 初速度
  · 加速度
  · 方向
```

### 6.3 能量分析三步

```text
① 找能量初态
  · 初动能
  · 初势能
  · 初其他能量

② 找能量末态
  · 末动能
  · 末势能
  · 末其他能量

③ 找能量转化
  · 哪个能量增加
  · 哪个能量减少
  · 减少的 = 增加的 + 损失
```

## 七、变式训练

### 7.1 变式三类型

```text
┌──────────┬────────────────────────┬──────────────┐
│ 类型      │ 描述                    │ 训练目标      │
├──────────┼────────────────────────┼──────────────┤
│ 一题多解  │ 一道题多种解法          │ 思维灵活性    │
│ 多题一解  │ 多道题同一种方法        │ 方法识别      │
│ 变条件    │ 改变题目条件/问题       │ 思维深度      │
└──────────┴────────────────────────┴──────────────┘
```

### 7.2 物理解题变式

```text
■ 改条件
  · 改水平面 → 斜面
  · 改光滑 → 粗糙
  · 改匀速 → 加速
  · 改平抛 → 斜抛

■ 改问题
  · 求速度 → 求时间
  · 求距离 → 求高度
  · 求最大 → 求最小

■ 改情境
  · 改平抛 → 类平抛
  · 改单摆 → 复摆
  · 改直流 → 交流
```

## 八、错因分析

### 8.1 物理解题错因

```text
┌──────────┬────────────────────────┐
│ 错因      │ 描述                    │
├──────────┼────────────────────────┤
│ 概念模糊  · 物理概念没掌握         │
│ 模型错误  · 模型选错/简化错        │
│ 受力错    · 受力分析错             │
│ 运动错    · 运动分析错             │
│ 公式错    · 公式用错               │
│ 计算错    · 算错                   │
│ 漏条件    · 漏掉条件               │
│ 多过程错  · 多过程分析错           │
└──────────┴────────────────────────┘
```

**与学生端物理错误DNA五维（P/C/F/R/T）的映射**（写回 `xiaozhi-physics-error-dna` 时必用）：

```text
本 SKILL 8 类错因        →  物理错误DNA 五维
────────────────────────────────────────────
概念模糊                →  C（概念混淆）
模型错误 / 受力错        →  P（图景建立：受力图/模型图）
运动错 / 漏条件 / 多过程错 →  R（过程分析）
公式错                  →  F（公式误用）
计算错                  →  T（数学工具）
```

> 说明：物理五维再经"物理错误DNA↔通用错题本"协议映射到通用四维；本表只解决"解题教学 8 类 → 物理五维"这一段的对齐。

### 8.2 五步对应错因

```text
· 审题错 → 漏条件
· 建模错 → 模型错误
· 过程错 → 受力错/运动错
· 列方程错 → 公式错
· 求解错 → 计算错
```
