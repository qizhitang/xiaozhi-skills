# 五步物理解题模板

> 配合 `xiaozhi-teach-physics-problem-guide` 使用。让解题从"套公式"到"建模型"。

## 1. 五步法总览

```text
① 审题：明确场景/已知/求
② 建模：选择/简化/建构
③ 过程分析：受力/运动/能量
④ 列方程：规律/关系/约束
⑤ 求解：数学/讨论/检验
+ 反思（贯穿始终）
```

## 2. 力学样板 · 滑块问题

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🧮 五步解题 · 滑块问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 题目
  质量 m=1kg 的滑块
  初速度 v0=2m/s
  受摩擦力 f=1N
  求 5 秒后滑块速度

■ 第一步：审题
  · 场景：滑块在水平面上减速
  · 已知：m=1kg, v0=2m/s, f=1N, t=5s
  · 求：5s 后速度

■ 第二步：建模
  · 质点模型
  · 水平面问题
  · 滑动摩擦

■ 第三步：过程分析
  · 受力：重力/支持力/摩擦力
  · 运动：匀减速
  · 能量：动能→摩擦生热

■ 第四步：列方程
  · ma = -f
  · a = -f/m = -1 m/s²
  · v = v0 + at
  · 因为滑块可能停下，先算停下时间
  · t 停 = v0/a = 2/1 = 2s < 5s
  · 所以 5s 后滑块已停下，v = 0

■ 第五步：求解
  · v = 0
  · 合理（滑块停下）

■ 反思
  · 学员常见错：忘记先判断是否停下
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 3. 力学样板 · 抛体问题

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🧮 五步解题 · 抛体问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 题目
  水平抛出小球
  初速度 v0=10m/s
  高度 h=5m
  求落地时间/水平距离/落地速度

■ 第一步：审题
  · 场景：平抛运动
  · 已知：v0=10m/s, h=5m, g=10m/s²
  · 求：t, x, v

■ 第二步：建模
  · 质点
  · 平抛运动
  · 忽略空气阻力

■ 第三步：过程分析
  · 水平：匀速
  · 竖直：自由落体

■ 第四步：列方程
  · 竖直：h = ½gt²
  · t = √(2h/g) = √(10/10) = 1s
  · 水平：x = v0 × t = 10 × 1 = 10m
  · 落地速度：vx=10, vy=gt=10
  · v = √(vx² + vy²) = √200 ≈ 14.1 m/s
  · 方向：θ = 45°

■ 第五步：求解
  · t=1s, x=10m, v=14.1m/s

■ 反思
  · 学员常见错：忘记用分解
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 4. 力学样板 · 弹簧振子

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🧮 五步解题 · 弹簧振子
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 题目
  弹簧振子 m=1kg, k=100N/m
  最大位移 A=0.1m
  求最大速度/周期/最大加速度

■ 第一步：审题
  · 场景：弹簧振子
  · 已知：m, k, A
  · 求：vmax, T, amax

■ 第二步：建模
  · 弹簧振子
  · 简谐振动

■ 第三步：过程分析
  · 能量守恒
  · 振子从最大位移到平衡位置

■ 第四步：列方程
  · 能量守恒：½kA² = ½mvmax²
  · vmax = √(k/m) × A = √100 × 0.1 = 1 m/s
  · 周期：T = 2π√(m/k) = 2π√(1/100) = 0.628s
  · 最大加速度：amax = (k/m) × A = 100 × 0.1 = 10 m/s²

■ 第五步：求解
  · vmax=1m/s, T=0.628s, amax=10m/s²

■ 反思
  · 学员常见错：忽略能量守恒
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 5. 电学样板 · 闭合电路

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🧮 五步解题 · 闭合电路
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 题目
  电源 E=12V, r=1Ω
  外电阻 R=5Ω
  求电流/路端电压/输出功率/总功率/效率

■ 第一步：审题
  · 场景：闭合电路
  · 已知：E=12V, r=1Ω, R=5Ω
  · 求：I, U端, P出, P总, η

■ 第二步：建模
  · 简单电路
  · 闭合电路欧姆定律

■ 第三步：过程分析
  · 总电阻：R+r=6Ω
  · 电流：I = E/(R+r) = 12/6 = 2A
  · 路端：U = IR = 2×5 = 10V
  · 输出：P出 = UI = 10×2 = 20W
  · 总功率：P总 = EI = 12×2 = 24W
  · 效率：η = P出/P总 = 20/24 = 83.3%

■ 第四步：列方程
  · I = E/(R+r)
  · U = E - Ir
  · P出 = UI
  · P总 = EI
  · η = P出/P总

■ 第五步：求解
  · I=2A, U=10V, P出=20W, P总=24W, η=83.3%

■ 反思
  · 学员常见错：忽略内阻
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 6. 热学样板 · 气体状态

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🧮 五步解题 · 气体状态
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 题目
  理想气体
  初状态：V1=2L, P1=1atm, T1=300K
  等温压缩到 V2=1L
  求 P2

■ 第一步：审题
  · 场景：等温变化
  · 已知：V1, P1, T1, V2
  · 求：P2

■ 第二步：建模
  · 理想气体
  · 等温变化
  · 玻意耳定律

■ 第三步：过程分析
  · 等温 → T 不变
  · 体积变小 → 压强变大

■ 第四步：列方程
  · 玻意耳定律：P1V1 = P2V2
  · P2 = P1V1/V2 = 1 × 2/1 = 2 atm

■ 第五步：求解
  · P2=2atm
  · 合理（体积变小，压强变大）

■ 反思
  · 学员常见错：等温/等压/等容混淆
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 7. 光学样板 · 透镜成像

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🧮 五步解题 · 透镜成像
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 题目
  凸透镜 f=10cm
  物距 u=30cm
  求像距 v/像的性质

■ 第一步：审题
  · 场景：凸透镜成像
  · 已知：f=10cm, u=30cm
  · 求：v, 像的性质

■ 第二步：建模
  · 薄透镜
  · 凸透镜成像

■ 第三步：过程分析
  · u > 2f
  · 像在 f < v < 2f
  · 倒立、缩小、实像

■ 第四步：列方程
  · 透镜公式：1/u + 1/v = 1/f
  · 1/30 + 1/v = 1/10
  · 1/v = 1/10 - 1/30 = 2/30
  · v = 15 cm

■ 第五步：求解
  · v=15cm
  · 倒立、缩小、实像

■ 反思
  · 学员常见错：忘记单位换算
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 8. 学员层次五步法

```text
■ 基础层
  · 重点：审题 + 列方程
  · 不强求：建模 + 反思

■ 中等层
  · 重点：审题 + 建模 + 过程 + 列方程
  · 加分：反思

■ 提升层
  · 重点：五步 + 反思 + 变式
  · 加分：拓展
```

## 9. 五步法错因

```text
· 审题错 → 漏条件
· 建模错 → 模型错误
· 过程错 → 受力错/运动错
· 列方程错 → 公式错
· 求解错 → 计算错
```

## 10. 教学建议

```text
■ 老师讲题
  · 显式呈现五步
  · 让学员"看到"五步
  · 避免直接给答案

■ 学员做题
  · 学员自己完成五步
  · 老师引导而不替代
  · 学员反思

■ 学员互评
  · 学员互相看五步
  · 学员互相提建议
  · 学员互相学习
```
