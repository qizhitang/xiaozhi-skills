# 物理解题变式设计

> 配合 `xiaozhi-teach-physics-problem-guide` 使用。从"情境变式 / 物理量变式 / 解法变式"三维度设计变式训练，让学员建立"模型识别-迁移应用"能力。

## 1. 变式设计 3 维度

```text
┌──────────┬────────────────┬──────────────────┐
│ 维度      │ 变化什么        │ 训练什么          │
├──────────┼────────────────┼──────────────────┤
│ 情境变式  │ 故事背景        │ 识别核心模型      │
│ 物理量变式│ 已知/未知      │ 应用公式能力      │
│ 解法变式  │ 多种解法        │ 一题多解能力      │
└──────────┴────────────────┴──────────────────┘
```

## 2. 维度 1：情境变式

### 2.1 定义

```text
【定义】保留核心模型，改变故事背景
【目的】训练学员"识别核心模型"，不被表面故事迷惑
```


## 3. 维度 2：物理量变式

### 3.1 定义

```text
【定义】保留情境和模型，改变已知量或未知量
【目的】训练学员"灵活应用公式"
```

## 4. 维度 3：解法变式

### 4.1 定义

```text
【定义】保留情境和模型，提供多种解法
【目的】训练学员"一题多解"能力，理解物理本质
```

### 4.2 解法变式样板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 母题
━━━━━━━━━━━━━━━━━━━━━━━━━━━
质量 m = 2 kg 物体从 5 m 高处自由下落，
落到地面时的动能是多少？
（g = 10 m/s²）

━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 解法 1：运动学 + 动能公式
━━━━━━━━━━━━━━━━━━━━━━━━━━━
· 求落地速度：v² = 2gh = 100，v = 10 m/s
· 求动能：Ek = ½mv² = ½ × 2 × 100 = 100 J
· 答案：100 J

━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 解法 2：机械能守恒
━━━━━━━━━━━━━━━━━━━━━━━━━━━
· 起点：Ep = mgh = 2 × 10 × 5 = 100 J, Ek = 0
· 终点：Ep = 0, Ek = ？
· 守恒：100 = Ek
· 答案：100 J

━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 解法 3：动能定理
━━━━━━━━━━━━━━━━━━━━━━━━━━━
· 合外力做功 = ΔEk
· W = mgh = 100 J
· ΔEk = 100 J
· 答案：100 J

━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 三种解法对比
━━━━━━━━━━━━━━━━━━━━━━━━━━━
  · 解法 1：步骤多但直观
  · 解法 2：最简洁
  · 解法 3：物理意义最清晰
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 5. 变式训练的难度梯度

```text
【阶段 1：单一维度变式】
  · 只变情境
  · 模型识别为主

【阶段 2：双维度变式】
  · 变情境 + 变物理量
  · 识别 + 应用

【阶段 3：三维度变式】
  · 变情境 + 变物理量 + 变解法
  · 综合训练
```

## 7. 变式训练常见误区

```text
误区 ① 变式只变表面
  · 学员做完母题就"会"了
  · 变式未触及核心
  → 应深挖模型核心

误区 ② 变式跨度太大
  · 学员跟不上
  · 失去学习兴趣
  → 应循序渐进

误区 ③ 变式题量过多
  · 学员负担过重
  · 思考变机械
  → 控制在 3-5 题

误区 ④ 没有反馈
  · 学员不知道自己哪里错
  · 错误得不到纠正
  → 应即时反馈
```
