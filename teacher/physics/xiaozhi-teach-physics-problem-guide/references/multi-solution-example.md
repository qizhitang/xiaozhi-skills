# 一题多解样板 · 滑块问题

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🧮 一题多解 · 滑块问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 题目
  滑块 m, v0
  受 f 摩擦
  求停止距离

■ 解法 1（牛顿 + 运动学）
  · a = -f/m
  · 0 = v0² - 2as
  · s = mv0²/(2f)

■ 解法 2（动能定理）
  · -fs = 0 - ½mv0²
  · s = mv0²/(2f)

■ 解法 3（能量守恒）
  · ½mv0² = fs
  · s = mv0²/(2f)

■ 比较
  · 牛顿：最通用
  · 动能定理：跳过过程
  · 能量守恒：物理意义清晰
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
