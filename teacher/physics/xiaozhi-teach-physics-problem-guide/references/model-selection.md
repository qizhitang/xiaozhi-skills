# 物理模型选择参考

> 配合 `xiaozhi-teach-physics-problem-guide` 使用。从"问题类型 → 模型选择 → 关键公式"三步走，帮助学员建立模型选择决策能力。

## 1. 7 类物理模型回顾

```text
┌──────┬──────────────┬──────────────────┐
│ #    │ 模型          │ 关键公式          │
├──────┼──────────────┼──────────────────┤
│ 1    │ 质点          │ 位置/速度/加速度  │
│ 2    │ 匀变速直线    │ v=v₀+at, s=v₀t+½at²│
│ 3    │ 受力平衡      │ ΣF=0              │
│ 4    │ 能量守恒      │ Ek+Ep=const       │
│ 5    │ 动量守恒      │ m₁v₁+m₂v₂=const   │
│ 6    │ 电路          │ I=U/R             │
│ 7    │ 场            │ F=qE, F=BIL       │
└──────┴──────────────┴──────────────────┘
```

## 2. 模型选择决策树

```text
学员面对问题
  │
  ├── 涉及什么物理量？
  │     ├── 运动量（位置/速度/加速度）→ 运动学
  │     ├── 受力 → 力学
  │     ├── 能量 → 能量守恒
  │     ├── 动量 → 动量守恒
  │     ├── 电流/电压/电阻 → 电路
  │     ├── 电荷/磁极 → 场
  │     └── 物质属性 → 物质观
  │
  ├── 物体运动状态？
  │     ├── 静止/匀速 → 平衡模型
  │     ├── 匀变速 → 匀变速模型
  │     ├── 圆周 → 圆周模型
  │     └── 振动 → 振动模型
  │
  ├── 系统是否孤立？
  │     ├── 孤立（无外力）→ 动量守恒
  │     ├── 只有重力 → 机械能守恒
  │     └── 其他 → 牛顿第二定律
  │
  └── 选最优模型 → 应用关键公式
```

## 3. 模型 1 选择：质点

```text
【判断标准】
  · 物体大小 << 运动范围？
  · 转动/形变可忽略？
  · 是 → 用质点模型
  · 否 → 不能用质点

【典型场景】
  · 公转的行星 ✓
  · 飞行的子弹 ✓
  · 转动的陀螺 ✗
  · 形变的弹簧 ✗

【关键公式】
  · x(t), v(t), a(t)
```

## 4. 模型 2 选择：匀变速直线运动

```text
【判断标准】
  · 加速度恒定？
  · 直线运动？
  · 是 → 用匀变速模型

【典型场景】
  · 自由落体 ✓
  · 水平面上恒力推动 ✓
  · 竖直上抛 ✓
  · 匀速圆周运动 ✗（不是匀变速）
  · 简谐振动 ✗（加速度变化）

【关键公式】
  · v = v₀ + at
  · s = v₀t + ½at²
  · v² - v₀² = 2as
  · 平均速度 = (v₀+v)/2
```

## 5. 模型 3 选择：受力平衡

```text
【判断标准】
  · 物体静止？
  · 物体匀速直线运动？
  · 合外力为零？
  · 是 → 用平衡模型

【典型场景】
  · 静止在斜面上的物体 ✓
  · 匀速下落的跳伞者 ✓
  · 静止的桥梁 ✓
  · 加速的火箭 ✗
  · 转弯的汽车 ✗

【关键公式】
  · ΣF = 0
  · Fx = 0, Fy = 0
  · 力的合成/分解
```

## 6. 模型 4 选择：能量守恒

```text
【判断标准】
  · 涉及功和能？
  · 机械能守恒条件？
  · 是 → 用能量守恒模型

【适用条件】
  · 只有重力做功
  · 只有弹力做功
  · 其他力不做功

【典型场景】
  · 自由落体 ✓
  · 抛体运动 ✓
  · 弹簧振子 ✓
  · 单摆 ✓
  · 滑块在粗糙斜面 ✗（有摩擦做功）

【关键公式】
  · Ek + Ep = const
  · W = ΔEk（动能定理）
  · W = -ΔEp（势能变化）
```

## 7. 模型 5 选择：动量守恒

```text
【判断标准】
  · 系统合外力为零？
  · 或内力 << 外力？
  · 是 → 用动量守恒模型

【典型场景】
  · 碰撞（弹性/非弹性）✓
  · 冲击 ✓
  · 爆炸 ✓
  · 反冲 ✓
  · 自由落体 ✗（单个物体，无"碰撞"）

【关键公式】
  · m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
  · 弹性碰撞：+ 动能守恒
  · 完全非弹性：+ 共同速度
```

## 8. 模型 6 选择：电路分析

```text
【判断标准】
  · 涉及电流、电压、电阻？
  · 是 → 用电路模型

【子模型选择】
  · 单电阻：I = U/R
  · 串联：I 相同，U 分配
  · 并联：U 相同，I 分配
  · 混联：综合分析

【典型场景】
  · 简单电路 ✓
  · 串并联混联 ✓
  · 滑动变阻器 ✓
  · 非纯电阻（如电动机）✗

【关键公式】
  · I = U/R
  · P = UI
  · 串联：R = R₁ + R₂
  · 并联：1/R = 1/R₁ + 1/R₂
```

## 9. 模型 7 选择：场

```text
【判断标准】
  · 涉及引力/电场/磁场？
  · 是 → 用场模型

【子模型选择】
  · 重力场：F = mg
  · 万有引力场：F = GMm/r²
  · 电场：F = qE
  · 磁场：F = BIL（电流受力）

【典型场景】
  · 物体在地球表面 → 重力场
  · 卫星运动 → 万有引力场
  · 带电粒子在电场中 → 电场
  · 电流在磁场中 → 磁场
```

## 10. 综合问题模型选择

```text
【综合问题】滑块从斜面滑下，撞上弹簧

  · 第 1 阶段：滑块下滑
    → 受力分析（重力、摩擦、支持力）
    → 运动学（匀变速）

  · 第 2 阶段：滑块撞弹簧
    → 能量守恒（机械能转化为弹性势能）
    → 或 动量定理（如果有冲击）

  · 第 3 阶段：滑块反弹
    → 能量守恒（弹性势能转化为动能）
    → 受力分析

【启示】综合问题需要"分阶段"使用不同模型
```

## 11. 模型选择训练 3 步

```text
第 1 步：识别
  · 学员读题 → 找出已知量和求量
  · 学员判断：这是什么类型的物理问题？

第 2 步：选择
  · 学员选：用什么模型？
  · 学员验证：模型的适用条件是否满足？

第 3 步：应用
  · 学员列出关键公式
  · 学员求解
  · 学员检验（量纲/物理意义）
```
