# 物理教案设计 · 精简版

> 详细版见 `SKILL.md`。

## 1. 一句话定位

物理不是教学生"记公式"，而是教学生"用物理观念看世界"——物理观念建构是核心。

## 2. 工作流

```text
物理观念 → 概念建构 → 规律教学 → 模型建构 → 应用训练 → 课堂小结
```

## 3. 五大物理观念

```text
物质观 / 运动观 / 能量观 / 相互作用 / 演化观
```

## 4. 概念建构三步

```text
引入（生活/演示/反例）
抽象（抓住关键/排除干扰）
应用（区分/联系/拓展）
```

## 5. 规律教学三步

```text
实验（演示/学生）
数据（处理/图像/规律）
归纳（数学表达/物理意义）
```

## 6. 物理模型分类

```text
质点 / 刚体 / 理想气体 / 点电荷 / 理想流体 / 弹簧振子 / 单摆
```

## 7. 应用三类

```text
生活应用 / 科技应用 / 前沿应用
```

## 8. 必触发场景

- 老师："这个物理概念怎么讲"
- 老师："物理教案怎么写"
- 老师："学员物理思维弱"
- 老师："物理概念怎么引入"
- 老师："物理模型怎么教"
- 老师："实验怎么进课堂"
