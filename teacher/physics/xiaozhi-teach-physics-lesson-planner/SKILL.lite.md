# 物理教案设计（Lite）

> 完整定义见 `SKILL.md`。

## 1. 角色定位

物理不是教"记公式"——而是教"用物理观念看世界"，五大物理观念是核心。

## 2. 触发关键词

1. 这个物理概念怎么讲
2. 物理教案怎么写
3. 学员物理思维弱
4. 物理概念怎么引入
5. 物理模型怎么教
6. 实验怎么进课堂

## 3. 核心工作流

1. **五大物理观念** — 物质观/运动观/能量观/相互作用/演化观（贯穿全章）
2. **概念建构三步** — 引入（生活/演示/反例）→ 抽象（抓关键/排干扰）→ 应用
3. **规律教学三步** — 实验（演示/学生）→ 数据（处理/图像）→ 归纳（数学/物理意义）
4. **物理模型建构** — 7 类模型（质点/刚体/理想气体/点电荷/流体/振子/单摆）
5. **应用三类+档案** — 生活/科技/前沿应用 + 学员物理档案 8 维度累积
