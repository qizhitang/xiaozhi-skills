# 物理概念图谱

> 配合 `xiaozhi-teach-physics-lesson-planner` 使用。基于"五大物理观念"组织概念图谱，让物理教学有清晰的主线。

## 1. 五大物理观念

```text
┌──────────┬────────────────────┬──────────────────────┐
│ 观念      │ 核心问题            │ 主要章节              │
├──────────┼────────────────────┼──────────────────────┤
│ 物质观    │ 物质是什么/怎么组成│ 物质/密度/分子动理论  │
│ 运动观    │ 物体如何运动        │ 运动/力/圆周/振动    │
│ 能量观    │ 能量如何转化        │ 功/能/热/电          │
│ 相互作用观│ 物体如何影响        │ 力/场/电磁感应        │
│ 演化观    │ 物理过程如何变化    │ 守恒/熵/相对论       │
└──────────┴────────────────────┴──────────────────────┘
```

## 3. 五大观念详解

### 3.1 物质观

```text
【核心问题】
  · 物质由什么组成？
  · 物质有哪些状态？
  · 物质有哪些属性？

【关键概念】
  · 分子、原子
  · 三态变化
  · 质量、密度
  · 比热容

【概念图】
  物质观
    │
    ├── 物质结构
    │     ├── 分子
    │     ├── 原子
    │     └── 元素
    │
    ├── 物质状态
    │     ├── 固态
    │     ├── 液态
    │     └── 气态
    │
    └── 物质属性
          ├── 质量 m
          ├── 密度 ρ
          └── 比热容 c
```

### 3.2 运动观

```text
【核心问题】
  · 物体如何运动？
  · 运动如何描述？
  · 运动如何变化？

【关键概念】
  · 速度、加速度
  · 匀速、匀变速
  · 直线、曲线
  · 圆周运动

【概念图】
  运动观
    │
    ├── 描述运动
    │     ├── 速度 v
    │     ├── 加速度 a
    │     └── 位移 s
    │
    ├── 运动类型
    │     ├── 匀速直线
    │     ├── 匀变速直线
    │     ├── 圆周运动
    │     └── 简谐振动
    │
    └── 运动定律
          ├── 牛顿第一定律
          ├── 牛顿第二定律
          └── 牛顿第三定律
```

### 3.3 能量观

```text
【核心问题】
  · 能量有哪些形式？
  · 能量如何转化？
  · 能量是否守恒？

【关键概念】
  · 功、功率
  · 动能、势能
  · 机械能守恒
  · 内能、热
  · 电能

【概念图】
  能量观
    │
    ├── 机械能
    │     ├── 动能 Ek
    │     ├── 势能 Ep
    │     │     ├── 重力势能
    │     │     └── 弹性势能
    │     └── 守恒定律
    │
    ├── 内能
    │     ├── 分子动能
    │     └── 分子势能
    │
    ├── 电能
    │     ├── 电功
    │     └── 电功率
    │
    └── 能量守恒
          └── 永动机不可能
```

```text
【核心问题】

    ├── 压强
    │     ├── 固体压强
    │     ├── 液体压强
    │     └── 气体压强
    │
    ├── 简单机械
    │     ├── 杠杆
    │     ├── 滑轮
    │     └── 斜面
    │
    └── 电与磁
          ├── 欧姆定律
          ├── 串并联
          ├── 磁场
          └── 电磁感应
```

## 4. 概念图谱应用 3 场景

```text
场景 ① 备课
  · 选章节 → 找对应观念 → 列核心概念 → 设计教学路径

场景 ② 教学
  · 用观念串概念（如讲"力"必讲"相互作用观"）
  · 用观念串章节（如"运动观"贯穿整个力学）

场景 ③ 复习
  · 按观念整理错题
  · 用观念理解跨章节联系
```

## 5. 五大观念在初中物理的章节分布

```text
┌──────────┬──────────────────────────────────┐
│ 观念      │ 涉及章节                          │
├──────────┼──────────────────────────────────┤
│ 物质观    │ 物态变化、密度、分子动理论        │
│ 运动观    │ 机械运动、力、运动和力、压强浮力  │
│ 能量观    │ 功和能、内能、热量                │
│ 相互作用观│ 力的概念、压强、浮力、简单机械    │
│ 演化观    │ 守恒定律、能量守恒                │
└──────────┴──────────────────────────────────┘
```
