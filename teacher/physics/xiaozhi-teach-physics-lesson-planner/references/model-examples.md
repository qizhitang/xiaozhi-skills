# 物理模型案例库

> 配合 `xiaozhi-teach-physics-lesson-planner` 使用。物理模型是连接实际与理论的桥梁，本文档提供 7 类核心模型的具体案例。

## 1. 7 类核心物理模型

```text
┌──────┬──────────────┬──────────────────────┐
│ #    │ 模型          │ 适用场景              │
├──────┼──────────────┼──────────────────────┤
│ 1    │ 质点          │ 物体大小可忽略时      │
│ 2    │ 匀变速直线    │ 恒定加速度直线运动    │
│ 3    │ 受力平衡      │ 静止或匀速直线运动    │
│ 4    │ 能量守恒      │ 涉及功和能的问题      │
│ 5    │ 动量守恒      │ 碰撞/冲击问题         │
│ 6    │ 电路分析      │ 涉及欧姆定律的问题    │
│ 7    │ 场（重力场/电 │ 涉及万有引力/电场/磁场│
│      │ 场/磁场）    │ 的问题               │
└──────┴──────────────┴──────────────────────┘
```

## 2. 模型 1：质点

### 2.1 模型定义

```text
【定义】把物体看成一个有质量的点，忽略其形状、大小

【适用条件】
  · 物体大小远小于运动范围
  · 物体转动可忽略
  · 物体形变可忽略

【不适用】
  · 研究物体转动时
  · 物体大小不能忽略时
  · 物体形变是研究主体时
```

### 2.2 案例

```text
【案例 1】地球绕太阳公转
  · 模型：质点
  · 理由：地球半径 << 日地距离

【案例 2】火车从北京到上海
  · 模型：质点
  · 理由：火车长度 << 京沪距离

【案例 3】地球自转
  · 模型：不是质点
  · 理由：研究转动时不能忽略形状

【案例 4】乒乓球飞行
  · 模型：可视为质点
  · 理由：研究飞行轨迹时，旋转可单独处理
```

## 3. 模型 2：匀变速直线运动

### 3.1 模型定义

```text
【定义】加速度恒定且方向与初速度共线的直线运动

【关键公式】
  · v = v₀ + at
  · s = v₀t + ½at²
  · v² - v₀² = 2as
  · s = (v + v₀)t / 2

【适用场景】
  · 自由落体（a = g）
  · 竖直上抛
  · 水平面上恒力推动
  · 斜面上恒力推动
```

### 3.2 案例

```text
【案例 1】汽车以 2 m/s² 加速
  · 已知：v₀ = 0, a = 2 m/s², t = 5 s
  · 求：v 和 s
  · 解：v = 10 m/s, s = 25 m

【案例 2】自由落体
  · 已知：h = 20 m, g = 10 m/s²
  · 求：t 和落地速度
  · 解：t = 2 s, v = 20 m/s

【案例 3】刹车问题
  · 已知：v₀ = 20 m/s, a = -5 m/s²
  · 求：停止时间
  · 解：t = 4 s
```

## 4. 模型 3：受力平衡

### 4.1 模型定义

```text
【定义】物体所受合外力为零，保持静止或匀速直线运动

【关键方程】
  · ΣF = 0
  · Fx = 0, Fy = 0（正交分解）

【适用场景】
  · 静止物体
  · 匀速直线运动
  · 物体在斜面上匀速下滑
  · 物体被多根绳悬挂
```

### 4.2 案例

```text
【案例 1】物体在斜面上匀速下滑
  · 已知：m = 10 kg, 斜面角度 30°, μ = ?
  · 解：μ = tan(30°) = √3/3

【案例 2】两人抬水
  · 已知：水重 100 N, 两端对称
  · 解：每人承担 50 N

【案例 3】吊灯被两根绳拉住
  · 已知：灯重 20 N, 绳对称 30°
  · 解：每根绳拉力 = 20/(2cos30°) N
```

## 5. 模型 4：能量守恒

### 5.1 模型定义

```text
【定义】在只有重力/弹力做功时，机械能总量不变

【关键公式】
  · Ek + Ep = const
  · ½mv² + mgh = const

【适用场景】
  · 自由落体
  · 抛体运动
  · 弹簧振子
  · 单摆
```

### 5.2 案例

```text
【案例 1】小球从 5 m 自由下落
  · 已知：h = 5 m, v₀ = 0, g = 10 m/s²
  · 求：落地速度
  · 解：v = √(2gh) = 10 m/s

【案例 2】滑块从斜面滑下
  · 已知：h = 4 m, m = 2 kg, g = 10 m/s²
  · 求：到斜面底的速度
  · 解：v = √(2gh) = √80 m/s ≈ 8.94 m/s
```

## 6. 模型 5：动量守恒

### 6.1 模型定义

```text
【定义】系统不受外力或合外力为零时，总动量不变

【关键公式】
  · m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'

【适用场景】
  · 碰撞（弹性/非弹性）
  · 冲击
  · 反冲
  · 爆炸
```

### 6.2 案例

```text
【案例 1】两球弹性碰撞
  · 已知：m₁ = 1 kg, v₁ = 2 m/s；m₂ = 1 kg, v₂ = 0
  · 求：碰后速度
  · 解：v₁' = 0, v₂' = 2 m/s（交换速度）

【案例 2】子弹打入木块
  · 已知：m₁ = 0.01 kg, v₁ = 200 m/s；m₂ = 1 kg, v₂ = 0
  · 求：共同速度
  · 解：v = (m₁v₁)/(m₁+m₂) ≈ 1.98 m/s
```

## 7. 模型 6：电路分析

### 7.1 模型定义

```text
【定义】基于欧姆定律分析串并联电路

【关键公式】
  · I = U/R
  · 串联：I 相同，U 分配与 R 成正比
  · 并联：U 相同，I 分配与 R 成反比
  · 功率：P = UI = I²R = U²/R
```

### 7.2 案例

```text
【案例 1】串联电路
  · 已知：R₁ = 2 Ω, R₂ = 3 Ω, U = 10 V
  · 求：I, U₁, U₂
  · 解：I = 2 A, U₁ = 4 V, U₂ = 6 V

【案例 2】并联电路
  · 已知：R₁ = 2 Ω, R₂ = 4 Ω, U = 12 V
  · 求：I₁, I₂, I总
  · 解：I₁ = 6 A, I₂ = 3 A, I总 = 9 A
```

## 8. 模型 7：场

### 8.1 模型定义

```text
【定义】物质存在的特殊形式，对放入其中的物体有力的作用

【三类场】
  · 重力场（万有引力）
  · 电场（电荷间相互作用）
  · 磁场（磁极/电流间相互作用）
```

### 8.2 案例

```text
【案例 1】重力场
  · 地球周围空间存在重力场
  · 重力 g = 9.8 m/s²（地面附近）

【案例 2】电场
  · 点电荷周围存在电场
  · 场强 E = F/q

【案例 3】磁场
  · 磁体/电流周围存在磁场
  · 磁感应强度 B
```

## 9. 模型选择决策树

```text
学员面对问题 → 涉及什么？
  · 物体运动 → 选运动学模型
  · 物体受力 → 选力学模型
  · 能量转化 → 选能量模型
  · 物体碰撞 → 选动量模型
  · 电流问题 → 选电路模型
  · 引力/电场/磁场 → 选场模型
```

## 10. 模型迁移训练

```text
【训练 1】同一问题不同模型
  · 例：自由落体
    · 运动学模型：v = gt
    · 能量模型：v = √(2gh)

【训练 2】不同问题同一模型
  · 例：能量守恒可用于：
    · 自由落体
    · 抛体运动
    · 弹簧振子
    · 单摆

【训练 3】综合问题模型选择
  · 例：斜面滑块
    · 运动学 + 受力分析 + 能量守恒
```
