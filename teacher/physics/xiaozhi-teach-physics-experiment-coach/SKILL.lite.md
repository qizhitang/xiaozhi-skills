# 物理实验教学指导 · 精简版

> 详细版见 `SKILL.md`。

## 1. 一句话定位

实验不是"走过场"，而是"真探究"——实验设计能力是核心。

## 2. 工作流

```text
实验目标 → 实验设计 → 实验实施 → 数据处理 → 结论归纳 → 实验报告 → 实验档案
```

## 3. 三类实验

```text
验证性：验证已知规律
探究性：探究未知规律
测量性：测量物理量
```

## 4. 实验设计四要素

```text
实验目的（验证/探究/测量）
实验原理（用什么规律）
实验器材（选什么器材）
实验步骤（怎么操作）
```

## 5. 变量控制

```text
自变量：主动改变
因变量：随自变量变化
控制变量：保持不变
一次只改一个变量
```

## 6. 数据处理三形式

```text
表格：整理数据
图像：散点图+拟合曲线
计算：平均值+误差
```

## 7. 实验报告结构

```text
1. 实验目的
2. 实验原理
3. 实验器材
4. 实验步骤
5. 数据记录
6. 数据分析
7. 实验结论
8. 反思讨论
```

## 8. 必触发场景

- 老师："实验怎么教"
- 老师："实验怎么设计"
- 老师："学员不会做实验"
- 老师："数据怎么处理"
- 老师："实验报告怎么写"
- 老师："演示实验怎么做"
