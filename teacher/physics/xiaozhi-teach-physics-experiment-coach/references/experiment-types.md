## 2. 类型 1：测量型实验

### 2.1 定义与目的

```text
【定义】用实验方法测量某个物理量
【目的】掌握测量方法，理解测量原理
【典型】测密度、测电阻、测速度
```

### 2.2 设计要点

```text
【核心】选择合适的测量工具
【步骤】
  ① 明确待测量
  ② 选择测量工具（精度匹配）
  ③ 设计测量步骤
  ④ 多次测量取平均
  ⑤ 误差分析

【案例】测金属块密度
  · 工具：天平、量筒
  · 步骤：测质量 m、测体积 V
  · 计算：ρ = m/V
  · 误差：量筒读数误差、天平精度
```

### 2.3 常见测量型实验

```text
┌────────────────────┬──────────────────────┐
│ 实验                │ 测量量                │
├────────────────────┼──────────────────────┤
│ 测金属块密度        │ 质量、体积            │
│ 测小石块密度        │ 质量、体积            │
│ 测盐水密度          │ 质量、体积            │
│ 测小灯泡电阻        │ 电压、电流            │
│ 测滑轮组机械效率    │ 拉力、物重、距离      │
│ 测小灯泡的电功率    │ 电压、电流            │
│ 测斜面机械效率      │ 拉力、物重、距离      │
└────────────────────┴──────────────────────┘
```

## 3. 类型 2：验证型实验

### 3.1 定义与目的

```text
【定义】用实验验证已学过的物理规律
【目的】加深对规律的理解
【典型】验证牛顿第二定律、验证机械能守恒
```

### 3.2 设计要点

```text
【核心】把抽象规律转化为可测量的物理量
【步骤】
  ① 列出待验证的规律
  ② 转化为可测量的关系
  ③ 设计实验方案
  ④ 测量数据
  ⑤ 比对预期与实际

【案例】验证牛顿第二定律
  · 规律：F = ma
  · 实验：控制 m 一定，F 变化测 a；控制 F 一定，m 变化测 a
  · 预期：F ∝ a, a ∝ 1/m
  · 结论：F = ma 成立
```

### 3.3 常见验证型实验

```text
┌────────────────────┬──────────────────┐
│ 实验                │ 验证规律          │
├────────────────────┼──────────────────┤
│ 验证牛顿第二定律    │ F = ma            │
│ 验证机械能守恒      │ Ep = Ek           │
│ 验证欧姆定律        │ I = U/R           │
│ 验证阿基米德原理    │ F浮 = ρ液gV排     │
│ 验证焦耳定律        │ Q = I²Rt          │
│ 验证杠杆平衡条件    │ F1L1 = F2L2       │
└────────────────────┴──────────────────┘
```

## 4. 类型 3：探究型实验

### 4.1 定义与目的

```text
【定义】通过实验探究未知的物理规律
【目的】培养科学探究能力
【典型】探究液体内部压强、探究电流与电压电阻关系
```

### 4.2 设计要点

```text
【核心】运用控制变量法
【步骤】
  ① 提出问题
  ② 猜想与假设
  ③ 设计实验（控制变量）
  ④ 收集数据
  ⑤ 分析数据
  ⑥ 得出结论
  ⑦ 评估

【案例】探究电流与电压、电阻的关系
  · 问题：电流与电压、电阻有什么关系？
  · 假设：I ∝ U, I ∝ 1/R
  · 实验：
    ① 控制 R 一定，改变 U，测 I
    ② 控制 U 一定，改变 R，测 I
  · 结论：I = U/R
```

### 4.3 常见探究型实验

```text
┌────────────────────┬──────────────────┐
│ 实验                │ 探究内容          │
├────────────────────┼──────────────────┤
│ 探究液体压强        │ 影响因素          │
│ 探究浮力大小        │ 影响因素          │
│ 探究杠杆平衡条件    │ F 与 L 的关系     │
│ 探究电流与电压电阻  │ 欧姆定律          │
│ 探究电磁铁磁性强弱  │ 影响因素          │
│ 探究导体电阻        │ 影响因素          │
│ 探究动能            │ 影响因素          │
└────────────────────┴──────────────────┘
```

## 5. 类型 4：演示型实验

### 5.1 定义与目的

```text
【定义】老师演示，学员观察的实验
【目的】直观展示物理现象
【典型】大气压存在、电磁感应现象
```

### 5.2 设计要点

```text
【核心】现象要明显，过程要直观
【步骤】
  ① 选择能产生明显现象的实验
  ② 设计演示过程
  ③ 引导学员观察
  ④ 引导学员思考

【案例】大气压存在
  · 实验：覆杯实验
  · 现象：水不流出
  · 引导：是什么托住了水？
  · 结论：大气压存在
```

## 6. 类型 5：设计型实验

### 6.1 定义与目的

```text
【定义】学员自己设计方案完成实验
【目的】培养综合实验能力
【典型】测未知电阻、估测大气压
```

### 6.2 设计要点

```text
【核心】开放性问题，多种解法
【步骤】
  ① 明确实验目的
  ② 列出可用器材
  ③ 设计方案（多个）
  ④ 选择最优方案
  ⑤ 实施
  ⑥ 误差分析

【案例】测量未知电阻
  · 方法 1：伏安法
  · 方法 2：等效替代法
  · 方法 3：电桥法
  · 选择：基于器材选择
```

## 7. 类型 6：课外型实验

### 7.1 定义与目的

```text
【定义】在课后、家庭、社区进行的实验
【目的】延伸学习，培养兴趣
【典型】自制密度计、自制电磁铁
```

### 7.2 设计要点

```text
【核心】安全、有趣、可操作
【步骤】
  ① 选择安全材料
  ② 简化操作
  ③ 引导观察
  ④ 撰写实验报告

【案例】自制密度计
  · 材料：吸管、橡皮泥
  · 步骤：吸管底部塞橡皮泥
  · 测试：放入不同液体
  · 结论：液体密度越大，吸管露出越多
```

## 8. 类型 7：仿真型实验

### 8.1 定义与目的

```text
【定义】用计算机软件模拟实验
【目的】弥补实物实验的不足
【典型】PhET 仿真、Algodoo
```

### 8.2 设计要点

```text
【核心】补充非可视化的实验
【步骤】
  ① 选择仿真软件
  ② 设置参数
  ③ 运行实验
  ④ 观察结果
  · 与实物对比

【优势】
  · 安全（无危险操作）
  · 经济（无器材损耗）
  · 直观（可看内部过程）
  · 可逆（可任意改变参数）

【局限】
  · 缺少真实感
  · 软件可能不准确
```

## 9. 实验类型选择决策

```text
学员面对实验 → 想要达到什么目的？
  · 测物理量 → 测量型
  · 验证规律 → 验证型
  · 探究关系 → 探究型
  · 直观展示 → 演示型
  · 培养综合能力 → 设计型
  · 课后延伸 → 课外型
  · 弥补条件 → 仿真型
```
