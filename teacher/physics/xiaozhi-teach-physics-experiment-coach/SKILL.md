# 物理实验教学指导 SKILL

> **一句话定位：** 实验不是"走过场"，而是"真探究"——实验设计能力是核心。

---

## ⚠️ 技术实现边界声明

> **关于"AI 实验"边界：** 本 SKILL 输出**实验设计框架**与**教学流程**；**不**替老师做实验。
>
> **关于"实验材料"边界：** 实验器材必须标注 safetyLevel；不涉及危险实验；不推荐学员在家做危险实验。
>
> **关于"AI 评分"边界：** 本 SKILL **不**替老师给实验报告评分；只提供**评估框架**和**改进建议**。

---

## 一、核心使命

物理实验教学常见的三个误区：

```text
误区① 实验=走过场：演示实验老师做，
        学生实验按"标准步骤"做，
        学员没真正"探究"。

误区② 实验=记数据：学员只关心"数据对不对"，
        不会分析数据、不会找规律。

误区③ 实验=验证：实验只是为了"验证"已知规律，
        没让学员"发现"规律。
```

本 SKILL 要解决的是：
- **让实验"有目标"**：验证/探究/测量 三类
- **让实验"有设计"**：变量控制+器材选择+步骤
- **让数据"有分析"**：表格+图像+误差
- **让报告"有深度"**：结构+规范+反思
- **让实验"有探究"**：让学员"发现"规律

## 四、实验目标（三类）

### 4.1 三类实验

```text
┌──────────┬────────────────────────┬──────────────┐
│ 类型      │ 描述                    │ 目的          │
├──────────┼────────────────────────┼──────────────┤
│ 验证性    │ 验证已知规律            │ 加深理解      │
│ 探究性    │ 探究未知规律            │ 培养能力      │
│ 测量性    │ 测量物理量              │ 学会方法      │
└──────────┴────────────────────────┴──────────────┘
```

### 4.2 学员层次实验

```text
■ 基础层
  · 验证性实验
  · 测量性实验
  · 步骤明确

■ 中等层
  · 验证性实验
  · 简单探究性实验
  · 步骤部分自主

■ 提升层
  · 探究性实验
  · 设计性实验
  · 完全自主
```

### 4.3 验证 vs 探究

```text
■ 验证性
  · 老师讲过规律
  · 实验是为了"证实"
  · 学员被动接受

■ 探究性
  · 规律还没讲
  · 实验是为了"发现"
  · 学员主动探究

■ 教学建议
  · 学员基础弱时，验证为主
  · 学员能力强时，探究为主
  · 验证+探究结合
```

## 五、实验设计

### 5.1 实验设计四要素

```text
① 实验目的
  · 验证/探究/测量什么

② 实验原理
  · 用什么物理规律

③ 实验器材
  · 选什么器材

④ 实验步骤
  · 怎么操作
```

### 5.2 变量控制

```text
┌──────────┬────────────────────────┐
│ 变量      │ 描述                    │
├──────────┼────────────────────────┤
│ 自变量    │ 主动改变的量            │
│ 因变量    │ 随自变量变化的量        │
│ 控制变量  │ 保持不变的量            │
└──────────┴────────────────────────┘
```

### 5.3 实验设计原则

```text
■ 变量控制
  · 一次只改变一个变量
  · 其他保持不变

■ 多次测量
  · 减少偶然误差
  · 数据更可靠

■ 范围合理
  · 实验数据有规律
  · 不会"出界"

■ 安全第一
  · 老师监督
  · 学员有保护
```

### 5.4 实验设计样板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
🧪 实验设计样板 · 探究欧姆定律
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 实验目的
  探究电流与电压、电阻的关系

■ 实验原理
  电流 I = 电压 V / 电阻 R
  在 R 不变时，I 与 V 成正比
  在 V 不变时，I 与 R 成反比

■ 实验器材
  · 电源
  · 电阻（多个）
  · 电流表
  · 电压表
  · 滑动变阻器
  · 开关
  · 导线

■ 变量
  · 自变量：电压/电阻
  · 因变量：电流
  · 控制变量：电阻/电压

■ 实验步骤
  ① 连接电路
  ② 测 R1 不变时，不同 V 对应 I
  ③ 测 V 不变时，不同 R 对应 I
  ④ 记录数据
  ⑤ 画图
  ⑥ 分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```


## 十三、行为准则

| ✅ 应该做 | ❌ 不能做 |
|---------|---------|
| 实验三类（验证/探究/测量） | 实验=走过场 |
| 变量控制 | 一次多变量 |
| 数据处理（表格/图像/计算） | 实验=记数据 |
| 结论归纳（规律/讨论/反思） | 结论=抄数据 |
| 报告规范 | 实验=验证 |
| 学员实验档案累积 | 学员不积累 |
| 学员化名 | 公开批评学员实验 |
| 标注 safetyLevel | 推荐危险实验 |
