## 三、Bloom 提问链速查（备课时快速查阅）

| 层次 | 关键词 | 提问模板 |
|------|--------|----------|
| 记忆 | 是什么 / 在哪 / 多少 | "[概念] 的定义是什么？" |
| 理解 | 能不能用自己的话 | "你能用一句话给同桌解释 [概念] 吗？" |
| 应用 | 用这个做一下 | "用 [方法] 解决 [具体问题]。" |
| 分析 | 为什么 / 分几类 | "[A] 和 [B] 有什么相同和不同？" |
| 评价 | 哪个好 / 对不对 | "这两种解法你选哪个？为什么？" |
| 创造 | 你能想出 / 设计 | "用 [知识] 设计一个 [情境] 方案。" |

候场追问生成器（每个问题配 1-2 个）：

```text
候场 A：从"是不是"到"为什么"
  原问：y=kx+b 中 k 表示什么？
  候场：k 的正负决定什么？

候场 B：边界条件探查
  原问：k 是什么？
  候场：如果 k=0 呢？这变成什么函数？

候场 C：迁移到生活
  原问：一次函数图象是什么样子？
  候场：你能举一个生活中 y=kx 的例子吗？
```

## 四、A/B/C 分层任务卡模板

```text
【A 层任务卡】《[课题]》
  任务 1：[基础任务，明确步骤]
  任务 2：[半示范任务，教师先做一步]
  任务 3：[判断/识别型任务]
  ⏱ 预计 [   ] 分钟，弹性 +[   ] 分钟
  教师关注：频繁巡视 + 单独提示

【B 层任务卡】《[课题]》
  任务 1：[独立完成型]
  任务 2：[应用型，需要迁移]
  任务 3：[对比分析型]
  ⏱ 预计 [   ] 分钟
  教师关注：重点关注中间生

【C 层任务卡】《[课题]》
  任务 1：[完成 B 层任务]
  任务 2：[跨章/跨学科迁移]
  任务 3：[开放设计型]
  ⏱ 预计 [   ] 分钟，硬性截止
  教师关注：巡视 + 1v1 拔高
```

## 五、备课检查清单

每次备课完成前，对照以下清单自检：

- [ ] 三维目标每条都配了评估证据
- [ ] 没有任何目标写"理解""掌握""体会"等空话
- [ ] Bloom 提问链至少覆盖 4 个层次
- [ ] 每个问题都有 1-2 个候场追问
- [ ] 六步时间分配总和 = 课时长度
- [ ] A/B/C 三层任务卡明确差异（不是同一题三次）
- [ ] 风险预案至少 2 条
- [ ] 课后反思要点已留好填写位置
- [ ] 教案中没有真实学生姓名
