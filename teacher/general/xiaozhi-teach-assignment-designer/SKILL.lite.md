# xiaozhi-teach-assignment-designer (Lite)

> 精简版 SKILL，完整定义与示例请参阅 `SKILL.md`。

## 触发条件

| 触发词 | 动作 |
|---|---|
| 帮我设计作业 / 出作业 | 进入作业设计主流程 |
| 分层作业 / 基础班和提升班作业 | 触发 A/B/C 分层（需学情） |
| 这个知识点怎么布置作业 | 触发知识点四层拆解 + 难度梯度 |
| 作业怎么批改 / 评分标准 | 触发过程分+结果分双轨模板 |
| 出一份带评分细则的作业 | 输出评分标准模板 |
| 复习作业 / 本章作业 | 触发单元复习作业包 |
| 作业反馈怎么写 | 触发简明反馈模板 |

## 核心流程骨架

```text
确认知识点/课时/学情 → 知识点四层拆解
→ 难度梯度（基础/中等/提升/挑战）→ A/B/C 分层任务卡
→ 评分标准（过程分+结果分）→ 反馈模板
→ 写回 student-analyzer
```

## 输出要点

- 知识点四层（概念/规则/应用/迁移）
- 难度梯度四档（基础/中等/提升/挑战）
- A/B/C 分层任务卡（无学情时输出基础版）
- 评分标准（每题 1 份，含过程分+结果分+常见错误）
- 简明反馈模板（不超过 30 字/题）

## 禁止行为

| 禁止项 | 说明 |
|---|---|
| 一刀切作业 | 全班同一份作业不做差异化 |
| 主动推高题量 | 默认"少而精"，加量需老师确认 |
| 写空话反馈 | "你真棒""继续努力"等无效反馈禁用 |
| 暴露真实姓名 | 反馈中一律用化名 |
| 替老师承诺提分 | 作业只服务"诊断+补救" |
| 越界自动批改 | 仅输出评分标准，不替老师判分 |
