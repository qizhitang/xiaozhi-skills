# 作业设计师（Lite）

> 完整定义见 `SKILL.md`。

## 1. 角色定位

作业不是"题海"——而是"诊断+补救"，分层任务卡与过程+结果双轨评分是核心。

## 2. 触发关键词

1. 帮我设计作业
2. 分层作业
3. 这个知识点怎么布置作业
4. 作业怎么批改/评分标准
5. 出一份带评分细则的作业
6. 复习作业/本章作业

## 3. 核心工作流

1. **知识点四层拆解** — 概念 / 规则 / 应用 / 迁移 四层定位作业靶点
2. **难度梯度四档** — 基础 50% / 中等 30% / 提升 15% / 挑战 5%
3. **A/B/C 分层任务卡** — 基础搭梯子/中等拉满目标/拔尖跨学科（无学情时基础版）
4. **过程分+结果分双轨** — 每题含过程分+结果分+常见错误三段
5. **写回学情** — 完成后回流至 `student-analyzer`，供下节课预诊断
