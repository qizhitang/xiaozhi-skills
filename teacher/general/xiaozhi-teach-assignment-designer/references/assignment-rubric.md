## 二、A/B/C 分层任务卡模板

### 2.1 A 层（基础）

```text
【A 层作业卡】《[课题]》
  ⏱ 预计 25-30 分钟，弹性 +10 分钟

  1. [   ]（基础-概念）
     课本例题 [N]（直接套用规则）
     预计用时：[   ] 分钟

  2. [   ]（基础-规则）
     课本练习 [N]（套用规则）
     预计用时：[   ] 分钟

  3. [   ]（基础-规则）
     课本练习 [N]（一步变形）
     预计用时：[   ] 分钟

  4. [   ]（基础-概念）
     看图填空（k>0/k<0 时图象走向）
     预计用时：[   ] 分钟

  📌 重点：把每一步"为什么"写清楚
```

### 2.2 B 层（中等）

```text
【B 层作业卡】《[课题]》
  ⏱ 预计 30-35 分钟

  1. A 层作业 1-2
  2. [   ]（中等-应用）
     变式题 [N]（参数符号变化）
     预计用时：[   ] 分钟
  3. [   ]（中等-应用）
     课本练习 [N]（用图象解应用题）
     预计用时：[   ] 分钟
  4. [   ]（提升-应用）
     综合题 [N]（一次函数与正比例函数对比）
     预计用时：[   ] 分钟

  📌 重点：写出思路，不只写答案
```

### 2.3 C 层（拔尖）

```text
【C 层作业卡】《[课题]》
  ⏱ 预计 35-40 分钟，硬性截止

  1. B 层作业 1-3
  2. [   ]（挑战-迁移）
     跨章综合：用一次函数解释物理中匀速运动
     预计用时：[   ] 分钟
  3. [   ]（挑战-迁移）
     开放题：设计一个能列出一次函数的生活情境
     预计用时：[   ] 分钟

  📌 重点：跨章联结的论证
```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 六、反馈话术库

### 6.1 给学生的简明反馈（≤ 30 字/题）

```text
■ 概念类
  "概念 X 的定义是 Y，请重读课本第 N 页"
  "你把 X 和 Y 概念搞混了，区别在于 A 和 B"
  "X 的核心特征是 A，你没说到"

■ 规则类
  "Y 规则的关键步骤是 Z，你跳过了哪步"
  "用 Y 规则时，先看 [条件] 再 [动作]"
  "X 公式的适用条件是 A，本题满足吗？"

■ 审题类
  "看清题目问的是 A 不是 B"
  "题目说'不正确的是'，你看清否定词了吗"
  "题目给了 [条件]，你用到了吗？"

■ 策略类
  "这类题用 X 方法更快，试试看"
  "你可以先把 [已知] 列出来，再选方法"
  "X 方法的标志是 [条件]，本题符合吗？"

■ 计算类
  "思路对，再算一遍"
  "X 步骤的运算是 A → B，你算成 C 了"
  "检查符号和单位"
```

### 6.2 给家长的反馈（脱敏版，≤ 200 字）

```text
[学生化名] 本次作业完成情况：
  · 提交：[N] 道 / 总 [N] 道
  · 错题：[N] 道（主要集中在 [知识点]）
  · 主要错因：[聚合错因]

建议家长配合：
  · 关注孩子 [具体行为] 时段
  · 与孩子约定 [1 个具体动作]
  · 不必焦虑：[降低家长焦虑的话术]
```

## 七、批改标记速查

| 标记 | 含义 | 用途 |
|------|------|------|
| ✓ | 完全正确 | 标在题号旁 |
| ▲ | 思路对但计算/书写错 | 标在错误处 |
| ? | 思路不清/不完整 | 标在关键步 |
| ✗ | 思路错误 | 标在错处 |
| ?✓ | 部分对，看具体情况 | 教师判断 |
| # | 超纲或独特解法 | 需教师判断 |

## 八、自检清单

每次输出作业前，AI 必须自检：

- [ ] 是否每题都对应一个明确的知识点层级
- [ ] 是否覆盖概念/规则/应用三层（④ 仅在有学情时）
- [ ] 分层任务卡是否有明确差异（不是同一题三次）
- [ ] 每题是否配可观测的评分标准
- [ ] 总题量是否符合"少而精"原则
- [ ] 反馈话术是否基于具体行为
- [ ] 写回 student-analyzer 时是否做了脱敏
- [ ] 是否避免了真实姓名

---

> **使用提示**：建议老师先快速列出 5-8 道原始题目，让 AI 用本框架重新组织；不要直接接受 AI 的第一版。
