## 二、候场追问速查表

| 类型 | 适用情境 | 模板 |
|------|---------|------|
| A 是不是→为什么 | 答案正确但浅 | "X 是这样，那 Y 呢？为什么？" |
| B 边界条件探查 | 学生没考虑特殊情况 | "如果 X 变成 Y，会怎样？" |
| C 迁移到生活 | 抽象概念 | "你能举一个生活中 X 的例子吗？" |
| D 让基础学生参与 | 题目太难 | "想象你在 [生活场景]，X 是什么？" |
| E 让拔尖学生深入 | 答案正确但简单 | "如果 Y 变化，X 会怎么变？" |

## 四、互动过渡话术库

```text
■ 全班 → 小组
  "接下来 [N] 分钟，请 [N] 人小组讨论 [X]，计时员看时间。"
  "请每组先分配角色：主持人、记录员、汇报员、计时员。"

■ 小组 → 全班
  "时间到。请第 [X] 组汇报员分享，其他组倾听并准备点评。"
  "其他组可以用一句话补充或不同意见。"

■ 讲授 → 提问
  "讲到这里，我想问大家一个关键问题：[X]"
  "我刚才讲得最快的那部分，谁能用一句话说清楚？"

■ 提问 → 练习
  "现在请大家独立完成 [X]，[N] 分钟后同桌互对。"
  "做完后请在题号旁画 ✓ 或 ✗，方便我巡视。"

■ 沉闷 → 活跃
  "我出一道抢答题，最快举手答对的小组加分。"
  "这次我们换一种方式——[小组讨论/同桌互讲/独立思考]。"

■ 一个活动结束
  "刚才这个环节大家表现很好，特别是 [具体表扬]。现在我们进入下一个环节。"
```

## 五、冷场应对 3 步走

```text
第 1 步：等 5-8 秒（不要立刻换问题）
  → 大部分学生需要时间思考
  → 在心里默数

第 2 步：等 10 秒还没人举手
  → 改问更简单的候场问题
  → "我再问简单一点：[候场问题]"

第 3 步：还是没人举手
  → 老师自问自答但故意留半句
  → "其实我心里有一个答案的起点，是 [X]，谁能接着说？"
  → 或者直接让同桌互讲 2 分钟，再回到全班
```

## 六、基础学生参与路径

```text
事前准备：
  · 提前给基础学生"小纸条"提示
  · 让基础学生先在同桌间小声回答
  · 老师巡视时优先听基础学生的回答

事中引导：
  · 老师听到正确版本后说"我刚才听到 XX 说得很好，请她/他分享"
  · 任何时候答对都给予"具体"表扬
  · 答错时不批评，只纠正

事后巩固：
  · 课后单独点评基础学生
  · 在作业分层中给基础学生更友好的任务
  · 让基础学生担任"计时员/记录员"等低门槛角色
```

## 七、反馈话术库

### 7.1 正面反馈（具体化）

```text
❌ 模糊："你答得不错"
✅ 具体："你抓住了'k>0'这个关键点，这正是图象上升的原因"

❌ 模糊："你真棒"
✅ 具体："你今天主动发言了 3 次，比上周多了 2 次"

❌ 模糊："讲得清楚"
✅ 具体："你用三步就把'为什么 k>0 图象上升'讲明白了，第二步那个'X 越大 Y 越大'最关键"
```

### 7.2 纠错反馈（不羞辱）

```text
❌ 否定："这都不对"
✅ 引导："你的思路有 X 部分是对的，但 Y 这步可能需要再想想"

❌ 嘲笑："这么简单都不会"
✅ 鼓励："这道题确实有难度，我们一起想想"

❌ 直接给答案："答案是 X，记住了吗"
✅ 引导："你离答案只差一步，是 X"
```

### 7.3 沉默应对

```text
学生说"不会"时：
  · "不会很正常，我们一起来想"
  · "那我给你一个提示：[X]"
  · "给你 30 秒想一下"

学生沉默时：
  · 等 5-8 秒，不要立刻换
  · "我再换一种方式问：[候场问题]"
  · "我刚才讲的那部分，谁能用自己的话说一遍？"
```

## 八、课堂节奏调控

```text
注意力曲线：
  0-10 min：高度集中（适合讲核心概念）
  10-25 min：开始分散（需要切换活动）
  25-35 min：再次集中（适合讲难点）
  35-45 min：疲劳期（适合总结、布置作业）

每 10-15 分钟切换一次活动形式
  · 全班 → 小组 → 同桌 → 独立 → 全班
  · 重点内容放在黄金时段
  · 难点要在第二次集中时讲
```

  · 沉默：[X]

  · 问题难度：[升/降]

> **使用提示**：建议老师课前 5 分钟浏览本提问链模板，挑选 6-8 问作为本节课核心提问；课中根据学生反应调用候场追问；课后 5 分钟内补完观察记录。
