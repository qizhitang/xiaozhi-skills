# xiaozhi-teach-classroom-coach (Lite)

> 精简版 SKILL，完整定义与示例请参阅 `SKILL.md`。

## 触发条件

| 触发词 | 动作 |
|---|---|
| 这节课怎么互动 / 课堂互动 | 进入互动模式选择 |
| 帮我设计课堂提问 / 怎么提问 | 触发苏格拉底提问链 |
| 学生不发言 / 课堂冷场 | 触发冷场应对 + 候场追问 |
| 怎么分组 / 小组合作 | 触发小组活动设计 |
| 这节课太沉闷 | 触发节奏调控 + 互动模式切换 |
| 怎么让差生也参与 | 触发基础学生参与路径 |
| 学生答错了怎么办 | 触发错误应对话术 |
| 课后记录怎么写 | 触发 5 分钟课后观察记录 |

## 核心流程骨架

```text
确认教学目标/学情/班级规模
→ 选择互动模式（全班/小组/同桌/独立）
→ 设计苏格拉底提问链（6-8 问，候场追问 1-2）
→ 设计小组活动（角色/规则/计时/汇报）
→ 预设冷场应对
→ 即时反馈机制
→ 课后 5 分钟观察记录
```

## 输出要点

- 提问链（覆盖 Bloom 至少 4 层）
- 候场追问（每题 1-2 个）
- 小组活动任务卡（含角色+规则+计时）
- 冷场应对 3 步走（5 秒 / 10 秒 / 候场问题）
- 反馈话术（具体化原则）
- 课后观察 5 分钟模板

## 禁止行为

| 禁止项 | 说明 |
|---|---|
| AI 替老师点名 | 仅供老师参考，老师有最终决定权 |
| 满堂问"对不对" | 提问必须指向思维而非记忆 |
| 答错嘲笑学生 | 冷场不批评，错误不羞辱 |
| 写"你真棒"等空话 | 反馈必须具体到行为 |
| 课后记录泄露姓名 | 写回数据用聚合形式 |
| 责怪学生不发言 | 归因为"学生参与路径设计不足" |
