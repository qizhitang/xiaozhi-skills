## 一、知识图谱四层结构

```text
┌────────────────────────────────────────────────────┐
│ 知识图谱四层                                        │
├────────────────────────────────────────────────────┤
│ ① 核心概念层（最底层）                              │
│   · 学生必须准确复述的术语、定义、规则               │
│   · 例：函数、变量、斜率、截距                        │
├────────────────────────────────────────────────────┤
│ ② 规则方法层（中层）                                │
│   · 学生必须熟练运用的方法                           │
│   · 例：待定系数法、列表法、描点法                   │
├────────────────────────────────────────────────────┤
│ ③ 综合应用层（上层）                                │
│   · 多个概念的组合应用                               │
│   · 例：用函数解决行程问题                           │
├────────────────────────────────────────────────────┤
│ ④ 跨科联结层（最高层）                              │
│   · 与其他学科/章节的联结                            │
│   · 例：一次函数 ↔ 物理匀速运动                     │
└────────────────────────────────────────────────────┘
```

## 四、单元复习 5 阶段模板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 单元复习计划 · [单元] · 共 [N] 课时
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 阶段 1：知识图谱建构（1-2 课时）
  · 学生画本章思维导图
  · 教师补全关键节点
  · 标记每个人的薄弱点

■ 阶段 2：分层突破（2-3 课时）
  · 重点：教师精讲 + 学生互讲
  · 难点：拆成小步骤 + 多角度举例
  · 非重点：快速过

■ 阶段 3：综合应用（1-2 课时）
  · 跨章节综合题
  · 真实情境建模
  · 小组合作探究

■ 阶段 4：错题回炉（1 课时）
  · 把本月 X 道错题重做
  · 重点关注 3 次同错的题
  · 错题归档

■ 阶段 5：单元测评（0.5-1 课时）
  · 闭卷/开卷测评
  · 立即讲评关键错题
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 五、考前分阶段策略

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📅 考前分阶段策略 · [考试]
━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 距考试 > 14 天：系统梳理模式
  · 重点：弱项专项突破，建立完整知识框架
  · 每日节奏：新知识 60% + 错题复习 30% + 主动回忆 10%

■ 距考试 7-14 天：强化整合模式
  · 重点：知识点联动，模拟综合练习
  · 每日节奏：综合练习 50% + 弱项强化 35% + 轻松复习 15%

■ 距考试 3-7 天：冲刺提炼模式
  · 重点：核心考点速览，错题清单最终确认
  · 每日节奏：核心要点复习 60% + 做题验证 30% + 心理建设 10%
  · ⚠️ 此阶段不引入新知识点

■ 距考试 1-3 天：信心建立模式
  · 重点：历史上做对过的同类题，建立上场信心
  · 每日节奏：熟悉题型 70% + 休息保状态 30%
  · ⚠️ 不出新题型，专练已会的

■ 考试当天：状态保持模式
  · 早上：基于学情摘要给鼓励话术
  · 深呼吸 + 准备齐全 + 信心入场
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 六、复习活动设计

### 6.1 主动回忆法

```text
■ 闭书回忆（5-10 分钟）
  · 学生不看课本，凭记忆列出本章所有概念/规则
  · 写不出的 = 真正的盲点
  · 教师巡视，记录高频"空白"

■ 同桌互考（10 分钟）
  · A 给 B 出一道本章题，B 答完换角色
  · 5 分钟 × 2 轮
  · 重点：让学生"出题"远比"做题"暴露盲点

■ 错题重做（10-15 分钟）
  · 拿错题本重做
  · 限时
  · 重点关注第 3 次做的题
```

### 6.2 概念网络法

```text
■ 思维导图（15-20 分钟）
  · 4 人小组共画本章思维导图
  · 用不同颜色标记：核心/重要/了解
  · 跨章节联结用虚线箭头表示

■ 概念对比表（10-15 分钟）
  · 横向对比易混概念
  · 例：一次函数 vs 二次函数 vs 反比例函数
  · 列：定义/图象/性质/典型应用
```

### 6.3 跨科联结法

```text
■ 数学 ↔ 物理
  · 一次函数 ↔ 匀速运动 s=vt
  · 二次函数 ↔ 匀变速 s=vt+½at²
  · 三角函数 ↔ 简谐运动

■ 数学 ↔ 经济
  · 一次函数 ↔ 成本-收益分析
  · 不等式 ↔ 利润最大化

■ 语文 ↔ 历史
  · 文言文 ↔ 历史背景
  · 诗词 ↔ 作者生平
```

### 6.4 错题档案法

```text
■ 三栏错题本
  · 左栏：原题
  · 中栏：当时错因
  · 右栏：现在的正确思路
  · 复习时只看右栏

■ 错题热力图
  · 按知识点排列错题
  · 红色 = 3 次以上错
  · 黄色 = 2 次
  · 绿色 = 已攻克
```

## 七、考前心理建设话术

### 7.1 推荐话术（老师对全班）

```text
考前 7 天：
  "复习已经完成 80%。剩下 7 天不是新任务，
  是把已经会的东西再确认一遍。"

考前 3 天：
  "最后 3 天只做 1 件事：翻错题本最后一遍，
  记得自己已经搞定了哪些题。"

考前 1 天：
  "明天考试，不用再想新内容了。
  把错题本翻最后一遍，记得自己攻克过的那些题。
  准备好 [考试用品清单]，早睡。
  明天上考场，你就是 [同学们这段时间努力] 的自己。"

考试当天早上：
  "今天考 [学科]。基于 [学情摘要]：
  ✅ [优势点] 已经掌握
  📌 [弱项] 还需注意：[一句话提醒]
  深呼吸，你已经准备好了。"
```

### 7.2 避免话术

```text
❌ "这次考试很重要"
❌ "考不好后果很严重"
❌ "你们再看看这些重点"
❌ "如果这次没考好..."
❌ "别人都已经..."
```

## 九、自检清单

每次输出复习计划前，AI 必须自检：

- [ ] 是否基于真实学情数据（student-analyzer）识别重难点
- [ ] 知识图谱是否四层都覆盖
- [ ] 复习活动是否包含主动回忆/概念网络/错题档案至少 2 种
- [ ] 阶段策略是否与时间倒推匹配
- [ ] 考前 3-7 天是否明确"不引入新知识点"
- [ ] 心理建设话术是否基于学习策略
- [ ] 写回数据是否做了脱敏
- [ ] 是否避免了承诺提分
- [ ] 是否避开了焦虑话术

> **使用提示**：建议老师先用本模板梳理知识图谱，再去填阶段计划；不要直接套用模板，必须基于本班学情做调整。
