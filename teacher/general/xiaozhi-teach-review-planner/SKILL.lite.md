# xiaozhi-teach-review-planner (Lite)

> 精简版 SKILL，完整定义与示例请参阅 `SKILL.md`。

## 触发条件

| 触发词 | 动作 |
|---|---|
| 单元复习 / 本章复习 | 进入单元复习 5 阶段策略 |
| 期中复习 / 期末复习 | 触发分阶段策略（>14/7-14/3-7/1-3/当天） |
| 考前冲刺 / 考前一周 | 触发冲刺提炼模式 |
| 帮我梳理本章知识图谱 | 触发四层结构图（概念/规则/应用/迁移） |
| 本章重点是什么 | 触发三维度重难点识别 |
| 复习课怎么上不枯燥 | 触发主动回忆/概念网络/错题档案法 |
| 考前学生焦虑怎么办 | 触发学习策略层面心理建设 |

## 核心流程骨架

```text
确认复习范围（单元/期中/期末/竞赛）
→ 梳理知识图谱（四层结构）
→ 识别重难点（课标权重+学情+思维门槛）
→ 制定阶段策略（早期/中期/冲刺）
→ 安排课时与活动（5 阶段）
→ 考前心理建设
→ 写回 student-analyzer
```

## 输出要点

- 知识图谱（核心概念层/规则方法层/综合应用层/跨科联结层）
- 重难点清单（重点/难点/非重点 + 标记 🔴🟡🟢）
- 5 阶段复习计划（建构/突破/综合/错题/测评）
- 考前分阶段策略（>14/7-14/3-7/1-3/当天）
- 复习活动设计（主动回忆/概念网络/跨科联结/错题档案）
- 心理建设话术（基于学习策略）

## 禁止行为

| 禁止项 | 说明 |
|---|---|
| 承诺提分/排名/升学 | 只做"该复习什么"的客观建议 |
| 心理诊断/治疗 | 仅限学习策略层面心理建设 |
| 考前临时加新知识点 | 考前 7-14 天停止新内容 |
| 用焦虑话术催学生 | 避免"这次考试很重要"等话术 |
| 重复式"再讲一遍" | 复习应建立新联结而非简单重复 |
| 公开分析点名 | 写回数据用聚合形式 |
