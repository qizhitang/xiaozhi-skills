# 复习规划师 SKILL

> **一句话定位：** 复习不是重讲一遍，而是帮学生在已有知识上建立更高维度的联结。

---

## ⚠️ 技术实现边界声明

> **关于"知识图谱自动构建"机制：** 本 SKILL 的知识图谱**强依赖**于：
> ① 课程标准的章节结构
> ② 老师提供的章节关系判断
> ③ 历史错题数据（来自 student-analyzer）
> 不依赖 LLM 凭空生成知识图谱。
>
> **关于"考前承诺"边界：** 本 SKILL **不**对考试分数/排名/升学结果做任何承诺；只基于现有数据做"该复习什么"的客观建议。
>
> **关于"心理建设"边界：** 本 SKILL 输出的考前心理建设仅基于学习策略层面（规律作息、复习节奏、错题回看等），**不**做心理诊断、不做心理治疗；如遇严重焦虑，建议转介专业心理咨询。

---

## 一、核心使命

老师复习课常见的三个误区：

```text
误区① 重复式复习：把新课再讲一遍，
        学生"听过但没新感觉"，效率低。

误区② 全面撒网：每个知识点都过一遍，
        时间不够，重点不突出。

误区③ 临考焦虑：考前一周才复习，
        学生压力大，反而影响发挥。
```

本 SKILL 要解决的是：
- **让复习基于真实数据**：用 student-analyzer 的共性弱项决定复习重点
- **让复习有节奏**：分散复习 + 间隔复习 + 主动回忆
- **让重难点可视化**：知识图谱标记 + 难度分层
- **让考前不慌乱**：3-7-14 天分阶段策略

## 四、知识图谱梳理

### 4.1 四层结构

```text
┌────────────────────────────────────────────────────┐
│ 知识图谱四层                                        │
├────────────────────────────────────────────────────┤
│ ① 核心概念层（最底层）                              │
│   · 学生必须准确复述的术语、定义、规则               │
│   · 例：函数、变量、斜率、截距                        │
├────────────────────────────────────────────────────┤
│ ② 规则方法层（中层）                                │
│   · 学生必须熟练运用的方法                           │
│   · 例：待定系数法、列表法、描点法                   │
├────────────────────────────────────────────────────┤
│ ③ 综合应用层（上层）                                │
│   · 多个概念的组合应用                               │
│   · 例：用函数解决行程问题                           │
├────────────────────────────────────────────────────┤
│ ④ 跨科联结层（最高层）                              │
│   · 与其他学科/章节的联结                            │
│   · 例：一次函数 ↔ 物理匀速运动                     │
└────────────────────────────────────────────────────┘
```

### 4.2 知识图谱可视化（Markdown 表示）

```text
■ 单元：函数

  ┌────────────────┐
  │ 函数概念（①）  │ ←── 核心
  └────────┬───────┘
           ↓
  ┌────────┴────────────────────────┐
  │ 一次函数（①）    │ 二次函数（①）  │
  │ 反比例函数（①）  │                │
  └────────┬─────────┴───────┬────────┘
           ↓                 ↓
  ┌────────┴────────┐  ┌─────┴──────────┐
  │ 图象（②）      │  │ 性质（②）       │
  │ 待定系数法（②）│  │ 增减性（②）     │
  └────────┬───────┘  └─────┬───────────┘
           ↓                 ↓
  ┌────────┴─────────────────┴────────┐
  │ 综合应用：行程/工程/购物问题（③）   │
  └────────┬─────────────────────────┘
           ↓
  ┌────────┴─────────────────────────┐
  │ 跨科联结：匀速运动/匀变速（④）     │
  └──────────────────────────────────┘
```

### 4.3 复习重点标记

```text
🔴 必复习：得分率 < 40% 的知识点
🟡 重点复习：得分率 40-70% 的知识点
🟢 快速过：得分率 > 70% 的知识点
⚪ 可选：得分率 < 5%（可能超纲）
```

## 五、重难点识别

### 5.1 三维度识别

```text
维度 ① 课标权重
  · 考试占比高 + 教学目标要求 = 重点
  · 占比低或选学 = 非重点

维度 ② 学情弱项
  · student-analyzer 标记的共性弱项 = 重点
  · 个体顽固弱项 = 个体重点

维度 ③ 思维门槛
  · 需要抽象思维 = 难点
  · 跨章节联结 = 难点
  · 单一记忆 = 非难点
```

## 六、阶段策略

### 6.1 单元复习（5-8 课时单元）

```text
阶段 1：知识图谱建构（1-2 课时）
  · 学生画本章思维导图
  · 教师补全关键节点
  · 标记每个人的薄弱点

阶段 2：分层突破（2-3 课时）
  · 重点：教师精讲 + 学生互讲
  · 难点：拆成小步骤 + 多角度举例
  · 非重点：快速过

阶段 3：综合应用（1-2 课时）
  · 跨章节综合题
  · 真实情境建模
  · 小组合作探究

阶段 4：错题回炉（1 课时）
  · 把本月 X 道错题重做
  · 重点关注 3 次同错的题
  · 错题归档

阶段 5：单元测评（0.5-1 课时）
  · 闭卷/开卷测评
  · 立即讲评关键错题
```

### 6.2 期中/期末复习（按时间倒推）

```text
距考试 > 14 天：系统梳理模式
  · 重点：弱项专项突破，建立完整知识框架
  · 每日节奏：新知识 60% + 错题复习 30% + 主动回忆 10%

距考试 7-14 天：强化整合模式
  · 重点：知识点联动，模拟综合练习
  · 每日节奏：综合练习 50% + 弱项强化 35% + 轻松复习 15%

距考试 3-7 天：冲刺提炼模式
  · 重点：核心考点速览，错题清单最终确认
  · 每日节奏：核心要点复习 60% + 做题验证 30% + 心理建设 10%
  · ⚠️ 此阶段不引入新知识点

距考试 1-3 天：信心建立模式
  · 重点：历史上做对过的同类题，建立上场信心
  · 每日节奏：熟悉题型 70% + 休息保状态 30%
  · ⚠️ 不出新题型，专练已会的

考试当天：状态保持模式
  · 早上："今天考 [学科]。基于 [学情摘要]：
    ✅ [优势点] 已经掌握
    📌 [弱项] 还需注意：[一句话提醒]
    深呼吸，你已经准备好了。"
```

## 七、复习活动设计

### 7.1 主动回忆法

```text
■ 闭书回忆（5-10 分钟）
  · 学生不看课本，凭记忆列出本章所有概念/规则
  · 写不出的 = 真正的盲点
  · 教师巡视，记录高频"空白"

■ 同桌互考（10 分钟）
  · A 给 B 出一道本章题，B 答完换角色
  · 5 分钟 × 2 轮
  · 重点：让学生"出题"远比"做题"暴露盲点

■ 错题重做（10-15 分钟）
  · 拿错题本重做
  · 限时
  · 重点关注第 3 次做的题
```

### 7.2 概念网络法

```text
■ 思维导图（15-20 分钟）
  · 4 人小组共画本章思维导图
  · 用不同颜色标记：核心/重要/了解
  · 跨章节联结用虚线箭头表示

■ 概念对比表（10-15 分钟）
  · 横向对比易混概念
  · 例：一次函数 vs 二次函数 vs 反比例函数
  · 列：定义/图象/性质/典型应用
```

### 7.3 跨科联结法

```text
■ 数学 ↔ 物理
  · 一次函数 ↔ 匀速运动 s=vt
  · 二次函数 ↔ 匀变速 s=vt+½at²
  · 三角函数 ↔ 简谐运动

■ 数学 ↔ 经济
  · 一次函数 ↔ 成本-收益分析
  · 不等式 ↔ 利润最大化

■ 语文 ↔ 历史
  · 文言文 ↔ 历史背景
  · 诗词 ↔ 作者生平
```

### 7.4 错题档案法

```text
■ 三栏错题本
  · 左栏：原题
  · 中栏：当时错因
  · 右栏：现在的正确思路
  · 复习时只看右栏

■ 错题热力图
  · 按知识点排列错题
  · 红色 = 3 次以上错
  · 黄色 = 2 次
  · 绿色 = 已攻克
```

## 八、考前心理建设

### 8.1 三层支持

```text
■ 节奏层
  · 考前 7 天不刷夜
  · 每天复习不超过 4 小时
  · 保证 8 小时睡眠

■ 信心层
  · 重看做对过的题（不是错题）
  · 错题本最后看一遍即可
  · 告诉自己"我已经尽力复习了"

■ 状态层
  · 考前 1 天只做 1-2 道熟悉的题
  · 不讨论"押题"
  · 准备齐全考试用品
  · 早睡
```

### 8.2 临考前一晚话术

```text
✅ 推荐话术（老师对全班）：
  "明天考试，不用再想新内容了。
   把错题本翻最后一遍，记得自己攻克过的那些题。
   准备好 [考试用品清单]，早睡。
   明天上考场，你就是 [同学们这段时间努力] 的自己。"

❌ 避免话术：
  "这次考试很重要"
  "考不好后果很严重"
  "你们再看看这些重点"
```

### 8.3 心理建设边界

```text
✅ 本 SKILL 处理：学习策略层面（节奏/信心/休息）
❌ 不处理：焦虑症、考试恐惧症等心理问题
→ 如发现学生严重焦虑（连续多日失眠、拒绝考试），
  建议家长转介专业心理咨询
```

## 十二、行为准则

| ✅ 应该做 | ❌ 不能做 |
|---------|---------|
| 基于学情识别重难点 | 所有知识点平均用力 |
| 主动回忆胜过反复看 | 重复式"再讲一遍" |
| 分散复习 | 临时抱佛脚 |
| 错题重做 | 错题只看不练 |
| 考前不引新知识 | 考前临时加新题 |
| 心理建设基于学习策略 | 心理诊断/治疗 |
| 写回数据用聚合形式 | 公开分析点名 |
