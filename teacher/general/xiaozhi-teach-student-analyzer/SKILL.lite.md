# xiaozhi-teach-student-analyzer (Lite)

> 精简版 SKILL，完整定义与示例请参阅 `SKILL.md`。

## 触发条件

| 触发词 | 动作 |
|---|---|
| 帮我分析班级成绩 / 班级整体什么水平 | 触发班级整体画像 |
| 全班错了一半 / X 知识点全军覆没 | 触发共性弱项识别 |
| 这个学生最近怎么了 / 这次退步严重 | 触发个体诊断（需 ≥ 3 次数据） |
| 对比上次和这次 / 趋势追踪 | 触发趋势对比 |
| 接下来重点讲什么 / 怎么调整教学 | 触发教学调整建议 |
| 哪些学生该进提升班 | 触发分层建议 |
| 跟家长客观说明学情 | 生成家长沟通素材（脱敏版） |

## 核心流程骨架

```text
收集数据 → 班级画像（分布/分层/难度/区分度）
→ 知识点热力图 → 共性弱项排序
→ 个体诊断卡（≥ 3 次数据）→ 教学调整建议
→ 写回 lesson-planner / classroom-coach / exam-designer
```

## 输出要点

- 班级画像：平均/中位数/标准差/优秀率/及格率/低分率/分布形态/难度 P/区分度 D
- 知识点热力图：🔴 < 40% / 🟡 40-70% / 🟢 > 70%
- 共性弱项排序（按严重度）
- 个体诊断卡（10 字段）
- 教学调整建议：讲/练/测 比例 + 分层任务

## 禁止行为

| 禁止项 | 说明 |
|---|---|
| 1 次数据下趋势结论 | 必须 ≥ 3 次同知识点失分才标"顽固弱项" |
| 写"理解""掌握"等空话目标 | 所有结论需可观测 |
| 预测未来分数 | 不做"提分 X 分"承诺 |
| 点名 | 公开报告用聚合数据，不写"小名" |
| 没有数据时编造 | 缺数据时输出"证据不足，需补测" |
| 替老师判断品行 | 成绩差不等于态度差 |
