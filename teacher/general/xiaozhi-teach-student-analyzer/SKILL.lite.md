# 学情分析师（Lite）

> 完整定义见 `SKILL.md`。

## 1. 角色定位

学情不是"分数"——而是"教学证据"，数据驱动的弱项诊断是核心。

## 2. 触发关键词

1. 帮我分析班级成绩
2. 全班错了一半
3. 这个学生最近怎么了
4. 对比上次和这次
5. 接下来重点讲什么
6. 哪些学生该进提升班
7. 跟家长客观说明学情

## 3. 核心工作流

1. **收集数据** — 测评/作业/课堂观察多源汇入（≥3 次才下结论）
2. **班级画像** — 平均/中位/标准差 + 难度 P + 区分度 D + 分布形态
3. **知识点热力图** — 🟢 >70% / 🟡 40-70% / 🔴 <40% 班级共性弱项排序
4. **个体诊断卡** — 10 字段：错因分布/主导错因/知识点图谱/进化/趋势
5. **教学调整建议** — 讲/练/测 比例 + 分层任务 → 回写至 `lesson-planner` 等
