# xiaozhi-teach-resource-library (Lite)

> 精简版 SKILL，完整定义与示例请参阅 `SKILL.md`。

## 触发条件

| 触发词 | 动作 |
|---|---|
| 帮我找一下 [X 讲义] / 有没有 [X 题型] | 触发关键词检索 |
| 这类错题怎么讲评 / 讲评怎么说 | 触发讲评话术库检索 |
| [化名] 适合什么资源 | 触发学员画像匹配 |
| 教过的案例 / 类似情况怎么处理 | 触发错因案例库检索 |
| 这个讲义/题库/案例存一下 | 触发资源入库流程 |
| 资源怎么分类 / 教学资源库 | 触发五大类分类说明 |
| 学员犯过这种错吗 / 类似错因 | 触发错因案例检索 |
| 这个教案可以给另一个学员用吗 | 触发复用与改编流程 |

## 核心流程骨架

```text
资源分类入库（讲义/题库/讲评/案例/教案）
→ 标签化（学科/章节/难度/学员类型）
→ 检索与匹配（关键词/学员画像/错因）
→ 复用与改编（4 步：检索→评估→调整→记录）
→ 写回 solo-dashboard
→ 持续更新（每次教学都是更新机会）
```

## 输出要点

- 资源五大类（讲义/题库/讲评/案例/教案）
- 标签化体系（7 维度：学科/章节/知识点/难度/学员类型/学段/自定义）
- 三种检索方式（关键词/学员画像/错因）
- 复用四步走（检索→评估→调整→记录）
- 改编四原则（改数/改问/改情境/改综合度）
- 版权管理四档（自有/改编/公开可引用/仅存索引）
- 案例脱敏规则（必脱敏 6 类字段）
- 资源生命周期（新入库→试用→优化→成熟→经典→归档）

## 禁止行为

| 禁止项 | 说明 |
|---|---|
| 存储未授权教辅原题 | 必须有 copyrightStatus 标注 |
| 学员案例不脱敏入库 | 化名 + 模糊年级 + 去除可识别细节 |
| 资源入库不打标签 | 入库即打主标签 + 推荐辅标签 |
| 资源库检索反查具体学员 | 仅基于画像/关键词/错因 |
| 资源库自动推送资源 | 老师按需检索复用 |
| 复用不调整适配 | 必须修改参数/难度/学员元素 |
| 用了不反馈 | 每次使用记录效果 |
