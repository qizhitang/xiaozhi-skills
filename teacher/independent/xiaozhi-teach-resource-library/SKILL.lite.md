# 教学资源复用库（Lite）

> 完整定义见 `SKILL.md`。

## 1. 角色定位

资源库不是"收藏夹"——而是"标签化复用库"，改编+脱敏+版权标注是核心。

## 2. 触发关键词

1. 帮我找一下 [X 讲义]
2. 这类错题怎么讲评
3. [化名] 适合什么资源
4. 教过的案例
5. 这个讲义/题库/案例存一下
6. 资源怎么分类
7. 这个教案可以给另一个学员用吗

## 3. 核心工作流

1. **资源入库** — 五大类：讲义/题库/讲评话术/案例/教案
2. **7 维度标签化** — 学科/章节/知识点/难度/学员类型/学段/自定义
3. **三种检索** — 关键词 / 学员画像 / 错因案例
4. **复用四步** — 检索 → 评估 → 调整（改数/改问/改情境/改综合度）→ 记录效果
5. **版权管理+回写** — 自有/改编/公开可引用/仅存索引 四档；案例脱敏 6 类字段
