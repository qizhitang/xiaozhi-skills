# 教学资源复用库 SKILL

> **一句话定位：** 独立教师最贵的资产不是学员名单，而是你积累的教学经验——把它从"脑子里"搬到"库里"，才算真的拥有。

---

## ⚠️ 技术实现边界声明

> **关于"版权"边界：** 资源库中**禁止**存储未授权的教辅原题、盗版扫描件；所有资源必须有 copyrightStatus 标注（自有/改编/公开可引用/仅存索引）。
>
> **关于"学员案例"边界：** 学员案例入库必须**完全脱敏**（化名/模糊年级/去除可识别细节）；不允许反向追溯到具体学员。
>
> **关于"自动推送"边界：** 资源库不主动推送资源到学员；老师根据需要检索复用。

---

## 一、核心使命

独立教师资源管理常见的三个误区：

```text
误区① 资源散落：讲义在电脑文件夹、题库在微信收藏、
        讲评话术在脑子里、错题案例在笔记本上，
        需要时找不到，找到时已过时。

误区② 从零开始：每个新学员/新学期都从零备课，
        之前教过的优秀案例、题库、讲评话术
        都没有积累，白白浪费。

误区③ 资源孤岛：优秀讲义、题库、讲评话术
        只在某个学员/某次课用一次，
        之后被遗忘。
```

本 SKILL 要解决的是：
- **让教学资源有"家"**：统一分类入库
- **让资源可检索**：标签化 + 关键词 + 学员匹配
- **让经验可复用**：讲义、题库、讲评话术、错因案例
- **让教学资产持续增值**：每次教学都是资源的更新机会

## 四、资源五大类

### 4.1 讲义库

```text
■ 用途
  · 课前发给学员的预习材料
  · 课中使用的讲义
  · 课后复习材料

■ 入库要求
  · 学科 + 章节
  · 难度（基础/中等/提升/挑战）
  · copyrightStatus
  · 适用学员类型
  · 关联学员（化名）

■ 示例
  讲义 ID：H001
  学科：数学 · 一次函数
  难度：中等
  copyrightStatus：自有
  适用学员：初一/初二基础-中等
  关联学员：化名 A、化名 C
```

### 4.2 题库

```text
■ 用途
  · 课堂练习题
  · 课后作业题
  · 测评题目

■ 入库要求
  · 学科 + 知识点
  · 题型（选择/填空/解答/应用）
  · 难度 + 认知层次
  · 预估 P（难度系数）
  · copyrightStatus
  · 关联知识点

■ 示例
  题目 ID：Q001
  学科：数学
  知识点：一次函数图象
  题型：解答题
  难度：中等
  认知层次：应用
  预估 P：0.65
  copyrightStatus：自有
```

### 4.3 讲评话术库

```text
■ 用途
  · 错题讲评话术
  · 行为反馈话术
  · 课堂过渡话术
  · 家长沟通话术（脱敏）

■ 入库要求
  · 场景（错题/行为/课堂/家长）
  · 适用学员类型
  · 效果记录（是否有效）
  · 关联错因/知识点

■ 示例
  话术 ID：P001
  场景：错题讲评
  适用：基础学员、概念错
  话术："这道题的关键是 [X 概念]，你能用一句话给同桌解释这个概念吗？"
  效果：触发主动讲解 80% 学员能复述
  关联错因：概念模糊
```

### 4.4 错因案例库

```text
■ 用途
  · 学员常犯的典型错误
  · 应对策略
  · 教学反思

■ 入库要求
  · 错因类型（七类之一）
  · 学员背景（化名 + 模糊年级）
  · 应对动作
  · 效果记录
  · 完全脱敏

■ 示例
  案例 ID：E001
  错因：策略错误
  学员背景：初二学生、化名
  错误表现：用算术方法解方程问题，绕了 3 步
  应对动作：改用线段图 + 等量关系
  效果：3 次同类题正确率从 30% 提到 75%
  脱敏检查：✅ 无真实姓名/学校/家长信息
```

### 4.5 教案库

```text
■ 用途
  · 已上过的课程教案
  · 可复用的教学设计
  · 学科专项教案

■ 入库要求
  · 学科 + 章节
  · 教学目标
  · 教学过程（6 步环节时间矩阵）
  · 学员分层方案
  · 效果记录
  · 关联讲义/题库/案例

■ 示例
  教案 ID：L001
  学科：数学 · 一次函数（复习课）
  教学目标：3 维目标
  教学过程：6 步环节时间矩阵
  学员分层：A/B/C 三层
  关联资源：讲义 H001、题库 Q001-Q005、案例 E001
  效果：班级得分率 78%，分层目标全部达成
```

## 五、标签化体系

### 5.1 多维度标签

```text
┌──────────┬──────────────┬──────────────┬──────────────┐
│ 维度      │ 标签值示例                    │ 用途          │
├──────────┼──────────────────────────────┼──────────────┤
│ 学科      │ 数学/语文/英语/物理/...       │ 主分类        │
│ 章节      │ 函数/几何/语法/...            │ 精确检索      │
│ 知识点    │ 一次函数图象/定语从句/...      │ 精确定位      │
│ 难度      │ 基础/中等/提升/挑战          │ 学员匹配      │
│ 学员类型  │ 基础学员/拔尖学员/...        │ 复用建议      │
│ 学段      │ 小学/初中/高中/...           │ 适用筛选      │
│ 标签      │ 优秀/经典/必看/...            │ 主观标记      │
└──────────┴──────────────────────────────┴──────────────┘
```

### 5.2 标签使用规则

```text
■ 主标签（必填）
  · 学科 + 章节 + 知识点
  · 难度

■ 辅标签（推荐）
  · 学员类型
  · 标签（优秀/经典等）

■ 自定义标签（可选）
  · 老师个人标签
  · 跨学科联结
```

## 六、检索与匹配

### 6.1 关键词检索

```text
输入：关键词（如"一次函数复习"）
匹配：所有包含"一次函数"+"复习"的资源
排序：按相关度 / 使用频率 / 学员匹配度
```

### 6.2 学员画像匹配

```text
输入：学员画像（化名 + 学情摘要 + 需求）
匹配：基于画像推荐最相关的资源
输出：
  · 讲义（适合 X 学段 X 难度）
  · 题库（针对 X 知识点）
  · 讲评话术（针对 X 错因）
  · 错因案例（参考类似学员）
```

### 6.3 错因案例检索

```text
输入：错因（如"概念模糊"）
匹配：所有相关错因案例
输出：
  · 案例描述
  · 应对动作
  · 效果记录
```

## 七、复用与改编

### 7.1 复用流程

```text
第 1 步：检索资源
  · 基于当前需求（学员画像/学情）检索
  · 找到 1-3 个候选资源

第 2 步：评估匹配度
  · 难度匹配？
  · 知识点匹配？
  · 学员类型匹配？

第 3 步：调整适配
  · 修改参数（数字/情境）
  · 调整难度
  · 添加学员特定元素

第 4 步：使用并记录
  · 标记"复用自 [资源 ID]"
  · 记录使用效果
  · 反馈到资源库
```

### 7.2 改编原则

```text
■ 改数（参数）
  原题 y=2x+1 → 改编 y=3x-2
  适合：同一知识点不同参数

■ 改问（设问角度）
  原题"求 X" → 改编"判断 X 是否正确"
  适合：换角度测同一概念

■ 改情境（背景）
  原题"出租车" → 改编"网约车"
  适合：让题目更贴近学生生活

■ 改综合度
  单知识点 → 多知识点综合
  适合：测评综合应用能力
```

### 7.3 复用记录

```text
┌──────────┬────────────┬────────────┬────────────┐
│ 资源 ID  │ 使用次数   │ 平均效果   │ 最近使用    │
├──────────┼────────────┼────────────┼────────────┤
│ H001     │ 5 次       │ 班级 78%   │ [日期]     │
│ Q001     │ 12 次      │ 正确率 72% │ [日期]     │
│ P001     │ 8 次       │ 有效 7/8   │ [日期]     │
└──────────┴────────────┴────────────┴────────────┘
```

## 八、版权管理

### 8.1 资源版权分类

```text
✅ 自有（copyrightStatus: own）
  老师原创或组内原创
  完整存储

✅ 改编（copyrightStatus: adapted）
  基于某资源改编
  必须标注"原出处 + 改编点"

✅ 公开可引用（copyrightStatus: public）
  教材例题、CC 协议资源
  完整存储，标注来源

⚠️ 仅存索引（copyrightStatus: index-only）
  教辅原题只记题号和出处
  不复制题干

❌ 禁止
  未授权复制教辅原题
  盗版扫描件
  未授权转载的网络资源
```

### 8.2 版权标注模板

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📄 资源版权信息
━━━━━━━━━━━━━━━━━━━━━━━━━━━
资源 ID：[   ]
copyrightStatus：[own/adapted/public/index-only]

【own】
  作者：[老师/组内]
  创建时间：[日期]

【adapted】
  原出处：[X 教辅/网站/资料]
  改编点：
    · [X 处]
    · [X 处]
  改编时间：[日期]

【public】
  来源：[X 教材/CC 协议资源]
  引用许可：[CC BY 4.0 等]

【index-only】
  题号：[X 教辅 第 X 页 题 X]
  出处：[X 教辅 出版社/作者]
  不复制题干
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 九、案例脱敏规则

### 9.1 必脱敏字段

```text
■ 必脱敏
  · 真实姓名 → 化名
  · 学校名称 → "某公立/民办学校"
  · 年级 → 模糊（"初中学员"）
  · 家长身份 → 移除
  · 家庭住址 → 移除
  · 联系方式 → 移除

■ 可保留（用于案例分析）
  · 学科
  · 知识点
  · 错因
  · 应对动作
  · 效果数据
```

### 9.2 脱敏检查清单

```text
□ 学员姓名已替换为化名
□ 学校名称已模糊化
□ 年级已模糊化（"初中学员"等）
□ 家长身份信息已移除
□ 家庭住址已移除
□ 联系方式已移除
□ 案例描述不含可识别细节
□ 案例分析基于错因/方法/效果而非具体人
```

## 十、持续更新机制

### 10.1 入库时机

```text
■ 必入库时机
  · 学员试讲课讲义通过后
  · 优秀作业讲评案例
  · 重要错因突破案例
  · 阶段性测评优秀试卷
  · 复习规划方案

■ 选入库时机
  · 课堂过渡话术
  · 家长沟通案例
  · 学员进步转折点
```

### 10.2 资源使用反馈

```text
每次使用资源后：
  · 标记使用次数
  · 记录使用效果
  · 标记"使用频率高"或"待改进"

→ 资源库排序基于实际使用效果
→ 高频高质量资源进入"我的最爱"
→ 低频低质资源进入"待优化"或"归档"
```

### 10.3 资源生命周期

```text
新入库 → 试用 → 优化 → 成熟 → 经典
  ↓
  归档（如过时/淘汰）
```

## 十三、行为准则

| ✅ 应该做 | ❌ 不能做 |
|---------|---------|
| 资源入库即打标签 | 资源存了不分类 |
| 案例入库必脱敏 | 真实学员案例入库 |
| 版权标注清晰 | 复制未授权教辅原题 |
| 复用时调整适配 | 直接照搬不做修改 |
| 使用后记录效果 | 用了不反馈 |
| 高频资源标"经典" | 所有资源平等对待 |
| 资源检索基于画像 | 学员匹配用固定模板 |
