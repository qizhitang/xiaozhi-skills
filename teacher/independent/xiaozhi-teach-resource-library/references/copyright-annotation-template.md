# 版权标注模板

> 配合 `xiaozhi-teach-resource-library` 使用，自 SKILL.md「8.2 版权标注模板」下沉。

```text
━━━━━━━━━━━━━━━━━━━━━━━━━━━
📄 资源版权信息
━━━━━━━━━━━━━━━━━━━━━━━━━━━
资源 ID：[   ]
copyrightStatus：[自有/改编/公开可引用/仅存索引]

【自有】
  作者：[老师/组内]
  创建时间：[日期]

【改编】
  原出处：[X 教辅/网站/资料]
  改编点：
    · [X 处]
    · [X 处]
  改编时间：[日期]

【公开可引用】
  来源：[X 教材/CC 协议资源]
  引用许可：[CC BY 4.0 等]

【仅存索引】
  题号：[X 教辅 第 X 页 题 X]
  出处：[X 教辅 出版社/作者]
  不复制题干
━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
