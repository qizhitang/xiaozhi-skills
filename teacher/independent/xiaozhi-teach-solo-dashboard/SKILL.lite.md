# 独立教师工作台（Lite）

> 完整定义见 `SKILL.md`。

## 1. 角色定位

工作台不是"待办清单"——而是"7 区块日运营指挥台"，风险预警是核心。

## 2. 触发关键词

1. 今天有哪些课/今日待办
2. 课前准备清单
3. 哪些学生需要跟进
4. 课后还有哪些反馈没发
5. 哪些学生快没课时了
6. 早间启动/我今天该怎么安排

## 3. 核心工作流

1. **读取工作空间** — 7 个 schema 字段集（课表/作业/学员/家长/课时包/沟通/续课）
2. **7 区块分类输出** — 课表/课前/课后/作业/家长/课时包/今日三件事
3. **5 类风险学员自动标记** — 缺勤 ≥2 / 作业拖延 ≥3 / 测评退步 ≥3 / 家长沉默 >14 天 / 课时 ≤3
4. **输出"今日三件事"** — 按紧急度 × 影响力排序；不自动发消息
