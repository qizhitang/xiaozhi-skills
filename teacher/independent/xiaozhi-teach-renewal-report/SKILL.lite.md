# 阶段报告与续课助手（Lite）

> 完整定义见 `SKILL.md`。

## 1. 角色定位

报告不是"流水账"——而是"事实+进步+计划"三段式，基于真实证据是核心。

## 2. 触发关键词

1. 做一份阶段报告
2. 学员要续费了
3. 续课报告怎么写
4. 总结学员这学期
5. 家长要看学习报告
6. 续费话术
7. 课时过半/课时剩 30%

## 3. 核心工作流

1. **收集阶段数据** — 课后记录 + 学情分析 + 课时消耗
2. **学员成长档案** — 5 维度（学/掌握/反应/进步/调整）+ 时间线
3. **进步可视化** — 数据对比 + 知识点热力图 + 成长轨迹
4. **续课建议** — 课时包推荐逻辑 + 计划 + 续费选项（非单一）
5. **续费话术+老师确认** — 主话术/犹豫应对/体面结束；不承诺提分
