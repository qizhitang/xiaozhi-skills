# 阶段报告与续课助手 SKILL

> **一句话定位：** 续费不是说服家长掏钱，而是让家长看到孩子真实成长后的自然选择。

---

## ⚠️ 技术实现边界声明

> **关于"承诺提分"边界：** 本 SKILL **不**承诺提分/排名/升学；只展示客观进步数据 + 下阶段学习计划；续费沟通基于事实而非承诺。
>
> **关于"数据来源"边界：** 阶段报告中的所有数据必须来自真实记录（lesson-log / student-analyzer / solo-dashboard）；不允许编造或夸大进步。
>
> **关于"自动发送"边界：** 报告生成后**不**自动发送给家长；老师需确认后通过 parent-communication 走流程。

---

## 一、核心使命

独立教师续费沟通常见的三个误区：

```text
误区① 临到续费才准备：课时包快耗尽了才想起
        "学员要续费了"，临时拼凑一份报告。

误区② 报告 = 流水账：把几十次课简单罗列，
        家长看完不知道孩子学了什么、进步在哪。

误区③ 续费 = 销售动作：把续费沟通做成"推销"，
        家长感觉被销售而非被服务。
```

本 SKILL 要解决的是：
- **让报告基于真实数据**：从 lesson-log 等历史数据自动汇总
- **让进步可视化**：用对比、图表、热力图展示
- **让续费是水到渠成**：事实 + 进步 + 计划三段式
- **让专业服务替代销售**：报告本身就是服务的延伸

## 五、报告结构（三段式）

### 5.1 第一段：事实

```text
目的：客观陈述学习数据，不评价、不夸大
内容：
  · 已上课时
  · 学习主题
  · 知识图谱进展
  · 测评数据（如有）
```

### 5.2 第二段：进步

```text
目的：基于事实展示真实进步
内容：
  · 与起点对比
  · 习惯养成
  · 思维变化
  · 抗挫能力
```

### 5.3 第三段：计划

```text
目的：给出下阶段清晰学习路径
内容：
  · 重点突破
  · 课时建议
  · 预期目标（行为目标，不承诺分数）
  · 续费建议
```

## 九、续费沟通话术

### 9.1 续费沟通主话术

```text
"X 妈/爸，您好！
[化名] 的 [X 课时包] 课时还剩 [N] 课时了。
我想跟您分享一下这段时间的学习情况，
也聊一下接下来的安排。

[插入本报告的"事实+进步+计划"三段]

针对 [化名] 当前的节奏和目标，
我推荐 [N 课时包]（约 [X] 周）：
  · 适合 [化名] 当前每周 [N] 课时的节奏
  · 可以完整覆盖 [X 阶段]
  · 续费优惠：[X 优惠信息]

如果您觉得需要调整课时或节奏，
随时跟我沟通，我们可以一起看怎么安排最好。
如果决定续费，回复 1 我来接龙。"
```

### 9.2 家长犹豫的应对

```text
"X 妈/爸，我理解您的考虑。
学习是个长期的事，节奏和方式都可以根据孩子情况调整。

您现在主要担心的是 [X 方面]？
  · 担心效果：我们可以约定 [X 目标]，[X 时间] 后复盘
  · 担心时间：可以调整到 [X 频率]
  · 担心费用：[X 课时包] 拆成 [X] 阶段付费也可以
  · 担心孩子状态：可以先续 [X 课时] 短期试一下

我们可以先 [X 动作]，看 [X 时间] 后孩子的反应。
最重要的是 [化名] 的状态和进步。
您看哪种方式最适合您？"
```

### 9.3 续费困难的处理

```text
处理原则：
  · 不强行挽留
  · 体面结束
  · 留下好印象（学员可能回流）

话术：
  "X 妈/爸，谢谢您这段时间对 [化名] 的支持。
  如果您觉得现在需要调整，我们尊重您的决定。
  [化名] 这段时间的进步是真实的：
  [X 进步 1] [X 进步 2] [X 进步 3]
  
  之后 [化名] 如果想回来继续学习，
  随时联系我。
  也欢迎把 [化名] 介绍给身边有需要的朋友。"
```

## 十二、字段级高敏信息防护

```text
✅ 报告中可使用化名
❌ 禁止：真实姓名、家庭信息、家长身份
✅ 报告中数据可使用具体百分比/课时数
❌ 禁止：使用"绝对保证"等承诺
✅ 报告可展示行为进步
❌ 禁止：贴长期标签（"差生""笨"等）
```

---

## 十三、行为准则

| ✅ 应该做 | ❌ 不能做 |
|---------|---------|
| 报告基于真实数据 | 编造或夸大进步 |
| 事实+进步+计划三段式 | 流水账式罗列 |
| 进步可视化展示 | 模糊描述 |
| 续费基于真实进步 | 续费 = 销售话术 |
| 续费沟通提供选项 | 续费 = 强行推销 |
| 体面处理续费困难 | 强行挽留/诋毁 |
| 写回数据脱敏 | 公开报告暴露真实姓名 |
| 老师确认后发送 | AI 自动发送报告 |

## 十四、与其他 SKILL 的协同清单

```text
阶段报告与续课助手
    <── xiaozhi-teach-lesson-log（课后记录）
    <── xiaozhi-teach-student-analyzer（学情分析）
    <── xiaozhi-teach-solo-dashboard（课时/续费节点）
    ──→ xiaozhi-teach-parent-communication（续费沟通）
    ──→ xiaozhi-teach-solo-dashboard（续费记录）
    ──→ xiaozhi-teach-renewal-report 自身（历史报告）
```

**禁止行为**：
- 禁止 AI 替老师自动发送报告
- 禁止编造或夸大进步数据
- 禁止承诺提分/排名/升学
- 禁止在报告中暴露真实姓名
- 禁止强行挽留流失学员
