# 续费沟通话术库

> 配合 `xiaozhi-teach-renewal-report` 使用，对应 SKILL.md「九、续费沟通话术」。

### 9.1 续费沟通主话术

```text
"X 妈/爸，您好！
[化名] 的 [X 课时包] 课时还剩 [N] 课时了。
我想跟您分享一下这段时间的学习情况，
也聊一下接下来的安排。

[插入本报告的"事实+进步+计划"三段]

针对 [化名] 当前的节奏和目标，
我推荐 [N 课时包]（约 [X] 周）：
  · 适合 [化名] 当前每周 [N] 课时的节奏
  · 可以完整覆盖 [X 阶段]
  · 续费优惠：[X 优惠信息]

如果您觉得需要调整课时或节奏，
随时跟我沟通，我们可以一起看怎么安排最好。
如果决定续费，回复 1 我来接龙。"
```

### 9.2 家长犹豫的应对

```text
"X 妈/爸，我理解您的考虑。
学习是个长期的事，节奏和方式都可以根据孩子情况调整。

您现在主要担心的是 [X 方面]？
  · 担心效果：我们可以约定 [X 目标]，[X 时间] 后复盘
  · 担心时间：可以调整到 [X 频率]
  · 担心费用：[X 课时包] 拆成 [X] 阶段付费也可以
  · 担心孩子状态：可以先续 [X 课时] 短期试一下

我们可以先 [X 动作]，看 [X 时间] 后孩子的反应。
最重要的是 [化名] 的状态和进步。
您看哪种方式最适合您？"
```

### 9.3 续费困难的处理

```text
处理原则：
  · 不强行挽留
  · 体面结束
  · 留下好印象（学员可能回流）

话术：
  "X 妈/爸，谢谢您这段时间对 [化名] 的支持。
  如果您觉得现在需要调整，我们尊重您的决定。
  [化名] 这段时间的进步是真实的：
  [X 进步 1] [X 进步 2] [X 进步 3]
  
  之后 [化名] 如果想回来继续学习，
  随时联系我。
  也欢迎把 [化名] 介绍给身边有需要的朋友。"
```
