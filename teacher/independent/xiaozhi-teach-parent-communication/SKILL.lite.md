# xiaozhi-teach-parent-communication (Lite)

> 精简版 SKILL，完整定义与示例请参阅 `SKILL.md`。

## 触发条件

| 触发词 | 动作 |
|---|---|
| 给家长发消息 / 帮我想个消息 | 触发场景化话术生成 |
| 怎么跟家长沟通 | 触发沟通三原则（具体+低焦虑+可操作） |
| 家长问成绩怎么回 | 触发具体化话术 + 1 对 1 私聊 |
| 家长焦虑怎么办 | 触发低焦虑话术 + 共同应对 |
| 续费怎么跟家长说 | 触发续费沟通模板 |
| 家长群里发什么 | 触发群公告模板（无个体信息） |
| 学员退步了怎么告诉家长 | 触发退步沟通模板 |
| 学员进步了怎么告诉家长 | 触发进步沟通模板 |
| 家长不配合 | 触发内部反思 + 温和沟通 |

## 核心流程骨架

```text
识别沟通场景（日常/节点/应急/续费）
→ 评估沟通目的（报喜/报忧/共商/续费）
→ 应用三原则生成话术
→ 沟通频率检查
→ 写回 solo-dashboard
→ 老师确认后发送
```

## 输出要点

- 4 类沟通场景（日常简报/节点/应急/续费）
- 三原则（具体化/低焦虑/可操作）
- 6 类典型场景话术（进步/退步/情绪/续费/焦虑/不配合）
- 家长群公告模板（无个体信息）
- 沟通频率管理（标准频率表）
- 沟通记录（写回 solo-dashboard）

## 禁止行为

| 禁止项 | 说明 |
|---|---|
| AI 替老师自动发送消息 | 所有话术需老师确认 |
| 群发具体学员表现 | 必须 1 对 1 私聊 |
| 暴露其他学员的分数/排名 | 群内禁止对比 |
| 使用焦虑话术（"再这样下去"） | 三原则：具体+低焦虑+可操作 |
| 老师当心理咨询师 | 情绪问题转介专业 |
| 暴露真实姓名/家庭信息 | 一律使用化名 |
| 续费时夸大效果 | 基于真实进步沟通 |
