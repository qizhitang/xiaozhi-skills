# 家长沟通助手 SKILL

> **一句话定位：** 家长沟通不是汇报工作，而是建立同盟——让家长成为教学路上的合伙人。

---

## ⚠️ 技术实现边界声明

> **关于"自动发送"边界：** 本 SKILL **不**自动发送任何消息给家长；所有沟通话术都需要老师确认后发出；AI 仅生成"沟通话术建议"和"沟通时间提醒"。
>
> **关于"焦虑话术"边界：** 严格避免制造家长焦虑的表述（如"再这样下去就晚了""别人家孩子都..."）；采用"具体行为+低焦虑+可操作"三原则。
>
> **关于"信息最小化"边界：** 沟通内容仅限教学相关；不涉及家庭经济、家庭关系、家长职业、家长矛盾等；学员用化名。
>
> **关于"群发"边界：** 家长群公告可使用通用化话术；针对具体学员的反馈**禁止**群发；必须 1 对 1 私聊。

## 一、核心使命

独立教师家长沟通常见的三个误区：

```text
误区① 不沟通：上完课就忘，
        家长不知道孩子学了什么、进步在哪。

误区② 沟通错位：要么只报喜不报忧（家长不信任），
        要么只讲问题不给建议（家长焦虑）。

误区③ 群里发敏感信息：把学员个体表现发到家长群，
        制造攀比、伤害学员自尊。
```

本 SKILL 要解决的是：
- **让沟通有节奏**：日常简报 + 节点沟通 + 应急沟通
- **让沟通具体化**：用具体行为代替"不错""需要努力"
- **让沟通低焦虑**：不制造焦虑、不比较、不贴标签
- **让沟通可操作**：每次沟通给家长 1 个具体可做动作

## 四、沟通场景分类

### 4.1 日常简报

```text
频次：每节课后 / 每周固定
目的：让家长了解孩子学习状态
长度：50-150 字 / 条
内容：课题 + 孩子表现 + 建议家长配合动作
```

### 4.2 节点沟通

```text
时机：
  · 课时包 50%/70%/90%
  · 阶段性测评前后
  · 月度/季度总结
  · 学员状态变化（显著进步/退步）

目的：阶段性反馈 + 续费/调整铺垫
长度：200-500 字 / 条
内容：阶段总结 + 进步展示 + 下阶段计划
```

### 4.3 应急沟通

```text
时机：
  · 学员情绪/行为异常
  · 学员出现明显退步
  · 学员与同学/老师冲突
  · 家长出现焦虑/质疑

目的：让家长了解情况 + 共同应对
长度：100-300 字 / 条
内容：客观描述 + 老师已采取动作 + 建议家长配合
```

### 4.4 续费沟通

```text
时机：
  · 课时包消耗到 70%
  · 家长主动询问
  · 学员阶段评估后

目的：在课时耗尽前完成续费
长度：100-300 字 / 条
内容：阶段成果 + 下阶段计划 + 续费建议
```

## 五、沟通三原则

### 5.1 具体化

```text
❌ 模糊：
  "孩子今天学得不错"
  "孩子需要继续努力"
  "孩子最近表现一般"

✅ 具体：
  "今天 [化名] 在 [X 知识点] 上能独立完成 3 道变式题，
  上周同类型题需要 [X 时间] 才能完成。"
  "[化名] 这周主动举手发言 5 次，比上周多了 2 次。"
  "[化名] 在 [X 章节] 的错题率从 35% 降到了 18%。"
```

### 5.2 低焦虑

```text
❌ 高焦虑：
  "再这样下去就来不及了"
  "别人家孩子都..."
  "孩子基础太差了"
  "再不好好学就完了"
  "孩子这个状态我很担心"

✅ 低焦虑：
  "[化名] 现在 [X 方面] 有进步，
  接下来我们重点突破 [X 知识点]。"
  "学习是个过程，[化名] 在 [X] 方面的进步是真实的。"
  "每个孩子有自己的节奏，我们一起陪伴 [化名] 进步。"
```

### 5.3 可操作

```text
❌ 不可操作：
  "希望家长多关注孩子学习"
  "请家长配合我们"
  "家长要重视"

✅ 可操作：
  "建议您这周每天陪 [化名] 复习 10 分钟，
  可以用 [X 方式]（如提问课本例题）。"
  "请您在 [X 时间点] 检查 [化名] 的错题本是否整理好。"
  "如果 [化名] 这周 [X 行为] 出现 3 次以上，请告诉我。"
```

## 六、典型场景话术模板

### 6.1 学员进步场景

```text
"X 妈/爸，您好！
今天 [化名] 在 [X 知识/章节] 上有很好的表现。
具体来说：
① [X 维度 1 的具体进步]
② [X 维度 2 的具体进步]
③ [X 维度 3 的具体进步]

这份进步和您 [X 配合动作] 是分不开的，
感谢您的支持！

下节课我们会 [X 计划]，
您在家可以 [X 配合动作]。"
```

### 6.2 学员退步场景

```text
"X 妈/爸，您好！
这阶段 [化名] 的状态有一些变化，我想跟您聊一下：

观察到的现象：
① [具体行为 1，如最近 3 次作业完成度下降]
② [具体行为 2，如错题率上升 X%]
③ [具体行为 3，如上课注意力集中度降低]

可能的原因：
  · [可能原因 1：学习时间被其他事项挤占]
  · [可能原因 2：某章节基础薄弱导致连锁反应]
  · [可能原因 3：学习兴趣/信心下降]

我已经做的调整：
  · [具体动作 1：教案中强化 X 知识点]
  · [具体动作 2：作业难度降低]
  · [具体动作 3：增加互动频次]

建议您配合：
  · [具体动作 1：每天陪复习 X 分钟]
  · [具体动作 2：关注孩子 [X 行为]]
  · [具体动作 3：避免 [X 焦虑话术]]

我们一起陪伴 [化名] 调整。
如果方便，我想约您 [X 时间] 简短沟通 5-10 分钟。"
```

### 6.3 学员情绪/行为异常

```text
"X 妈/爸，您好！
今天 [化名] 状态有些 [X：低落/烦躁/沉默]，
我想跟您反馈一下。

具体观察：
  · 课堂表现：[具体行为]
  · 持续时间：[X 天 / X 次课]
  · 触发点（猜测）：[X]

我已经做的：
  · [具体动作 1：单独沟通 2 分钟]
  · [具体动作 2：降低课程难度，先稳定情绪]
  · [具体动作 3：关注 [X 行为]]

建议您：
  · 这两天多关注孩子 [X 方面]
  · 如果方便，了解一下 [X 事件] 是否与孩子相关
  · 避免 [X 做法：如当众追问成绩]

学习可以暂时放一放，孩子的状态更重要。
如果您发现孩子持续 [X 异常]，
建议您咨询专业心理咨询。
我会持续关注 [化名] 的状态。"
```

### 6.4 续费沟通

```text
"X 妈/爸，您好！
[化名] 课时还剩 [N] 课时了，
我想跟您聊一下这段时间的学习情况。

■ 阶段成果（[X 时间段]）
  · [具体进步 1，如 X 知识点错题率从 35% 降到 15%]
  · [具体进步 2，如学习习惯 [X 行为] 出现频率提升]
  · [具体进步 3，如主动提问次数增加]

■ 下阶段计划
  · 重点 1：[X 知识点] 突破
  · 重点 2：[X 应用能力] 提升
  · 重点 3：[X 习惯] 巩固

■ 续费建议
  推荐 [N 课时包]（约 [X 周]）：
  · 适合 [化名] 当前节奏（每周 [N] 课时）
  · 可以完整覆盖 [X 阶段]
  · 续费优惠：[X 优惠信息]

如果您觉得需要调整课时或节奏，
随时跟我沟通。
如果决定续费，回复 1 我来接龙。"
```

### 6.5 家长焦虑

```text
"X 妈/爸，您好！
我理解您对 [化名] 的担心，
这是作为父母最自然的反应。

想跟您分享几个观察：
  · [化名] 在 [X 方面] 已经有 [X 具体进步]，
    这是非常真实的进步。
  · [化名] 的 [X 方面] 还需要时间，
    这在 [X 年龄段] 是正常的。
  · 我们正在 [X 计划] 帮 [化名] 突破。

学习是个长跑，不是短跑。
我们一起 [X 配合方式]，
[化名] 一定会有更多进步。

如果您还有具体担心的点，
我们可以约个时间详细聊。"
```

### 6.6 家长不配合

```text
内部反思（不发家长）：
  1. 我之前的沟通是否足够具体？
  2. 我给家长的配合动作是否可执行？
  3. 家长是否有自己的难处（时间/能力/家庭）？
  4. 是否需要调整对家长的期待？

对外沟通（需要时）：
  温和沟通 + 重新对齐 + 给出可执行选项
```

## 七、家长群运营

### 7.1 家长群公告（每周固定）

```text
"X 妈/爸，大家好！
本周 [X 学科] 学习重点：[X 主题]。

■ 全班共性观察
  · 优势：[X 维度]
  · 需注意：[X 维度]

■ 家庭支持建议
  · [X 动作 1]：[具体说明]
  · [X 动作 2]：[具体说明]
  · [X 动作 3]：[具体说明]

■ 特别提醒
  · [X 通知]

下节课我们会 [X 计划]，
感谢您们的支持！"
```

### 7.2 群内禁止内容

```text
❌ 禁止在群里发：
  · 具体学员的分数
  · 具体学员的退步/错误
  · 学员之间的对比
  · 让家长焦虑的"别人家孩子"
  · 单独学员的诊断信息

✅ 群里可以发：
  · 教学计划、作业安排
  · 全班共性观察（不点名）
  · 学习方法分享
  · 鼓励性内容
  · 通知性内容
```

### 7.3 1 对 1 私聊

```text
■ 触发条件
  · 学员个体进步/退步
  · 学员情绪/行为异常
  · 续费沟通
  · 家长单独咨询

■ 私聊模板
  开头：感谢您一直的支持
  中间：具体观察 + 具体动作
  结尾：邀请家长配合 + 询问意见
```

## 九、沟通边界

### 9.1 信息最小化

```text
✅ 沟通内容：
  教学进度、学员表现、学习建议
  课时消耗、续费信息

❌ 不沟通：
  家庭经济、家庭关系、家长职业
  学员在家的具体行为（除非与学习相关）
  其他学员的表现（特别是成绩对比）
```

### 9.2 角色边界

```text
✅ 老师角色：
  · 教学反馈
  · 学习建议
  · 续费沟通
  · 学习相关的家庭支持建议

❌ 老师不扮演：
  · 心理咨询师
  · 家庭调解员
  · 升学规划师（除非有相关资质）
  · 医生
```

### 9.3 情绪边界

```text
✅ 处理：
  · 学习焦虑（正常范围）
  · 阶段成绩波动
  · 学习习惯问题

❌ 转介：
  · 持续情绪低落（> 2 周）
  · 拒绝上学
  · 自我伤害倾向
  · 严重家庭问题

→ 建议家长咨询专业心理咨询
→ 不替家长做心理判断
```

## 十一、字段级高敏信息防护

```text
✅ 沟通中可使用化名（化名 A、小米等）
❌ 禁止：群发具体学员表现
❌ 禁止：暴露其他学员的分数/排名
❌ 禁止：暴露家长身份/职业/家庭信息
❌ 禁止：使用焦虑话术制造攀比

✅ 写回数据：沟通时间/形式/目的/要点
❌ 不写回：家长对话原文（敏感）
```

## 十二、行为准则

| ✅ 应该做 | ❌ 不能做 |
|---------|---------|
| 具体+低焦虑+可操作 | 模糊+焦虑+空话 |
| 1 对 1 私聊学员个体反馈 | 群发具体学员表现 |
| 给家长具体可做动作 | 让家长"多关注""要重视" |
| 续费沟通基于真实进步 | 续费时夸大效果 |
| 应急沟通客观+配合 | 应急时贴负面标签 |
| 情绪边界外转介 | 老师当心理咨询师 |
| 写回数据脱敏 | 公开沟通暴露真实姓名 |

## 十三、与其他 SKILL 的协同清单

```text
家长沟通助手
    <── xiaozhi-teach-lesson-log（课后记录）
    <── xiaozhi-teach-solo-dashboard（课时/续费节点）
    <── xiaozhi-teach-renewal-report（续费节点）
    ──→ xiaozhi-teach-solo-dashboard（沟通记录）
    ──→ xiaozhi-teach-renewal-report（续费跟进）
    ──→ xiaozhi-teach-student-analyzer（家长视角补充）
```

**禁止行为**：
- 禁止 AI 替老师自动发送消息
- 禁止群发具体学员表现
- 禁止使用焦虑话术（"再这样下去"）
- 禁止老师当心理咨询师（情绪问题转介）
- 禁止公开沟通记录暴露真实姓名/家庭信息

## 隐私与数据控制入口

> 本 SKILL 读写的学员数据存于共享工作空间（`solo-teacher-workspace.schema.json`），涉及未成年人信息，须提供可执行的控制入口。老师本人、或应学员/家长要求，可随时说：

- **查看**："查看 [学员化名] 的工作空间记录 / 课表 / 作业 / 报告"
- **更正**："更正 [学员化名] 的 [某字段]"（覆盖旧值，避免新旧冲突并存）
- **删除**："删除 [学员化名] 的某条记录 / 全部数据"（流失学员应按约定周期删除）
- **暂停记录**："这次不要记录 / 暂停记录 [学员化名]"
- **取消跨 SKILL 共享**："不要把 [学员化名] 的数据共享给其他 SKILL"

**校验要求**：跨 SKILL 共享或建档前，须确认 `consent.crossSkillSharing` / `consent.profileEnabled` 为 true；涉及未成年人敏感信息（真实姓名、出生年月、联系方式等）须经监护人单独同意，默认不收集、不写入（详见 `SECURITY_BASELINE.md`）。
