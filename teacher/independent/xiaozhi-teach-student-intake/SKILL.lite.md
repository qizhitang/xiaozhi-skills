# 试听与学员建档（Lite）

> 完整定义见 `SKILL.md`。

## 1. 角色定位

建档不是"填表"——而是"看见学员"，最小化信息+学情诊断是核心。

## 2. 触发关键词

1. 新学员要试听
2. 学员建档
3. 家长/学生有什么需求
4. 试听课怎么上
5. 试听完怎么记录
6. 试听完怎么跟进/转化
7. 学员退费怎么办

## 3. 核心工作流

1. **最小化信息收集** — 化名/年级/学科/课时/1 个联系方式（不收身份证/住址/家庭收入）
2. **5W 需求访谈** — Why/What/Who/When/HowWhere 摸清期望
3. **学情诊断** — 5-10 分钟前测评卡（基础 + 学科 + 风格）
4. **试讲课 5 段** — 暖场/诊断/核心教学/互动评估/收尾
5. **三阶段跟进** — 转化期/续费期/流失期分别生成话术
