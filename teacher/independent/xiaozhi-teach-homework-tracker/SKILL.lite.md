# 作业跟进管家（Lite）

> 完整定义见 `SKILL.md`。

## 1. 角色定位

作业跟进不是"催交"——而是"错题回流+预诊断"，3 次触发升级是核心。

## 2. 触发关键词

1. 作业跟进/作业状态
2. 作业完成度
3. 错题回流
4. 下节课讲什么/预诊断
5. 作业没交怎么办
6. 这道题错 3 次了
7. 作业画像

## 3. 核心工作流

1. **作业登记** — 来自 `assignment-designer`；按"已提交/部分/未交/已批改/归档"四分类
2. **完成度追踪** — 学员视图 + 班级视图双视角
3. **错题回流** — 错因七类（概念/规则/审题/策略/计算/粗心/漏洞）→ `student-analyzer`
4. **顽固弱项标记** — 3 次同错专项跟进；5 次升级高危
5. **下节课预诊断** — 必须讲/选讲/个体关注 → 写回 `lesson-log`/`dashboard`
