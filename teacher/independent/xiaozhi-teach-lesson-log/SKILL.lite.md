# xiaozhi-teach-lesson-log (Lite)

> 精简版 SKILL，完整定义与示例请参阅 `SKILL.md`。

## 触发条件

| 触发词 | 动作 |
|---|---|
| 课后总结 / 课后记录 | 触发 5 维度结构化记录 |
| 今天这节课记录 | 触发课后记录 + 课时预扣 |
| 学员今天学得怎么样 | 触发 5 维度记录 + 学员视角总结 |
| 复盘 / 这节课上得怎么样 | 触发有效性评估 + 调整方向 |
| 学习轨迹 | 触发累积轨迹视图 |
| 课时扣减 / 扣一课时 | 触发课时预扣（24h 可调） |
| 下节课应该讲什么 | 触发衔接建议生成 |
| 学员进步 | 触发进步对比 + 轨迹点 |

## 核心流程骨架

```text
课后 5 分钟内触发 → 5 维度结构化记录
→ 学习轨迹更新 → 课时消耗登记（预扣 24h）
→ 学情更新 → 下节课衔接建议
→ 课后家长简报（可选）
```

## 输出要点

- 5 维度课后记录（学/掌握/反应/进步/调整）
- 5 分钟即时记录原则（窗口 1/2/3/4）
- 学习轨迹视图（知识图谱/习惯/抗挫/转折点）
- 课时预扣机制（24h 老师可调）
- 衔接建议（必做/选做/节奏/素材）
- 家长简报（进步/退步/情绪/节点）

## 禁止行为

| 禁止项 | 说明 |
|---|---|
| 拖过 24 小时才记录 | 5 分钟内补录是核心原则 |
| 写"今天讲了 X 章节"流水账 | 必须 5 维度结构化 |
| AI 替老师自动发家长简报 | 简报需老师确认后发送 |
| 在记录中暴露真实姓名 | 一律使用化名 |
| 简报贴负面标签 | 用具体行为描述代替评价 |
| 课时消耗不透明 | 预扣 24h 老师可调 |
