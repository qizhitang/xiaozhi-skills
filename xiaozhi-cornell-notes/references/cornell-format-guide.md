# 康奈尔笔记格式详解与各科模板

---

## 一、康奈尔笔记标准格式说明

### 三栏结构尺寸参考

```
┌──────────────────────────────────────────────────────┐
│  课题：____________  日期：________  学科：________  │
├────────────┬─────────────────────────────────────────┤
│            │                                         │
│  左栏      │  右栏（笔记区）                          │
│  线索问题区 │  ← 这里是课堂上实际记录的内容            │
│            │                                         │
│  约占页面  │  约占页面                               │
│  宽度30%   │  宽度70%                                │
│            │                                         │
│  课后填写  │  课上快速记录                            │
│            │                                         │
├────────────┴─────────────────────────────────────────┤
│  底部总结区（约占页面高度的20%）                       │
│  一句话核心：                                         │
└──────────────────────────────────────────────────────┘
```

### 三栏填写时机

| 栏位 | 填写时机 | 填写者 | 内容标准 |
|-----|---------|-------|---------|
| 右栏（笔记区） | 课堂实时记录 | 学生 | 快速、不求完美，关键信息不遗漏 |
| 左栏（线索区） | 课后24小时内 | AI提炼+学生确认 | 每个知识点转化为一个自测问题 |
| 底部总结 | 课后24小时内 | AI提炼+学生修改 | 一句话≤30字，覆盖本页核心 |

## 二、各科笔记记录要点

### 数学课笔记

**右栏记录重点：**
```
· 定义/定理的文字表述（不只是公式符号）
· 推导过程的关键步骤（用→标注逻辑流）
· 例题的完整解题过程
· 老师强调的"注意点"或"易错点"
· 适用条件（什么情况下用这个方法）
```

**左栏线索问题示例（数学）：**
```
Q: 一次函数和正比例函数的区别？
Q: y=kx+b中，k和b各影响图像的什么？
Q: 什么情况下函数过第一、三象限？
Q: 这道例题如果条件变成…，方法还适用吗？
```

**底部总结示例（数学）：**
```
"一次函数y=kx+b，k决定倾斜方向和速度，b决定起点高度（y轴截距）"
```

---

### 语文课笔记

**右栏记录重点：**
```
· 课文核心段落的赏析角度
· 修辞手法 + 效果 + 情感（三件套）
· 文章主旨的关键语句
· 写作技法要点（开头/结尾/过渡方式）
· 古诗词的背景和情感脉络
```

**左栏线索问题示例（语文）：**
```
Q: 这篇文章的主旨是什么？能用一句话概括吗？
Q: 第X段用了什么修辞手法，有什么表达效果？
Q: 作者为什么在这里用这个词而不是另一个词？
Q: 这种写法的固定答题格式是什么？
```

**底部总结示例（语文）：**
```
"《背影》以'背影'为线索，用细节描写表达父子间含蓄深沉的爱"
```

---

### 英语课笔记

**右栏记录重点：**
```
· 语法规则的核心条件（如：第三人称单数+s/es的规则）
· 词汇的例句（不只是中文意思）
· 句式结构的框架（用_填空方式记录）
· 阅读文章的段落大意
· 老师造的示范句
```

**左栏线索问题示例（英语）：**
```
Q: 现在完成时和一般过去时有什么区别？各用什么情境？
Q: "interest" 作形容词是 interested 还是 interesting？怎么区别？
Q: 这个词的近义词是什么？它们之间有什么细微差别？
```

**底部总结示例（英语）：**
```
"现在完成时 have/has+过去分词，强调过去动作对现在的影响，常与already/yet/ever连用"
```

---

### 物理/化学笔记

**右栏记录重点：**
```
· 物理量的符号和单位（成对记录）
· 公式+适用条件（缺一不可）
· 实验步骤和注意事项
· 图示或电路图（尽量画出来）
```

**左栏线索问题示例（物理）：**
```
Q: 欧姆定律的公式是什么？各字母代表什么？
Q: 电阻和温度的关系是什么？举一个生活例子？
Q: 这个实验的误差主要来自哪里？
```

## 三、AI提炼线索问题的质量标准

### 好的线索问题（✅）

```
特征：
· 可以从右栏笔记中找到答案
· 引导思考，不是简单的是/否问题
· 覆盖了该知识点的核心而不是细枝末节
· 有助于检验是否真正理解

示例：
✅ "一次函数图像向上倾斜还是向下，由哪个参数决定？"
✅ "这道题的解题思路，适用于哪一类题型？"
✅ "这个定理成立的前提条件是什么？"
```

### 差的线索问题（❌）

```
特征：
· 答案不在笔记里（要额外查资料）
· 过于简单，不需要思考
· 过于开放，无法从笔记找到明确答案

示例：
❌ "一次函数有什么用？"（太宽泛）
❌ "k的值是多少？"（要看具体题目，不是规律性问题）
❌ "你觉得这个知识点难吗？"（主观感受，不是知识检验）
```

## 五、电子笔记 vs 手写笔记的使用建议

| 场景 | 推荐格式 | 原因 |
|-----|---------|-----|
| 数学公式推导 | 手写拍照 | 符号和图示更自然 |
| 语文文本分析 | 手写拍照 | 批注和标记更灵活 |
| 英语单词和例句 | 电子文档 | 方便搜索和复制 |
| 物理实验记录 | 手写拍照 | 图示和数据表格 |
| 复习整理 | 电子文档 | 方便修改和关键词搜索 |

> 无论哪种格式，拍照上传给小龙虾的操作方式相同——
> AI都能识别内容并完成三栏提炼。
