# 听力话题模板与材料生成指南

> 本文档为 `xiaozhi-english-listening-trainer` 的配套参考资源，提供分年级分兴趣话题的听力材料生成模板、调节维度参数与提示词框架。

---

## 一、话题分类体系

### 三大场景分类

```
场景①：校园场景（最熟悉、最常见）
  → 学生每天接触，词汇可及度高
  → 适合入门级和日常练习

场景②：生活场景（较熟悉、需一定策略表达）
  → 购物/家庭/出行等日常活动
  → 适合进阶级练习

场景③：信息获取场景（有认知挑战）
  → 天气预报/学校通知/旅行公告
  → 适合训练"从听中提取关键信息"的能力
```

## 二、校园场景话题模板

### 话题①：课堂讨论（Classroom Discussion）

**适用学段：** 初中及以上

**材料生成参数：**

```
话题关键词：teacher, student, question, discuss, opinion, explain
语速：初一120词/分 | 初二140词/分 | 初三160词/分
句长：初一8-12词/句 | 初二10-15词/句 | 初三12-18词/句
生词密度：初一≤5% | 初二≤8% | 初三≤10%
```

**材料结构模板：**

```
开头（30秒）：引入话题
  "Today in class, we're going to talk about [topic].
   Let me first ask—what do you already know about this?"

主体（2分钟）：讨论展开
  - 老师提出问题
  - 学生A回答（简单观点）
  - 老师追问或补充
  - 学生B提出不同意见
  - 讨论逐步深入

结尾（30秒）：总结或思考题
  "So, what we discussed today is...
   Think about this question for next time: [问题]"
```

**示例片段（初二难度）：**

```
Teacher: "Today we're talking about technology in education.
          Do you think computers help students learn better?"

Student A: "Yes, I think so. We can find information quickly
           and watch educational videos online."

Teacher: "That's a good point. But Student B, you seem
          to disagree. What's your opinion?"

Student B: "I think computers can be distracting.
           Sometimes students play games instead of
           doing their homework."

Teacher: "Both sides make sense. The key question is:
          How can we use technology wisely without
          letting it distract us? Think about that
          for our next class."
```

### 话题②：学校社团（School Club）

**适用学段：** 初中

**材料生成参数：**

```
话题关键词：club, join, activity, practice, perform, organize, member
语速：初一120词/分 | 初二140词/分
句长：初一8-10词/句 | 初二10-14词/句
```

**材料结构模板：**

```
开头：社团招新场景
  "Welcome to the school club fair! We have many clubs
   you can join this semester."

主体：介绍3个社团 + 学生提问
  - 社团A介绍（音乐社）
  - 社团B介绍（科技社）
  - 社团C介绍（体育社）
  - 学生提问："When do they meet? / Can I join two clubs?"

结尾：决定和期待
  "I think I'll join the [X] club. I'm excited to start!"
```

### 话题③：课后计划（After-school Plan）

**适用学段：** 小学高年级+初中

**材料生成参数：**

```
话题关键词：after school, homework, library, practice, meet, finish, plan
语速：小学100词/分 | 初一120词/分
句长：小学6-8词/句 | 初一8-12词/句
```

**材料结构模板：**

```
开头：放学后的对话
  "What are you doing after school today?"

主体：讨论各自计划
  - 学生A：先去图书馆做作业

  - 两人讨论要不要一起学习
  - 确定时间地点

结尾：约定
  "Let's meet at the library at 4:30.
   See you there!"
```

## 三、生活场景话题模板

### 话题④：购物（Shopping）

**适用学段：** 初中及以上

**材料生成参数：**

```
话题关键词：buy, price, discount, size, color, quality, compare, recommend
语速：初二140词/分 | 初三160词/分
句长：初二10-14词/句 | 初三12-16词/句
```

**材料结构模板：**

```
开头：进店场景
  "Hi, I'm looking for a gift for my friend's birthday.
   Can you help me find something nice?"

主体：比较产品 + 决定购买
  - 店员介绍3个选项
  - 学生询问价格、质量、颜色
  - 比较优劣
  - 做出决定

结尾：购买完成
  "I'll take this one. It's perfect!
   Thank you for your help."
```

## 四、信息获取场景话题模板

### 话题⑦：天气预报（Weather Report）

**适用学段：** 初中及以上

**材料生成参数：**

```
话题关键词：temperature, sunny, cloudy, rain, forecast, degree, weekend, condition
语速：初二130词/分 | 初三150词/分
句长：初二8-12词/句 | 初三10-15词/句
特点：信息密集，需要快速提取关键数据
```

**材料结构模板：**

```
开头：播报格式
  "Good morning. Here's your weather forecast for today
   and the coming weekend."

主体：逐天播报
  - Today: [温度]°C, [天气状况], [特殊提醒]
  - Tomorrow: [温度]°C, [天气状况]
  - Weekend: [温度]°C, [天气状况], [出行建议]

结尾：总结提醒
  "Remember to bring an umbrella on Thursday.
   Have a great day!"
```

**训练重点：** 数字听辨（温度、时间）+ 天气词汇

### 话题⑧：学校通知（School Notice）

**适用学段：** 初中

**材料生成参数：**

```
话题关键词：announce, schedule, change, important, remind, exam, event, attention
语速：初二130词/分
句长：10-14词/句
特点：有多个关键信息点，需要逐条记录
```

**材料结构模板：**

```
开头：通知开头
  "Attention, students. Here are some important announcements

主体：多条通知
  ① 考试时间变更通知
  ② 社团活动安排
  ③ 图书馆开放时间调整
  ④ 校运会日期确认

结尾：提醒
  "Please check the school website for more details.
   If you have questions, visit the school office."
```

**训练重点：** 关键信息提取（日期、时间、地点）+ 注意力分配

### 话题⑨：旅行公告（Travel Announcement）

**适用学段：** 初中高级+高中

**材料生成参数：**

```
话题关键词：departure, boarding, gate, delay, luggage, announcement, passenger, flight
语速：初三150词/分
句长：12-16词/句
特点：正式语体，信息精确，有紧急性
```

**材料结构模板：**

```
开头：机场/车站公告格式
  "Attention, passengers. This is an important announcement
   for all travelers on [航班/车次号]."

主体：公告内容
  - 航班/车次信息
  - 变更或延误说明
  - 新的出发时间/登机口
  - 乘客需要做什么

结尾：指引
  "Please proceed to Gate [X] for boarding.
   Thank you for your patience."
```

**训练重点：** 正式语体识别 + 紧急信息快速抓取

## 五、调节维度详解

### 维度①：句长调节

```
句长分级标准：
  入门级（小学/初一）：6-8词/句 → 简单句为主
    例："I like basketball. It is fun."

  中级（初二）：10-14词/句 → 含1个复合句
    例："I like basketball because it helps me stay healthy and active."

  高级（初三以上）：12-18词/句 → 含2-3个复合句
    例："Although I used to prefer watching sports on TV,
         I've recently started playing basketball every weekend,
         which has made me much more active."

调节方法：
  → 降低句长：拆分复合句为2-3个简单句
  → 提高句长：合并简单句，添加连接词（because, although, while）
```

### 维度②：生词密度调节

```
生词密度分级：
  ≤5%：几乎无新词 → 初一/入门级
    例：350词材料中仅约17个新词

  8-10%：适量挑战 → 初二/中级
    例：350词材料中约28-35个新词

  10-15%：明显挑战 → 初三+高级
    例：350词材料中约35-50个新词

调节方法：
  → 降低密度：用同义词替换新词（如 "exquisite" → "very beautiful"）
  → 提高密度：将熟悉词替换为同义新词（如 "important" → "significant"）
  → 新词必须有上下文线索（不能孤立出现）
```

### 维度③：话题熟悉度调节

```
熟悉度分级：
  高（学生日常接触的话题）：
    → 校园生活、家庭、日常活动
    → 理解难度低，适合训练流利度

  中（学生了解但不常接触的话题）：
    → 科技、环保、社会问题
    → 需要一定背景知识，适合训练推断能力

  低（学生很少接触的话题）：
    → 专业领域、抽象哲学、国际政治
    → 除非学生特别感兴趣，不建议使用

调节方法：
  → 降低难度：将话题与学生的日常生活建立连接
    例：讲"环保" → 从"你每天看到的垃圾分类"切入
  → 提高难度：引入更多专业术语和抽象概念
```

### 维度④：语速调节

```
语速分级：
  初学/初一：约100-120词/分钟
    → 短句、停顿明显、重复关键信息

  初中中等/初二：约140词/分钟
    → 正常语速、少量停顿

  初中高级/初三：约160词/分钟
    → 接近自然语速、连续句子、较少重复

  高级/高中：约180词/分钟
    → 完全自然语速、含连读和省音

调节方法：
  → 降低语速：在关键信息前加停顿、重复核心词
  → 提高语速：减少重复、增加句子间的连接
```

## 六、材料生成提示词框架

### 标准生成提示词

```
请生成一段约3分钟（350-400词）的英语听力材料。

参数：
- 目标学段：[初一/初二/初三/高中]
- 话题：[话题名称]
- 语速：[100/120/140/160/180词/分钟]
- 词汇范围：基于学生的词汇DNA，核心词≥70%来自已学词汇
- 新词数量：3-5个，必须在上下文中可推断含义
- 句长：[6-8/10-14/12-18词/句]

结构要求：
- 开头（30秒）：引入话题，使用熟悉词汇
- 主体（2分钟）：展开内容，包含3-5个新词（有上下文线索）
- 结尾（30秒）：总结或提出思考问题

输出格式：
1. 完整材料文本
2. 新词注释（3-5个新词的释义+例句）
3. 理解测验（3道简单问题）
4. 关键句标注（2-3个值得关注的句子结构）
```

### 卡壳诊断专用提示词

```
当学生反馈"没听懂"时，生成诊断版材料：

请将刚才的听力材料按以下方式拆解：
- 将材料分成4-5个片段，每个片段独立播放
- 每个片段标注难度级别（入门/中级/高级）
- 标注每个片段可能卡壳的原因：
  ① 词义不明（新词导致）
  ② 句子结构复杂（长句导致）
  ③ 语速/连读问题（节奏导致）
- 提供每个片段的慢速版本（语速降低30%）
```

## 七、话题选择决策流程

```
听力材料话题选择流程：

① 检查学习DNA → 有记录的兴趣话题？
   → YES：使用档案话题（兴趣匹配度最高）
   → NO：进入②

② 检查口语DNA → 有最近聊过的话题？
   → YES：用同话题但不同角度的听力材料（复习+延伸）
   → NO：进入③

③ 使用本话题库 → 按学生年级选择场景类别
   → 初一/小学：优先校园场景①②③
   → 初二：校园+生活场景混合
   → 初三以上：三个场景轮换，增加信息获取场景比重

④ 生成参数调整：
   → 第一次练习：降低难度（入门级参数）
   → 学生反馈"太简单"：提高一个难度级别
   → 学生反馈"太难"：降低难度+增加上下文线索
```
