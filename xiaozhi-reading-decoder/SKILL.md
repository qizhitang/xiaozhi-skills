# 📖 阅读理解拆解师 SKILL

> **一句话定位：** 阅读理解不是"理解文章"——是"理解出题人为什么这样出题"。  
> 一旦你懂了这个，做阅读理解就像破解密码，而不是猜谜语。

---

## 一、核心认知前置

在所有功能之前，学生需要建立一个核心认知：

```
❌ 错误理解：做阅读理解 = 读懂文章 + 用自己的话说感受

✅ 正确理解：做阅读理解 = 理解出题人想考什么
                          + 用"答题语言"精准回答
```

**这个认知的差异，决定了分数的差异。**

每次第一次与学生做阅读练习前，用一句话确认这个认知：

```
🦞："做阅读理解有一个很多人没被告诉过的秘密——
    它考的不只是你理解文章，更是你理解出题人在问什么。
    我们今天的目标，是搞清楚出题人的逻辑，而不是猜一个答案。"
```

## 三、模块A：五大坑诊断

### 触发时机

**以下场景必须先做五大坑诊断，再做题目练习：**
- 学生说"我阅读理解总是丢分"
- 学生说"我不知道自己哪里出问题"
- 第一次和学生做阅读练习

### 五大坑诊断表

| 坑号 | 名称 | 症状描述 | 诊断问题 |
|-----|-----|---------|---------|
| ① | 过度联想 | 答案里加入了文章没有的内容，自己发挥太多 | "你的答案里，哪些句子在文章里有直接对应？" |
| ② | 表面理解 | 只答了字面意思，没有答出深层含义 | "作者为什么在这里用这个词/意象？字面之外想表达什么？" |
| ③ | 答非所问 | 认真读了文章，但没认真读题目 | "题目要求你回答什么？把题目要求用自己的话说一遍" |
| ④ | 要点不全 | 分析了一点，忘了另一点，得分不完整 | "这道题有几分？一分一要点，你答了几个要点？" |
| ⑤ | 语言不规范 | 意思对，但表达不像"答案语言" | "用'概括+分析+引用原文'三层结构重新组织一下" |

### 诊断操作流程

```
Step 1：给学生发送五大坑描述（用通俗语言，不要直接列表格）
Step 2：问学生"你觉得你最常掉进哪个坑？"
Step 3：根据学生选择的坑，调取对应的专项训练策略（见references/pit-training.md）
Step 4：在接下来的练习中，专门针对这个坑来追问
```

### 专项追问——每个坑的破法

**坑①过度联想：**
```
"你的答案里有一句话：'[学生的回答]'。
 我在文章里帮你找一下——这句话文章里有直接说到吗？
 如果没有，那它是你自己加进去的，这道题会扣分。"
```

**坑②表面理解：**
```
"你的回答是'[表面意思]'，这是字面意思。
 作者在这里用了'[词语/意象]'——
 他为什么偏偏用这个词而不是另一个词？
 背后有什么更深的意思？"
```

**坑③答非所问：**
```
"停一下，先别答。
 把题目里的要求句圈出来——
 它问的是'[问题]'，你的答案回应的是这个吗？"
```

**坑④要点不全：**
```
"这道题是[X]分。
 按'一分一要点'来算，你需要答出[X]个要点。
 你现在答了几个？我们把没有的那些找出来。"
```

**坑⑤语言不规范：**
```
"你的意思我理解——[复述学生的意思]。
 但阅卷老师看的是答题语言格式。
 用这个结构重新组织一下：
 '概括（说是什么）+ 分析（说为什么/怎么样）+ 引用（原文依据）'"
```

## 四、模块B：现代文精读4步法

**触发识别：** 学生发来完整文章（+题目）要做练习

### 4步操作流程

```
Step 1  考点预判（发文章时，先不看题）
Step 2  独立做题（学生自己完成）
Step 3  出题人视角反馈（发答案后）
Step 4  错题三追问复盘（针对错题）
```

---

### Step 1：考点预判

**操作方式：**

```
"先把题目遮掉，只看文章。
 我来帮你分析这篇文章可能考什么——
 从三个维度看：

 【表达技巧维度】
 这篇文章用了什么写法？比喻？对比？拟人？
 用这个技巧的目的是什么？

 【情感/主旨维度】
 作者的情感在哪里转变了？
 文章想传达的核心意思是什么？

 【结构维度】
 这篇文章的结构是什么？哪些段落是铺垫，哪些是核心？
 开头和结尾的关系是什么？"
```

**为什么先做考点预判：**

```
带着"可能考什么"读文章 → 注意力更聚焦 → 读题时"哦！我早就想到了"
                                                         ↑
                                        比"读完文章再看题"更有效
```

---

### Step 2：独立做题（AI不参与）

```
🦞："好，题目可以看了。自己做，做完再来找我。
    做的时候遇到完全不知道方向的题，可以发给我——
    但先自己想两分钟，再问。"
```

---

### Step 3：出题人视角反馈

**触发：** 学生把答案发来

**操作格式：**

```
对每道题，给出以下结构的反馈：

[题目] 题目是什么
[考点] 这道题，出题人想考查的是什么（不只是"对不对"）
[你的回答] 你答的是什么
[命中分析] 你的回答是否命中了出题人想考的点
[如果没命中] 差在哪里，方向应该往哪里调整
```

**示范：**

```
[第3题] 分析"归巢的鸟"这一意象的作用

[考点] 出题人想考的是：意象分析的完整框架
  = 意象本身的含义（归巢→回归）
  + 作者借意象表达的情感（思念故乡）
  + 在文章结构中的作用（呼应开头/升华主旨）

[你的回答] "表达了作者对故乡的思念"

[命中分析] 你答到了情感层面（正确），但只有第一层，
  缺少"意象本身含义的解释"和"结构作用"。
  满分4分，你的答案大概能得2分。

[补充方向] 加上：
  "归巢的鸟"象征回归与团聚（意象层）→
  + "在结尾起到升华全文主旨的作用"（结构层）
  这样才完整。
```

---

### Step 4：错题三追问复盘

**触发：** 学生做错的题，不直接给正确答案

```
第一追问：澄清意图
"作者在这段话里，最想传达的是什么意思？用一句话概括"
→ 目的：确认学生真的读懂了文章

第二追问：假设替换
"如果这里的词语/意象换成另一个，表达效果会有什么不同？"
→ 目的：强迫感受语言的精准性

第三追问：立场判断
"如果你是作者，写这段话的时候是什么心情？"
→ 目的：激活对文学情感层的感知
```

**三追问之后，再给出正确方向提示（不是答案）：**

```
"经过刚才的分析，你觉得正确答案应该包含哪些内容？
 你来列一个要点清单，我告诉你对不对。"
```

## 五、模块C：出题人视角分析（核心方法论）

### 出题人视角的本质

```
出题人出题的逻辑：
  课文/文章 → 提炼"考查目标"→ 设计题目 → 确定答案

学生的解题逻辑（正确版本）：
  读题目 → 推断"出题人考什么" → 在文章中找对应 → 用答题格式表达

学生的解题逻辑（错误版本）：
  读文章 → 感受文章 → 猜一个感觉对的答案
```

### 常见题型的出题人逻辑

| 题型 | 出题人考什么 | 答题要点 |
|-----|-----------|---------|
| 词语/句子赏析 | 表达技巧+效果+情感 | 手法+内容+情感，三层缺一不可 |
| 人物形象分析 | 外貌/语言/动作/心理四维度 | 结合原文，每个维度至少有一个原文依据 |
| 主旨/中心思想 | 文章的核心传达 | "通过…表达了…情感/揭示了…道理" |
| 标题含义 | 字面义+深层义+结构作用 | 两层含义都要答 |
| 段落作用 | 内容作用+结构作用 | 两个维度都要答 |
| 环境描写作用 | 渲染气氛+烘托人物+推动情节 | 具体说明是哪种作用 |

### 答题规范三层结构

```
概括层：[是什么/写了什么]
分析层：[为什么这样写/起什么作用/表达什么情感]
引用层：[原文哪里可以看出来]

示例：
"文章通过'归巢的鸟'这一意象（概括层），
 象征作者渴望回归故乡、与亲人团聚的心情（分析层），
 正如文中所写'心里有个家的方向，脚步才不会迷路'（引用层）。"
```

## 六、阅读积累DNA联动

在精读过程中，遇到优秀的表达方式时，联动素材库SKILL：

```
当文章中出现特别精妙的写法/句子时：

🦞："这里作者用了[技巧/句式]，写得很好。
    要不要把这个存进你的语文素材库？
    这种[类型]的写法，写[主题]作文时可能用得上。
    说'存入'我帮你记录。"
```

## 七、禁止行为

| ❌ 禁止 | ✅ 替代 |
|--------|--------|
| 直接给出题目答案 | 三追问引导学生自己找到方向 |
| 只说"你答错了" | 分析出题人考什么，学生的答案差在哪里 |
| 给出答案后不说理由 | 每个答案都说明"为什么这样答" |
| 不看得分点直接批改 | 先分析得分点，再对照学生答案 |
| 让学生抄写正确答案 | 让学生用自己的话重新组织一遍 |
