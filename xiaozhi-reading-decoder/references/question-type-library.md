# 现代文各题型出题逻辑与答题模板详解

---

## 一、记叙文阅读题型库

### 题型①：词语/句子赏析
**出题人考点：** 表达手法识别 + 效果分析 + 情感挖掘

**答题模板：**
```
运用了[手法]的手法，
写出了[内容/形象]，
表达了作者[情感/态度]。
```

**常见追问（当学生只答了一层时）：**
- 只说了手法：→ "用这个手法写出了什么效果？"
- 只说了情感：→ "文章用了什么表达方式让你感受到这个情感？"
- 忘了引用原文：→ "文章里哪句话可以证明你的分析？"

---

### 题型②：人物形象分析
**出题人考点：** 四维度综合（外貌/语言/动作/心理）

**答题模板：**
```
[人物]是一个[特点1]、[特点2]的人。
从[原文依据1]可以看出他/她[特点1]；
从[原文依据2]可以看出他/她[特点2]。
```

**追问提示（当学生只给结论没给依据时）：**
```
"你说他'善良'——文章里哪句话、哪个动作让你看出来的？
 人物形象分析，每一个特点都要有原文依据。"
```

---

### 题型③：环境描写作用
**出题人考点：** 内容作用 + 结构作用（二选一或两者都有）

**内容作用清单：**
- 渲染气氛（渲染了XX的氛围）
- 烘托人物心情（烘托了人物XX的心情）
- 交代时间/地点/季节背景

**结构作用清单：**
- 引出下文（为下文XX情节做铺垫）
- 与前文呼应
- 推动情节发展

**追问（当学生只答了一个维度）：**
```
"这道题问的是'作用'，一般包含内容作用和结构作用两个维度。
 你答了内容作用，结构上这段描写还有什么用？"
```

---

### 题型④：标题含义
**出题人考点：** 字面义 + 深层义（+ 结构作用，如有）

**答题模板：**
```
表层含义：[字面上指的是什么]
深层含义：[象征/暗示/引申的含义]
（如果标题与文章结构紧密相关，补充：）
结构作用：[贯穿全文的线索/点明主旨]
```

---

### 题型⑤：段落/句子在结构上的作用

**开头段的作用：**
- 引出下文（说明功能）
- 开门见山，点明主旨
- 制造悬念，吸引读者
- 奠定全文感情基调

**中间段的作用：**
- 过渡承上启下
- 补充说明，丰富内容
- 为后文情节/论证做铺垫

**结尾段的作用：**
- 总结全文
- 呼应开头（首尾呼应）
- 升华主旨，深化中心
- 言有尽而意无穷（留白效果）

## 二、议论文阅读题型库

### 题型①：论点识别
**出题人考点：** 能否准确找到并概括论点

**追问（当学生找错论点时）：**
```
"论点通常出现在开头、结尾、或小标题处。
 论点是一个完整的判断句，能单独成立。
 你找的这句话，能单独表达一个完整观点吗？"
```

### 题型②：论证方法分析
**常见论证方法：**

| 方法 | 识别标志 | 作用表述 |
|-----|---------|---------|
| 举例论证 | "例如""比如"具体事例 | 具体有力地证明了…，增强了说服力 |
| 道理论证 | 引用名言、理论 | 引用XX的话，有权威性地证明了… |
| 对比论证 | 两种情况对比 | 通过XX与XX的对比，突出了… |
| 比喻论证 | 用比喻讲道理 | 生动形象地说明了…，通俗易懂 |

---

## 三、说明文阅读题型库

### 题型①：说明对象及其特点
**答题格式：**
```
这篇文章说明的对象是[XX]，
其主要特点是[特点1]、[特点2]、[特点3]。
```

### 题型②：说明方法判断
**识别+作用的完整答法：**
```
运用了[说明方法]的说明方法，
[具体说明了什么内容]，
使说明更[准确/生动/具体/有说服力]。
```

### 题型③：说明顺序判断
- 时间顺序：有明确的时间词（年代、季节、先后）
- 空间顺序：有方位词（从上到下、由内而外）
- 逻辑顺序：从现象到原因、从整体到局部、从主到次

## 四、高频失分点速查表

| 失分情形 | 根本原因 | 纠正方法 |
|---------|---------|---------|
| 词语赏析只说手法不说效果 | 误以为"指出手法"就答完了 | 牢记：手法+效果+情感，三层缺一不可 |
| 人物分析给结论无原文依据 | 只答印象，未读原文 | 找原文，每个特点配一句原文 |
| 段落作用漏答一个维度 | 不知道要答两个维度 | 先问自己：内容作用？结构作用？ |
| 字数过少，要点不全 | 不知道按分值估算要点数 | 几分题→几个要点，分值即要点数 |
| 主观感受代替客观分析 | 没有从出题人视角思考 | 问自己：出题人考的是什么客观知识点？ |
