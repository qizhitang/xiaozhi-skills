# 语文错因维度表

> 本文档为语文学科包的核心参考资源，定义语文（阅读理解+写作+文言文+语言运用）的五类错因维度体系、子类型分类、跨维度关联规则与追踪标准。为 `xiaozhi-reading-decoder`、`xiaozhi-writing-coach`、`xiaozhi-classical-revival`、`xiaozhi-grammar-tracker` 提供统一的错因分类框架。

---

## 一、维度总览

```
语文错因维度体系（五维×多子类型）

维度①  阅读认知错误 ── 信息获取层（读的方式有问题）
维度②  答题规范错误 ── 表达层（懂的但答不规范）
维度③  写作思维错误 ── 创作层（写的方式有问题）
维度④  文言文理解错误 ── 古语层（古汉语解码偏差）
维度⑤  语言运用错误 ── 语感层（现代汉语语病/表达偏差）

每道错题必须归入且只归入一个主维度。
如存在跨维度因素，记录为"主维度+次要维度"。
```

## 二、维度①：阅读认知错误

**定义：** 阅读过程中信息获取或理解的系统性偏差——不是"没读懂文章"，而是"读懂了但读的方式有问题"。

**核心认知前置：** 阅读理解不是"理解文章"——是"理解出题人为什么这样出题"。这个认知偏差是维度①的底层根因。

### 子类型分类

| 子类型ID | 名称 | 典型表现 | 对应坑号 | 高发文体 | 关联参考 |
|---------|------|---------|---------|---------|---------|
| RC01 | 脱离文本发散 | 答案加入文章没有的内容 | 坑① | 散文 | → pit-training §坑① |
| RC02 | 表面理解 | 只答字面意思，忽略深层含义 | 坑② | 散文/记叙文 | → pit-training §坑② |
| RC03 | 答非所问 | 认真读了文章但没认真读题目 | 坑③ | 所有文体 | → pit-training §坑③ |
| RC04 | 要点不全 | 只答了部分要点，得分不完整 | 坑④ | 议论文/说明文 | → pit-training §坑④ |
| RC05 | 文体特征忽略 | 所有文章都用同一种方式读 | 坑②变体 | 散文→议论文转换时 | → pit-training §坑② |
| RC06 | 考点预判缺失 | 没有带着"可能考什么"去读文章 | — | 所有文体 | → SKILL.md §四 Step1 |
| RC07 | 定位能力薄弱 | 在原文中找不到题目对应位置 | — | 长文 | — |

**严重程度分级：**

```
轻度：RC06 → 方法缺失但理解力好
  → 处理：训练考点预判习惯（读前先问"这段可能考什么"）

中度：RC01、RC03、RC04 → 答题方法层面系统性偏差
  → 处理：专项训练（原文锚定法/题目审读法/要点提取法）
  → 工具：pit-training.md 对应坑训练步骤

重度：RC02、RC05 → 理解深度层面问题（表面理解+文体忽略）
  → 这是最底层的坑 → 如果不解决RC02/RC05，其他坑的训练效果受限
  → 处理：文体区分训练 + 深层含义追问（苏格拉底三追问）
```

**根治方法映射：**

```
RC01 原文锚定法：
  → 答案每句话标注原文对应 → 无对应的删除 → 训练"只有原文能说的才算答案"

RC03 题目审读法：
  → 先圈出题目关键词 → 确认题目问的是什么 → 再组织答案方向

RC04 要点提取法：
  → 一分一要点 → 从原文逐段扫描 → 每个要点附原文对应段落

RC02/RC05 文体区分法：
  → 散文考情感+意象+结构 → 议论文考论点+论据+论证 → 说明文考方法+信息
  → 不同文体使用不同读法和答题格式
```

## 三、维度②：答题规范错误

**定义：** 文章理解正确但答题表达不符合"阅卷语言"——意思对但格式不对。

**判断标准：** 学生能说出正确意思，但写出来的答案不符合评分标准的格式要求。

### 子类型分类

| 子类型ID | 名称 | 典型表现 | 高发题型 | 关联参考 |
|---------|------|---------|---------|---------|
| AR01 | 缺少概括层 | 只有分析没有概括 | 意象分析/人物分析 | → pit-training §坑⑤ |
| AR02 | 缺少引用层 | 分析到位但没有原文依据 | 所有分析类题型 | → pit-training §坑⑤ |
| AR03 | 口语化表达 | 意思对但表达像聊天 | 所有题型 | → pit-training §坑⑤ |
| AR04 | 术语使用不当 | 用了错误的分析术语 | 赏析题/手法分析 | → question-type-library |
| AR05 | 结构混乱 | 先分析后概括/引用穿插 | 所有多要点题 | → pit-training §坑⑤ |
| AR06 | 主观情感代入过深 | 答案变成个人感悟而非客观分析 | 散文/记叙文 | → pit-training §坑⑤ |

**严重程度分级：**

```
轻度：AR03 → 口语化但不影响得分要点
  → 处理：温和提示"这里可以换一个更正式的说法"

中度：AR01、AR02、AR05 → 结构缺失导致得分不完整
  → 处理：答题三层结构训练（概括→分析→引用）
  → 工具：pit-training §坑⑤的三层补全练习

重度：AR04、AR06 → 术语/情感层面系统性偏差
  → AR04→术语速查训练 + 出题人逻辑对照
  → AR06→主观/客观区分训练 + 原文锚定法回炉
```

**答题规范三层结构（核心方法论）：**

```
概括层：[是什么/写了什么] → 一句话概括核心
分析层：[为什么这样写/起什么作用/表达什么情感] → 说明作用/意义
引用层：[原文哪里可以看出来] → 附原文句子作为依据

缺失诊断：
  只有概括 → 补分析层和引用层
  只有分析 → 补概括层和引用层
  只有引用 → 补概括层和分析层
  三层都有 → 结构完整

升级路径：
  初级："表达了思念" → 只有概括
  中级："通过意象表达了思念" → 概括+分析
  高级："文章通过'归巢的鸟'这一意象，象征作者渴望回归故乡的心情，
         正如文中所写'心里有个家的方向'" → 三层完整
```

## 四、维度③：写作思维错误

**定义：** 写作过程中的思维偏差——不是"不会写字"而是"不知道怎么想、怎么组织、怎么深入"。

**判断标准：** 学生能写完一篇文章但质量不高，核心问题在于思维层面而非语言层面。

### 子类型分类

| 子类型ID | 名称 | 典型表现 | 高发文体 | 关联参考 |
|---------|------|---------|---------|---------|
| WT01 | 论点模糊 | 观点人人都能说，没有个人立场 | 议论文 | → writing-rubric §论点层 |
| WT02 | 论据空洞 | 举了名人但不具体；只堆事实不建立因果 | 议论文 | → writing-rubric §论据层 |
| WT03 | 论证单一 | 只有一个论证角度，缺乏反驳/对立视角 | 议论文 | → debate-script-guide |
| WT04 | 情感笼统 | 只写"我很开心/难过"，缺乏具体感受 | 记叙文 | → writing-rubric §情感层 |
| WT05 | 细节缺失 | 场景描写笼统，动作描写简单 | 记叙文 | → writing-rubric §细节层 |
| WT06 | 详略失当 | 重要场景简短，次要场景详细 | 记叙文 | → writing-rubric §结构层 |
| WT07 | 说明顺序混乱 | 时间/空间/逻辑顺序中途混用 | 说明文 | → writing-rubric §逻辑顺序层 |
| WT08 | 审题偏离 | 作文主题偏离题目要求 | 所有文体 | — |
| WT09 | 连词低效 | "然后""还有""另外"频繁出现 | 所有文体 | → writing-rubric §语言层 |
| WT10 | 开头/结尾套路 | "众所周知"开头、"总而言之"结尾 | 所有文体 | → writing-rubric §语言层 |

**严重程度分级：**

```
轻度：WT09、WT10 → 语言层面小问题
  → 处理：指出低效连词/套话，建议替换
  → 联动写作教练：直接修改建议

中度：WT01、WT02、WT04、WT05 → 思维深度不足
  → 处理：苏格拉底四连追问（澄清→检验→对立→推向）
  → WT01/WT02 → 议论文专项追问
  → WT04/WT05 → 记叙文专项追问

重度：WT03、WT06、WT07、WT08 → 结构层面系统性偏差
  → WT03 → 辩论赛制训练（引入对立视角）
  → WT06 → 详略训练（先列"重要→次要"排序再写）
  → WT07 → 说明文结构训练（先定顺序框架再写）
  → WT08 → 审题训练（先解构题目要求再写）
```

**写作五步法中的维度对应：**

```
写作五步法各步骤主要防的错误维度：

Step 1 头脑风暴 → 主要防 WT08(审题偏离) + WT01(论点模糊)
  → 头脑风暴中必须包含"题目到底要我写什么"的确认

Step 2 逻辑检查 → 主要防 WT03(论证单一) + WT06(详略失当)
  → 逻辑检查必须包含"有没有反面/对立视角"

Step 3 动笔写作 → 主要防 WT04(情感笼统) + WT05(细节缺失)
  → 写作过程中遇到笼统处要追问"具体是什么样的"

Step 4 AI阅读 → 主要防 WT09(连词低效) + WT10(开头结尾套路)
  → AI阅读环节重点检查语言层面的低效表达

Step 5 精炼升级 → 主要防 WT02(论据空洞) + AR类(答题规范)
  → 精炼环节重点升级论证深度和表达精确度
```

## 五、维度④：文言文理解错误

**定义：** 古汉语解码偏差——文言词汇/句式/文化背景理解不到位。

**判断标准：** 学生读文言文时出现理解偏差，核心原因是古今语言差异而非现代文阅读能力问题。

### 子类型分类

| 子类型ID | 名称 | 典型表现 | 高发学段 | 关联SKILL |
|---------|------|---------|---------|----------|
| CL01 | 实词古今异义 | "妻子"古义=妻+子；今义=妻 | 初中+ | → classical-revival |
| CL02 | 虚词功能不清 | "之""其""而""以"功能混淆 | 初中+ | → classical-revival |
| CL03 | 特殊句式未识别 | 判断句/省略句/倒装句未还原 | 高中 | → classical-revival |
| CL04 | 词类活用遗漏 | 名词用作动词未识别 | 高中 | → classical-revival |
| CL05 | 文化背景缺失 | 不理解古代制度/典故/礼仪 | 全学段 | → classical-author-profiles |
| CL06 | 文意整体理解偏差 | 能翻译字面但不懂文章整体意图 | 初中+ | → reading-decoder |
| CL07 | 意象/典故不理解 | "折柳""望月"等固定意象含义不熟 | 高中 | → classical-revival |

**严重程度分级：**

```
轻度：CL05、CL07 → 知识储备不足
  → 处理：文化背景补充 + 意象典故速查
  → 联动文言文复活计划：补充作者画像和时代背景

中度：CL01、CL02 → 词汇/虚词层面问题
  → 处理：古今对照表 + 虚词功能归类训练
  → 识别"古今异义"高频词组
  → 联动文言文复活计划：实词古今对照 + 虚词功能速查

重度：CL03、CL04、CL06 → 句式/活用/整体理解层面系统性偏差
  → CL03→特殊句式还原训练（判断句=…也；被动句=为…所…）
  → CL04→词类活用识别训练（名词+宾语→名词作动词）
  → CL06→联动阅读理解拆解师（文意整体分析能力）
```

## 五、维度⑤：语言运用错误

**定义：** 现代汉语语病和表达偏差——句子有语法问题或表达不够精准。

**判断标准：** 学生写的/说的中文句子有结构性语病，或表达不够准确精炼。

### 子类型分类

| 子类型ID | 名称 | 典型表现 | 高发学段 | 关联参考 |
|---------|------|---------|---------|---------|
| LG01 | 词语搭配不当 | "增加效率"(应为"提高")；"损失利益"(应为"损害") | 初中+ | → grammar-error-library §类型① |
| LG02 | 句子成分残缺 | "经过老师的帮助，使我明白了道理"(缺主语) | 初中+ | → grammar-error-library §类型② |
| LG03 | 句子成分赘余 | "大约…左右"；"原因是由…造成的" | 初中+ | → grammar-error-library §类型③ |
| LG04 | 语序不当 | 关联词位置错；定语顺序错 | 初中+ | → grammar-error-library §类型④ |
| LG05 | 逻辑混乱 | 因果倒置；自相矛盾 | 高中 | → grammar-error-library |
| LG06 | 表意不明 | 代词指代不清；歧义句 | 高中 | → grammar-error-library |

**严重程度分级：**

```
轻度：LG03 → 成分赘余（最容易发现也最容易修改）
  → 处理：同义词重复→删一个；"防止…不"→去掉否定词

中度：LG01、LG02 → 搭配不当和成分残缺（需要结构意识）
  → LG01→搭配速查训练（动宾搭配/形名搭配）
  → LG02→缩句检验法（"谁+做什么/怎么样"→主语找到没有？）

重度：LG04、LG05、LG06 → 语序/逻辑/歧义（深层结构问题）
  → LG04→关联词配对检查 + 定语排序规则
  → LG05→因果逻辑验证法
  → LG06→代词指代检验法
```

## 七、跨维度关联规则

```
维度间常见交叉（语文特别多RC和AR交叉）：

RC01(脱离文本发散) + AR06(主观情感代入过深)：
  → 两个都是"自己加进去的"问题
  → RC01是信息获取层面(读的时候加了自己的理解)
  → AR06是表达层面(答的时候写了自己的感悟)
  → 判断：文章理解对了吗？如果对了，只是表达方式问题 → AR06为主
  → 如果理解本身就有偏差 → RC01为主
  → 通用根治：原文锚定法（两个坑同时解决）

RC03(答非所问) + RC04(要点不全)：
  → 不看题目要求 → 自然漏掉要点
  → RC03是方向问题 → RC04是数量问题
  → 优先治RC03 → RC04会随题目审读改善而自然改善

RC02(表面理解) + AR02(缺少引用层)：
  → 表面理解 → 答案只有概括 → 缺少深层分析和引用
  → RC02是理解深度问题 → AR02是表达结构问题
  → 先治RC02(理解深度) → AR02会随分析能力提升而改善

WT08(审题偏离) + RC03(答非所问)：
  → 写作审题偏离和阅读答非所问是同一个底层问题
  → 不仔细读题目要求 → 不按题目要求写作/答题
  → 通用根治：题目关键词圈出法 + 审题确认法

CL06(文意整体理解偏差) + RC02(表面理解)：
  → 文言文整体理解偏差 → 可能是现代文阅读能力本身的问题
  → 如果现代文阅读也有坑② → 先治RC02 → 再治CL06
  → 如果现代文阅读没问题 → CL06是文言文特有问题 → 专项训练

优先处理顺序：
  RC类(阅读认知) > AR类(答题规范) > WT类(写作思维) > CL类(文言文) > LG类(语言运用)
  → 阅读认知是底层 → 答题规范是表达层 → 写作思维是创作层 → 文言文和语言运用是专项
```

## 八、语文专项SKILL维度分配

```
各语文SKILL主要负责的维度：

阅读理解拆解师：RC类（主） + AR类（辅，答题规范）
  → 五大坑诊断覆盖：RC01-RC05
  → 答题三层结构训练覆盖：AR01-AR06

写作教练：WT类（主） + AR类（辅，写作中的表达规范） + LG类（辅，语病）
  → 三维批改法覆盖：WT01-WT10（按文体）
  → 苏格拉底四连追问覆盖：WT01/WT02/WT03
  → AI辩论赛覆盖：WT03(论证单一)

文言文复活计划：CL类（主） + RC类（辅，文言文中的阅读理解问题）
  → 实词/虚词/句式/活用/文化背景/意象覆盖：CL01-CL07

语法追踪器（语文语法）：LG类（主）
  → 语病六大类型覆盖：LG01-LG06

数据流转规则：
  → 阅读理解中发现写作思维问题 → 推送到写作教练
  → 写作中发现阅读理解问题 → 推送到阅读理解拆解师
  → 写作中发现语病 → 推送到语法追踪器
  → 阅读文言文时发现理解偏差 → 推送到文言文复活计划

不产生重复记录：
  → 学习DNA档案中语文科目只记录主维度错误
  → 跨维度问题 → 主维度SKILL记录完整，辅维度SKILL只标记关联
```

## 九、顽固弱项追踪标准

```
同一子类型（相同维度+子类型ID+具体坑/错误模式）出现次数追踪：

出现1次 → 标记"偶发"，低优先级
出现2次 → 标记"初步弱项"，下次练习提醒
出现≥3次 → 标记"顽固弱项"，启动专项突破

顽固弱项突破方法（按维度）：
  RC类顽固弱项 → 对应坑的3步专项训练 + 3次练习验证
  AR类顽固弱项 → 答题三层结构反复练习 + 自检清单
  WT类顽固弱项 → 苏格拉底四连追问深度介入 + 辩论赛训练
  CL类顽固弱项 → 古今对照表 + 虚词功能归类 + 作者画像补充
  LG类顽固弱项 → 语病口诀速记 + 专项练习题库(每类5道)

攻克标准：
  RC类：连续3次练习中该坑未出现
  AR类：连续3次自发使用三层结构(不靠提醒)
  WT类：连续2次写作中该思维偏差未出现
  CL类：同类文言文理解正确率>80%
  LG类：同类语病识别准确率>90%

月报统计维度：
  → 五维度占比饼图
  → 各维度内子类型排序
  → 顽固弱项清单（含出现次数+状态+突破方法）
  → 跨维度交叉分析（如有）
  → 文体维度交叉：不同文体下各维度错误分布（散文/议论文/说明文）
```
