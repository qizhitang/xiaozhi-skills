# 物理数学工具自检清单

> 本文档为 `xiaozhi-physics-error-archive` 的参考资源，帮助学生区分"物理问题"和"数学问题"，并提供物理常用的数学工具自检方法。

---

## 核心原则

```
在物理题中犯了数学错误，根源可能在两个地方：

1. 物理层问题：图景不清→公式选错→计算方向错
   → 解决方案：回到图景建立和概念理解

2. 数学层问题：图景清楚、概念正确、公式对，但数学执行出错
   → 解决方案：用本清单进行自检

判断方法：
  重新审题后，你能正确说出：
  - 应该画什么图？（图景OK）
  - 用什么规律/公式？（概念+公式OK）
  - 列出正确的方程？（分析OK）
  但计算结果不对 → 数学工具问题（T类）
```

## 一、单位换算自检

### 1.1 常用物理单位换算表

| 物理量 | 常见单位 | 国际单位 | 换算关系 | 常见错误 |
|--------|---------|---------|---------|---------|
| 长度 | cm, mm, km | m | 1km=10³m, 1cm=10⁻²m, 1mm=10⁻³m | — |
| 面积 | cm² | m² | 1cm²=10⁻⁴m² | ❌写成10⁻²m² |
| 体积 | cm³, L, mL | m³ | 1cm³=10⁻⁶m³, 1L=10⁻³m³ | ❌写成10⁻³m³ |
| 速度 | km/h | m/s | 1km/h=1/3.6 m/s | ❌直接除以3.6出错 |
| 密度 | g/cm³ | kg/m³ | 1g/cm³=10³kg/m³ | ❌写成1kg/m³ |
| 压强 | mmHg, atm | Pa | 1atm≈1.01×10⁵Pa | — |
| 电能 | kW·h | J | 1kW·h=3.6×10⁶J | ❌写成3.6×10³J |
| 功率 | kW | W | 1kW=10³W | — |

### 1.2 单位换算三步自检法

```
Step 1：审题后立即统一为国际单位制
  → 在题目数据旁标注换算后的值
  → 例：v=72km/h → v=20m/s

Step 2：计算过程中保持单位参与运算
  → 不要只算数字，带上单位一起算
  → 例：F=ma=2kg×3m/s²=6N

Step 3：最终结果检查单位是否合理
  → 力的单位应该是N（kg·m/s²）
  → 功的单位应该是J（N·m）
  → 如果算出力的单位是J，说明公式用错了
```

## 二、比例运算自检

### 2.1 串联分压与并联分流

```
串联电路分压：
  U₁/U₂ = R₁/R₂
  → 电阻大的分担电压多

并联电路分流：
  I₁/I₂ = R₂/R₁  ← 注意反比！
  → 电阻大的分流少

自检方法：
  写出比例式后，代入一个简单值验证
  例：R₁=2Ω, R₂=4Ω串联
  U₁/U₂ = 2/4 = 1/2 ✓（电阻小的分压少）
```

### 2.2 比例运算常见错误

| 错误类型 | 错误示例 | 正确做法 |
|---------|---------|---------|
| 交叉相乘方向错 | a/b=c/d → a×d=b×c写成a×c=b×d | 画十字交叉线 |
| 正比反比搞混 | I₁/I₂=R₂/R₁写成I₁/I₂=R₁/R₂ | 先定性判断：电阻大→电流小（反比） |
| 连比出错 | R₁:R₂:R₃=1:2:3时R₁占比写成1/2 | R₁占比=1/(1+2+3)=1/6 |

## 三、方程组求解自检

### 3.1 物理中的方程组典型场景

```
场景一：电路多状态分析
  开关断开：U = I₁(R₁+R₂)
  开关闭合：U = I₂×R₁
  两个方程两个未知数 → 联立求解

场景二：浮力+密度
  G = mg = ρVg
  F浮 = ρ液gV排
  平衡条件：F浮 = G → 联立

场景三：杠杆平衡
  F₁L₁ = F₂L₂
  F₁ = ρ₁V₁g
  代入后求解
```

### 3.2 方程求解自检法

```
Step 1：列出所有方程，标注编号
Step 2：确认未知数个数 = 方程个数
Step 3：代入消元时，每步只做一次运算
Step 4：求出结果后代回原方程验证
  → 左边 = ？右边 = ？相等则正确
```

## 四、几何关系自检

### 4.1 物理中的几何关系

| 场景 | 几何关系 | 常见错误 |
|------|---------|---------|
| 杠杆力臂 | 从支点到力的作用线的垂直距离 | ❌量到力的作用点 |
| 光路角度 | 入射角=入射光线与法线的夹角 | ❌量成与镜面的夹角 |
| 斜面分解 | 重力沿斜面分力=mg sinθ | ❌写成mg cosθ |
| 滑轮组 | 绳子段数n与力的关系F=G/n | ❌数错绳子段数 |

### 4.2 力臂找法三步自检

```
Step 1：找到支点O
Step 2：沿力的方向画力的作用线（虚线延长）
Step 3：从O向力的作用线作垂线 → 垂线段长度就是力臂

自检关键：
  力臂是"点到线的距离"，不是"点到点的距离"
  → 如果你的力臂连在力的箭头上，那很可能是错的
```

## 五、图像读取自检

### 5.1 常见物理图像

| 图像类型 | 横轴 | 纵轴 | 斜率含义 | 面积含义 |
|---------|------|------|---------|---------|
| s-t图 | 时间 | 位移 | 速度 | 无直接物理意义 |
| v-t图 | 时间 | 速度 | 加速度 | 位移 |
| I-U图 | 电压 | 电流 | 电阻的倒数 | 无直接物理意义 |
| m-V图 | 体积 | 质量 | 密度 | 无直接物理意义 |

### 5.2 图像读取三步自检

```
Step 1：先看坐标轴——横轴是什么？纵轴是什么？单位是什么？
Step 2：再看斜率——斜率代表什么物理量？斜率变大说明什么？
Step 3：最后看面积——面积有没有物理意义？如果有，代表什么？

常见错误：
  ❌ 把v-t图的斜率说成速度（实际是加速度）
  ❌ 把I-U图的斜率说成电阻（实际是电阻的倒数1/R）
  ❌ 忘记检查坐标轴单位，直接读数
```

## 六、科学计数法与精度自检

### 6.1 科学计数法常见错误

| 错误类型 | 错误示例 | 正确做法 |
|---------|---------|---------|
| 指数符号错 | 10⁻³写成10³ | 先判断大小：0.001应该是负指数 |
| 运算出错 | 10³×10²=10⁶ | 10³×10²=10⁵（指数相加） |
| 中间步骤取整过早 | 3.14×2.7=8.5→后续计算 | 保持至少3位有效数字 |

### 6.2 精度处理原则

```
1. 中间计算保持3-4位有效数字
2. 最终结果按题目要求保留位数
3. g取9.8还是10？看题目要求
4. 最终结果一定要做数量级判断：
   人的体重应该是几百N → 如果算出几万N，肯定算错了
   汽车速度应该是几十m/s → 如果算出几千米/s，肯定算错了
```
