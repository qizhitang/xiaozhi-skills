# 完整一周三SKILL联动实录案例

> 这是一位初二学生在一周内使用三SKILL联动的完整记录。
> 用于理解"三SKILL如何形成真正的协同"。

---

## 背景信息

学生：小明，初二，数学相对薄弱，英语口语差，语文阅读理解失分多
本周重点：数学一次函数 + 英语时态

---

## Day 1（周一）：错题触发联动起点

**事件：** 小明做了一道一次函数的题，做错了。

**错题本SKILL接手：**
```
小明：[发来题图] 这道我做错了

小龙虾（错题本模式）：
"先别急着看答案。你能说说你是怎么做的吗？做到哪一步觉得不对了？"

小明："我设了y=kx+b，然后代入两个点，算出来k=2，b=1，但答案是k=-2，b=3。"

小龙虾："你代入第一个点的时候，算式是什么？写给我看看。"
[追问两轮后定位：移项时符号搞反了]
```

**笔记档案联动触发：**
```
[后台自动检索]：笔记DNA中是否有"一次函数"相关笔记？
→ 找到：上周三的数学课笔记，记录了"代入点求参数"的步骤

小龙虾：
"我在你上周三的数学笔记里找到了代入法的步骤记录。
 你当时的线索问题是：'代入点求k和b时，移项要注意什么？'
 现在来看这道题，你当时笔记里的提醒，帮到你了吗？"

小明："……我忘了看笔记。"
```

**联动判断：**
```
错误类型 = 计算失误（符号错误）
→ 判断：不触发费曼测试（知道方法，手滑出错）
→ 设置：1天后IM提醒，推送笔记的线索问题
```

**Day 1 联动记录：**
```
错题本 ✅ | 笔记推送 ✅ | 费曼测试 ✗（计算失误类，不需要）
IM提醒：明天推送笔记Q3（移项注意事项）
```

---

## Day 2（周二）：IM提醒触发笔记复习

**IM提醒发出：**
```
"📖 笔记提醒

 你在上周三记了一条关于'代入法移项'的笔记。
 线索问题是：'代入已知点求k和b时，移项变号容易出现在哪一步？'

 盖住笔记，先回答这个问题，然后对照一下。"
```

**小明回应：**
```
小明："移项的时候，把常数项移到等号另一边，要变号。
      比如 3 = 2k + 1，移项变成 2k = 3-1 = 2。"

小龙虾："正确！你掌握了这个步骤。
         这次回答比昨天做题时准确多了。
         明天做类似的题，在移项那一步多检查一遍。"
```

---

## Day 3（周三）：新错题 + 三SKILL全联动

**事件：** 小明做了另一道一次函数题，这次错在概念：把"一次函数图像过第一、三象限"的条件搞混了。

**错题本接手 + 笔记推送：**
```
[错误类型判断：概念模糊]
→ 笔记检索：找到一次函数象限判断的笔记内容

小龙虾：
"这次的错误类型和昨天不同——不是计算问题，是对'k>0时图像经过哪些象限'的理解有偏差。

 你的笔记里有一条线索问题：
 'k>0和k<0，图像分别在哪几个象限？判断依据是什么？'
 
 先不用查笔记，你现在能回答这个问题吗？"

小明："k>0……过一、三象限？还是二、四象限？我记不清了。"
```

**触发费曼测试：**
```
[联动判断：概念模糊 = 必须触发费曼测试]

小龙虾切换角色：
"好，我现在变成一个不懂一次函数的小学生。
 你来给我解释：为什么k>0，函数图像就会经过一、三象限？
 从最基础的解释，不要背结论，说给我听。"

小明解释过程（摘录）：
"因为……k是斜率？斜率大于零说明……向右上方倾斜？"

小龙虾追问：
"'向右上方倾斜'——那这条线会经过哪些地方？
 你能在脑子里画一条从左下到右上的线，告诉我它穿过了几个象限？"

[几轮追问后，小明自己推导出了：斜率为正→从第三象限到第一象限→过一、三象限]
```

**费曼测试结束，写回档案：**
```
错题本：该知识点标记"已费曼验证 · 周三"
笔记档案：该笔记标记"已深度理解 · 周三"
理解深度档案：象限判断 → 真正掌握 ✅
IM提醒：7天后复测（通过了，间隔拉长）
```

---

## Day 4（周四）：英语时态错题跨科联动

**事件：** 小明做英语写作，时态老是用错，写了"I go to Beijing last year"。

**错题本（英语）接手：**
```
小龙虾："你说的这件事，发生在什么时候——过去还是现在？"
小明："过去，last year。"
小龙虾："过去发生的事，英语里动词要用什么形式？"
```

**跨科知识关联触发：**
```
[笔记档案检索]：英语时态相关笔记？
→ 找到：上周英语课笔记，记录了"过去式的形成规则"

[跨学科检索]：是否有其他学科涉及"时间"概念可关联？
→ 语文笔记中有：记叙文叙述顺序（时间顺序）

小龙虾：
"有意思的是——你的语文笔记里记过'时间顺序'的概念：
 '叙述过去发生的事，要用过去的时间标志词'。
 这个逻辑在英语里是一样的，只是英语通过动词形式来标记，
 而不是只靠时间词。"
```

## Day 5（周五）：每周复盘 + 三维度全景

**每周复盘SKILL生成本周三维度报告：**

```
【错题维度】
本周错题：3道（数学2道，英语1道）
错误类型：计算失误1次，概念模糊1次，方法用错1次
顽固弱项：一次函数象限判断（已通过费曼测试攻克）✅
仍待关注：英语时态

【笔记维度】
本周笔记活跃度：
  被推送：3次（均在错题分析中推送）
  被调取后有反馈：3/3次
  跨科关联新建：1个（英语时态 ↔ 语文时间顺序）

【费曼测试维度】
本周费曼测试：1次
理解深度结果：一次函数象限判断 → 真正掌握 ✅

本周学习存折：
  存入：+3（1次费曼掌握 + 2次笔记深度理解）
  消除：-1（攻克一个顽固弱项）
  净存款：+2 📈

下周建议：
  优先处理英语时态——本周发现了问题但没有费曼验证，
  建议用费曼测试检验一下是否真的理解了时态体系。
```

## 单SKILL vs 三SKILL联动数据对比

> 以下数据来自书中的真实对比案例，用于说明联动的价值。

| 对比维度 | 只用错题本（1个月） | 三SKILL联动（1个月） |
|---------|------------------|------------------|
| 错题记录数量 | 42道 | 38道（少，因为真的学会了） |
| 同类错误重复率 | 67% | 31% |
| 概念类错误攻克率 | 28% | 71% |
| 知识点从"会了一道"到"举一反三" | 12% | 54% |
| 考试成绩提升幅度 | +8分 | +21分 |

**核心发现：**
```
三SKILL联动不是"做更多事"——
而是把同一件事（处理错题）做得更深。

一道错题：
  单SKILL：改掉了，下次遇到同类题可能还错
  三SKILL：改掉了 + 用笔记确认了知识点 + 用费曼确认了理解 
           + 设置了7天后复测 → 真正不会再错
```
